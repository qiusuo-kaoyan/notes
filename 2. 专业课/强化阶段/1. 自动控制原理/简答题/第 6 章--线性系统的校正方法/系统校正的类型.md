## 🎯 一、校正（补偿）总体概念

**目的：**  
通过改变系统结构（增加环节）或参数，使系统的动态性能和稳态精度满足设计指标。

**常见分类（按结构位置）**：

> 串联校正（Series / Cascade）  
> 反馈校正（Feedback）  
> 前馈校正（Feedforward）  
> 复合校正（Combined）

---

## 🧩 二、四种校正方式完整对比表

|校正方式|基本结构位置|主要作用|典型形式|优点|缺点|适用场景|
|---|---|---|---|---|---|---|
|**① 串联校正**|与被控对象串联在前向通道中|改善系统的**动态性能**（相位裕度、超调、速度）及**稳态精度**|超前校正、滞后校正、超前–滞后组合|设计方法成熟；物理实现简单；频率特性易分析|可能影响低频精度与高频稳定性；易受干扰|最常用：要求改善相位裕度、超调、响应速度等|
|**② 反馈校正**|在反馈支路中引入修正环节|改善系统**稳定性**、**动态特性**，可实现**负反馈校正**（稳定增强）或**正反馈校正**（灵敏度提高）|加速度反馈、速度反馈、滞后反馈等|稳定性好、抗扰性强|结构复杂；反馈量获取困难；容易引入噪声|要求高稳定性、高精度（如位置、伺服系统）|
|**③ 前馈校正**|在主通道输入支路旁增加一条补偿路径，绕过反馈环节|提高系统**跟踪性能**、**消除静差**、**加快响应**|前馈补偿网络、扰动补偿|可提前“预知”扰动；改善跟踪性能；不影响稳定性|对对象模型依赖大；参数偏差敏感|有可测扰动或期望信号的系统，如扰动补偿控制|
|**④ 复合校正**|综合多种方式（常见：前馈+反馈，或串联+反馈）|综合改善**稳态 + 动态 + 抗扰**性能|前馈–反馈组合、串联–反馈复合|性能全面，补偿能力强|实现复杂、参数调整难|对性能要求极高的系统（自动驾驶、伺服定位）|

## 🧠 三、形象理解（结构图记忆）

```
1️⃣ 串联校正：直接放在主通道前面
  R(s) → [ 校正器C(s) ] → [ 对象G(s) ] → Y(s)

2️⃣ 反馈校正：放在反馈支路
  R(s) → G(s) → Y(s)
                ↑
             [ H(s)校正器 ]

3️⃣ 前馈校正：额外从输入绕到输出前端
  R(s) → G(s)
   ↘ [ 前馈D(s) ] →

4️⃣ 复合校正：多种组合，例如前馈 + 串联 或 前馈 + 反馈
```

## 📘 六、考试答题模板（简答题版）

> **校正的分类与作用：**  
> 按校正环节在系统中的位置不同，可分为串联校正、反馈校正、前馈校正和复合校正四种。
> 
> - **串联校正**：通过在开环前向通道加入补偿网络改善系统的动态性能与稳态精度。
>     
> - **反馈校正**：在反馈通道加入校正环节，以改善系统稳定性、灵敏度和抗扰性能。
>     
> - **前馈校正**：在输入信号与控制对象之间增加并行校正支路，提前补偿扰动或误差。
>     
> - **复合校正**：综合使用以上方法，兼顾稳态、动态和抗扰三方面性能。
>

```

位置：反馈支路中

口诀：前馈先一步，误差减半路

⚡ 复合校正
位置：多环节混合（常为前馈+反馈）
作用：兼顾稳态、动态与抗扰
口诀：复合最全面，性能皆周全
──────────────────────────────
总记：串动反稳前抗扰，复合全能统改好。
```
