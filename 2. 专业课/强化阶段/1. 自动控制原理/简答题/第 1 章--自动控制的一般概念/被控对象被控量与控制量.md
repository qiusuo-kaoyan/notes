
# 🎉 三个口诀总结

贴在脑子里，一辈子不会再错：

```
被控量 = 传感器测的量  
被控对象 = 产生该量的物理设备  
控制量 = 执行器被调节的量
```

---

# ✅ 一、先给出这道题的正确答案（为了校准概念）

我们按图（a）与图（b）分别来：

---

## （a）根据物料出口温度来调节冷却水流量

- **被控对象**：热交换器（Heat Exchanger）
    
- **被控量**：冷物料的出口温度（TT 测的温度）
    
- **控制量（操纵量）**：冷却水阀门开度（调节进入热交换器的冷却水量）
    

---

## （b）根据物料流量来调节冷却水入口流量

- **被控对象**：热交换器
    
- **被控量**：冷却水流量（FT 测的）
    
- **控制量（操纵量）**：冷却水阀门开度
    

---

你可以看到：不论(a)(b)，**被控对象一直是热交换器**。  
差别只是在 **控制目标不同**（温度 OR 流量）。

# 🔥 二、最重要的：如何“永不出错”？（三步法！）

下面是学生最缺乏的一点：  
**你必须按顺序分析：先被控量 → 再被控对象 → 最后控制量。**

---

# ⭐【三步法】——看到任何控制系统图都能做对

---

# **① 找被控量（最重要）——看 “传感器测的是什么”**

你看到什么测量仪表？

- TT（温度变送器）→ 被控量是温度
    
- FT（流量计）→ 被控量是流量
    
- PT（压力计）→ 被控量是压力
    

**口诀：被控量 = 测量量。**

在图（a），TT测温度 → 被控量 = 温度  
在图（b），FT测流量 → 被控量 = 流量

---

# **② 找被控对象——看“流经过被控量的装置是什么”**

被控量在哪里产生？  
一定是在一个“物理设备”中。

常见对象：锅炉、反应釜、加热炉、热交换器、蒸发器、泵、机械臂……

例如：

- 温度是在热交换器出口产生 → 被控对象 = 热交换器
    
- 压力在管道内建立 → 被控对象 = 管道
    
- 流量在管道或泵中变化 → 被控对象 = 泵 / 阀后管道
    

**口诀：被控对象 = 产生被控量的物理设备。**

---

# **③ 找控制量（操纵量）——看“控制器输出控制谁？”**

看控制器连着哪个执行器：

- 阀门 → 阀门开度
    
- 电机 → 电机电压或转速
    
- 泵 → 流量
    
- 加热器 → 加热功率
    

常见执行器：阀门（最常见）、加热器、风机、电机、泵……

**口诀：控制量 = 控制器直接操纵的执行器变量。**

在这道题：  
执行器都是“冷却水阀门” → 控制量 = 阀门开度

# 🧠 四、我们再用这三步来复盘本题（你会瞬间通透）

---

## （a）

**第1步：看测量装置 TT → 测温度 → 被控量 = 温度**  
**第2步：温度在哪产生？热交换器出口 → 被控对象 = 热交换器**  
**第3步：执行器是谁？冷却水阀门 → 控制量 = 阀门开度**

→ 完整准确！

---

## （b）

第1步：看测量装置 FT → 测流量 → 被控量 = 流量  
第2步：流量从哪里变化？阀门调节进入热交换器的冷却水 → 被控对象 = 热交换器  
第3步：执行器是谁？冷却水阀门 → 控制量 = 阀门开度
