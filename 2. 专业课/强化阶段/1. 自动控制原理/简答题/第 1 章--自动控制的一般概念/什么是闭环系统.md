## 🎯 一、定义：什么是闭环系统？

> **闭环系统**是指在控制系统中，  
> **系统的输出会被测量并反馈回输入端**，  
> 用来调整控制作用，从而使输出更接近期望值（给定值）。

简而言之：

> **定义:** 将被控量按==负反馈==与给定量比较得到==偏差==, 并对偏差进行==纠正消除==的控制系统. 又称反馈控制.
> **特点:** 需要控制的是==被控量==, 测量的是==被控量相对给定值==的偏差. 系统根据偏差进行控制. 只要被控量相对给定值存在偏差, 系统就会==自动纠偏==.

---

## ⚙️ 二、结构框图理解

最经典的闭环系统框图如下👇：

```
      +---------+
R(s) →|   比较   |→ e(s) →[ 控制器 Gc(s) ]→[ 被控对象 Gp(s) ]→ Y(s)
      | (求误差) |←--------[ 反馈 H(s) ]<---------------------+
```

**信号说明：**

- (R(s))：期望输入（设定值）
    
- (Y(s))：系统输出（实际值）
    
- (E(s) = R(s) - H(s)Y(s))：误差信号（输入与输出的差）
    
- (H(s))：反馈环节（测量或比例反馈）
    
- (G_c(s))：控制器（放大、调节）
    
- (G_p(s))：被控对象（系统本体）

## 🧠 三、通俗生活比喻

### 🌡️ 例1：空调的温控系统

- 你设定目标温度 25℃（→输入 (R)）
    
- 房间当前温度（→输出 (Y)）
    
- 温控器测得房间温度，与设定比较 → 得到误差 (E=R-Y)
    
- 控制器根据误差决定空调制冷/制热功率（→控制输入 (u)）
    
- 房间温度随之调整（→输出变化，再反馈）
    

✅ 这就是一个**闭环控制系统**。  
因为系统会自动检测输出，并根据误差自行调整控制量。

---

### 🪀 例2：小孩骑自行车保持平衡

- “保持竖直”是目标（输入）
    
- 眼睛+内耳感受倾斜角度（反馈）
    
- 大脑计算误差（你歪向哪边）
    
- 手臂调整车把（控制输入）
    
- 车重新平衡（输出变化）
    

这整个过程不断循环——这就是“反馈闭环”。

## 📐 四、数学形式（连续系统）

若开环传递函数为：  
[  
G(s) = G_c(s)G_p(s)  
]  
反馈为 (H(s))，则闭环传递函数为：

[  
\frac{Y(s)}{R(s)} = \frac{G(s)}{1 + G(s)H(s)}  
]

其中：

- 分子 (G(s))：系统的放大部分；
    
- 分母 (1 + G(s)H(s))：反映反馈作用；
    
- 分母=0 就是系统的**闭环特征方程**。

## 🧩 五、闭环系统的特点（非常重要）

|特性|说明|优点/缺点|
|---|---|---|
|**1. 具有反馈作用**|输出返回输入形成误差信号|能自动修正偏差|
|**2. 抗扰动能力强**|外界扰动通过反馈被抑制|系统稳健|
|**3. 稳定性可控**|通过反馈可改善极点位置|合理设计则稳定性提高|
|**4. 动态性能可调整**|可通过调节反馈参数改变响应速度、超调、稳态误差|灵活性高|
|**5. 稳态误差小**|反馈不断逼近目标值|提高精度|
|**6. 结构复杂、成本高**|增加测量和反馈环节|成本和风险上升|
|**7. 可能引入振荡**|反馈信号相位滞后过大会导致自激|稳定性分析必须谨慎|

## 🔍 六、闭环 vs 开环 对比表

|比较项目|开环系统|闭环系统|
|---|---|---|
|反馈存在|❌ 无反馈|✅ 有反馈|
|输出影响输入|否|是|
|抗扰性|弱|强|
|精度|受参数变化影响|稳态误差小|
|稳定性|结构固定、难调|可通过反馈改善|
|实现复杂度|简单、成本低|复杂、成本高|
|举例|洗衣机定时控制|温控空调、自动驾驶|

## 🧠 七、从控制思想角度理解

> 闭环系统的本质思想是：  
> **“感知 → 比较 → 调整 → 修正” 的自我纠偏机制。**

这种自我修正让系统能够在干扰、参数变化下仍维持期望性能。  
所以：

- 古典控制（PID、根轨迹）主要是闭环思想；
    
- 现代控制（状态反馈）本质上也是闭环：  
    (u = -Kx) 中状态 (x) 就是“内部反馈信号”。

## 📘 八、总结学习卡

```
🎯 闭环系统（Closed-loop System）

定义：
  输出信号经反馈回路与输入比较，
  由误差决定控制作用的系统。

数学形式：
  Y(s)/R(s) = G(s) / [1 + G(s)H(s)]

特点：
  ✅ 自动修正偏差（误差反馈）
  ✅ 抗扰动能力强
  ✅ 稳态误差小
  ✅ 动态性能可调
  ⚠️ 成本高、可能引入振荡

生活例子：
  空调温控、自行车保持平衡、自动巡航、稳压电源。

一句话记忆：
  “开环靠命令，闭环靠反馈；
   输出回头看，误差自己改。”
```
