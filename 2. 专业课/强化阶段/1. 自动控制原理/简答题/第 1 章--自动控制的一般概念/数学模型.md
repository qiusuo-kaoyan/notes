# 🎯 **简答题：控制系统为什么要建立数学模型？**

控制系统建立数学模型的原因是：

1. **定量分析系统特性**  
    通过模型可准确分析系统的稳定性、动态响应、稳态误差等性能指标。
    
2. **便于设计控制器**  
    控制器（如 PID、状态反馈、校正网络）都需要依赖数学模型才能进行参数设计与优化。
    
3. **预测系统行为，进行仿真**  
    数学模型能让我们在计算机中模拟各种输入、扰动和参数变化下的系统行为。
    
4. **便于系统优化和改进**  
    可对模型进行线性化、化简，或用于鲁棒性分析，指导实际工程系统的改造和优化。
    
5. **统一语言，便于交流与分析**  
    数学模型提供定量、严谨的描述方式，使工程人员能够统一标准地讨论系统问题。
    

**一句话总结：**

> 建立数学模型是为了“可计算、可分析、可设计、可仿真”。

# 🎯 **简答题：控制系统常见的数学模型有哪些？**

常见的控制系统数学模型包括：

1. **微分方程模型**
    
    - 最基本的动态描述形式
        
    - 直接反映系统输入–输出关系
        
2. **传递函数模型（Laplace域模型）**
    
    - 线性定常系统的最常用模型
        
    - 用于稳定性、频率响应、校正等分析
        
3. **状态空间模型**
    
    - 适用于多输入多输出 (MIMO) 系统
        
    - 可描述高阶、非线性、时变系统
        
    - 现代控制理论的基础
        
4. **方框图法、信号流图（结构模型）**
    
    - 描述系统结构、信号路径
        
    - 便于分解系统、化简结构
        
5. **频率特性模型（Bode 图、Nyquist 图）**
    
    - 从频域角度描述系统
        
    - 用于校正、相位裕度、带宽分析
        
6. **数值模型 / 离散模型（差分方程）**
    
    - 数字控制系统使用
        
    - 基于采样与数字算法
        
7. **非线性模型（描述函数、相平面）**
    
    - 用于分析死区、饱和等非线性系统
