# 🎯 一、什么是三频段理论？（一句话概念）

> **三频段理论**是频域分析中用于说明开环频率特性在不同频率段（低、中、高）对闭环系统性能（稳态、动态、噪声抑制）影响的规律。

简单说：  
开环幅频曲线的 **低频段、中频段、高频段** 分别决定闭环系统的：

- **低频段 → 稳态性能（稳态误差）**
    
- **中频段 → 动态性能（超调、稳定性、阻尼）**
    
- **高频段 → 随机噪声、鲁棒性、传感器噪声的放大**
    

这就是三频段理论的核心！

---

# 🔥 二、三频段分别对应什么闭环性能？

下面是考研最经典的**表格式总结**（考场可直接写👇）

---

# ⭐（1）低频段（ω 很小）——反映稳态误差特性

### 📌 低频段对应闭环系统的：

- **稳态误差**（本段增益越高，误差越小）
    
- **扰动抑制能力**（低频扰动如负载变化）
    

### 📌 原因：

低频段的开环增益决定了系统的“系统类型”（Type）  
→ 即系统能否跟踪阶跃/斜坡/抛物线。

### 📌 要求：

- **低频段开环增益要大**（比如 >20 dB），这样稳态误差小。
    

---

# ⭐（2）中频段（过渡带，≈截止频率 ωc）——反映动态性能

这是最重要的区域！

### 📌 中频段对应闭环系统的：

- **相角裕度 / 幅值裕度**（稳定性）
    
- **超调量**
    
- **快速性、响应速度**
    
- **阻尼比**
    

### 📌 原因：

在 ω=ωc 附近开环幅值从 >1 下降到 <1  
→ 决定闭环极点的实际位置  
→ 决定系统到底是欠阻尼、临界阻尼还是过阻尼。

### 📌 工程要求（典型）：

- 相角裕度 ≥ 40°～60°
    
- 截止频率 ωc 越高，响应越快（但不能太高，否则高频噪声太大）
    

一句话：

> **中频段决定动态性能（超调、稳定性、响应速度）。**

---

# ⭐（3）高频段（ω→∞）——反映噪声放大与鲁棒性

### 📌 高频段对应闭环系统的：

- **抗噪声能力**（传感器噪声通常是高频）
    
- **鲁棒性**（高频模型误差）
    
- **执行器磨损/疲劳**
    
- **是否放大高频微分噪声**
    

### 📌 现象：

如果开环增益在高频段不下降，  
→ 高频噪声（如传感器白噪声）会被放大  
→ 闭环输出会“抖动”甚至引起振荡。

### 📌 工程要求：

- 高频段开环增益应 **迅速衰减**（比如 -20 dB/dec、-40 dB/dec）
    

一句话：

> **高频段决定闭环系统的抗噪声能力与鲁棒性。**

---

# 🧠 三、三频段理论的整体总结图

```
        频率 ↑ →
 ┌───────────────────────────────┐
 │      低频段      中频段      高频段      │
 │  稳态误差      动态性能      噪声/鲁棒性 │
 │  (大增益好)   (频带决定速度) (增益要低) │
 └───────────────────────────────┘
```

# 🎓 四、可直接写在试卷上的考研简答版（30 秒背完）

> 在频域分析中，根据开环频率特性的不同频率区间对闭环性能的影响，可将其分为三个频段：
> 
> 1. **低频段**：开环增益的大小决定稳态误差和低频扰动抑制能力。低频增益越大，稳态误差越小。
>     
> 2. **中频段**：包括截止频率附近的区域，决定系统的动态性能，如稳定性、相角裕度、超调量和响应速度。
>     
> 3. **高频段**：高频开环增益决定系统的噪声抑制能力和鲁棒性。高频增益应尽量低，以避免放大高频噪声并增强鲁棒性。
>
