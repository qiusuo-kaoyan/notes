# 一句可背诵的考场版（直接背）

> 选取包围右半平面的闭合 s 路径（包含虚轴从 −j∞ 到 +j∞ 与在右半平面的闭合大半圆，若穿过 jω 轴极点则做小弧避开）；把开环复函数 (G(s)H(s)) 在该路径上映射到复平面，得到奈奎斯特曲线。记开环在右半平面的极点数为 (P)，奈奎斯特曲线对点 (-1) 的逆时针净绕数为 (N)，则闭环在右半平面的极点数（即 (1+G(s)H(s)=0) 的根数）为 (Z=N+P)。若 (Z=0) 则闭环稳定，否则不稳定。

> （注：计数时逆时针为正，顺时针为负；若教材用 (R=P-Z) 表示绕数，二者等价但符号/定义不同，注意统一符号。）
---

# 记忆口诀（简短，考场速记）

> **“先数 P，画图数 N，算 Z=N+P，Z=0 稳定”**。  
> 并记住：**N 是对 −1 的逆时针净绕数**（逆时针为正）。
---

# 考场答题模板（可直接抄/背）

1. 写出闭环特征方程：(F (s)=1+G (s) H (s)=0)。
    
2. 指出取包围右半平面的闭合路径并说明如有 jω 极点需避开（小半圆）。
    
3. 将该路径在映射 (G (s) H (s)) 下得到奈奎斯特曲线，记对 −1 的逆时针净绕数为 (N)。
    
4. 记开环在 RHP 的极点数为 (P)。根据幅角原理有 (N=Z-P)（或 (Z=N+P)），故 (Z=N+P)。
    
5. 若 (Z=0)，所有闭环极点都在左半平面，闭环系统稳定；否则不稳定。
---

# 逐步详解（考试答题时每一步要写清楚）

1. **设定问题对象**：稳定性判据针对闭环特征方程 (F(s)=1+G(s)H(s)=0)，闭环稳定 ⇔ 所有 (F(s)) 的根（闭环极点）在左半平面 ⇔ 右半平面根数 (Z=0)。
    
2. **取路径（Contour）**：在复 s 平面选取一条包围整个右半平面（RHP）的闭合路径 Γ（常取虚轴从 −j∞ → +j∞ 再连接一个半径很大的右半圆回到 −j∞）；若开环在虚轴上有极点，则在该点做小弧绕过（向右半平面做小半圆）。
    
3. **映射（映像曲线）**：把路径 Γ 在映射 (W(s)=G(s)H(s)) 下映射到 W 平面，得到奈奎斯特曲线（Nyquist plot）。等价地，画出 (G(j\omega)H(j\omega)) 从 (\omega=0) 到 (\infty) 的曲线并加上大半圆的映像（通常大半圆的位置可由渐近分析确定）。
    
4. **定义三个关键量**：
    
    - (P)：开环 (G(s)H(s)) 在右半平面的极点数（可从表达式直接判断）；
        
    - (N)：奈奎斯特曲线对点 (-1+0j) 的**逆时针净绕数**（逆时针为正，顺时针为负）；
        
    - (Z)：闭环系统在右半平面的零点数，即 (F(s)=0) 的 RHP 根数（也就是闭环极点数）。
        
5. **幅角原理 / 关系式**：由复变映射的幅角原理有（标准形式）  
$$
    N = Z - P \quad\Longrightarrow\quad Z = N + P.  
$$
    （等价的表述：有些教材把 (R=P-Z) 或者把绕数符号取反，注意符号约定；上式使用“逆时针为正”的约定。）
    
6. **判定稳定性**：计算出 (Z)（用上式），若 (Z=0) ⇒ 闭环稳定；若 (Z>0) ⇒ 闭环有 (Z) 个 RHP 极点，不稳定。
    

---

# 常见注意点与易错点（考试一定写清楚）

- **符号约定**：明确“逆时针为正”，并在答题中写清你用的符号关系（是 (N=Z-P) 还是他书写的 (R=P-Z)）。不同教材符号可能不同，答题时要统一并说明。
    
- **P 的计算**：(P) 是开环 (G(s)H(s)) 在 RHP 的极点个数（不是闭环极点）。
    
- **虚轴极点**：若开环有 jω 轴极点（如积分项 s=0 或纯虚根），在绘制奈奎斯特时必须在 s 平面对路径做小弧避开（通常向右小弧），并考虑该小弧的映像对绕数的贡献。
    
- **大半圆项**：大半圆映像常靠极限估计（|s|→∞），很多典型环节大半圆会趋于原点或某方向，考试时说明其处理方法即可。
    
- **绕数的方向**：绕 -1 顺时针一次即 (N=-1)。记数要“净值”——正的减去负的。
    
- **教材差异**：有些教材直接说“R=0 则稳定”，那是 **在前提 P=0（开环无 RHP 极点）时** 的简化结论。考试回答要注意注明前提条件。
    

---

# 简单示例（帮助记忆）

- 若开环无 RHP 极点（(P=0)），且奈奎斯特曲线不绕 −1（(N=0)），则 (Z=N+P=0)，闭环稳定。
    
- 若 (P=1)（开环有 1 个 RHP 极点），奈奎斯特曲线逆时针绕 −1 一次（(N=1)），则 (Z=N+P=2)，闭环有 2 个 RHP 极点（不稳定）。
    

（注：上面第二例仅示意计数结果，实际画图要看曲线如何绕 −1，绕数可为负数。）

## 🎯 **奈奎斯特判据·金句速记卡**

🧩 **核心原理：**

> 依据**幅角原理**，通过分析开环频率特性 ( G(s)H(s) ) 的**绕圈数**，判断闭环系统是否稳定。

---

### ⚙️ **判据步骤口诀：**

> **先数 P，画图数 N，算 Z = N + P，Z = 0 稳定。**

---

### 🧭 **关键定义：**

|符号|含义|判断方法|
|:-:|:--|:--|
|(P)|开环系统右半平面极点数|看 (G(s)H(s)) 的极点位置|
|(N)|奈奎斯特曲线绕 (-1) 的逆时针净圈数|逆时针为正，顺时针为负|
|(Z)|闭环系统右半平面极点数|(Z = N + P)|

---

### ✅ **稳定性判断：**

> 若 (Z = 0)，闭环系统稳定；  
> 若 (Z > 0)，闭环系统不稳定。

---

### ⚠️ **易混点：**

- “R=0 稳定”仅当 **P=0（开环稳定）** 时才成立。
    
- 绕数方向要说明：**逆时针为正**。
    
- 虚轴极点需**小弧避开**，其映像需计入绕数。
