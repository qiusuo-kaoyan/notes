## 🎯 一、三频段理论总览

> 👉 思想核心：  
> 系统的**开环频率特性**可分为低频段、中频段、高频段三个区间。  
> 每个频段都反映了闭环系统在**时域性能**上的不同方面。

## 🧠 三、各频段的物理直觉理解

### ① **低频段 → 稳态性能**

- 输入信号变化缓慢（阶跃、斜坡等）
    
- 频率低 → 系统几乎来得及完全跟踪输入
    
- 开环低频增益越大，输出越接近输入  
    → **稳态误差越小**
    
- 决定因素：系统的积分环节数量（系统类型）
    

> **一句话：低频段看误差。**

---

### ② **中频段 → 动态性能**

- 频率接近系统响应带宽（约在 ($\omega_c$) 附近）
    
- 这一段决定系统过渡过程的“速度、超调、稳定性”
    
- 相位裕度大 → 系统阻尼大 → 超调小、稳定
    
- 相位裕度小 → 超调大、振荡强、易不稳
    

> **一句话：中频段看稳定。**

---

### ③ **高频段 → 抗干扰性能**

- 高频信号对应快速变化或噪声
    
- 若系统高频增益不够衰减（平缓甚至上升） → 放大噪声
    
- 若滚降足够（−20dB/dec、−40dB/dec） → 高频干扰被滤掉
    
- 高频段衰减越快 → 抗噪能力越强，但响应速度可能变慢
    

> **一句话：高频段看抗扰。**

## 📈 四、从波特图角度看三频段

```
  幅值(dB)
    │                   ／＼       ← 中频段（稳定性）
    │                  ／   ＼
    │   ────────────          ← 低频段（稳态精度）
    │
    │                         ＼＼＼
    │                          ＼＼  ← 高频段（抗干扰）
────┴───────────────────────────────→ log ω
           ↑         ↑
         低频       ωc≈ωg（增益交叉频率）
```

- 左边平坦：低频增益高 → 稳态误差小
    
- 中间穿 0 dB：相位裕度、超调、稳定性
    
- 右边陡降：高频滚降快 → 噪声抑制好

## 🔁 五、控制设计时三频段的平衡关系

|目标|方法|影响|
|---|---|---|
|提高稳态精度（低频段）|提高开环增益|可能降低相位裕度，稳定性下降|
|改善稳定性（中频段）|加相位超前（超前补偿）|提升相位裕度，改善超调|
|增强抗干扰（高频段）|加滞后环节或滤波|增加高频衰减，响应变慢|

> **设计核心：三段性能要协调——不能顾此失彼。**

## 📘 六、总结背诵卡

|频段|看什么|指标|一句话记忆|
|---|---|---|---|
|低频段|稳态性能|($K_p,K_v,K_a$)，系统类型|**低频保精度**|
|中频段|动态性能|相位裕度、增益裕度、超调|**中频保稳定**|
|高频段|抗干扰|滚降斜率、噪声抑制|**高频防干扰**|

> **口诀：低频保精度，中频保稳定，高频防干扰。**
