太好了👏！四阶系统的劳斯判据确实是考研控制原理中**高频考点**之一，  
尤其是考简答题或选择题时，很多人会记不清那个“不等式条件”。  
下面我们系统地讲清楚它的**推导、判断条件、物理意义、快速记忆法**👇

---

## 🎯 一、四阶系统的标准形式

写成一般特征方程形式：

[  
a_0 s^4 + a_1 s^3 + a_2 s^2 + a_3 s + a_4 = 0  
]

假定所有系数 (a_i > 0)。  
我们的目标是判断系统是否**稳定（所有根实部 < 0）**。

---

## ⚙️ 二、劳斯表的构造

四阶系统的劳斯表如下：

|s次幂|系数|
|:-:|:-:|
|(s^4)|(a_0)  (a_2)  (a_4)|
|(s^3)|(a_1)  (a_3)  0|
|(s^2)|(\dfrac{a_1 a_2 - a_0 a_3}{a_1})  (\dfrac{a_1 a_4 - a_0·0}{a_1}=\dfrac{a_4 a_1}{a_1}=a_4)|
|(s^1)|(\dfrac{b_1 a_3 - a_1 a_4}{b_1})  0|
|(s^0)|(a_4)|

其中 (b_1 = \dfrac{a_1 a_2 - a_0 a_3}{a_1})。

---

## ✅ 三、稳定条件（逐列分析）

劳斯判据：**第一列全为正 → 系统稳定。**

因此必须满足：

[  
\begin{cases}  
a_0 > 0,\  
a_1 > 0,\  
b_1 = a_1 a_2 - a_0 a_3 > 0,\  
\dfrac{b_1 a_3 - a_1 a_4}{b_1} > 0,\  
a_4 > 0.  
\end{cases}  
]

---

## ✳️ 四、化简为四阶系统的判据核心不等式

由上式第4个条件：

[  
b_1 a_3 - a_1 a_4 > 0  
]

代入 (b_1 = a_1 a_2 - a_0 a_3)，得到：

[  
(a_1 a_2 - a_0 a_3)a_3 - a_1^2 a_4 > 0  
]

---

### 🔹 四阶系统稳定的充要条件为：

[  
\boxed{  
\begin{cases}  
a_0 > 0, \  
a_1 > 0, \  
a_2 > 0, \  
a_3 > 0, \  
a_4 > 0, \  
a_1 a_2 > a_0 a_3, \  
a_1 a_2 a_3 > a_0 a_3^2 + a_1^2 a_4  
\end{cases}  
}  
]

---

## 🧠 五、快速记忆法（口诀）

> ✅ **三阶：**  
> [  
> a_1 a_2 > a_0 a_3  
> ]
> 
> ✅ **四阶：**  
> [  
> a_1 a_2 a_3 > a_0 a_3^2 + a_1^2 a_4  
> ]

可以用口诀帮助记忆👇

> 🔹 “三阶一乘一减，四阶多一层连；  
> 三看两项差，四看三项和。”
> 
> 或者更口语一点：
> 
> > 三阶看乘积差，  
> > 四阶加个平方压。

---

## 🧩 六、例题验证

给定：  
[  
s^4 + 4s^3 + 6s^2 + 4s + 1 = 0  
]

（这其实是 ((s+1)^4=0)，稳定）

检验：

- (a_0=1,a_1=4,a_2=6,a_3=4,a_4=1)
    
- (a_1 a_2 - a_0 a_3 = 24 - 4 = 20 > 0)
    
- (a_1 a_2 a_3 - a_0 a_3^2 - a_1^2 a_4 = 4×6×4 - 1×16 - 16×1 = 96 - 32 = 64 > 0)
    

✅ 满足 → 系统稳定。

---

## 📘 七、临界稳定与不稳定情况

|条件|状态|
|---|---|
|所有条件成立|稳定|
|某条件等号成立|临界稳定（有虚轴根）|
|任一条件不成立|不稳定（右半平面根）|

---

## 🧾 八、总结学习卡

```
🎯 四阶系统劳斯判据速记卡

特征方程：
  a0 s⁴ + a1 s³ + a2 s² + a3 s + a4 = 0

Routh稳定条件：
  ① a0, a1, a2, a3, a4 > 0
  ② a1·a2 > a0·a3
  ③ a1·a2·a3 > a0·a3² + a1²·a4

含义：
  - 前两项保证无符号变化；
  - 最后一项防止出现实部为正的复根对。

考试口诀：
  “三阶一差式，四阶加平方；
   先看a1a2大于a0a3，
   再看a1a2a3大于a0a3²+a1²a4。”

```
