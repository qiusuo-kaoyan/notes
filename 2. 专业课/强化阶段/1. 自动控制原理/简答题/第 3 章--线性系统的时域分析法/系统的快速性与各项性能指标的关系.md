下面给你一个**非常清晰、考研可直接写的总结**，把  
**系统快速性 ↔ 谐振次数 ↔ 截止频率（穿越频率 ωc）↔ 相位裕度 PM**  
之间的关系，一次性讲透。

---

# ⭐ 先给出一条总主线（最重要！）

> **快速性由带宽（≈截止频率 ωc）决定；  
> 带宽越高 → 越容易振荡 → 谐振次数增加 → 相位裕度 PM 降低。**

一句话：

```
想要系统快，就提高带宽；带宽提升会削弱相位裕度，使系统更容易振荡。
```

---

# ⭐ 一、系统快速性与截止频率（穿越频率 ωc）

- 截止频率/穿越频率 ωc 越高 → 系统带宽越大
    
- 带宽越大 → 系统能响应更快的信号
    
- 因此 **系统快速性 ∝ ωc**
    

📌 记忆：

> “带宽大，系统快。”

---

# ⭐ 二、截止频率 ωc 与 谐振峰值/振荡次数

当 ωc 提高时：

- 对应频率附近的相角可能不足
    
- 频率响应会出现 **峰值增强（谐振峰值 Mr ↑）**
    
- 时域表现为 **振荡次数增加、超调增大**
    

因此：

> **ωc ↑ → Mr ↑ → 振荡次数 ↑ → 系统更抖**

📌 记忆：

> “想跑得快，就抖得更厉害。”

---

# ⭐ 三、截止频率 ωc 与 相位裕度 PM 的关系（核心！）

为了提高 ωc，通常会增大系统增益 K，这将造成：

- Nyquist 曲线被“拉向右”（靠近 -1 点）
    
- → 与 -180° 相角交叉的点更靠近 0 dB 线
    
- → 相位裕度 PM 减少
    

即：

> **ωc ↑ → PM ↓**

相位裕度越低，系统越容易振荡，稳定性越差。

📌 记忆：

> “带宽高，一般相位裕度就低。”

---

# ⭐ 四、谐振次数 与 相位裕度 PM 的关系（经典二阶系统关系）

对于近似二阶系统，有经验公式：

- PM ≈ 60° → 阻尼比 ≈ 0.7 → 振荡几乎没有
    
- PM ≈ 45° → 阻尼比 ≈ 0.5 → 有明显振荡
    
- PM < 30° → 振荡显著、甚至濒临不稳定
    

所以：

> **振荡次数 ↑ → PM ↓**

或

> **PM ↑ → 振荡次数 ↓**

---

# ⭐ 用一个超级清晰的链条总结：

```
加快系统 → 提高带宽 ωc
提高带宽 → 谐振峰值 Mr ↑ → 振荡次数 ↑ → 超调量 ↑
提高带宽 → 相位裕度 PM ↓ → 稳定性下降
```

# ⭐ 考研简答题标准答案（背即可）

```
系统的快速性主要由截止频率（穿越频率 ωc）决定，ωc 越高，
系统带宽越大，响应越快。

但随着 ωc 提高，系统的相位裕度 PM 通常减小，使频率响应出现更
高的谐振峰值 Mr，表现为时域中超调量增大、振荡次数增加。

因此：快速性↑ → 带宽↑ → 振荡↑ → 相位裕度↓ → 稳定性下降。
```
