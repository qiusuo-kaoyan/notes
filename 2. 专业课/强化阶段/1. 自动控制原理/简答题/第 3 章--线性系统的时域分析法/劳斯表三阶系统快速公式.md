——对于**三阶系统**的稳定性，确实有一个**快速判断公式**，  
就是你说的那个：  
[  
a_1 a_2 - a_0 a_3 > 0  
]  
下面我带你完整梳理一下这个公式的**来历、含义、适用条件**，以及如何在考试时**快速使用和避免错误**👇

---

## 🎯 一、先从三阶系统的特征方程说起

一般三阶系统的**闭环特征方程**写作：  
[  
a_0 s^3 + a_1 s^2 + a_2 s + a_3 = 0  
]  
（其中所有 (a_i > 0)）

我们要判断系统是否稳定（即全部根在左半平面）。

---

## ⚙️ 二、构造劳斯（Routh）判据表

劳斯判据的原理是：  
系统稳定 ⇔ 劳斯表第一列的所有元素同号（通常为正）。

对于三阶方程，劳斯表写成：

|s次幂|系数|
|:-:|:-:|
|(s^3)|(a_0)  (a_2)|
|(s^2)|(a_1)  (a_3)|
|(s^1)|(\dfrac{a_1 a_2 - a_0 a_3}{a_1})  0|
|(s^0)|(a_3)|

---

## ✅ 三、稳定条件（逐行分析）

为了所有根都在左半平面（系统稳定），  
劳斯表的**第一列**必须全为正：

1️⃣ (a_0 > 0)  
2️⃣ (a_1 > 0)  
3️⃣ (\dfrac{a_1 a_2 - a_0 a_3}{a_1} > 0)  
  ⇔ (a_1 a_2 - a_0 a_3 > 0)  
4️⃣ (a_3 > 0)

---

### ✳️ 因此三阶系统稳定的**充要条件**是：

[  
\boxed{  
\begin{cases}  
a_0 > 0 \  
a_1 > 0 \  
a_2 > 0 \  
a_3 > 0 \  
a_1 a_2 - a_0 a_3 > 0  
\end{cases}  
}  
]

---

## 🧠 四、这个快速公式的含义

> (\boldsymbol{a_1 a_2 - a_0 a_3 > 0})  
> 是**三阶劳斯稳定条件的核心不等式**，  
> 它保证了系统不会出现实部为正的根。

它反映了一个**动态平衡关系**：

- (a_1 a_2) 代表系统“阻尼项 × 弹性项”的综合效果；
    
- (a_0 a_3) 则代表“惯性项 × 零频增益”的作用。  
    当 (a_1 a_2 > a_0 a_3) 时，阻尼足够大，可以压制振荡；  
    当两者相等或后者更大时，系统会临界或失稳。
    

---

## ⚡ 五、考试中快速判断步骤

> 给定三阶特征方程：  
> (a_0 s^3 + a_1 s^2 + a_2 s + a_3 = 0)

**稳定判断三步法：**

1️⃣ 看所有系数是否为正（(a_i > 0)）  
→ 否则肯定不稳定。

2️⃣ 用公式 (a_1 a_2 - a_0 a_3 > 0) 检查。  
→ 若成立 → 稳定；  
→ 若等号成立 → 临界稳定（虚轴根）；  
→ 若小于 0 → 不稳定。

3️⃣ 完成✅（三阶题秒杀！）

---

## 📘 六、举个例子验证一下

设特征方程：  
[  
s^3 + 5s^2 + 8s + 4 = 0  
]

则：

- (a_0 = 1)
    
- (a_1 = 5)
    
- (a_2 = 8)
    
- (a_3 = 4)
    

检查：  
[  
a_1 a_2 - a_0 a_3 = 5×8 - 1×4 = 40 - 4 = 36 > 0  
]  
✅ 且所有系数为正 → 系统稳定。

---

## 🧩 七、临界稳定情况

若：  
[  
a_1 a_2 - a_0 a_3 = 0  
]

→ 劳斯表中第三行第一项为 0，  
说明系统存在**一对共轭纯虚根**，  
即系统**处于临界稳定状态**（持续振荡）。

---

## 🧾 八、总结学习卡

```
🎯 三阶系统稳定判据速记卡

特征方程：
  a0 s³ + a1 s² + a2 s + a3 = 0

Routh稳定条件：
  a0, a1, a2, a3 > 0
  且 a1·a2 − a0·a3 > 0

意义：
  - a1a2 > a0a3 → 稳定
  - a1a2 = a0a3 → 临界稳定（虚根）
  - a1a2 < a0a3 → 不稳定

考试口诀：
  “三阶系统看正号，再检 a1a2 大于 a0a3。”

例：
  s³ + 4s² + 6s + 2 → 4×6−1×2=22>0 → 稳定 ✅
```
