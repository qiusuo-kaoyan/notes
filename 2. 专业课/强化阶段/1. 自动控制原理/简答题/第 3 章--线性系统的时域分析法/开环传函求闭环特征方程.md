非常棒👏！这个问题非常关键，是考研自动控制原理里常见的“陷阱点”之一。  
你问的——

> “闭环特征方程 = 开环分子 + 开环分母，是否要求该反馈为单位负反馈？”  
> 答案是：✅ **是的，只有在单位负反馈系统中才成立。**

下面我们系统地解释「为什么」👇

---

## 🎯 一、从一般反馈系统出发

考虑最一般的线性反馈系统：

```
          +───[ G(s) ]───+
     R(s) →──┤            ├──→ Y(s)
              +────[ H(s) ]◄───+
```

其中：

- (G(s))：开环前向传递函数（控制器 × 被控对象）
    
- (H(s))：反馈环节传递函数（测量或反馈元件）
    
- 符号为 **负反馈**
    

---

### 1️⃣ 闭环传递函数（一般形式）

系统输出与输入的关系为：  
[  
\frac{Y(s)}{R(s)} = \frac{G(s)}{1 + G(s)H(s)}  
]

---

### 2️⃣ 闭环特征方程（定义）

特征方程定义为使分母为零的方程：  
[  
1 + G(s)H(s) = 0  
]

这才是闭环系统的特征方程。  
注意，这个式子在定义上与反馈类型、反馈系数都有关系。

---

## ⚙️ 二、在“单位负反馈”时的特殊形式

如果是**单位负反馈系统**：  
[  
H(s) = 1  
]

则闭环传递函数变为：  
[  
\frac{Y(s)}{R(s)} = \frac{G(s)}{1 + G(s)}  
]

假设开环传递函数为：  
[  
G(s) = \frac{N(s)}{D(s)}  
]  
其中 (N(s)) 为分子多项式，(D(s)) 为分母多项式。

---

### ✳️ 闭环特征方程推导：

代入上式：  
[  
1 + G(s) = 0 \Rightarrow 1 + \frac{N(s)}{D(s)} = 0  
]  
[  
\Rightarrow D(s) + N(s) = 0  
]

✅ 所以，只有当反馈环节是 **单位负反馈** 时，  
才有这个**漂亮简洁的结论：**

> [  
> \text{闭环特征方程} = \text{开环分母} + \text{开环分子}  
> ]

---

## 🚫 三、如果反馈不是单位反馈（H(s)≠1）会怎样？

若 (H(s) \neq 1)，例如 (H(s) = k_H) 或 (H(s) = \frac{M(s)}{N(s)})，  
那么：

[  
1 + G(s)H(s) = 1 + \frac{N(s)}{D(s)} H(s)  
]  
[  
\Rightarrow D(s) + N(s)H(s) = 0  
]

所以此时特征方程是：  
[  
D(s) + N(s)H(s) = 0  
]

而不是 (D(s) + N(s))。

---

### 举个例子（看出区别）：

|反馈类型|特征方程形式|是否 = 分子 + 分母|
|---|---|---|
|**单位负反馈** (H(s)=1)|(D(s)+N(s)=0)|✅ 是|
|**比例反馈** (H(s)=k)|(D(s)+kN(s)=0)|❌ 否|
|**一般反馈** (H(s)=H(s))|(D(s)+N(s)H(s)=0)|❌ 否|
|**正反馈**|(D(s)-N(s)H(s)=0)|❌ 号变负|

---

## 🧠 四、直观理解

- “开环分母 + 开环分子”这种写法，其实是单位负反馈的**特例**。
    
- 一旦反馈不是1，反馈通路也会引入极点或零点，系统的特征方程自然不同。
    
- 所以这只是一个简化记忆法，不能无脑套。
    

---

## 📘 五、总结学习卡

```
🎯 闭环特征方程与单位负反馈关系

一般闭环系统：
  Y(s)/R(s) = G(s) / [1 + G(s)H(s)]
  特征方程： 1 + G(s)H(s) = 0

若 G(s) = N(s)/D(s) → 特征方程为：
  D(s) + N(s)H(s) = 0

✅ 若为单位负反馈 H(s)=1：
  → 特征方程简化为 D(s) + N(s) = 0

❌ 若 H(s) ≠ 1 或为正反馈：
  → 特征方程为 D(s) ± N(s)H(s) = 0（不再是分子+分母）

🧠 一句话记忆：
  “分母加分子，只在单位负反馈；H(s)≠1，就得乘上它。”
```
