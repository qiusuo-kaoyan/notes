### **题目：** 绘制震荡环节的频率特性 (对数幅频、对数相频) 及幅相特性曲线
![[2-23.png]]
![[2-23-1.png]]
**核心环节：** 震荡环节，其传递函数为 $G(s) = \frac{1}{T^2s^2 + 2\zeta Ts + 1}$，其中 $0 \le \zeta < 1$ (阻尼比)。

#### **一、 频率特性 (对数幅频特性和对数相频特性)**

1.  **求频率特性表达式:**
    *   令 $s = j\omega$ 代入传递函数：
        $G(j\omega) = \frac{1}{T^2(j\omega)^2 + 2\zeta T(j\omega) + 1} = \frac{1}{1 - \omega^2 T^2 + j2\zeta \omega T}$
    *   **幅频特性 (模值):**
        $A(\omega) = |G(j\omega)| = \frac{1}{|(1 - \omega^2 T^2) + j(2\zeta \omega T)|} = \frac{1}{\sqrt{(1 - \omega^2 T^2)^2 + (2\zeta \omega T)^2}}$
    *   **相频特性 (相角):**
        $\phi(\omega) = \angle G(j\omega) = \angle 1 - \angle ((1 - \omega^2 T^2) + j(2\zeta \omega T))$
        $= 0 - \arctan\left(\frac{2\zeta \omega T}{1 - \omega^2 T^2}\right) = -\arctan\left(\frac{2\zeta \omega T}{1 - \omega^2 T^2}\right)$

2.  **绘制对数幅频特性曲线 (渐近线):** $L(\omega) = 20 \log_{10} A(\omega)$
    *   $L(\omega) = 20 \log_{10} \left(\frac{1}{\sqrt{(1 - \omega^2 T^2)^2 + (2\zeta \omega T)^2}}\right) = -10 \log_{10} \left((1 - \omega^2 T^2)^2 + (2\zeta \omega T)^2\right)$
    *   **低频段 ( $\omega T \ll 1$ ):**
        当 $\omega T \ll 1$ 时，$\omega^2 T^2 \approx 0$，$2\zeta \omega T \approx 0$。
        $L(\omega) \approx -10 \log_{10} (1^2 + 0) = -10 \log_{10} (1) = 0 \text{ dB}$
        这是一条在 0dB 处的水平线。
    *   **高频段 ( $\omega T \gg 1$ ):**
        当 $\omega T \gg 1$ 时，$(1 - \omega^2 T^2)^2 \approx (-\omega^2 T^2)^2 = \omega^4 T^4$，$ (2\zeta \omega T)^2 $ 项相对 $\omega^4 T^4$ 可以忽略（因为 $\zeta < 1$）。
        $L(\omega) \approx -10 \log_{10} (\omega^4 T^4) = -40 \log_{10} (\omega T)$
        这是一条斜率为 **-40 dB/十倍频** 的斜线。
    *   **转折频率:**
        观察传递函数分母形式 $T^2s^2 + 2\zeta Ts + 1$，通常取自然振荡频率（或无阻尼振荡频率）作为名义上的转折频率 $\omega_n = \frac{1}{T}$。
        在转折频率处，渐近线的交点位于 $L(\omega_n) = -40 \log_{10} (\frac{1}{T} \cdot T) = -40 \log_{10} (1) = 0 \text{ dB}$。
    *   **谐振峰值 (实际曲线):**
        当 $0 < \zeta < \frac{1}{\sqrt{2}} \approx 0.707$ 时，实际的对数幅频特性曲线在转折频率附近会出现一个峰值（谐振峰）。
        谐振频率 $\omega_r = \omega_n \sqrt{1-2\zeta^2} = \frac{1}{T}\sqrt{1-2\zeta^2}$ (仅当 $\zeta < \frac{1}{\sqrt{2}}$ 时存在)。
        谐振峰值 $M_r = \frac{1}{2\zeta\sqrt{1-\zeta^2}}$ (幅值)，对数值为 $20\log_{10}M_r$。
        **考试中绘制渐近线时，一般不要求画出这个峰值，除非特别说明。**

3.  **绘制对数相频特性曲线:** $\phi(\omega) = -\arctan\left(\frac{2\zeta \omega T}{1 - \omega^2 T^2}\right)$
    *   $\omega = 0$: $\phi(0) = -\arctan(0) = 0^\circ$
    *   $\omega = \frac{1}{T}$ (转折频率 $\omega_n$):
        $\phi(\frac{1}{T}) = -\arctan\left(\frac{2\zeta \frac{1}{T} T}{1 - (\frac{1}{T} T)^2}\right) = -\arctan\left(\frac{2\zeta}{1 - 1}\right) = -\arctan\left(\frac{2\zeta}{0}\right)$
        当 $2\zeta > 0$，分母为 $0^-$ (因为 $\omega$ 从小于 $1/T$ 趋近于 $1/T$ 时，$1-\omega^2T^2 > 0$；从大于 $1/T$ 趋近于 $1/T$ 时，$1-\omega^2T^2 < 0$。严格来说，当 $\omega = 1/T$ 时，分母为0，若 $2\zeta \neq 0$，则参数为 $\pm\infty$ )
        因此，在 $\omega = 1/T$ 处，相角跳变为 $-90^\circ$ (从 $0^\circ$ 经过)。
        $\phi(\omega_n) = -90^\circ$
    *   $\omega \to \infty$:
        $1 - \omega^2 T^2 \to -\infty$
        $2\zeta \omega T \to +\infty$
        $\frac{2\zeta \omega T}{1 - \omega^2 T^2} \approx \frac{2\zeta \omega T}{-\omega^2 T^2} = -\frac{2\zeta}{\omega T} \to 0^-$ ==(从负值趋近于0)==
        $\phi(\infty) = -\arctan(0^-) = -180^\circ$
    *   曲线从 $0^\circ$ 开始，经过转折频率处的 $-90^\circ$，逐渐趋近于 $-180^\circ$。

**数学公式表达 (对数频率特性):**
*   **对数幅频:**
    $$ L(\omega) = -10 \log_{10} \left((1 - \omega^2 T^2)^2 + (2\zeta \omega T)^2\right) \quad [dB] $$
*   **对数相频:**
    $$ \phi(\omega) = -\arctan\left(\frac{2\zeta \omega T}{1 - \omega^2 T^2}\right) \quad [^\circ] $$

**举例 (震荡环节，假设 $T=1, \zeta=0.5$):**
*   转折频率: $\omega_n = \frac{1}{1} = 1 \text{ rad/s}$
*   低频段: $L(\omega) = 0 \text{ dB}$
*   高频段: 斜率为 -40 dB/十倍频，过点 $(\omega_n, 0 \text{ dB})$ (渐近线)
*   相频: $\phi(0)=0^\circ$, $\phi(1)=-90^\circ$, $\phi(\infty)=-180^\circ$

#### **二、 幅相特性曲线 (Nyquist 图)**

1.  **利用幅频和相频特性分析关键点:**
    *   $\omega = 0^+$:
        $A(0^+) = \frac{1}{\sqrt{(1-0)^2 + 0}} = 1$
        $\phi(0^+) = -\arctan(0) = 0^\circ$
        起点：$(1, 0)$ 或 $1 \angle 0^\circ$
    *   $\omega = \frac{1}{T}$ (转折频率 $\omega_n$):
        $A(\frac{1}{T}) = \frac{1}{\sqrt{(1-1)^2 + (2\zeta \frac{1}{T} T)^2}} = \frac{1}{\sqrt{0 + (2\zeta)^2}} = \frac{1}{2\zeta}$
        $\phi(\frac{1}{T}) = -90^\circ$ (如前述分析)
        点：$(\frac{1}{2\zeta} \angle -90^\circ)$ 或 $(0, -\frac{1}{2\zeta})$
    *   $\omega \to \infty$:
        $A(\infty) = \lim_{\omega \to \infty} \frac{1}{\sqrt{(1-\omega^2 T^2)^2 + (2\zeta \omega T)^2}} = 0$
        $\phi(\infty) = -180^\circ$ (如前述分析)
        终点：原点 $(0,0)$，以 $-180^\circ$ 的相角（即沿负实轴）趋近。

2.  **化为实部和虚部形式 (通常较复杂，不常用于手绘复杂震荡环节的精确形状):**
    *   $G(j\omega) = \frac{1}{(1-\omega^2 T^2) + j(2\zeta \omega T)} = \frac{(1-\omega^2 T^2) - j(2\zeta \omega T)}{((1-\omega^2 T^2) + j(2\zeta \omega T))((1-\omega^2 T^2) - j(2\zeta \omega T))}$
    *   实部: $X(\omega) = \text{Re}[G(j\omega)] = \frac{1-\omega^2 T^2}{(1-\omega^2 T^2)^2 + (2\zeta \omega T)^2}$
    *   虚部: $Y(\omega) = \text{Im}[G(j\omega)] = \frac{-2\zeta \omega T}{(1-\omega^2 T^2)^2 + (2\zeta \omega T)^2}$
    *   根据 $\zeta$ 的不同，曲线形状会有所不同。
        *   当 $\zeta$ 较小时，曲线会在第二、三象限形成一个“鼓包”，然后再趋近原点。
        *   虚部 $Y(\omega)$ 始终为负或零（因为 $2\zeta \omega T \ge 0$）。

**曲线图示:**
*   **对数幅频特性 (渐近线):** 从 0dB 水平线开始，在 $\omega = 1/T$ 处转折，以 -40dB/十倍频的斜率下降。
*   **对数相频特性:** 从 $0^\circ$ 开始，平滑下降，经过 $(\frac{1}{T}, -90^\circ)$，趋近于 $-180^\circ$。
*   **幅相特性曲线:**
    *   从 $(1,0)$ 点开始。
    *   当 $\omega$ 增加，曲线进入第四象限。
    *   在 $\omega = 1/T$ 时，到达 $(0, -\frac{1}{2\zeta})$，即与负虚轴相交。
    *   随着 $\omega$ 继续增加，曲线进入第三象限（如果 $1-\omega^2T^2 < 0$）。
    *   最终以 $-180^\circ$ 的相角（沿负实轴方向）趋近于原点 $(0,0)$。
    *   曲线不穿过正实轴（除了起点）。

**注意相角范围：** 相角从 $0^\circ$ 变化到 $-180^\circ$。

#### **三、总结**

1.  **核心步骤：** $s=j\omega$ 代入，求幅频和相频表达式，再根据特性绘制曲线。
2.  **对数特性关键点：** 转折频率 ($\omega_n=1/T$)、低频段 (0dB)、高频段斜率 (-40dB/十倍频)。
3.  **幅相特性关键点：** 起点 ($1\angle 0^\circ$)、终点 ($0\angle -180^\circ$)、与负虚轴交点 ($\omega=1/T$ 时, 幅值为 $1/(2\zeta)$, 相角为 $-90^\circ$)。
4.  **阻尼比 $\zeta$ 的影响：**
    *   影响幅频特性在转折频率附近的峰值。
    *   影响幅相特性曲线的“饱满”程度。
5.  **震荡环节特性：** 0dB 低频渐近线，-40dB/十倍频高频渐近线，相角从 $0^\circ$ 变化到 $-180^\circ$。
