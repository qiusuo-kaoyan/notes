#每日一题 #自动控制原理

### **题目：** 已知开环传递函数和单位反馈系统结构图，求系统的根轨迹 (根轨迹绘制)
![[2-3-1.png]]
![[2-3-2.png]]
**给定：**

*   开环传递函数：$G(s)H(s) = \frac{K}{s(s+1)(s+2)}$
*   单位反馈系统

**求：** 绘制系统的根轨迹图。

#### **一、 根轨迹绘制的基本规则 (八条规则)**

1.  **根轨迹方程:**
    *   $1 + G(s)H(s) = 0$
    *   本题：$1 + \frac{K}{s(s+1)(s+2)} = 0$

2.  **分支数、起点和终点:**
    *   **分支数 (n):** 等于开环极点的个数。
        *   本题开环极点：$p_1=0, p_2=-1, p_3=-2$ (3个)
        *   所以根轨迹有 **3** 条分支。
    *   **起点:** 根轨迹始于开环极点。
        *   本题起点：$0, -1, -2$
    *   **终点:** 根轨迹终于开环零点。若开环零点个数 $m < n$，则有 $n-m$ 条分支趋向无穷远处。
        *   本题开环零点：无 (m=0)
        *   所以 3 条分支均趋向无穷远处。

3.  **实轴上的根轨迹:**
    *   **规则:** 实轴上某一区域，若其右边开环零、极点总数为奇数，则该区域是根轨迹的一部分。
    *   本题实轴上根轨迹分布：
        *   $[0, -1]$ (右侧1个极点 $p_1=0$)
        *   $(-\infty, -2]$ (右侧3个极点 $p_1=0, p_2=-1, p_3=-2$)
    *   **注意:** 区间端点通常包含在根轨迹内（作为起点或分离/汇合点）。

4.  **根轨迹的对称性:**
    *   根轨迹关于实轴对称。(考试时绘图体现即可，无需文字说明)

5.  **根轨迹的渐近线:** (当 $n > m$ 时存在)
    *   **渐近线条数:** $n-m$
        *   本题：$3 - 0 = 3$ 条渐近线。
    *   **渐近线与实轴正方向的夹角 $\phi_a$:**
        $\phi_a = \frac{(2k+1)\pi}{n-m}$,  $k = 0, 1, ..., n-m-1$
        *   本题 ($k=0,1,2$): $\phi_a = \frac{\pi}{3}, \pi, \frac{5\pi}{3}$ (即 $\pm 60^\circ, 180^\circ$)
    *   **渐近线的交点 $\sigma_a$:**
        $\sigma_a = \frac{\sum_{i=1}^{n} p_i - \sum_{j=1}^{m} z_j}{n-m}$
        *   本题：$\sigma_a = \frac{(0) + (-1) + (-2) - 0}{3-0} = -1$

6.  **根轨迹的分离点和汇合点:**
    *   **求解方法:** 由根轨迹方程 $1+G(s)H(s)=0$ 导出 $K = -1/G(s)H(s)$，然后令 $\frac{dK}{ds} = 0$，求解 $s$。
    *   本题：$K = -s(s+1)(s+2) = -(s^3 + 3s^2 + 2s)$
        $\frac{dK}{ds} = -(3s^2 + 6s + 2) = 0$
        解得：$s_1 \approx -0.423$, $s_2 \approx -1.577$
    *   **校验:** 只有位于实轴根轨迹上的点才是有效的分离点或汇合点。
        *   $s_1 = -0.423$ 位于 $[0, -1]$ 区间，是分离点。
        *   $s_2 = -1.577$ 不在实轴根轨迹上，舍去。

7.  **根轨迹的出射角和入射角:**
    *   **适用情况:** 当开环传递函数存在复数极点或复数零点时，需要计算。
    *   本题开环零极点均为实数，故**无需计算**出射角和入射角。

8.  **根轨迹与虚轴的交点:**
    *   **方法一 (令 $s=j\omega$):** 将 $s=j\omega$ 代入闭环特征方程 $1+G(s)H(s)=0$，令实部和虚部分别为零，求解 $\omega$ 和 $K$。
        *   闭环特征方程：$s(s+1)(s+2) + K = 0 \Rightarrow s^3 + 3s^2 + 2s + K = 0$
        *   代入 $s=j\omega$: $(j\omega)^3 + 3(j\omega)^2 + 2(j\omega) + K = 0$
            $-j\omega^3 - 3\omega^2 + 2j\omega + K = 0$
            $(K - 3\omega^2) + j(2\omega - \omega^3) = 0$
        *   令虚部为零：$2\omega - \omega^3 = 0 \Rightarrow \omega(2-\omega^2) = 0$
            解得 $\omega_1 = 0$ (对应实轴上的点，通常不视为与虚轴的交点), $\omega_2 = \sqrt{2}$, $\omega_3 = -\sqrt{2}$
        *   令实部为零 (代入 $\omega = \pm\sqrt{2}$): $K - 3(\pm\sqrt{2})^2 = 0 \Rightarrow K - 6 = 0 \Rightarrow K = 6$
        *   与虚轴的交点为：$s = \pm j\sqrt{2}$，此时 $K=6$。
    *   **方法二 (劳斯判据):** 利用劳斯稳定判据，找到使系统临界稳定的 $K$ 值和对应的振荡频率 $\omega$。

#### **二、 绘制根轨迹图**

**步骤：**

1.  在 s 平面上标出开环零点 (用 $\circ$ 表示) 和开环极点 (用 $\times$ 表示)。
2.  根据规则3，画出实轴上的根轨迹。
3.  画出渐近线 (交点和角度)。
4.  标出分离点/汇合点。
5.  标出与虚轴的交点。
6.  (若有) 根据出射角/入射角方向，从复数极点/零点出发/汇入。
7.  连接各段轨迹，使其从开环极点出发，止于开环零点或沿渐近线趋于无穷。注意根轨迹的平滑性和对称性。
8.  标注关键点 (极点、零点、分离点、与虚轴交点、渐近线交点) 的坐标和对应的 K 值 (与虚轴交点处的 K 值)。

**本题根轨迹图关键信息汇总：**

*   **开环极点:** $0, -1, -2$
*   **开环零点:** 无
*   **实轴根轨迹:** $[0, -1]$, $(-\infty, -2]$
*   **渐近线:** 3条，交点 $\sigma_a = -1$，角度 $\pm 60^\circ, 180^\circ$
*   **分离点:** $s \approx -0.423$
*   **与虚轴交点:** $s = \pm j\sqrt{2}$ (此时 $K=6$)

**绘制示意 (文字描述):**

*   从 $s=0$ 和 $s=-1$ 出发的两条根轨迹在实轴上相向而行，在 $s \approx -0.423$ 处分离。
*   分离后，两条轨迹对称地离开实轴，分别与虚轴交于 $s = j\sqrt{2}$ 和 $s = -j\sqrt{2}$。
*   之后，这两条轨迹分别趋向于角度为 $60^\circ$ 和 $-60^\circ$ 的渐近线。
*   从 $s=-2$ 出发的根轨迹沿负实轴 (也是 $180^\circ$ 渐近线) 趋向无穷远处。

#### **三、总结**

1.  **核心知识点:** 根轨迹绘制的八条基本规则。
2.  **解题流程:** 严格按照八条规则逐步分析计算，最后综合信息绘图。
3.  **关键计算:** 渐近线参数、分离点/汇合点、与虚轴交点。
4.  **绘图要求:** 清晰标注关键点和参数，轨迹平滑对称。
