### **题目：** 证明[[根轨迹]]的复数根部分为一特定圆 (根轨迹方程证明)
![[4. 每日一题/基础阶段/自控/清华鲁老师 25(1-200)/Picture Set/2-7.png]]
**已知：**
*   开环传递函数：$G(s)H(s) = \frac{K(s+2)}{s(s+1)}$

**求证：**
*   系统的复数根轨迹部分是以 $(-2, j0)$ 为圆心，以 $\sqrt{2}$ 为半径的圆。

#### **一、 核心思路**

1.  ==**根轨迹定义:** 根轨迹是闭环系统特征方程的根，随某一参数（通常是开环增益K）从0到∞变化时，在s平面上形成的轨迹。==
2.  **证明方法:**
    *   写出闭环特征方程。
    *   解出特征根的表达式（含参数K）。
    *   令特征根 $s = x + jy$ (x为实部，y为虚部)。
    *   **通过代数运算，<font color="#c00000">消去参数K</font>，得到一个只包含 x 和 y 的方程。**
    *   判断该方程是否为题目所要求的圆的方程。

#### **二、 解题步骤**

1.  **写出闭环特征方程:**
    *   $1 + G(s)H(s) = 0$
    *   $1 + \frac{K(s+2)}{s(s+1)} = 0$
    *   $s(s+1) + K(s+2) = 0$
    *   $s^2 + s + Ks + 2K = 0$
    *   $s^2 + (K+1)s + 2K = 0$

2.  **求解特征根 $s_{1,2}$ (使用求根公式):**
    *   $s = \frac{-(K+1) \pm \sqrt{(K+1)^2 - 4 \cdot 1 \cdot 2K}}{2 \cdot 1}$
    *   $s = \frac{-(K+1) \pm \sqrt{K^2 + 2K + 1 - 8K}}{2}$
    *   $s = \frac{-(K+1) \pm \sqrt{K^2 - 6K + 1}}{2}$
    *   $s = -\frac{1}{2}(K+1) \pm \frac{1}{2}\sqrt{K^2 - 6K + 1}$

3.  **令 $s = x + jy$ (实部与虚部):**
    *   当根为复数根时，根号下的部分为负，即 $K^2 - 6K + 1 < 0$。
    *   实部 $x = -\frac{1}{2}(K+1)$
    *   虚部 $y = \pm \frac{1}{2}\sqrt{-(K^2 - 6K + 1)} = \pm \frac{1}{2}\sqrt{-K^2 + 6K - 1}$

4.  **消去参数K，得到 x 和 y 的关系式:**
    *   **方法一 (题目已知圆心半径):**
        *   题目要求证明的圆方程为：$(x - (-2))^2 + (y - 0)^2 = (\sqrt{2})^2$
        *   即：$(x+2)^2 + y^2 = 2$
        *   将 $x = -\frac{1}{2}(K+1)$ 代入 $x+2$:
            $x+2 = -\frac{1}{2}(K+1) + 2 = -\frac{K}{2} - \frac{1}{2} + 2 = -\frac{K}{2} + \frac{3}{2} = \frac{3-K}{2}$
        *   计算 $(x+2)^2$:
            $(x+2)^2 = (\frac{3-K}{2})^2 = \frac{9 - 6K + K^2}{4}$
        *   计算 $y^2$:
            $y^2 = (\pm \frac{1}{2}\sqrt{-K^2 + 6K - 1})^2 = \frac{1}{4}(-K^2 + 6K - 1) = \frac{-K^2 + 6K - 1}{4}$
        *   计算 $(x+2)^2 + y^2$:
            $(x+2)^2 + y^2 = \frac{9 - 6K + K^2}{4} + \frac{-K^2 + 6K - 1}{4}$
            $= \frac{9 - 6K + K^2 - K^2 + 6K - 1}{4} = \frac{8}{4} = 2$
        *   得到 $(x+2)^2 + y^2 = 2$，与目标圆方程一致，证毕。

    *   **方法二 (题目未给出圆心半径，仅要求证明是圆):**
        *   从 $x = -\frac{1}{2}(K+1)$ 中==解出 $K$==:
            $2x = -(K+1)$
            $K+1 = -2x$
            $K = -2x - 1$
        *   将 $K = -2x - 1$ 代入 $y^2 = \frac{-K^2 + 6K - 1}{4}$:
            $4y^2 = -(-2x-1)^2 + 6(-2x-1) - 1$
            $4y^2 = -(4x^2 + 4x + 1) - 12x - 6 - 1$
            $4y^2 = -4x^2 - 4x - 1 - 12x - 6 - 1$
            $4y^2 = -4x^2 - 16x - 8$
        *   两边同除以4:
            $y^2 = -x^2 - 4x - 2$
        *   整理成圆的标准方程形式:
            $x^2 + 4x + y^2 + 2 = 0$
            $(x^2 + 4x + 4) + y^2 - 4 + 2 = 0$
            $(x+2)^2 + y^2 - 2 = 0$
            $(x+2)^2 + y^2 = 2$
        *   此为圆心在 $(-2, 0)$，半径为 $\sqrt{2}$ 的圆方程，证毕。

#### **三、 数学公式与概念**

*   **闭环特征方程:** $1 + G(s)H(s) = 0$
*   **二次方程求根公式:** 对于 $ax^2+bx+c=0$, 其根为 $x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$
*   **复平面:** 以实轴为横轴，虚轴为纵轴的坐标平面。
*   **圆的标准方程:** $(x-a)^2 + (y-b)^2 = r^2$，其中 $(a,b)$ 为圆心， $r$ 为半径。
*   **参数方程与普通方程的转化:** 消去参数是关键。

#### **四、 补充说明 (根轨迹绘制一般步骤 - 本题未直接考察)**

虽然本题是证明题，但提到了根轨迹绘制，这里简要列出绘制根轨迹的一般步骤：
1.  确定根轨迹的起点（开环极点）和终点（开环零点）。
2.  确定实轴上的根轨迹段。
3.  确定根轨迹的渐近线（如果零极点个数不等）。
4.  确定根轨迹的分离点与会合点。
5.  确定根轨迹与虚轴的交点（劳斯判据或令 $s=j\omega$）。
6.  确定出射角与入射角（如果存在复数开环零极点）。
7.  大致描绘根轨迹。

**举例:**

*   **对于本题的 $K^2 - 6K + 1 < 0$ (产生复数根的条件):**
    解不等式 $K^2 - 6K + 1 = 0$ 的根为 $K = \frac{6 \pm \sqrt{36-4}}{2} = 3 \pm \sqrt{8} = 3 \pm 2\sqrt{2}$。
    所以，当 $3 - 2\sqrt{2} < K < 3 + 2\sqrt{2}$ 时，系统有复数根，根轨迹的复数部分在此K值范围内形成。
*   **如果题目给的是一阶系统，则不会有复数根轨迹部分。** 例如 $G(s)H(s) = \frac{K}{s+1}$，特征方程 $s+1+K=0$, $s = -1-K$，根轨迹永远在实轴上。
*   **如果题目是证明其他形状的轨迹**，例如直线，那么消去K后得到的x和y的关系式应为线性方程。

#### **五、总结**

1.  **核心方法：** 利用根轨迹的定义，通过代数运算消去参数K，得到根的实部与虚部之间的关系。
2.  **关键步骤：**
    *   正确写出闭环特征方程。
    *   准确求解特征根表达式。
    *   巧妙地消去参数K。
3.  **两种证明思路：**
    *   若已知圆心和半径，直接代入验证。
    *   若未知圆心和半径，通过消参推导圆方程。
4.  **数学基础：** 熟悉二次方程求解、复数表示、圆的方程等。
5.  **题目类型：** 此类题目不常考，但可以加深对根轨迹本质的理解。
