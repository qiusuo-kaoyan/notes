### **题目：** 参数根轨迹绘制 (参数根轨迹)
![[2-6.png]]
![[2-6-1.png]]
**已知：**

*   开环传递函数：$G(s)H(s) = \frac{10(Ts+1)}{s(s+2)}$ (视频中写的是 $T_1$, 为了统一用 $T$)
*   参数 K 已知为 10。
*   要求绘制当参数 T 从 0 到 $\infty$ 变化时的根轨迹。

**核心问题：** 参数 T 不在标准的根轨迹方程的增益位置。

#### **一、 参数根轨迹绘制的核心思想**

1.  **化为标准形式:** 将包含参数 T 的闭环特征方程变换为 $1 + K_p G_p(s)H_p(s) = 0$ 的形式，其中 $K_p$ 是与参数 T 相关的等效增益，$G_p(s)H_p(s)$ 是等效的开环传递函数。
    *   **步骤：**
        1.  写出系统的闭环特征方程。
        2.  将包含参数 T 的项移到等号一侧，不包含 T 的项移到另一侧。
        3.  整理成 $1 + \frac{\text{含T的项}}{\text{不含T的项}} = 0$ 的形式。

#### **二、 本题的参数根轨迹标准形式转换**

1.  **闭环特征方程:**
    $1 + G(s)H(s) = 0$
    $1 + \frac{10(Ts+1)}{s(s+2)} = 0$
    $s(s+2) + 10(Ts+1) = 0$
    $s^2 + 2s + 10Ts + 10 = 0$

2.  **分离参数 T:**
    $10Ts = -(s^2 + 2s + 10)$

3.  **化为标准形式:**
    $1 + \frac{10Ts}{s^2 + 2s + 10} = 0$
    或者
    $1 + T \cdot \frac{10s}{s^2 + 2s + 10} = 0$

    此时，等效的开环传递函数为 $G_p(s)H_p(s) = \frac{10s}{s^2 + 2s + 10}$，等效的根轨迹增益为 T (从 0 到 $\infty$ 变化)。

    **注：** 视频中是将 $s^2 + 2s + 10$ 除过去，得到 $1 + \frac{10Ts}{s^2 + 2s + 10} = 0$。这里等效增益为 $K_T = 10T$。  两种形式本质相同，只是增益的定义不同。后续计算按视频中的标准形式进行。

#### **三、 根轨迹绘制步骤 (按标准根轨迹绘制法则)**

以等效开环传递函数 $G_p(s)H_p(s) = \frac{10Ts}{s^2 + 2s + 10}$ 为例，其中等效增益 $K_T = 10T$。

1.  **起始点和终止点 (等效开环零极点):**
    *   **等效开环零点 (m):** 分子 $10s = 0 \Rightarrow z_1 = 0$。 (m=1)
    *   **等效开环极点 (n):** 分母 $s^2 + 2s + 10 = 0 \Rightarrow p_{1,2} = \frac{-2 \pm \sqrt{4-40}}{2} = -1 \pm j3$。 (n=2)
    *   根轨迹分支数：$n=2$ 条。
    *   根轨迹始于等效开环极点 $p_1, p_2$，一条终于等效开环零点 $z_1$，另一条终于无穷远处。

2.  **实轴上的根轨迹:**
    *   等效开环零极点在实轴上的分布：只有一个零点 0。
    *   实轴上 $(-\infty, 0]$ 区间右侧有奇数个实轴零极点，故 $(-\infty, 0]$ 为根轨迹。

3.  **渐近线:**
    *   渐近线条数：$n-m = 2-1=1$ 条。
    *   渐近线倾角：$\phi_a = \frac{(2k+1)\pi}{n-m} = \frac{(2k+1)\pi}{1}$。取 $k=0$，则 $\phi_a = \pi$。
    *   渐近线与实轴交点：$\sigma_a = \frac{\sum p_i - \sum z_j}{n-m} = \frac{(-1+j3) + (-1-j3) - 0}{1} = -2$。
    *   (注意：视频中说 $n-m=1$ 时渐近线意义不大，但计算方法是标准的)

4.  **分离点/汇合点:**
    *   由 $1 + G_p(s)H_p(s) = 0$，得到 $1 + \frac{K_T s}{s^2 + 2s + 10} = 0$，则 $K_T = -\frac{s^2 + 2s + 10}{s}$。
    *   令 $\frac{dK_T}{ds} = 0$ (或者由 $T = -\frac{s^2+2s+10}{10s}$，令 $\frac{dT}{ds} = 0$)
    *   $\frac{dT}{ds} = -\frac{1}{10} \cdot \frac{(2s+2)s - (s^2+2s+10)\cdot 1}{s^2} = -\frac{1}{10} \cdot \frac{2s^2+2s - s^2-2s-10}{s^2} = -\frac{1}{10} \cdot \frac{s^2-10}{s^2} = 0$
    *   解得 $s^2 - 10 = 0 \Rightarrow s = \pm \sqrt{10}$。
    *   根据实轴根轨迹分布，分离点/汇合点应在 $(-\infty, 0]$ 上，所以取 $s = -\sqrt{10}$。
    *   将 $s = -\sqrt{10}$ 代回 T 的表达式：$T = -\frac{(-\sqrt{10})^2 + 2(-\sqrt{10}) + 10}{10(-\sqrt{10})} = -\frac{10 - 2\sqrt{10} + 10}{-10\sqrt{10}} = -\frac{20 - 2\sqrt{10}}{-10\sqrt{10}} = \frac{20 - 2\sqrt{10}}{10\sqrt{10}} = \frac{2\sqrt{10} - 2}{10} = \frac{\sqrt{10}-1}{5} \approx 0.432$。

5.  **与虚轴的交点:**
    *   令 $s = j\omega$ 代入闭环特征方程 $s^2 + 2s + 10Ts + 10 = 0$。
    *   $(j\omega)^2 + 2(j\omega) + 10T(j\omega) + 10 = 0$
    *   $-\omega^2 + j2\omega + j10T\omega + 10 = 0$
    *   实部：$-\omega^2 + 10 = 0 \Rightarrow \omega^2 = 10 \Rightarrow \omega = \pm \sqrt{10}$。
    *   虚部：$2\omega + 10T\omega = 0 \Rightarrow \omega(2+10T) = 0$。
    *   若 $\omega \neq 0$，则 $2+10T = 0 \Rightarrow T = -0.2$。
    *   因为 $T \in [0, \infty)$，所以 $T=-0.2$ 不在此范围内，说明除了原点（开环零点），根轨迹与虚轴无其他交点。
        *   **注意：** 当 $\omega = 0$ 时，即 $s=0$，是开环零点，T 可以为任意值。
    *   视频中的解法：$1 + \frac{10Ts}{s^2 + 2s + 10} = 0 \Rightarrow s^2 + 2s + 10 + 10Ts = 0$
        令 $s=j\omega$: $-\omega^2 + 2j\omega + 10 + 10Tj\omega = 0$
        实部：$-\omega^2 + 10 = 0 \Rightarrow \omega = \pm \sqrt{10}$
        虚部：$2\omega + 10T\omega = 0 \Rightarrow \omega(2+10T)=0$.
        若 $\omega = \sqrt{10}$, 则 $2+10T=0 \Rightarrow T=-1/5$. 此 T 值不在 [0, $\infty$) 范围内。
        若 $\omega = 0$, 则 $10=0$, 矛盾。但 $s=0$ 是开环零点, T 趋于无穷时会到达。
        所以除了原点 (开环零点), 没有与虚轴的交点.

6.  **出射角/入射角:**
    *   等效开环极点为复数极点 $p_1 = -1+j3$ 和 $p_2 = -1-j3$。等效开环零点为 $z_1 = 0$。
    *   **出射角 $\theta_{p1}$ (从极点 $p_1 = -1+j3$ 出发):**
        $\theta_{p1} = (2k+1)\pi + (\angle (p_1-z_1)) - (\angle (p_1-p_2))$
        $\angle (p_1-z_1) = \angle (-1+j3 - 0) = \angle (-1+j3) = \pi - \arctan(\frac{3}{1})$
        $\angle (p_1-p_2) = \angle ((-1+j3) - (-1-j3)) = \angle (j6) = \frac{\pi}{2}$
        $\theta_{p1} = \pi + (\pi - \arctan(3)) - \frac{\pi}{2} = \frac{3\pi}{2} - \arctan(3)$
        $\arctan(3) \approx 71.57^\circ \approx 0.397\pi$
        $\theta_{p1} \approx 1.5\pi - 0.397\pi = 1.103\pi \approx 198.43^\circ$
        (或者用另一种常用的公式: $\sum \angle(s-z_i) - \sum \angle(s-p_j) = (2k+1)\pi$)
        对于极点 $p_1$: $\angle(p_1-z_1) - \angle(p_1-p_2) - \theta_{p1}' = (2k+1)\pi$  (其中 $\theta_{p1}'$ 为从 $p_1$ 点出发指向根轨迹的切线与实轴正方向的夹角, 即出射角)。
        $(\pi - \arctan(3)) - \frac{\pi}{2} - \theta_{p1}' = \pi$  (假设 $k=0$)
        $\theta_{p1}' = (\pi - \arctan(3)) - \frac{\pi}{2} - \pi = -\frac{\pi}{2} - \arctan(3) \approx -90^\circ - 71.57^\circ = -161.57^\circ$
        此为视频中的计算方法。

    *   **出射角 $\theta_{p2}$ (从极点 $p_2 = -1-j3$ 出发):**
        由于对称性，$\theta_{p2} = -\theta_{p1}' = 161.57^\circ$。

7.  **绘制根轨迹图:**
    *   根据以上信息绘制根轨迹草图。两条分支从 $p_1, p_2$ 出发，以计算出的出射角方向离开。
    *   在实轴上的 $s=-\sqrt{10}$ 处汇合。
    *   一条分支沿实轴向右到达原点 $z_1=0$ (当 $T \to \infty$)。
    *   另一条分支沿实轴向左，趋向于 $-\infty$ (渐近线方向为 $\pi$)。

**数学公式与根轨迹性质总结：**

*   **根轨迹方程:** $1 + K G(s)H(s) = 0$
*   **参数根轨迹标准形式:** $1 + K_p G_p(s)H_p(s) = 0$
*   **开环零点 $z_i$:** $G_p(s)H_p(s)$ 的零点。
*   **开环极点 $p_j$:** $G_p(s)H_p(s)$ 的极点。
*   **实轴上的根轨迹:** 某段右边实轴零极点个数为奇数。
*   **渐近线倾角:** $\phi_a = \frac{(2k+1)\pi}{n-m}$
*   **渐近线交点:** $\sigma_a = \frac{\sum p_j - \sum z_i}{n-m}$
*   **分离点/汇合点:** $\frac{dK_p}{ds} = 0$ (或 $\frac{d(\text{参数})}{ds} = 0$)
*   **与虚轴交点:** 令 $s=j\omega$ 代入闭环特征方程，实部虚部分别为零。
*   **出射角 (从极点 $p_k$):** $\theta_{pk} = (2q+1)\pi + \sum_{i=1}^m \arg(p_k-z_i) - \sum_{j=1, j\neq k}^n \arg(p_k-p_j)$
    (视频中使用的是: $\sum \psi_z - \sum \psi_p = (2k+1)\pi$, 其中 $\psi$ 是向量角，出射角为待求量)

#### **四、 总结**

1.  **参数根轨迹核心:** 将原系统特征方程转化为标准根轨迹方程形式 $1+K_p G_p(s)H_p(s)=0$。
2.  **等效变换:** 找出等效开环传递函数 $G_p(s)H_p(s)$ 和等效根轨迹增益 $K_p$ (与原参数相关)。
3.  **标准绘制法则:** 转换后，按标准的根轨迹绘制法则进行绘制。
4.  **关键步骤回顾:**
    *   确定等效开环零极点。
    *   确定实轴根轨迹。
    *   计算渐近线。
    *   计算分离点/汇合点。
    *   计算与虚轴交点。
    *   计算出射角/入射角 (特别是当有复数开环零极点时)。
5.  **参数范围:** 注意参数 T 的取值范围 $[0, \infty)$ 对根轨迹的影响。
