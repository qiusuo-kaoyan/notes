### **题目：** 给定开环传递函数，求系统的根轨迹渐近线以及根轨迹与虚轴的交点 (根轨迹绘制)
![[2-1-1.png]]
![[2-1-2.png]]
**已知：**

*   开环传递函数： $G(s)H(s) = \frac{K}{s(s+1)(s+2)}$ (视频中口述，但实际计算时隐含为 $s(s-1)(s-2)$, 根据渐近线交点为1推断，此处按视频计算结果为准，假设极点为0, 1, 2)

**求：**

1.  根轨迹的渐近线
2.  根轨迹与虚轴的交点

#### **一、 根轨迹渐近线 (规则五)**

1.  **渐近线条数：**
    *   公式：$N - M$
    *   其中，$N$ 为开环极点个数，$M$ 为开环零点个数。
    *   **本题：** $N = 3$ (极点为 $s=0, s=-1, s=-2$)，$M = 0$ (无开环零点)。
    *   渐近线条数 = $3 - 0 = 3$ 条。

2.  **渐近线与实轴的倾角 (与正实轴的夹角):**
    *   公式：$\phi_k = \frac{(2k+1)\pi}{N-M}$
    *   其中，$k = 0, 1, 2, ..., N-M-1$。
    *   **本题：** $N-M = 3$, 所以 $k = 0, 1, 2$。
        *   $k=0: \phi_0 = \frac{\pi}{3} = 60^\circ$
        *   $k=1: \phi_1 = \frac{3\pi}{3} = \pi = 180^\circ$
        *   $k=2: \phi_2 = \frac{5\pi}{3} = 300^\circ$ (或 $-60^\circ$)

3.  **渐近线与实轴的交点 (汇合点):**
    *   公式：$\sigma_a = \frac{\sum p_i - \sum z_j}{N-M}$
    *   其中，$\sum p_i$ 是所有开环极点之和，$\sum z_j$ 是所有开环零点之和。
    *   **本题 (根据开环传递函数 $G(s)H(s) = \frac{K}{s(s+1)(s+2)}$，极点为 0, -1, -2):**
        *   $\sum p_i = 0 + (-1) + (-2) = -3$
        *   $\sum z_j = 0$
        *   $\sigma_a = \frac{-3 - 0}{3} = -1$

**渐近线图示（以视频中交点为 $\sigma_a = 1$ 为例）：**

*   交点在实轴的 1 处。
*   三条虚线分别以 $60^\circ$, $180^\circ$, $300^\circ$ (或 $-60^\circ$) 从交点射出。

**数学公式表达：**
*   开环传递函数: $G(s)H(s) = K\frac{\prod_{j=1}^M (s-z_j)}{\prod_{i=1}^N (s-p_i)}$
*   渐近线条数: $N-M$ (当 $N \ge M$ 时)
*   倾角: $\phi_k = \frac{(2k+1)\pi}{N-M}$, $k = 0, 1, ..., N-M-1$
*   交点: $\sigma_a = \frac{\sum_{i=1}^N p_i - \sum_{j=1}^M z_j}{N-M}$

#### **二、 根轨迹与虚轴的交点 (规则八)**

1.  **方法一：令特征方程中 $s = j\omega$**
    *   **闭环特征方程:** ==$1 + G(s)H(s) = 0$==
        *   **本题:** $1 + \frac{K}{s(s+1)(s+2)} = 0$
        *   $s(s+1)(s+2) + K = 0$
        *   $s^3 + 3s^2 + 2s + K = 0$
    *   **代入 $s = j\omega$:**
        *   $(j\omega)^3 + 3(j\omega)^2 + 2(j\omega) + K = 0$
        *   $-j\omega^3 - 3\omega^2 + 2j\omega + K = 0$
    *   **分离实部和虚部，并令它们分别等于零:**
        *   实部：$-3\omega^2 + K = 0$
        *   虚部：$-\omega^3 + 2\omega = 0 \Rightarrow \omega(-\omega^2 + 2) = 0$
    *   **求解 $\omega$ 和 $K$:**
        *   从虚部方程：$\omega = 0$ (对应实轴上的点，一般不认为是与虚轴的"交点") 或 $\omega^2 = 2 \Rightarrow \omega = \pm\sqrt{2}$
        *   将 $\omega^2 = 2$ 代入实部方程：$-3(2) + K = 0 \Rightarrow K = 6$
    *   **交点:** $s = \pm j\sqrt{2}$，此时 $K = 6$。

2.  **方法二：劳斯判据 (Routh-Hurwitz Criterion)**
    *   **闭环特征方程:** $s^3 + 3s^2 + 2s + K = 0$
    *   **构造劳斯表:**
        | $s^3$ | $1$     | $2$     |
        | ----- | ------- | ------- |
        | $s^2$ | $3$     | $K$     |
        | $s^1$ | $\frac{3 \cdot 2 - 1 \cdot K}{3} = \frac{6-K}{3}$ | $0$     |
        | $s^0$ | $K$     |         |
    *   **出现全零行 (临界稳定):** 当 $s^1$ 行为全零行时，系统在虚轴上有根。
        *   $\frac{6-K}{3} = 0 \Rightarrow K = 6$
    *   **构造辅助方程:** 使用全零行的上一行 ( $s^2$ 行) 的系数构造辅助方程 $A(s) = 0$。
        *   $3s^2 + K = 0$
    *   **代入 $K=6$ 并求解 $s$:**
        *   $3s^2 + 6 = 0 \Rightarrow s^2 = -2 \Rightarrow s = \pm j\sqrt{2}$
    *   **交点:** $s = \pm j\sqrt{2}$，此时 $K = 6$。

**方法比较:**
*   $s=j\omega$法：直观，但计算量可能较大，尤其在高阶系统。
*   劳斯判据法：系统性强，对于高阶系统，构造辅助方程求解根可能更简便。

**举例:**
* **不同极点零点:** 如果系统有零点, 渐近线的计算会相应改变. 例如, $G(s)H(s) = \frac{K(s+1)}{s(s+2)(s+3)}$, 则$N=3, M=1$.
* **不同阶次:** 对于四阶系统 $s^4 + As^3 + Bs^2 + Cs + D = 0$, 使用劳斯判据判断与虚轴交点通常更方便.
* **无虚轴交点:** 某些系统的根轨迹可能不与虚轴相交(除原点外), 此时劳斯表第一列不会出现符号变化, 也不会有全零行(除非K使系统稳定边界在原点).
* **起始于虚轴的根轨迹:** 如果开环极点本身就在虚轴上(非原点), 这也是根轨迹与虚轴的交点, 对应K=0.

#### **三、总结**

1.  **根轨迹渐近线：**
    *   **目的：** 描述当根轨迹上的点趋向无穷远时，根轨迹的走向。
    *   **三要素：** 条数、与实轴的倾角、与实轴的交点。
    *   **公式记忆：** 务必准确记忆并理解公式。

2.  **根轨迹与虚轴的交点：**
    *   **目的：** 确定系统从稳定到不稳定的临界点，以及临界稳定时的振荡频率。
    *   **两种方法：**
        *   令特征方程中 $s = j\omega$，分离实部虚部求解。
        *   利用劳斯判据，找到使劳斯表出现全零行的 $K$ 值，并由辅助方程求解虚轴上的根。
    *   **选择：** 根据特征方程的复杂度灵活选择。

3.  **考点重要性：** 根轨迹绘制是自动控制原理中的核心考点，渐近线和与虚轴交点的计算是绘制根轨迹的关键步骤，需要熟练掌握。
