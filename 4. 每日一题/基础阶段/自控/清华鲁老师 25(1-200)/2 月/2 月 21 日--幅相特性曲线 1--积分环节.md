#### **一、 核心概念回顾**

1.  **频率特性:**
    *   **定义:** 系统在正弦输入信号作用下，稳态输出与输入信号之间的幅值比和相位差随频率 $\omega$ 变化的特性。
    *   **获取方法:** 在传递函数 $G(s)$ 中，令 $s = j\omega$，得到频率特性函数 $G(j\omega)$。
    *   $G(j\omega) = A(\omega)e^{j\phi(\omega)}$
        *   $A(\omega)$: 幅频特性 (幅值比)
        *   $\phi(\omega)$: 相频特性 (相位差)

2.  **对数频率特性 (Bode 图):**
    *   **对数幅频特性 ($L(\omega)$):**
        *   **定义:** 幅频特性 $A(\omega)$ 的对数量，通常用分贝 (dB) 表示。
        *   **公式:** $L(\omega) = 20 \log_{10} A(\omega)$ (dB)
        *   **横坐标:** 频率 $\omega$ 的对数分度 (logarithmic scale)，但仍标以 $\omega$ 的值。即横坐标上等间距的点，其 $\omega$ 值为十倍关系 (如 0.1, 1, 10, 100)。
        *   **纵坐标:** $L(\omega)$，单位 dB。
    *   **对数相频特性 ($\phi(\omega)$):**
        *   **定义:** 相频特性 $\phi(\omega)$ 与频率 $\omega$ 的对数分度的关系曲线。
        *   **横坐标:** 同对数幅频特性。
        *   **纵坐标:** $\phi(\omega)$，单位通常为度 (°) 或弧度 (rad)。

3.  **幅相特性曲线 (Nyquist 图 或 极坐标图):**
    *   **定义:** 当频率 $\omega$ 从 0 变化到 $+\infty$ 时，频率特性 $G(j\omega)$ 在复平面上描绘出的轨迹。
    *   **表示方法:** 以 $G(j\omega)$ 的实部 $Re[G(j\omega)]$ 为横坐标，虚部 $Im[G(j\omega)]$ 为纵坐标。
    *   **几何意义:** 曲线上每一点到原点的距离代表该频率下的幅值 $A(\omega)$，该点与原点的连线与正实轴的夹角代表该频率下的相角 $\phi(\omega)$。

**数学公式表达：**
* 频率特性函数: $G(j\omega) = |G(j\omega)| e^{j\angle G(j\omega)} = A(\omega) e^{j\phi(\omega)}$
* 幅频特性: $A(\omega) = |G(j\omega)|$
* 相频特性: $\phi(\omega) = \angle G(j\omega)$

#### **二、 积分环节 $G(s) = \frac{1}{s}$ 的频率特性分析**

1.  **求频率特性表达式 $G(j\omega)$:**
    *   令 $s = j\omega$，则 $G(j\omega) = \frac{1}{j\omega}$
    *   化简: $G(j\omega) = \frac{1}{j\omega} = \frac{-j}{\omega} = 0 - j\frac{1}{\omega}$

2.  **求幅频特性 $A(\omega)$ 和相频特性 $\phi(\omega)$:**
    *   **幅频特性:** $A(\omega) = |G(j\omega)| = |\frac{1}{j\omega}| = \frac{1}{|j\omega|} = \frac{1}{\omega}$
    *   **相频特性:** $\phi(\omega) = \angle G(j\omega) = \angle (\frac{1}{j\omega}) = \angle (1) - \angle (j\omega) = 0 - 90^\circ = -90^\circ$
        *   (或者: $\angle (-j\frac{1}{\omega}) = -90^\circ$，因为 $\frac{1}{\omega} > 0$)

3.  **绘制对数幅频特性曲线 $L(\omega)$:**
    *   $L(\omega) = 20 \log_{10} A(\omega) = 20 \log_{10} (\frac{1}{\omega}) = -20 \log_{10} \omega$ (dB)
    *   **特性分析:**
        *   当 $\omega = 1$ rad/s 时，$L(1) = -20 \log_{10} 1 = 0$ dB。曲线过点 (1, 0dB)。
        *   当 $\omega$ 增大 10 倍 (一个十倍频程) 时，$L(10\omega) = -20 \log_{10} (10\omega) = -20 (\log_{10} 10 + \log_{10} \omega) = -20 - 20 \log_{10} \omega = L(\omega) - 20$ dB。
        *   **结论:** 对数幅频特性是一条斜率为 **-20 dB/十倍频程** 的直线，且过点 ($\omega=1$, 0dB)。

4.  **绘制对数相频特性曲线 $\phi(\omega)$:**
    *   $\phi(\omega) = -90^\circ$ (或 $-\frac{\pi}{2}$ rad)
    *   **特性分析:** 相角恒为 -90°，与频率 $\omega$ 无关。
    *   **结论:** 对数相频特性是一条在 -90° 处的水平直线。

5.  **绘制幅相特性曲线 (Nyquist 图):**
    *   $G(j\omega) = 0 - j\frac{1}{\omega}$
    *   实部: $Re[G(j\omega)] = 0$
    *   虚部: $Im[G(j\omega)] = -\frac{1}{\omega}$
    *   **特性分析:**
        *   当 $\omega \to 0^+$ (频率从正方向趋近于 0) 时：
            *   $A(\omega) = \frac{1}{\omega} \to +\infty$ (幅值无穷大)
            *   $\phi(\omega) = -90^\circ$ (相角恒定)
            *   $Im[G(j\omega)] = -\frac{1}{\omega} \to -\infty$
        *   当 $\omega \to +\infty$ (频率趋近于无穷大) 时：
            *   $A(\omega) = \frac{1}{\omega} \to 0$ (幅值趋近于 0)
            *   $\phi(\omega) = -90^\circ$ (相角恒定)
            *   $Im[G(j\omega)] = -\frac{1}{\omega} \to 0^-$ (从负方向趋近于0)
    *   **结论:** 幅相特性曲线是复平面上 **负虚轴** 的一部分，起点在负虚轴的无穷远处($\omega \to 0^+$)，终点在原点($\omega \to +\infty$)。箭头方向从下往上指向原点。

*   **Bode 图:**
    *   **幅频图:** 一条过 (1, 0dB) 点，斜率为 -20dB/dec 的直线。
    *   **相频图:** 一条在 -90° 的水平直线。
*   **Nyquist 图:** 沿负虚轴从 $-\infty j$ 指向原点 $0$ 的一条射线。

**举例:**

*   **复杂系统的Bode图绘制**: 如果一个系统由多个典型环节串联而成,其总的对数幅频特性是各个环节对数幅频特性之和,总的相频特性是各个环节相频特性之和. 因此掌握积分环节的特性对于分析复杂系统至关重要.
*   **Nyquist稳定性判据**: 积分环节的Nyquist图不包围(-1, j0)点, 这在利用Nyquist稳定性判据分析系统稳定性时会用到.
*   **频率响应与时间响应**: 积分环节的频率特性反映了它对不同频率正弦输入的响应.例如,对低频信号,幅值响应很大;对高频信号,幅值响应很小.

#### **三、 总结**

1.  **绘制流程:**
    1.  传递函数 $G(s) \xrightarrow{s=j\omega} G(j\omega)$ (频率特性表达式)
    2.  $G(j\omega) \rightarrow A(\omega), \phi(\omega)$ (幅频、相频特性)
    3.  $A(\omega) \rightarrow L(\omega) = 20 \log_{10} A(\omega)$ (对数幅频特性)
    4.  绘制 $L(\omega)$ 和 $\phi(\omega)$ (Bode 图)
    5.  分析 $\omega \to 0^+$ 和 $\omega \to +\infty$ 时的 $G(j\omega)$ 行为，绘制幅相特性曲线 (Nyquist 图)。

2.  **积分环节 $G(s) = \frac{1}{s}$ 的关键特性:**
    *   **对数幅频:** 斜率为 -20 dB/十倍频程，过 (1, 0dB)。
    *   **对数相频:** 恒为 -90°。
    *   **幅相曲线:** 负虚轴，从 $-\infty j$ 指向原点。

3.  **重要性:** 掌握典型环节的频率特性是分析复杂系统频率特性的基础。
