**考察知识点：** 规则七——根轨迹的出射角与入射角

#### **一、 基本概念**

1.  **出射角与入射角的产生条件：**
    *   ==当开环传递函数存在**复数极点**或**复数零点**时，根轨迹会存在出射角或入射角。==
    *   **出射角 (Departure Angle):** 根轨迹离开**开环复数极点**处的切线与实轴正方向的夹角。
    *   **入射角 (Arrival Angle):** 根轨迹进入**开环复数零点**处的切线与实轴正方向的夹角。

2.  **根轨迹的起始点与终止点:**
    *   根轨迹起始于开环极点。
    *   根轨迹终止于开环零点 (或无穷远处)。

#### **二、 出射角的计算公式**

设待求出射角的开环复数极点为 $p_j$。

$\theta_{p_j} = (2k+1)\pi + \sum_{i=1}^{m} \angle (p_j - z_i) - \sum_{i=1, i \neq j}^{n} \angle (p_j - p_i)$

其中：

*   $\theta_{p_j}$：极点 $p_j$ 处的出射角。
*   $k = 0, \pm 1, \pm 2, ...$ (通常取 $k=0$，即使用 $\pi$ 进行计算)。
*   $m$：开环零点的个数。
*   $n$：开环极点的个数。
*   $z_i$：第 $i$ 个开环零点。
*   $p_i$：第 $i$ 个开环极点。
*   $\sum_{i=1}^{m} \angle (p_j - z_i)$: **所有开环零点**到该复数极点 $p_j$ 的向量与实轴正方向的夹角之和 (即零点对该极点贡献的相角和)。
*   $\sum_{i=1, i \neq j}^{n} \angle (p_j - p_i)$: **除 $p_j$ 自身以外的所有其他开环极点**到该复数极点 $p_j$ 的向量与实轴正方向的夹角之和 (即其他极点对该极点贡献的相角和)。

**简化记忆公式:**

$\theta_{出射角} = \pi + (\text{所有零点到该复数极点的相角和}) - (\text{所有其他极点到该复数极点的相角和})$

**注意：** 角度的正负取决于向量与实轴正方向的夹角，逆时针为正，顺时针为负。实际计算中，通常先计算出锐角，再根据象限确定实际角度。

#### **三、 入射角的计算公式**

设待求入射角的开环复数零点为 $z_j$。

$\phi_{z_j} = (2k+1)\pi - \sum_{i=1}^{m, i \neq j} \angle (z_j - z_i) + \sum_{i=1}^{n} \angle (z_j - p_i)$

**简化记忆公式:**

$\phi_{入射角} = \pi - (\text{所有其他零点到该复数零点的相角和}) + (\text{所有极点到该复数零点的相角和})$

**与出射角公式的区别：**

*   符号不同：出射角公式中零点贡献为正，其他极点贡献为负；入射角公式中其他零点贡献为负，极点贡献为正。
*   计算对象不同：出射角是针对复数极点，入射角是针对复数零点。

#### **四、 题目实例解析 (视频中的例子)**

**系统特征方程:** (视频中未直接给出特征方程，而是直接给了开环零极点)

*   **开环零点:** $z_1 = 0$, $z_2 = -4$
*   **开环极点:** $p_1 = -1+j$, $p_2 = -1-j$

**求复数极点 $p_1 = -1+j$ 的出射角 $\theta_{p_1}$:**

1.  **在复平面中标出零极点。**
2.  **计算零点到 $p_1$ 的相角:**
    *   $\angle(p_1 - z_1) = \angle((-1+j) - 0) = \angle(-1+j)$。该向量在第二象限，与实轴负方向夹角为 $\pi/4$，所以与实轴正方向夹角为 $3\pi/4$。
    *   $\angle(p_1 - z_2) = \angle((-1+j) - (-4)) = \angle(3+j)$。该向量在第一象限，$\tan^{-1}(1/3)$。 (视频中称之为 $\text{arc tan}(1/3) \approx 0.322 \text{ rad}$)

3.  **计算其他极点到 $p_1$ 的相角:**
    *   $\angle(p_1 - p_2) = \angle((-1+j) - (-1-j)) = \angle(2j)$。该向量沿虚轴正方向，与实轴正方向夹角为 $\pi/2$。

4.  **代入出射角公式:**
    $\theta_{p_1} = \pi + [\angle(p_1 - z_1) + \angle(p_1 - z_2)] - [\angle(p_1 - p_2)]$
    $\theta_{p_1} = \pi + [3\pi/4 + \tan^{-1}(1/3)] - [\pi/2]$
    $\theta_{p_1} = \pi + 0.75\pi + 0.322 - 0.5\pi$
    $\theta_{p_1} = 1.25\pi + 0.322 \approx 3.927 + 0.322 = 4.249 \text{ rad}$
    (视频中直接给出的结果为：$\theta_{p_1} = \pi + (3\pi/4 + \text{arctan}(1/3)) - \pi/2$  最后结果是 $\approx -2.032 \text{ rad}$ 或 $\approx -116.5^\circ$。 这与我根据其描述和公式计算的结果不同，可能是视频中的计算有误，或者我对视频中某个细节理解有偏差。**演讲稿中公式是 $\theta_{p_j} = (2k+1)\pi + \sum \angle (p_j - z_i) - \sum \angle (p_j - p_i)$，取 $k=-1$ 时，是 $-\pi$。** 如果按视频中的 $(2k+1)\pi$ 取 $-\pi$，则：
    $\theta_{p_1} = -\pi + [3\pi/4 + \tan^{-1}(1/3)] - [\pi/2]$
    $\theta_{p_1} = -\pi + 2.356 + 0.322 - 1.571 = -3.141 + 2.356 + 0.322 - 1.571 = -2.034 \text{ rad}$。 这个结果与视频中一致。**所以公式中的 $(2k+1)\pi$ 应该理解为根据具体情况选择 $\pi$ 或 $-\pi$ 等，以使出射角落在合理范围内，或者说通常用 $\pm 180^\circ$。** )

5.  **利用对称性求 $p_2 = -1-j$ 的出射角 $\theta_{p_2}$:**
    由于根轨迹关于实轴对称，所以 $\theta_{p_2} = -\theta_{p_1} \approx 2.032 \text{ rad}$。

**绘制出射角示意图：** 在复平面上，从 $p_1$ 点出发，与实轴正方向成 $\theta_{p_1}$ 角画出切线方向。

#### **五、 总结**

1.  **出射角/入射角的条件：** 存在开环复数零极点。
2.  **公式应用：**
    *   **出射角：** $\theta = (2k+1)\pi + \sum \angle (\text{极点}-\text{零点}) - \sum \angle (\text{极点}-\text{其他极点})$
    *   **入射角：** $\phi = (2k+1)\pi - \sum \angle (\text{零点}-\text{其他零点}) + \sum \angle (\text{零点}-\text{极点})$
    *   **简化记忆：** 通常取 $(2k+1)\pi$ 为 $\pi$ 或 $-\pi$。 出射角是 “加零减极”，入射角是 “减零加极”。
3.  **对称性：** 共轭复根的出射角/入射角关于实轴对称。
4.  **几何意义：** 出射角/入射角决定了根轨迹离开复数极点或进入复数零点的初始方向。
5.  **注意事项:** 务必正确计算各个向量的相角 (大小和符号)。
