### **题目：** 已知单位反馈系统开环传递函数，求系统根轨迹的分离点 (根轨迹绘制规则应用)
![[1-31.png]]
![[1-31-1.png]]
**已知：**

*   单位反馈系统
*   开环传递函数：$G(s) = \frac{K}{s(s+1)(s+2)}$

**求：** 系统根轨迹的分离点。

#### **一、 根轨迹绘制基本规则回顾 (重要基础)**

在求解分离点之前，讲座回顾了根轨迹绘制的八条基本规则，这些规则是理解和解决此类问题的基础。

1.  **规则一：根轨迹基本方程 (首1型/零极点形式)**
    *   将传递函数化为 $1 + KG_0(s)H_0(s) = 0$ 或 $K = -\frac{1}{G_0(s)H_0(s)}$ 的形式。
    *   $G_0(s)H_0(s)$ 表示开环传递函数中除去增益 $K$ 的部分，并化为首1型（分子分母最高次项系数为1）。
    *   本题开环传递函数已是首1型：$G(s)H(s) = \frac{K}{s(s+1)(s+2)}$ (单位反馈 H(s)=1)。

2.  **规则二：根轨迹的分支数、起点与终点**
    *   **分支数：** 等于系统的阶数 $n$（开环传递函数分母多项式的最高次数）。
        *   本题：$n=3$，有3条根轨迹分支。
    *   **起点：** 开环传递函数的极点。
        *   本题：$p_1=0, p_2=-1, p_3=-2$。
    *   **终点：** 包括 $m$ 个有限零点（开环零点）和 $n-m$ 个无限零点（终止于无穷远处）。
        *   本题：开环零点数 $m=0$，所以有 $3-0=3$ 个无限零点。

3.  **规则三：根轨迹在实轴上的分布**
    *   根轨迹在实轴上某一段存在的条件是：该段右侧实轴上开环零、极点总数为奇数。
        *   本题开环极点：$0, -1, -2$。
        *   实轴上的根轨迹分布：$(-\infty, -2]$ 和 $[-1, 0]$。
        *   **图示说明:**
            ```
            <--|-----|-----|-----|--> 实轴
              -2    -1     0
            ^^^         ^^^       (根轨迹存在区域)
            ```
        *   在 $(0, +\infty)$ 右侧零极点数为0 (偶数)，不存在根轨迹。
        *   在 $(-1, 0)$ 右侧零极点数为1 (奇数)，存在根轨迹。
        *   在 $(-2, -1)$ 右侧零极点数为2 (偶数)，不存在根轨迹。
        *   在 $(-\infty, -2)$ 右侧零极点数为3 (奇数)，存在根轨迹。

4.  **规则四：根轨迹的对称性**
    *   根轨迹关于实轴对称。

5.  **规则五：根轨迹的渐近线**
    *   当 $K \to \infty$ 时，根轨迹趋向于渐近线。
    *   渐近线与实轴的交点 (会合点/质心)：$\sigma_a = \frac{\sum{p_i} - \sum{z_i}}{n-m}$
    *   渐近线与实轴的夹角：$\phi_a = \frac{(2k+1)\pi}{n-m}$，其中 $k=0, 1, 2, ..., n-m-1$。
    *   (本题未详细展开计算，后续题目会单独讲解)

6.  **规则六：根轨迹的分离点与会合点**
    *   **分离点:** 两条或多条根轨迹从实轴上某一点分离，进入复平面。
    *   **会合点:** 两条或多条根轨迹从复平面会合到实轴上某一点。
    *   (本题核心考点)

7.  **规则七：根轨迹的出射角和入射角**
    *   **出射角:** 根轨迹离开开环复数极点的角度。
    *   **入射角:** 根轨迹进入开环复数零点的角度。
    *   (本题未涉及，后续题目会单独讲解)

8.  **规则八：根轨迹与虚轴的交点**
    *   用于判断系统的临界稳定状态和稳定域。
    *   (本题未涉及，后续题目会单独讲解)

#### **二、 根轨迹分离点/会合点的求解方法 (核心)**

1.  **方法一：基于 $K$ 对 $s$ 求导 (演讲稿中称为方法一)**
    *   **步骤：**
        1.  由根轨迹方程 ==$1 + G(s)H(s) = 0$== 得到 $K = - \frac{1}{G_0(s)H_0(s)}$。
        2.  对 $K$ 关于 $s$ 求导，即 $\frac{dK}{ds} = 0$。
        3.  解出方程的根 $s$，这些根即为可能的分离点或会合点。
        4.  **检验：** 只有落在实轴根轨迹上的点才是有效的分离点/会合点。
    *   **本题应用：**
        1.  $K = -s(s+1)(s+2) = -(s^3 + 3s^2 + 2s)$
        2.  $\frac{dK}{ds} = -(3s^2 + 6s + 2) = 0$
        3.  $3s^2 + 6s + 2 = 0$
        4.  解得：$s_1 = -0.423$, $s_2 = -1.577$
        5.  **检验：**
            *   实轴根轨迹为 $(-\infty, -2]$ 和 $[-1, 0]$。
            *   $s_1 = -0.423$ 位于 $[-1, 0]$ 区间内，是有效的分离点。
            *   $s_2 = -1.577$ 不在任何实轴根轨迹段上，舍去。
        *   因此，分离点为 $s = -0.423$。
    *   **分离点处的K值:** 将 $s = -0.423$ 代回 $K = -(s^3 + 3s^2 + 2s)$，可求得 $K = 0.385$。这个值在判断系统特性（如出现共轭复根的K范围）时有用。

2.  **方法二：基于零极点求和公式 (演讲稿中称为方法二，更适用于无零点或简单零点情况)**
    *   **公式 (无开环零点时):** $\sum_{i=1}^{n} \frac{1}{s-p_i} = 0$
    *   **公式 (有开环零点时):** $\sum_{i=1}^{n} \frac{1}{s-p_i} = \sum_{j=1}^{m} \frac{1}{s-z_j}$
        (注意: 视频中讲到如果右边有零点，是 $1/(D+z_j)$，这个表述是错误的，应该是 $1/(s-z_j)$ 移项到左边变为负号，或者等式右边为零点项之和)
        **修正后的公式 (更通用):** $\sum_{i=1}^{n} \frac{1}{s-p_i} - \sum_{j=1}^{m} \frac{1}{s-z_j} = 0$
    *   **本题应用 (无开环零点):**
        1.  极点为 $p_1=0, p_2=-1, p_3=-2$。
        2.  $\frac{1}{s-0} + \frac{1}{s-(-1)} + \frac{1}{s-(-2)} = 0$
        3.  $\frac{1}{s} + \frac{1}{s+1} + \frac{1}{s+2} = 0$
        4.  通分：$(s+1)(s+2) + s(s+2) + s(s+1) = 0$
        5.  展开：$(s^2+3s+2) + (s^2+2s) + (s^2+s) = 0$
        6.  $3s^2 + 6s + 2 = 0$
        7.  解得结果与方法一相同。

**两种方法的选择：**

*   方法一 (求导法) 更为通用，尤其当开环传递函数有分子（即有零点）时，表达式可能更复杂，但求导是标准操作。
*   方法二 (零极点求和) 在没有开环零点或零点较少时，形式上可能更简洁。但当零极点较多时，通分计算量也会增大。
*   个人习惯和题目具体形式会影响方法的选择。

#### **三、 总结与注意事项**

1.  **方法选择：** 两种求解分离点的方法都可以使用，根据个人习惯和题目特点选择。
2.  **结果取舍 (重要)：** 求出的 $s$ 值必须位于实轴根轨迹上才是有效的分离点或会合点，否则需要舍去。
3.  **分离点K值：** 求解分离点对应的 $K$ 值，有助于分析系统特性随 $K$ 变化的规律（如出现共轭复根、稳定性等）。
4.  **考点整合：** 实际考试中通常不会单独考分离点，而是作为绘制完整根轨迹图的一部分。
