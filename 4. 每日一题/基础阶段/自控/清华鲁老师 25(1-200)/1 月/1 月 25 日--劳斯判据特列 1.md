### **题目：** 已知系统特征方程，用劳斯判据判断系统稳定性 (特例：第一列出现零) (劳斯判据应用题)

![[1-25.png]]

**已知：** 系统的特征方程。
**要求：** 用劳斯判据判断系统的稳定性，特别是当劳斯表第一列出现零元素，且该零元素所在行其他元素不为零的情况。

#### **一、 劳斯判据基本步骤 (回顾)**

1.  **必要条件判断：**
    *   特征方程的**所有系数均为正**。
    *   特征方程**不缺项**。
    *   **注意：** 满足必要条件不一定稳定，但不满足必要条件一定不稳定。

2.  **列劳斯表：**
    *   根据特征方程的系数排列劳斯表。
    *   **第一行：** 按 $s$ 的偶次幂系数排列 (从最高次开始)。
    *   **第二行：** 按 $s$ 的奇次幂系数排列 (从最高次开始)。
    *   **后续行计算公式：**
        假设当前计算行为 $s^k$ 行，其前两行为 $s^{k+2}$ 和 $s^{k+1}$ 行。
        $s^k$ 行的第一个元素 $b_1 = -\frac{1}{a_1} \begin{vmatrix} a_0 & a_2 \\ a_1 & a_3 \end{vmatrix} = \frac{a_1 a_2 - a_0 a_3}{a_1}$
        $s^k$ 行的第二个元素 $b_2 = -\frac{1}{a_1} \begin{vmatrix} a_0 & a_4 \\ a_1 & a_5 \end{vmatrix} = \frac{a_1 a_4 - a_0 a_5}{a_1}$
        以此类推。其中 $a_0, a_1$ 分别为 $s^{k+2}$ 和 $s^{k+1}$ 行的第一个元素。

3.  **稳定性判断：**
    *   观察劳斯表**第一列**元素的符号。
    *   如果第一列所有元素**均为正**，则系统稳定。
    *   如果第一列元素出现**符号改变**，则系统不稳定，且符号改变的次数等于特征方程在右半 s 平面根的个数。

#### **二、 劳斯判据特例：第一列出现零元素**

**情况描述：** 在列劳斯表的过程中，某一行的第一个元素计算结果为零，但该行其他元素不全为零。

**处理方法：**

1.  **用 $\epsilon$ 替代：** 将该零元素用一个**任意小的正数 $\epsilon$** (epsilon) 替代。
2.  **继续计算：** 用包含 $\epsilon$ 的表达式继续完成劳斯表的计算。
3.  **判断符号：** 判断劳斯表第一列中包含 $\epsilon$ 的项，在 $\epsilon \to 0^+$ 时的符号。
    *   根据 $\epsilon \to 0^+$ 时各项的符号，判断第一列元素的符号是否改变。
    *   符号改变的次数即为系统在右半 s 平面根的个数。

**数学公式表达：**
* **特征方程 (示例):** $a_n s^n + a_{n-1} s^{n-1} + ... + a_1 s + a_0 = 0$
* **劳斯表 (示例，假设 $s^2$ 行为特例行):**
    ```
    s^4 | a_4   a_2   a_0
    s^3 | a_3   a_1   0
    s^2 | b_1   b_2   0    <-- 假设 b_1 计算为 0
    s^1 | c_1   0
    s^0 | d_1
    ```
    若 $b_1 = \frac{a_3 a_2 - a_4 a_1}{a_3} = 0$，且 $b_2 = \frac{a_3 a_0 - a_4 \cdot 0}{a_3} = a_0 \neq 0$
    则令 $b_1 = \epsilon$。
    继续计算 $c_1 = \frac{\epsilon a_1 - a_3 b_2}{\epsilon}$。
    判断 $\lim_{\epsilon \to 0^+} c_1$ 的符号。

**举例 (视频中的例子):**
特征方程: $s^4 + s^3 + 2s^2 + 2s + 3 = 0$

1.  **必要条件：** 系数均为正，不缺项，满足。

2.  **列劳斯表：**
    ```
    s^4 | 1   2   3
    s^3 | 1   2   0
    s^2 | 0   3   0   <-- 第一列出现零，该行非零元素 3
    ```
    用 $\epsilon$ 替代 0：
    ```
    s^4 | 1   2   3
    s^3 | 1   2   0
    s^2 | ε   3   0
    s^1 | (2ε - 3)/ε   0
    s^0 | 3
    ```

3.  **稳定性判断：**
    *   第一列元素： $1, 1, \epsilon, \frac{2\epsilon - 3}{\epsilon}, 3$
    *   当 $\epsilon \to 0^+$ 时：
        *   $\epsilon > 0$
        *   $\frac{2\epsilon - 3}{\epsilon} \approx \frac{-3}{\epsilon} \to -\infty$ (负号)
    *   第一列符号序列： 正, 正, 正, 负, 正。
    *   符号改变两次 (从 $\epsilon$ 到 $\frac{2\epsilon - 3}{\epsilon}$，从 $\frac{2\epsilon - 3}{\epsilon}$ 到 $3$)。
    *   结论：系统不稳定，有两个根在右半 s 平面。

**另一个例子:**
特征方程: $s^3 + s^2 + s + 1 = 0$
1. **必要条件:** 满足
2. **列劳斯表:**
   ```
   s^3 | 1   1
   s^2 | 1   1
   s^1 | 0   0  <-- 第一列为0, 且该行全为0 (这是另一种特例，需要构造辅助多项式，本题未涉及)
   ```
   如果题目是 $s^3 + s^2 + s + K = 0$, 要求K为何值时稳定。
   ```
   s^3 | 1   1
   s^2 | 1   K
   s^1 | (1-K)/1   0
   s^0 | K
   ```
   要稳定，则 $1>0, 1>0, 1-K>0, K>0$, 所以 $0 < K < 1$.
   如果 $K=1$, 则 $s^1$ 行第一个元素为0, 但该行其他元素也为0 (只有一个元素). 此时需要构造辅助多项式 $A(s) = s^2+1=0$, $s=\pm j$, 系统临界稳定.

#### **三、总结**

1.  **核心考点：** 劳斯判据的特例处理——第一列出现零元素，且该行其他元素不全为零。
2.  **解决方法：** 用任意小的正数 $\epsilon$ 替代零元素，继续完成劳斯表，并分析 $\epsilon \to 0^+$ 时第一列元素的符号。
3.  **重要性：** 虽然考频不高，但作为劳斯判据的完整知识点，需要掌握。
4.  **区分其他特例：** 注意与“某一行元素全为零”的特例区分开，后者需要构造辅助多项式。
