### **题目：** 给定系统结构图、传递函数、参考输入和扰动输入，求参考输入和扰动输入作用下的稳态误差 (稳态误差计算题)

**考察知识点：** 稳态误差和稳态误差系数

#### **一、 预备知识：稳态误差的定义与求解**

**1. 误差信号 E(s) 的定义 (针对参考输入):**
   *   $E(s) = R(s) - C(s)$  (其中 $C(s)$ 为实际输出)
   *   对于单位负反馈系统，且反馈环节为H(s), 输出 $C(s) = E(s)G_{开环}(s)H(s)$ (假设$C(s)=Y(s)$)
   *   推导可得误差的传递函数： $E(s) = \frac{R(s)}{1 + G_{开环}(s)H(s)}$
      *  **注意:** 视频中推导为 $E(s) = R(s) - Y(s)$，而 $Y(s) = G_1(s)G_2(s)E(s)$ （假设反馈H(s)=1且误差在G1之前）。  如果反馈为H(s)，则 $Y(s)=C(s)$， $E(s) = R(s) - H(s)Y(s)$, $Y(s)=G_1(s)G_2(s)E(s)$。
      *  最终，对于参考输入 $R(s)$，误差 $E_r(s) = \frac{R(s)}{1 + G_1(s)G_2(s)H(s)}$ (这里G代表 $G_1G_2$)

**2. 稳态误差 $e_{ss}$ 的计算 (终值定理):**
   *   $e_{ss} = \lim_{t \to \infty} e(t) = \lim_{s \to 0} sE(s)$

#### **二、 参考输入作用下的稳态误差求解**

**方法一：直接根据定义和终值定理**

1.  **写出开环传递函数 $G_{开}(s)$:**  $G_{开}(s) = G_1(s)G_2(s)H(s)$
2.  **写出误差的表达式 $E_r(s)$:** $E_r(s) = \frac{R(s)}{1 + G_{开}(s)}$ (假设 H(s)=1，如视频中推导)
    *   **更通用(考虑H(s)):** $E_r(s) = \frac{R(s)}{1 + G_1(s)G_2(s)H(s)}$
3.  **应用终值定理求稳态误差 $e_{ssr}$:**
    $e_{ssr} = \lim_{s \to 0} s E_r(s) = \lim_{s \to 0} \frac{s R(s)}{1 + G_1(s)G_2(s)H(s)}$

**方法二：稳态误差系数法 (与输入信号类型相关)**

*   **适用条件:** 单位负反馈系统或可等效为单位负反馈系统。
*   **开环传递函数标准型:** $G_{开}(s) = \frac{K \prod (T_i s + 1)}{s^v \prod (T_j s + 1)}$，其中 $v$ 为系统型别。

1.  **位置误差系数 $K_p$ (针对阶跃输入):**
    *   **输入:** $r(t) = A \cdot 1(t)$  (A 为阶跃幅度)  $\implies R(s) = \frac{A}{s}$
    *   **定义:** $K_p = \lim_{s \to 0} G_{开}(s)$
    *   **稳态误差:** $e_{ssr} = \frac{A}{1 + K_p}$

2.  **速度误差系数 $K_v$ (针对斜坡输入):**
    *   **输入:** $r(t) = A \cdot t$  (A 为斜坡信号的幅值/斜率) $\implies R(s) = \frac{A}{s^2}$
    *   **定义:** $K_v = \lim_{s \to 0} s G_{开}(s)$
    *   **稳态误差:** $e_{ssr} = \frac{A}{K_v}$

3.  **加速度误差系数 $K_a$ (针对抛物线/加速度输入):**
    *   **输入:** $r(t) = \frac{A}{2} t^2$ (A 为抛物线信号的幅值) $\implies R(s) = \frac{A}{s^3}$
    *   **定义:** $K_a = \lim_{s \to 0} s^2 G_{开}(s)$
    *   **稳态误差:** $e_{ssr} = \frac{A}{K_a}$

**使用误差系数法的注意事项：**
*   必须根据输入信号的类型选择对应的误差系数。
*   公式中的 A 是输入信号的幅值，对于单位输入，A=1。

#### ==**三、 扰动输入作用下的稳态误差求解**==

**1. 扰动作用下的输出 $Y_d(s)$:**
   *   根据系统结构图和梅逊增益公式（或[[信号流图]]化简）求出扰动 $D(s)$ 到输出 $Y(s)$ 的传递函数 $G_d(s) = \frac{Y_d(s)}{D(s)}$。
   *   视频中给出的扰动输出为：$Y_d(s) = \frac{G_2(s)D(s)}{1 + G_1(s)G_2(s)H(s)}$

**2. 扰动作用下的误差 $E_d(s)$:**
   *   **定义:** $E_d(s) = 0 - C_d(s) = -Y_d(s)$ (因为参考输入为0时，理想输出为0，误差就是负的扰动引起的输出)
        *  **更准确地:** 扰动作用下的误差，通常指扰动对系统输出的影响。如果误差定义点在输入端，且参考输入为0，则 $E_d(s) = R(s) - C_d(s)_{反馈点} = 0 - H(s)Y_d(s)$
        *  视频中给出的误差定义方式：$E_d(s) = \frac{-G_2(s) D(s)}{1 + G_1(s)G_2(s)H(s)}$ (这里是假设误差直接就是负的输出，或者H(s)=1且误差就是输出)
        *  **需要注意:** 视频中推导的 $E_d(s)$ 似乎是直接将 $Y_d(s)$ 取负，这在某些误差定义下是成立的，但需要明确误差的测量点。如果 $E(s) = R(s) - C(s)$，且 $R(s)=0$，则 $E_d(s) = -C_d(s)$。

**3. 扰动作用下的稳态误差 $e_{ssd}$ (终值定理):**
   $e_{ssd} = \lim_{s \to 0} s E_d(s) = \lim_{s \to 0} \frac{-s G_2(s) D(s)}{1 + G_1(s)G_2(s)H(s)}$

**扰动稳态误差求解注意事项：**
*   注意 $E_d(s)$ 表达式中的负号。
*   分子是扰动输入点到输出点之间的传递函数乘以扰动输入 $D(s)$。
*   分母是 $1 + G_{开环}(s)H(s)$。

#### **四、 题目具体分析**

**给定系统：** (结构图略，根据描述 G1, G2, H 和扰动点 D)
*   $G_1(s)$
*   $G_2(s)$
*   $H(s)$
*   参考输入 $R(s) = \frac{5}{s^2}$ (斜坡输入，幅值为5)
*   扰动输入 $D(s)$ (视频中未明确给出扰动类型，但在计算中代入了具体值，假设也为某种形式的输入)

**1. 参考输入下的稳态误差 $e_{ssr}$**

   *   **开环传递函数:** $G_{开}(s) = G_1(s)G_2(s)H(s) = \frac{1}{s(s+1)}$ (根据视频计算结果反推)
   *   **方法一 (直接法):**
      $e_{ssr} = \lim_{s \to 0} \frac{s (\frac{5}{s^2})}{1 + \frac{1}{s(s+1)}} = \lim_{s \to 0} \frac{\frac{5}{s}}{1 + \frac{1}{s(s+1)}} = \lim_{s \to 0} \frac{5s(s+1)}{s(s(s+1)+1)} = \lim_{s \to 0} \frac{5(s+1)}{s(s+1)+1} = \frac{5}{1} = 5$
      **视频中计算结果是 0.05，与我这里推导不一致，需要核对视频中的 G1, G2, H 具体表达式。**
      根据视频计算过程：$G_{开}(s)H(s) = \frac{100}{s(s+1)}$， $R(s) = \frac{5}{s^2}$
      $e_{ssr} = \lim_{s \to 0} \frac{s \cdot \frac{5}{s^2}}{1 + \frac{100}{s(s+1)}} = \lim_{s \to 0} \frac{\frac{5}{s}}{\frac{s(s+1)+100}{s(s+1)}} = \lim_{s \to 0} \frac{5(s+1)}{s(s+1)+100} = \frac{5 \cdot 1}{100} = 0.05$
      (看来视频中 $G_1(s)G_2(s)H(s) = \frac{100}{s(s+1)}$)

   *   **方法二 (误差系数法):**
      *   输入为斜坡信号，幅值 A = 5。使用速度误差系数 $K_v$。
      *   $K_v = \lim_{s \to 0} s G_{开}(s)H(s) = \lim_{s \to 0} s \frac{100}{s(s+1)} = \lim_{s \to 0} \frac{100}{s+1} = 100$
      *   $e_{ssr} = \frac{A}{K_v} = \frac{5}{100} = 0.05$

**2. 扰动输入下的稳态误差 $e_{ssd}$**

   *   **扰动传递函数 (从D到Y，再到误差E):** 假设 $G_2(s) = \frac{s+1}{s+1}$ (只是为了凑视频结果, 实际需要看图),  $D(s) = \frac{3}{s}$ (假设扰动为幅值为3的阶跃)
   *   $e_{ssd} = \lim_{s \to 0} \frac{-s G_2(s) D(s)}{1 + G_1(s)G_2(s)H(s)}$
   *   假设视频中 $G_2(s) = \frac{0.03(s+1)}{s+1}$ (瞎猜的,为了凑结果-0.03,真实G2看图), $D(s)=\frac{1}{s}$ (单位阶跃扰动)
   *   $e_{ssd} = \lim_{s \to 0} \frac{-s \cdot \frac{0.03(s+1)}{s+1} \cdot \frac{1}{s}}{1 + \frac{100}{s(s+1)}} = \lim_{s \to 0} \frac{-0.03}{\frac{s(s+1)+100}{s(s+1)}} = \lim_{s \to 0} \frac{-0.03 s(s+1)}{s(s+1)+100} = 0$
      **视频结果为 -0.03， 这意味着分子在 $s \to 0$ 时不为0，或者 $G_2(s)$ 的形式使得分子与分母的s的阶次有特定关系。**
      **根据视频公式：** $e_{ssd} = \lim_{s \to 0} s \left( \frac{-G_2(s)D(s)}{1+G_1(s)G_2(s)H(s)} \right)$
      为了得到 -0.03，假设 $D(s) = \frac{N}{s}$ (阶跃扰动)
      $\lim_{s \to 0} \frac{-s G_2(s) \frac{N}{s}}{1 + \frac{100}{s(s+1)}} = \lim_{s \to 0} \frac{-N G_2(s) s(s+1)}{s(s+1)+100}$
      如果 $G_2(0)$ 是一个常数 $K_{G2}$，则 $e_{ssd} = \frac{-N \cdot K_{G2} \cdot 0 \cdot 1}{0+100} = 0$
      **这表明视频中计算扰动稳态误差时， $G_2(s)$ 或者 $D(s)$ 的形式比较特殊，或者公式应用上与标准形式略有差异。或者分母的 $s$ 被约掉了。**
      **重新看视频公式：$e_{ssd} = \lim_{s \to 0} s \cdot \frac{-G_2(s)D(s)}{1+G_1G_2H}$**
      **如果扰动点在 $G_1$ 和 $G_2$ 之间，且 $H(s)=1$。**
      **视频中代入的 $G_2(s)$ 和 $D(s)$ 具体是什么，是关键。**
      假设 $G_1(s) = \frac{100}{s}$, $G_2(s) = \frac{1}{s+1}$, $H(s)=1$. 扰动作用在 $G_2(s)$ 的输入端。
      则 $Y_d(s) = \frac{G_2(s)D(s)}{1+G_1(s)G_2(s)} = \frac{\frac{1}{s+1}D(s)}{1+\frac{100}{s(s+1)}}$
      $e_{ssd} = \lim_{s \to 0} s \left( - \frac{\frac{1}{s+1}D(s)}{1+\frac{100}{s(s+1)}} \right) = \lim_{s \to 0} - \frac{s D(s)}{(s+1)(1+\frac{100}{s(s+1)})} = \lim_{s \to 0} - \frac{s^2 D(s)}{s(s+1)+100}$
      如果 $D(s) = \frac{3}{s}$ (阶跃扰动，幅值为3), $e_{ssd} = \lim_{s \to 0} - \frac{3s}{s(s+1)+100} = 0$.
      如果 $D(s) = \frac{3}{s^2}$ (斜坡扰动，幅值为3), $e_{ssd} = \lim_{s \to 0} - \frac{3}{s(s+1)+100} = -\frac{3}{100} = -0.03$.
      **所以，题目中的扰动 $D(s)$ 应该是幅值为3的斜坡输入，即 $D(s) = \frac{3}{s^2}$。**

**数学公式回顾：**

*   参考输入稳态误差： $e_{ssr} = \lim_{s \to 0} \frac{s R(s)}{1 + G_{开}(s)H(s)}$
*   扰动输入稳态误差： $e_{ssd} = \lim_{s \to 0} \frac{-s G_{扰动 \to 输出}(s) D(s)}{1 + G_{开}(s)H(s)}$
    (其中 $G_{扰动 \to 输出}(s)$ 是从扰动点到系统输出的传递函数)

#### **五、总结**

1.  **核心知识点：** 稳态误差定义、终值定理、稳态误差系数法（位置、速度、加速度）、扰动作用下稳态误差的计算。
2.  **求解参考输入稳态误差：**
    *   直接法：根据定义和终值定理。
    *   误差系数法：根据输入类型选择对应系数。
3.  **求解扰动输入稳态误差：**
    *   确定扰动点到输出的传递函数。
    *   应用终值定理，注意误差表达式的负号。
4.  **注意事项：**
    *   稳态误差系数法有其适用条件和对应的输入类型。
    *   扰动稳态误差的分子部分是从扰动点到输出的传递函数。
    *   准确识别系统结构图中的各个环节和信号作用点。
    *   **根据视频计算结果反推，题目中的扰动输入 $D(s)$ 应该是 $\frac{3}{s^2}$ (幅值为3的斜坡输入)，参考输入 $R(s) = \frac{5}{s^2}$。开环传递函数 $G_1G_2H = \frac{100}{s(s+1)}$。扰动作用在 $G_2$ 的输入端。**

这个总结提炼了稳态误差计算的核心方法和公式，并通过对视频中题目计算过程的分析（包括一些必要的反推和假设，因为视频中未给出所有G(s)的完整形式），明确了具体题目中可能的参数和输入类型。强调了不同方法的使用条件和注意事项。
