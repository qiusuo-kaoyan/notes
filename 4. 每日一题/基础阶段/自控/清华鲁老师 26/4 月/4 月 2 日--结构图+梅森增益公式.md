#自动控制原理 #每日一题 
***
### **题目：** 已知系统由下列方程组描述，绘制系统结构框图并求闭环传递函数 (框图绘制与梅森公式)
![[1. 控制科学备考/4. 每日一题/自控/清华鲁老师 26/Picture Set/4-2.png]]
**已知方程组：**

1.  $X_1(s) = R(s)G_1(s) - G_1(s)[G_7(s) - G_8(s)]C(s)$
2.  $X_2(s) = G_2(s)[X_1(s) - G_6(s)X_3(s)]$
3.  $X_3(s) = X_2(s) - G_5(s)C(s)$
4.  $C(s) = G_4(s)X_3(s)$

**要求：**
1.  绘制系统结构框图。
2.  求闭环传递函数 $C(s)/R(s)$。

#### **一、 绘制系统结构框图**

**核心技巧：倒着画 (从输出开始)**

1.  **从输出 $C(s)$ 开始：**
    *   根据方程 (4)：$C(s) = G_4(s)X_3(s)$
    *   画出输出 $C(s)$，其前一个信号是 $X_3(s)$，经过传递函数 $G_4(s)$。
2.  **找到 $X_3(s)$：**
    *   根据方程 (3)：$X_3(s) = X_2(s) - G_5(s)C(s)$
    *   $X_3(s)$ 是 $X_2(s)$ 减去一个反馈信号。反馈信号由 $C(s)$ 经过 $G_5(s)$ 得到。
    *   在 $X_3(s)$ 前画一个求和点，输入为 $X_2(s)$ (正)，反馈信号为 $G_5(s)C(s)$ (负)。
3.  **找到 $X_2(s)$：**
    *   根据方程 (2)：$X_2(s) = G_2(s)[X_1(s) - G_6(s)X_3(s)]$
    *   $X_2(s)$ 是一个信号经过 $G_2(s)$ 得到的。这个信号是 $X_1(s)$ 减去一个反馈信号。反馈信号由 $X_3(s)$ (已存在于图中) 经过 $G_6(s)$ 得到。
    *   在 $X_2(s)$ 前画一个传递函数 $G_2(s)$，其输入是一个求和点。求和点的输入为 $X_1(s)$ (正)，反馈信号为 $G_6(s)X_3(s)$ (负)。
    *(注意：为了简化后续步骤，实际绘制时会将 $X_3$ 直接从已画出的 $X_3$ 点引出)*

4.  **找到 $X_1(s)$：**
    *   根据方程 (1)：$X_1(s) = R(s)G_1(s) - G_1(s)[G_7(s) - G_8(s)]C(s)$
    *   $X_1(s)$ 是一个求和点。其正向输入是 $R(s)$ 经过 $G_1(s)$。其负向反馈是由 $C(s)$ (已存在于图中) 经过 $G_1(s)[G_7(s) - G_8(s)]$ (可以看作一个整体的反馈传递函数 $G_1(G_7-G_8)$)。
    *   在 $X_1(s)$ 前画一个求和点。正向输入是 $R(s)$ 经过 $G_1(s)$。负向反馈从 $C(s)$ 引出，经过 $G_1(G_7-G_8)$。
**最终结构框图 (示意，以视频讲解为准)：**

```
      +-------G6-------+
      |                |
      v                | -
R ---G1-->O-->X1-->O---G2-->X2-->O---X3---G4--> C
          ^     -   ^         -   ^
          |         |             |
          +---G1(G7-G8)---+       +----G5----+
                          |                  |
                          +------------------+
```
*注：上图仅为示意，以视频中鲁老师画的为准。O代表求和点。*

#### **二、 求解闭环传递函数 (梅森增益公式)**

**梅森增益公式：** $T = \frac{C(s)}{R(s)} = \frac{\sum_{k} P_k \Delta_k}{\Delta}$

1.  **确定前向通路 $P_k$ 及其增益：**
    *   只有一条前向通路：$R(s) \rightarrow G_1 \rightarrow X_1 \rightarrow G_2 \rightarrow X_2 \rightarrow X_3 \rightarrow G_4 \rightarrow C(s)$
    *   $P_1 = G_1 G_2 G_4$

2.  **确定所有独立回路 $L_i$ 及其增益：**
    *   **回路1 (L1):** $X_1 \rightarrow G_2 \rightarrow X_2 \rightarrow X_3 \xrightarrow{-G_6} X_1$ (反馈到 $X_1$ 前的求和点)
        $L_1 = -G_2 G_6$ (注意，这里的反馈是到$X_1$和$G_2$之间的求和点, $X_3$经过$G_6$后反馈)
        **更正 (根据视频讲解的框图和回路分析):**
        回路1: $X_1 \rightarrow G_2 \rightarrow X_2 \rightarrow \text{Sum1} \rightarrow X_3 \xrightarrow{-G_6 \text{到 Sum2}} \text{Sum2} \rightarrow X_1$  (这条路径更清晰的是 $X_3$ 经过 $G_6$ 反馈到 $X_1$ 和 $G_2$ 之间的求和点，与 $X_1$ 比较)
        所以 $L_1 = -G_2G_6$ (这条回路中，信号流过$G_2$和$G_6$)
    *   **回路2 (L2):** $X_2 \rightarrow X_3 \rightarrow G_4 \rightarrow C(s) \xrightarrow{-G_5} X_2$ (反馈到 $X_2$ 和 $X_3$ 之间的求和点)
        $L_2 = -G_4 G_5$
    *   **回路3 (L3):** $R(s) \rightarrow G_1 \rightarrow X_1 \rightarrow G_2 \rightarrow X_2 \rightarrow X_3 \rightarrow G_4 \rightarrow C(s) \xrightarrow{-(G_1(G_7-G_8))} \text{输入端的求和点}$
        $L_3 = -G_1 G_2 G_4 (G_1(G_7-G_8))$  **这里鲁老师的讲解是反馈支路直接是 $G_1(G_7-G_8)$，乘以整个前向通路中的 $G_2G_4$ 和输入端的 $G_1$**
        **根据视频的回路分析和最后公式，鲁老师指的第三个回路是：**
        $L_3 = -G_1 G_2 G_4 (G_7-G_8)$  (反馈信号 $G_1(G_7-G_8)C(s)$ 与 $R(s)G_1$ 在第一个求和点比较，所以回路增益是 $G_1 \cdot G_2 \cdot G_4 \cdot [-(G_7-G_8)G_1]$ 是不准确的。鲁老师直接写的是 $-(G_7-G_8)G_1G_2G_4$)

        **我们重新梳理视频中鲁老师的三个回路：**
        *   $L_1 = -G_2 G_6$ (围绕 $X_1, G_2, X_3$ 形成的内部小环)
        *   $L_2 = -G_4 G_5$ (围绕 $X_2, X_3, G_4, C$ 形成的内部小环)
        *   $L_3 = -G_1 G_2 G_4 (G_7-G_8)$ (从输入到输出再反馈到最前端的整体大环，反馈路径增益为 $G_1(G_7-G_8)$)
        * *注：鲁老师在视频中第三个回路的表达为 $L_3 = -G_1 G_2 G_4 (G_7-G_8)$，这代表反馈路径的传递函数为 $G_1(G_7-G_8)$，它与前向通路 $G_1G_2G_4$ 形成了这个大回路。*

3.  **确定特征式 $\Delta$：**
    $\Delta = 1 - (\sum L_i) + (\sum L_i L_j) - (\sum L_i L_j L_k) + ...$
    其中 $\sum L_i L_j$ 是所有两两不接触回路增益之积的和。
    *   观察框图，三个回路之间是否存在不接触的情况。
        *   $L_1$ 和 $L_2$ 是否接触？是，它们共享 $X_2, X_3$ 路径。
        *   $L_1$ 和 $L_3$ 是否接触？是，它们共享 $G_1, X_1, G_2, X_2, X_3, G_4$ 路径。
        *   $L_2$ 和 $L_3$ 是否接触？是，它们共享 $G_1, X_1, G_2, X_2, X_3, G_4, C$ 路径。
        *   **视频中结论：** 不存在两两不接触的回路。
    *   因此，$\Delta = 1 - (L_1 + L_2 + L_3)$
        $\Delta = 1 - (-G_2G_6 - G_4G_5 - G_1G_2G_4(G_7-G_8))$
        $\Delta = 1 + G_2G_6 + G_4G_5 + G_1G_2G_4(G_7-G_8)$

4.  **确定 $\Delta_k$：**
    $\Delta_k$ 是从 $\Delta$ 中去掉与第 $k$ 条前向通路接触的回路后得到的。
    *   由于只有一条前向通路 $P_1$，且 $P_1$ 与所有三个回路 $L_1, L_2, L_3$ 都接触。
    *   所以，$\Delta_1 = 1$ (去掉所有接触回路后，只剩下1)。

5.  **计算闭环传递函数 $T$：**
    $T = \frac{P_1 \Delta_1}{\Delta} = \frac{G_1 G_2 G_4 \times 1}{1 + G_2G_6 + G_4G_5 + G_1G_2G_4(G_7-G_8)}$
    $T = \frac{G_1 G_2 G_4}{1 + G_2G_6 + G_4G_5 + G_1G_2G_4(G_7-G_8)}$

#### **三、 总结**

1.  **绘制结构框图技巧：** 从输出开始，根据方程组逐步向前绘制，确定信号流向和求和点。
2.  **梅森增益公式步骤：**
    *   找前向通路 $P_k$。
    *   找所有独立回路 $L_i$。
    *   计算特征式 $\Delta$ (注意不接触回路)。
    *   计算 $\Delta_k$ (去掉与 $P_k$ 接触的回路)。
    *   代入公式 $T = \frac{\sum P_k \Delta_k}{\Delta}$。
3.  **注意点：**
    *   准确识别回路，特别是反馈信号的作用点和符号。
    *   仔细判断回路之间是否接触。
    *   在绘制框图时，求和点的“+”、“-”号要明确。
