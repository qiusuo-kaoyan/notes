### **题目：** 已知系统结构图，写出系统的输出表达式 (梅森增益公式应用题)
![[4-5.png]]
![[4-5-1.png]]
**已知：** 一个包含输入 $R(s)$ 和扰动 $N(s)$ 的系统结构图。
**求：** 系统的输出表达式 $C(s)$。

#### **一、 核心概念与方法**

1.  **叠加原理:** 对于线性系统，当有多个输入（包括扰动）时，总输出等于各个输入单独作用时产生的输出之和。
    *   因此，可以分别计算输入 $R(s)$ 对输出 $C(s)$ 的贡献和扰动 $N(s)$ 对输出 $C(s)$ 的贡献，然后相加。

2.  **梅森增益公式 ([[Mason's Gain Formula]]):** 用于求解复杂[[信号流图]]（或等效的[[方框图]]）的传递函数。
    *   **公式:** $P = \frac{1}{\Delta} \sum_{k} P_k \Delta_k$
        *   $P$: 从指定输入到指定输出的总传递函数。
        *   $P_k$: 从输入到输出的第 $k$ 条前向通路的增益。
        *   $\Delta$: 系统的特征式 (行列式)，计算公式为：
            $\Delta = 1 - \sum L_i + \sum L_i L_j - \sum L_i L_j L_k + ...$
            *   $\sum L_i$: 所有单个独立回路的增益之和。
            *   $\sum L_i L_j$: 所有两两互不接触的回路增益乘积之和。
            *   $\sum L_i L_j L_k$: 所有三三互不接触的回路增益乘积之和，以此类推。
        *   $\Delta_k$: 对于第 $k$ 条前向通路 $P_k$ 的余因子式，它是将 $\Delta$ 中所有与 $P_k$ 相接触的回路增益置为零后得到的。

**数学公式表达 (本题应用):**

*   输出 $C(s) = C_R(s) + C_N(s)$
    *   $C_R(s)$: 输入 $R(s)$ 单独作用时的输出。
    *   $C_N(s)$: 扰动 $N(s)$ 单独作用时的输出。
*   传递函数 $G_R(s) = \frac{C_R(s)}{R(s)}$
*   传递函数 $G_N(s) = \frac{C_N(s)}{N(s)}$
*   最终输出 $C(s) = G_R(s)R(s) + G_N(s)N(s)$

#### **二、 解题步骤**

**步骤一：计算输入 $R(s)$ 对输出 $C(s)$ 的传递函数 $G_R(s) = \frac{C(s)}{R(s)}$ (令 $N(s) = 0$)**

1.  **识别前向通路 $P_k$ (从 $R(s)$ 到 $C(s)$):**
    *   $P_1 = G_2 G_4$
    *   $P_2 = G_3 G_4$
    *   $P_3 = G_1 G_2 G_4$

2.  **识别独立回路 $L_i$:**
    *   $L_1 = -G_2 G_4$ (负号来自反馈求和点)
    *   $L_2 = -G_3 G_4$ (负号来自反馈求和点)

3.  **计算系统特征式 $\Delta$:**
    *   $\sum L_i = L_1 + L_2 = -G_2 G_4 - G_3 G_4$
    *   这两个回路 $L_1$ 和 $L_2$ 是相互接触的（共享 $G_4$ 和求和点），因此没有两两不接触的回路。
    *   $\Delta = 1 - (L_1 + L_2) = 1 - (-G_2 G_4 - G_3 G_4) = 1 + G_2 G_4 + G_3 G_4$

4.  **计算各前向通路的余因子式 $\Delta_k$:**
    *   **对于 $P_1 = G_2 G_4$:** $P_1$ 与 $L_1$ 和 $L_2$ 都接触。因此，在 $\Delta$ 中将 $L_1$ 和 $L_2$ 置为零。
        $\Delta_1 = 1 - (0 + 0) = 1$
    *   **对于 $P_2 = G_3 G_4$:** $P_2$ 与 $L_1$ 和 $L_2$ 都接触。
        $\Delta_2 = 1 - (0 + 0) = 1$
    *   **对于 $P_3 = G_1 G_2 G_4$:** $P_3$ 与 $L_1$ 和 $L_2$ 都接触。
        $\Delta_3 = 1 - (0 + 0) = 1$

5.  **应用梅森公式计算 $G_R(s)$:**
    $G_R(s) = \frac{P_1 \Delta_1 + P_2 \Delta_2 + P_3 \Delta_3}{\Delta} = \frac{G_2 G_4 \cdot 1 + G_3 G_4 \cdot 1 + G_1 G_2 G_4 \cdot 1}{1 + G_2 G_4 + G_3 G_4}$
    $G_R(s) = \frac{G_2 G_4 + G_3 G_4 + G_1 G_2 G_4}{1 + G_2 G_4 + G_3 G_4}$

**步骤二：计算扰动 $N(s)$ 对输出 $C(s)$ 的传递函数 $G_N(s) = \frac{C(s)}{N(s)}$ (令 $R(s) = 0$)**

1.  **识别前向通路 $P_k$ (从 $N(s)$ 到 $C(s)$):**
    *   $P_1' = G_4$ (注意这里的 $P_1'$ 与步骤一的 $P_1$ 不同)

2.  **识别独立回路 $L_i$:** 回路与输入点无关，因此回路与步骤一相同。
    *   $L_1 = -G_2 G_4$
    *   $L_2 = -G_3 G_4$

3.  **计算系统特征式 $\Delta$:** 与步骤一相同。
    $\Delta = 1 + G_2 G_4 + G_3 G_4$

4.  **计算各前向通路的余因子式 $\Delta_k$:**
    *   **对于 $P_1' = G_4$:** $P_1'$ 与 $L_1$ 和 $L_2$ 都接触。
        $\Delta_1' = 1 - (0 + 0) = 1$

5.  **应用梅森公式计算 $G_N(s)$:**
    $G_N(s) = \frac{P_1' \Delta_1'}{\Delta} = \frac{G_4 \cdot 1}{1 + G_2 G_4 + G_3 G_4}$
    $G_N(s) = \frac{G_4}{1 + G_2 G_4 + G_3 G_4}$

**步骤三：写出总输出表达式 $C(s)$**

$C(s) = G_R(s)R(s) + G_N(s)N(s)$
$C(s) = \left( \frac{G_2 G_4 + G_3 G_4 + G_1 G_2 G_4}{1 + G_2 G_4 + G_3 G_4} \right) R(s) + \left( \frac{G_4}{1 + G_2 G_4 + G_3 G_4} \right) N(s)$
$C(s) = \frac{(G_2 G_4 + G_3 G_4 + G_1 G_2 G_4)R(s) + G_4 N(s)}{1 + G_2 G_4 + G_3 G_4}$

**举例 (概念性):**

*   **更复杂的回路:** 如果存在两个互不接触的回路 $L_a$ 和 $L_b$，则 $\Delta$ 中会有 $+L_a L_b$ 项。
*   **前向通路与回路不接触:** 如果某条前向通路 $P_x$ 与回路 $L_y$ 不接触，则在计算 $\Delta_x$ 时，$L_y$ 仍然保留在 $\Delta_x$ 的表达式中。例如，若 $P_x$ 与 $L_1$ 接触，与 $L_2$ 不接触，则 $\Delta_x = 1 - L_2$ (假设只有这两个回路)。

#### **三、 总结**

1.  **核心方法:** 叠加原理 + 梅森增益公式。
2.  **梅森公式应用步骤:**
    1.  识别所有从输入到输出的前向通路及其增益 $P_k$。
    2.  识别所有独立回路及其增益 $L_i$。
    3.  计算系统特征式 $\Delta$ (注意互不接触的回路)。
    4.  计算每条前向通路的余因子式 $\Delta_k$ (去掉与该通路接触的回路)。
    5.  代入梅森公式。
3.  **多输入处理:** 分别计算每个输入对应的传递函数，然后根据叠加原理合并。
4.  **注意点:**
    *   反馈点的符号会影响回路增益的正负。
    *   仔细判断回路之间、前向通路与回路之间是否接触。
