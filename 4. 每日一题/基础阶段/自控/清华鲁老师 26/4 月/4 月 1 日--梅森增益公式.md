### **题目：** 已知系统结构图，求输出表达式 (梅森公式应用)
![[4. 每日一题/基础阶段/自控/清华鲁老师 26/Picture Set/4-1.png]]
![[4-1-2.png]]


**已知信息:**

*   系统结构图，包含两个输入 $U_1(s)$, $U_2(s)$ 和一个输出 $Y_1(s)$。
*   各个环节的传递函数为 $G_1, G_2, G_3$。
*   存在多个反馈回路和前向通路。

**要求:** 写出系统的输出表达式 $Y_1(s)$。

#### **一、 核心方法与概念**

1.  **叠加原理:** 对于多输入系统，总输出等于各个输入单独作用时产生的输出之和。
    *   $Y_1(s) = Y_{1(U_1)}(s) + Y_{1(U_2)}(s)$
    *   其中 $Y_{1(U_1)}(s)$ 是当 $U_2(s)=0$ 时，$U_1(s)$ 产生的输出。
    *   其中 $Y_{1(U_2)}(s)$ 是当 $U_1(s)=0$ 时，$U_2(s)$ 产生的输出。

2.  **梅森增益公式 (Mason's Gain Formula):** 用于复杂系统结构图求解传递函数。
    *   **公式:** $P(s) = \frac{1}{\Delta} \sum_{k} P_k \Delta_k$
    *   **符号解释:**
        *   $P(s)$: 从指定输入到指定输出的总传递函数。
        *   $P_k$: 第 $k$ 条前向通路的增益。
        *   $\Delta$: 系统的特征式 (或称行列式)，计算公式如下：
            $\Delta = 1 - \sum L_i + \sum L_i L_j - \sum L_i L_j L_l + ...$
            其中：
            *   $\sum L_i$: 所有单个独立回路增益之和。
            *   $\sum L_i L_j$: 所有两两互不接触的回路增益乘积之和。
            *   $\sum L_i L_j L_l$: 所有三三互不接触的回路增益乘积之和，以此类推。
        *   $\Delta_k$: 对于第 $k$ 条前向通路 $P_k$，从 $\Delta$ 中去掉所有与 $P_k$ 相接触的回路后得到的特征式。

#### **二、 求解步骤 (以本题为例)**

**第一步：计算 $U_1(s)$ 单独作用时的输出 $Y_{1(U_1)}(s)$ (令 $U_2(s)=0$)**

1.  **找出所有独立回路 (Loops) $L_i$:**
    *   $L_1 = -G_1$ (上部回路，经过 $G_1$ 和单位负反馈)
    *   $L_2 = -G_2 G_3$ (下部回路，经过 $G_2, G_3$ 和单位负反馈)
    *   $L_3 = (-G_1)(-G_2) = G_1 G_2$ (大回路，经过 $G_1$, $U_1$ 处的比较点，再经过 $G_2$ 和 $U_2$ 处的比较点返回)
    *   **注意:** 中间的两个求和点本身不直接构成回路，它们是回路路径上的一部分。

2.  **找出所有互不接触的回路对/组:**
    *   $L_1$ 与 $L_2$ 互不接触。
    *   $L_1 L_2 = (-G_1)(-G_2 G_3) = G_1 G_2 G_3$

3.  **计算系统特征式 $\Delta$:**
    *   $\Delta = 1 - (L_1 + L_2 + L_3) + (L_1 L_2)$
    *   $\Delta = 1 - (-G_1 - G_2 G_3 + G_1 G_2) + (G_1 G_2 G_3)$
    *   $\Delta = 1 + G_1 + G_2 G_3 - G_1 G_2 + G_1 G_2 G_3$

4.  **找出从 $U_1(s)$ 到 $Y_1(s)$ 的所有前向通路 $P_k$ 并计算其对应的 $\Delta_k$:**
    *   **通路 1 ($P_{11}$):** $U_1 \rightarrow G_1 \rightarrow Y_1$
        *   $P_{11} = G_1$
        *   与 $P_{11}$ 接触的回路有 $L_1, L_3$。
        *   不与 $P_{11}$ 接触的回路是 $L_2 = -G_2 G_3$。
        *   $\Delta_{11} = 1 - L_2 = 1 - (-G_2 G_3) = 1 + G_2 G_3$
    *   **通路 2 ($P_{12}$):** $U_1 \rightarrow G_1 \rightarrow \text{比较点} \rightarrow -G_2 \rightarrow \text{比较点} \rightarrow Y_1$
        *   $P_{12} = G_1 (-G_2) = -G_1 G_2$
        *   这条通路与所有回路 ($L_1, L_2, L_3$) 都接触 (因为 $L_3$ 包含了 $P_{12}$ 的路径的一部分，而 $L_1$ 和 $L_2$ 也因为共用节点而接触)。
        *   $\Delta_{12} = 1$
    *   **通路 3 ($P_{13}$):** $U_1 \rightarrow G_1 \rightarrow \text{比较点} \rightarrow -G_2 \rightarrow G_3 \rightarrow \text{比较点} \rightarrow Y_1$
        *   $P_{13} = G_1 (-G_2) G_3 = -G_1 G_2 G_3$
        *   这条通路与所有回路 ($L_1, L_2, L_3$) 都接触。
        *   $\Delta_{13} = 1$

5.  **计算 $U_1(s)$ 作用下的传递函数 $T_1(s) = \frac{Y_{1(U_1)}(s)}{U_1(s)}$:**
    *   $T_1(s) = \frac{P_{11}\Delta_{11} + P_{12}\Delta_{12} + P_{13}\Delta_{13}}{\Delta}$
    *   $T_1(s) = \frac{G_1(1+G_2G_3) - G_1G_2 - G_1G_2G_3}{1 + G_1 + G_2G_3 - G_1G_2 + G_1G_2G_3}$
    *   分子化简: $G_1 + G_1G_2G_3 - G_1G_2 - G_1G_2G_3 = G_1 - G_1G_2$
    *   $T_1(s) = \frac{G_1 - G_1G_2}{1 + G_1 + G_2G_3 - G_1G_2 + G_1G_2G_3}$

**第二步：计算 $U_2(s)$ 单独作用时的输出 $Y_{1(U_2)}(s)$ (令 $U_1(s)=0$)**

1.  **独立回路 $L_i$ 和特征式 $\Delta$ 不变。** (与第一步相同)

2.  **找出从 $U_2(s)$ 到 $Y_1(s)$ 的所有前向通路 $P_k$ 并计算其对应的 $\Delta_k$:**
    *   **通路 1 ($P_{21}$):** $U_2 \rightarrow G_2 \rightarrow \text{比较点} \rightarrow Y_1$
        *   $P_{21} = G_2$ (注意视频中这里有个负号，但从图上看U2直接进入G2前面的求和点，若上方反馈为负，则U2应为正输入。视频讲解时说“负号G2”，可能是指从某处反馈过来的路径，但作为U2的直接前向通路应为正。以视频中的最终结果为准，其P1=G2是正确的)
        *   与 $P_{21}$ 接触的回路有 $L_2, L_3$。
        *   不与 $P_{21}$ 接触的回路是 $L_1 = -G_1$。
        *   $\Delta_{21} = 1 - L_1 = 1 - (-G_1) = 1 + G_1$
          (视频中这里 $\Delta_1=1$, 理由是去掉 $P_{21}$ (G2向上)后不形成回路。这需要仔细看图，如果 $P_{21}$ 路径非常简单且不参与其他回路，则成立。但如果 $P_{21}$ 的节点被其他回路共用，则会影响。按视频的说法，$\Delta_{21}=1$)
    *   **通路 2 ($P_{22}$):** $U_2 \rightarrow G_2 \rightarrow G_3 \rightarrow \text{比较点} \rightarrow Y_1$
        *   $P_{22} = G_2 G_3$
        *   与 $P_{22}$ 接触的回路有 $L_2, L_3$。
        *   不与 $P_{22}$ 接触的回路是 $L_1 = -G_1$。
        *   $\Delta_{22} = 1 - L_1 = 1 - (-G_1) = 1 + G_1$

3.  **计算 $U_2(s)$ 作用下的传递函数 $T_2(s) = \frac{Y_{1(U_2)}(s)}{U_2(s)}$:**
    *   $T_2(s) = \frac{P_{21}\Delta_{21} + P_{22}\Delta_{22}}{\Delta}$
    *   $T_2(s) = \frac{G_2(1) + G_2G_3(1+G_1)}{1 + G_1 + G_2G_3 - G_1G_2 + G_1G_2G_3}$ (按视频中 $\Delta_{21}=1, \Delta_{22}=1+G_1$)
    *   分子化简: $G_2 + G_2G_3 + G_1G_2G_3$
    *   $T_2(s) = \frac{G_2 + G_2G_3 + G_1G_2G_3}{1 + G_1 + G_2G_3 - G_1G_2 + G_1G_2G_3}$

**第三步：计算总输出 $Y_1(s)$**

*   $Y_1(s) = T_1(s)U_1(s) + T_2(s)U_2(s)$
*   $Y_1(s) = \frac{G_1 - G_1G_2}{1 + G_1 + G_2G_3 - G_1G_2 + G_1G_2G_3} U_1(s) + \frac{G_2 + G_2G_3 + G_1G_2G_3}{1 + G_1 + G_2G_3 - G_1G_2 + G_1G_2G_3} U_2(s)$

#### **三、 关键点与难点**

1.  **正确识别所有独立回路 $L_i$:** 这是计算 $\Delta$ 的基础。
2.  **正确识别所有互不接触的回路对/组:** 影响 $\Delta$ 的高阶项。
3.  **正确识别所有从指定输入到指定输出的前向通路 $P_k$:** 不能遗漏，也不能重复。
4.  **正确判断 $P_k$ 与各个回路的接触情况，以确定 $\Delta_k$:**
    *   如果前向通路 $P_k$ 与某个回路 $L_i$ **不共享任何节点或支路**，则 $L_i$ 及其组合项会出现在 $\Delta_k$ 的计算中。
    *   如果 $P_k$ 与所有回路都接触，则 $\Delta_k = 1$。
    *   **这是本题的难点，也是梅森公式应用中容易出错的地方。** 视频中强调了“去掉前向通路后，结构图中是否还存在回路”来判断 $\Delta_k$。

#### **四、总结**

1.  **梅森公式是处理复杂结构图的有力工具。**
2.  **细心和耐心是关键：** 识别回路、通路，判断接触情况都需要仔细观察。
3.  **理解 $\Delta_k$ 的含义：** “当通路 $P_k$ 存在时，系统中其他不与 $P_k$ 冲突的回路所形成的特征式”。
4.  **多加练习：** 熟练掌握梅森公式的应用需要通过大量练习。
5.  **检查：** 计算完成后，可以尝试用方框图化简的方法验证简单部分的传递函数，以增加准确性。
