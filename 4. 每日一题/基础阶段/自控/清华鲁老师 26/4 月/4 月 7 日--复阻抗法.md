### **题目：** 已知电网络如图所示，求该电网络的传递函数 (电路的复阻抗法)

![[4. 每日一题/基础阶段/自控/清华鲁老师 26/Picture Set/4-7.png]]

**电路图描述：** 一个由电阻和电容组成的二端口网络。
*   输入电压 $u_I(t)$
*   输出电压 $u_O(t)$
*   电路结构：$R_1$ 与 $C_1$ 并联，构成阻抗 $Z_1$；$R_2$ 与 $C_2$ 并联，构成阻抗 $Z_2$。$Z_1$ 和 $Z_2$ 串联后接入输入电压 $u_I(t)$，输出电压 $u_O(t)$ 取自 $Z_2$ 两端。

#### **一、 核心方法：复阻抗法 (S域分析)**

1.  **将电路元件转换到 S 域 (复频域):**
    *   电阻 $R \rightarrow R$
    *   电容 $C \rightarrow \frac{1}{Cs}$
    *   电感 $L \rightarrow Ls$ (本题未涉及)
    *   电压 $u(t) \rightarrow U(s)$
    *   电流 $i(t) \rightarrow I(s)$

2.  **计算各部分复阻抗:**
    *   **$Z_1(s)$ ( $R_1$ 并联 $C_1$ ):**
        $\frac{1}{Z_1(s)} = \frac{1}{R_1} + \frac{1}{1/(C_1s)} = \frac{1}{R_1} + C_1s = \frac{1+R_1C_1s}{R_1}$
        所以，$Z_1(s) = \frac{R_1}{1+R_1C_1s}$
    *   **$Z_2(s)$ ( $R_2$ 并联 $C_2$ ):**
        同理，$Z_2(s) = \frac{R_2}{1+R_2C_2s}$

3.  **利用S域的电路分析方法 (如电压分配、节点电压法、回路电流法等):**
    *   本题中，$Z_1(s)$ 和 $Z_2(s)$ 串联，输出电压 $U_O(s)$ 是 $Z_2(s)$ 上的分压。
    *   总电流 $I(s) = \frac{U_I(s)}{Z_1(s) + Z_2(s)}$
    *   输出电压 $U_O(s) = I(s) \cdot Z_2(s)$

4.  **求解传递函数 $G(s) = \frac{U_O(s)}{U_I(s)}$:**
    *   根据电压分配原理：
        $G(s) = \frac{U_O(s)}{U_I(s)} = \frac{Z_2(s)}{Z_1(s) + Z_2(s)}$

**数学公式表达 (复阻抗):**
*   电阻: $Z_R = R$
*   电容: $Z_C = \frac{1}{Cs}$
*   电感: $Z_L = Ls$
*   并联阻抗: $\frac{1}{Z_{parallel}} = \frac{1}{Z_a} + \frac{1}{Z_b} \implies Z_{parallel} = \frac{Z_a Z_b}{Z_a + Z_b}$
*   串联阻抗: $Z_{series} = Z_a + Z_b$

#### **二、 题目解答**

1.  **代入 $Z_1(s)$ 和 $Z_2(s)$ 的表达式:**
    $G(s) = \frac{\frac{R_2}{1+R_2C_2s}}{\frac{R_1}{1+R_1C_1s} + \frac{R_2}{1+R_2C_2s}}$

2.  **化简表达式:**
    *   分子分母同乘以 $(1+R_1C_1s)(1+R_2C_2s)$:
        $G(s) = \frac{R_2(1+R_1C_1s)}{R_1(1+R_2C_2s) + R_2(1+R_1C_1s)}$
    *   展开分母:
        $G(s) = \frac{R_2 + R_1R_2C_1s}{R_1 + R_1R_2C_2s + R_2 + R_1R_2C_1s}$
    *   整理得到 (与视频中答案一致，只是视频中分子没有展开):
        $G(s) = \frac{R_2(1+R_1C_1s)}{(R_1+R_2) + R_1R_2(C_1+C_2)s}$

**最终传递函数 (视频中给出的形式，分母整理为按 $s$ 的幂次排列):**
$$ G(s) = \frac{R_2(1+R_1C_1s)}{R_1R_2(C_1+C_2)s + (R_1+R_2)} $$
如果将分子也展开，则为：
$$ G(s) = \frac{R_1R_2C_1s + R_2}{R_1R_2(C_1+C_2)s + (R_1+R_2)} $$

**举例 (简化情况):**
*   如果只有 $R_1$ 和 $R_2$ 串联，输出从 $R_2$ 取，则 $G(s) = \frac{R_2}{R_1+R_2}$。
*   如果 $R_1$ 和 $C_1$ 串联，再与 $R_2$ 串联，输出从 $R_2$ 取，则 $Z_1 = R_1 + \frac{1}{C_1s}$, $Z_2 = R_2$。
    $G(s) = \frac{R_2}{R_1 + \frac{1}{C_1s} + R_2} = \frac{R_2C_1s}{(R_1+R_2)C_1s + 1}$。

#### **三、总结**

1.  **复阻抗法的优势：**
    *   直接在 S 域进行分析，避免了列写和求解微分方程的繁琐过程。
    *   对于一些结构清晰的电路，求解传递函数更为快捷。
2.  **核心步骤：**
    *   元件 S 域转换。
    *   计算各部分复阻抗。
    *   应用 S 域电路分析法则（如电压分配）。
    *   代数化简。
3.  **适用性：** 特别适用于求解线性时不变电路的传递函数。
4.  **与微分方程法的关系：** 复阻抗法本质上是拉氏变换在电路分析中的直接应用，其结果与通过微分方程法得到的结果是一致的。
