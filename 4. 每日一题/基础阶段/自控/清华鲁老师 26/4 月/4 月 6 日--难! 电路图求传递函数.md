**电路图包含：** 电阻 $R_1, R_2$，电容 $C_1, C_2$，输入电压 $u_R(t)$，输出电压 $u_C(t)$ (即 $u_{C2}(t)$)。

#### **一、 写出微分方程组**

1.  **定义中间变量：**
    *   电流：$i_1(t)$ (流过 $R_1$)，$i_2(t)$ (流过 $R_2$ 和 $C_2$)
    *   电容电压：$u_{C1}(t)$ (电容 $C_1$ 两端电压)，$u_{C2}(t)$ (电容 $C_2$ 两端电压，即输出 $u_C(t)$)

2.  **根据电路原理列写方程：**
    *   **KVL (回路电压定律):**
        *   $u_R(t) = i_1(t)R_1 + u_{C1}(t)$
        *   $u_{C1}(t) = i_2(t)R_2 + u_{C2}(t)$
    *   **KCL (节点电流定律):**
        *   $i_1(t) = i_2(t) + i_{C1}(t)$ (其中 $i_{C1}(t)$ 为流过电容 $C_1$ 的电流)
    *   **元件特性方程 (电容):**
        *   $i_{C1}(t) = C_1 \frac{du_{C1}(t)}{dt}$
        *   $i_2(t) = C_2 \frac{du_{C2}(t)}{dt}$

3.  **化简与代换，得到关于输入 $u_R(t)$ 和输出 $u_C(t)$ (即 $u_{C2}(t)$) 的微分方程：**
    *   将 $u_{C2}(t)$ 替换为 $u_C(t)$。
    *   逐步消去中间变量 $i_1(t), i_2(t), u_{C1}(t)$。
        *   $i_2(t) = C_2 \frac{du_C(t)}{dt}$
        *   $u_{C1}(t) = R_2 C_2 \frac{du_C(t)}{dt} + u_C(t)$
        *   $i_{C1}(t) = C_1 \frac{d}{dt} [R_2 C_2 \frac{du_C(t)}{dt} + u_C(t)] = R_2 C_1 C_2 \frac{d^2u_C(t)}{dt^2} + C_1 \frac{du_C(t)}{dt}$
        *   $i_1(t) = i_2(t) + i_{C1}(t) = C_2 \frac{du_C(t)}{dt} + R_2 C_1 C_2 \frac{d^2u_C(t)}{dt^2} + C_1 \frac{du_C(t)}{dt}$
    *   最终代入第一个 KVL 方程：
        $u_R(t) = R_1 [C_2 \frac{du_C(t)}{dt} + R_2 C_1 C_2 \frac{d^2u_C(t)}{dt^2} + C_1 \frac{du_C(t)}{dt}] + [R_2 C_2 \frac{du_C(t)}{dt} + u_C(t)]$
    *   整理得到：
        $u_R(t) = R_1 R_2 C_1 C_2 \frac{d^2u_C(t)}{dt^2} + (R_1 C_1 + R_1 C_2 + R_2 C_2) \frac{du_C(t)}{dt} + u_C(t)$

**数学公式表达 (微分方程):**
$$ R_1 R_2 C_1 C_2 \frac{d^2u_C(t)}{dt^2} + (R_1 C_1 + R_1 C_2 + R_2 C_2) \frac{du_C(t)}{dt} + u_C(t) = u_R(t) $$

#### **二、 绘制系统结构图 (难点)**

1.  **将微分方程组（或中间过程的方程）进行拉氏变换 (零初始条件):**
    *   $U_R(s) = I_1(s)R_1 + U_{C1}(s)$
    *   $U_{C1}(s) = I_2(s)R_2 + U_{C2}(s)$  (即 $U_C(s)$)
    *   $I_1(s) = I_2(s) + I_{C1}(s)$
    *   $I_{C1}(s) = C_1 s U_{C1}(s)$
    *   $I_2(s) = C_2 s U_{C2}(s)$ (即 $C_2 s U_C(s)$)

2.  **根据拉氏变换后的代数方程组，从输出 $U_C(s)$ 开始，反向或逐步构建结构图：**
    *   **核心思路：** 将方程变形，表达出信号之间的加减和比例积分微分关系。
    *   从 $U_R(s) = I_1(s)R_1 + U_{C1}(s)$ 入手，可以看作 $U_{C1}(s) = U_R(s) - I_1(s)R_1$。 如果我们想让前向通路为1，那么输出 $U_C(s)$ 可以看作是 $U_R(s)$ 减去若干反馈信号。
    *   **视频中的思路 (一种可能的画法，结构图不唯一):**
        *   主前向通路认为是 1，即 $U_C(s)$ 直接由 $U_R(s)$ 减去反馈信号得到。
        *   $U_C(s) = U_R(s) - I_1(s)R_1 - I_2(s)R_2$  (这是由原始 KVL 方程 $u_R(t) = i_1(t)R_1 + i_2(t)R_2 + u_{C2}(t)$ 变形得到，**注意视频中此处有笔误，原始方程第三项是 $U_{C1}$ 或 $U_{C2}$，根据不同回路选择。 如果选 $u_R = i_1R_1 + i_2R_2 + u_{C2}$ 这个方程并不直接成立，需要基于最外层回路 $u_R = i_1R_1 + u_{C1}$，而 $u_{C1} = i_2R_2 + u_{C2}$ 来构建）**
        *   **正确的出发点应该是：** $U_C(s) = U_{C1}(s) - I_2(s)R_2$  和 $U_{C1}(s) = U_R(s) - I_1(s)R_1$
        *   **反馈支路1 (与 $I_2(s)R_2$ 相关):**
            *   $I_2(s) = C_2 s U_C(s)$。所以从 $U_C(s)$ 引出，经过 $C_2 s$，得到 $I_2(s)$，再经过 $R_2$，得到 $I_2(s)R_2$ 作为反馈。
        *   **反馈支路2 (与 $I_1(s)R_1$ 相关):**
            *   $I_1(s) = I_2(s) + I_{C1}(s) = I_2(s) + C_1 s U_{C1}(s)$
            *   $U_{C1}(s) = I_2(s)R_2 + U_C(s)$
            *   所以，$I_1(s) = I_2(s) + C_1 s [I_2(s)R_2 + U_C(s)]$
            *   从 $I_2(s)$ 引出，乘以 $R_2$，与 $U_C(s)$ 相加得到 $U_{C1}(s)$。
            *   $U_{C1}(s)$ 乘以 $C_1 s$ 得到 $I_{C1}(s)$。
            *   $I_{C1}(s)$ 与 $I_2(s)$ 相加得到 $I_1(s)$。
            *   $I_1(s)$ 乘以 $R_1$ 作为反馈。
    *   **视频中简化的反馈画法 (等效后的样子，不是直接推导过程):**
        该画法是将所有与输出 $U_C(s)$ 相关的项都看作从 $U_C(s)$ 引出的反馈。
        *   $U_C(s) = U_R(s) - (\text{反馈项1}) - (\text{反馈项2}) - ...$
        *   需要仔细分析微分方程的拉氏变换形式，找出各项如何由 $U_C(s)$ 经过不同环节得到。
        *   例如：$I_2(s)R_2 = (C_2 s U_C(s)) R_2 = (R_2 C_2 s) U_C(s)$，所以有一个反馈通路 $H_1(s) = R_2 C_2 s$。
        *   $I_1(s)R_1 = R_1 (I_2(s) + C_1 s U_{C1}(s)) = R_1 (C_2 s U_C(s) + C_1 s (R_2 C_2 s U_C(s) + U_C(s)))$
                 $= (R_1 C_2 s + R_1 R_2 C_1 C_2 s^2 + R_1 C_1 s) U_C(s)$。
                 所以有反馈通路 $H_2(s) = R_1 C_2 s + R_1 R_2 C_1 C_2 s^2 + R_1 C_1 s$。
        *   **注意：** 视频中绘制结构图时，为了让前向通路为1，将 $u_R(t) = i_1R_1 + i_2R_2 + u_C$ 变形为 $u_C = u_R - i_1R_1 - i_2R_2$。这个 $i_2R_2$ 项实际上是 $u_{C1} - u_C$。这个画法比较巧妙，但推导过程需要清晰。

**结构图特点 (视频中的画法):**
*   前向通路为 1。
*   所有其他部分均为从输出 $U_C(s)$ 引出的反馈通路。

**绘制结构图的通用思路：**
1. 列写所有独立的方程（KVL, KCL, 元件特性）。
2. 对所有方程进行拉氏变换。
3. 选择一个输出变量。
4. 从输入开始，或者从输出反推，逐步连接各个环节和求和点，表示方程关系。
5. 每个方程对应结构图的一部分。

#### **三、 通过等效变换求系统传递函数**

1.  **识别反馈回路：** 从绘制的结构图中，识别出所有的独立反馈回路。
2.  **逐个化简反馈回路：**
    *   对于单个负反馈回路，其等效传递函数为 $G_{eq}(s) = \frac{G(s)}{1+G(s)H(s)}$ (其中 G(s) 是前向通路，H(s) 是反馈通路)。
    *   对于视频中前向通路为1，反馈通路为 $H_i(s)$ 的情况，每个反馈支路可以看作与主通路并联后再进行总反馈，或者理解为输出 $U_C(s) = U_R(s) \cdot \frac{1}{1 + H_{total}(s)}$，其中 $H_{total}(s)$ 是所有反馈支路传递函数的总和。
3.  **化简串并联环节：**
    *   串联：$G_{eq}(s) = G_1(s)G_2(s)$
    *   并联：$G_{eq}(s) = G_1(s) \pm G_2(s)$
4.  **重复上述步骤，直到得到最终的 $U_C(s)/U_R(s)$。**

**传递函数的最终形式 (根据第一问微分方程直接拉氏变换得到):**
从 $R_1 R_2 C_1 C_2 \frac{d^2u_C(t)}{dt^2} + (R_1 C_1 + R_1 C_2 + R_2 C_2) \frac{du_C(t)}{dt} + u_C(t) = u_R(t)$
进行拉氏变换 (零初始条件):
$R_1 R_2 C_1 C_2 s^2 U_C(s) + (R_1 C_1 + R_1 C_2 + R_2 C_2) s U_C(s) + U_C(s) = U_R(s)$
所以，传递函数为：
$$ \frac{U_C(s)}{U_R(s)} = \frac{1}{R_1 R_2 C_1 C_2 s^2 + (R_1 C_1 + R_1 C_2 + R_2 C_2) s + 1} $$
**视频中给出的分母形式与此一致 (视频中分子为1，分母为 $1 + s^2R_1R_2C_1C_2 + sR_1C_1 + sC_2R_2 + sR_1C_2$ )，这表明视频中结构图的等效化简结果应与此一致。**

**视频中反馈支路的具体表达式 (对应传递函数分母的各项，除了1以外):**
*   $H_1(s) = s R_1 C_1$
*   $H_2(s) = s C_2 R_2$
*   $H_3(s) = s^2 R_1 R_2 C_1 C_2$
*   $H_4(s) = s R_1 C_2$

这些反馈项对应于拉氏变换后 $U_C(s)$ 的系数项。

#### **四、总结**

1.  **微分方程求解：** 熟练运用 KVL、KCL 和元件特性方程。
2.  **结构图绘制 (难点)：**
    *   基于拉氏变换后的代数方程组。
    *   思路不唯一，但最终传递函数唯一。
    *   细心分析信号流向和运算关系。
3.  **等效变换：**
    *   掌握基本反馈回路、串并联环节的化简公式。
    *   逐层化简，条理清晰。
4.  **细心和耐心：** 此类题目计算量和推导过程相对复杂，需要细心。
5.  **梅森公式 ([[Mason's Gain Formula]]):** 虽然题目要求等效变换，但梅森公式是另一种求解复杂结构图传递函数的有效方法。
