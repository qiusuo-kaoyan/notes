### **题目：** 求解给定系统框图的传递函数 $C(s)/R(s)$ (系统框图化简与传递函数求解)
![[3-30.png]]
**考察知识点：** 系统结构图的化简、典型环节的识别、反馈系统传递函数的计算。

#### **一、 解题思路**

1.  **简化框图：** 对于不常见的或者看起来比较复杂的框图，首要任务是将其**简化**或**等效变换**为我们熟悉的标准形式，如：
    *   串联环节
    *   并联环节
    *   负反馈环节
    *   正反馈环节

2.  **识别反馈类型：** 准确判断反馈是正反馈还是负反馈，反馈信号的引出点和引入点。

3.  **分步求解：** 对于嵌套的反馈或者复杂的串并联结构，可以先求解局部环节的等效传递函数，再逐步求解整体的传递函数。

#### **二、 视频中例题的化简与求解步骤**

**原始框图分析：**

*   **输入:** $R(s)$
*   **输出:** $C(s)$
*   **前向通路:** $G_1(s)$ 和 $G_2(s)$ 串联。
*   **反馈通路1:** 从 $G_1(s)$ 的输出端引出，经过 $H_1(s)$，反馈到 $R(s)$ 的输入端（形成一个内部反馈）。
*   **反馈通路2:** 从 $C(s)$ （即 $G_2(s)$ 的输出端）引出，经过 $H_2(s)$，再经过 $H_1(s)$，反馈到 $R(s)$ 的输入端（形成一个外部反馈）。

**化简步骤 (视频中的思路)：**

1.  **简化内部反馈环：**
    *   将 $R(s)$ 经过 $G_1(s)$ 以及 $H_1(s)$ 的反馈视为一个独立的负反馈环节。
    *   输入：$R(s)$
    *   前向通路：$G_1(s)$
    *   反馈通路：$H_1(s)$
    *   此内部反馈环的等效传递函数 $G_{eq1}(s)$ 为：
        $G_{eq1}(s) = \frac{G_1(s)}{1 + G_1(s)H_1(s)}$

2.  **重新构建系统框图：**
    *   此时系统可以看作是 $G_{eq1}(s)$ 与 $G_2(s)$ 串联作为新的前向通路。
    *   新的前向通路 $G_{open}(s) = G_{eq1}(s) G_2(s) = \frac{G_1(s)G_2(s)}{1 + G_1(s)H_1(s)}$
    *   外部反馈通路为 $H_1(s)H_2(s)$ （注意：视频中此处描述为 $H_2$ 经过 $H_1$ 反馈，因此是 $H_1H_2$）。

3.  **求解整体负反馈系统的传递函数：**
    *   前向通路: $G_{open}(s) = \frac{G_1(s)G_2(s)}{1 + G_1(s)H_1(s)}$
    *   反馈通路: $H_{feedback}(s) = H_1(s)H_2(s)$
    *   整体系统的传递函数 $\Phi(s) = \frac{C(s)}{R(s)}$ 为：
        $\Phi(s) = \frac{G_{open}(s)}{1 + G_{open}(s)H_{feedback}(s)}$
        $\Phi(s) = \frac{\frac{G_1G_2}{1 + G_1H_1}}{1 + \frac{G_1G_2}{1 + G_1H_1} H_1H_2}$

4.  **化简表达式：**
    $\Phi(s) = \frac{\frac{G_1G_2}{1 + G_1H_1}}{\frac{(1 + G_1H_1) + G_1G_2H_1H_2}{1 + G_1H_1}}$
    $\Phi(s) = \frac{G_1G_2}{1 + G_1H_1 + G_1G_2H_1H_2}$

**数学公式总结 (常用反馈系统传递函数)：**

*   **负反馈系统：**
    $\Phi(s) = \frac{G(s)}{1 + G(s)H(s)}$
*   **正反馈系统：**
    $\Phi(s) = \frac{G(s)}{1 - G(s)H(s)}$
*   **单位反馈系统 (负反馈, H(s)=1)：**
    $\Phi(s) = \frac{G(s)}{1 + G(s)}$

**另一种化简思路 (逐步向后合并或梅逊增益公式，此处不展开，视频中未使用)：**
对于更复杂的图，有时直接套用反馈公式可能不直观，可以考虑使用[[信号流图]]和梅逊增益公式，或者逐步进行等效变换（如将求和点、引出点移动等）。

#### **三、 总结与注意事项**

1.  **核心步骤：** 简化框图 -> 识别典型环节 -> 分步或整体求解。
2.  **关键技能：**
    *   准确识别前向通路和反馈通路。
    *   ==正确判断反馈的极性（正/负）。==
    *   熟练运用典型反馈系统的传递函数公式。
3.  **化简技巧：**
    *   从内到外，逐层化简嵌套的反馈环。
    *   注意反馈信号的引出点和反馈点的准确位置。
    *   对于复杂的反馈路径，要明确反馈信号具体经过了哪些环节。
4.  **代数运算：** 在化简传递函数表达式时，注意代数运算的准确性，避免计算错误。
5.  **真题特点：** 很多院校喜欢考察对标准反馈形式进行微小改动或组合的框图，测试学生对基本概念的灵活运用能力。

**举例：**

*   **例1 (简单负反馈):**
    ```
      R(s) -->(+)-->[ G(s) ]-- C(s)
              ^          |
              |---[ H(s) ]<--
    ```
    $\Phi(s) = \frac{G(s)}{1+G(s)H(s)}$

*   **例2 (串联后反馈):**
    ```
      R(s) -->(+)-->[ G1(s) ]-->[ G2(s) ]-- C(s)
              ^                         |
              |--------[ H(s) ]---------<--
    ```
    等效前向通路 $G_{eq}(s) = G_1(s)G_2(s)$
    $\Phi(s) = \frac{G_1(s)G_2(s)}{1+G_1(s)G_2(s)H(s)}$

*   **例3 (内部反馈再串联，类似本题局部):**
    ```
      R(s) -->(+)-->[ G1(s) ]--[ G2(s) ]-- C(s)
              ^          |
              |---[ H1(s) ]<--
    ```
    先化简内部反馈：$G_{internal}(s) = \frac{G_1(s)}{1+G_1(s)H_1(s)}$
    整体传递函数：$\Phi(s) = G_{internal}(s) G_2(s) = \frac{G_1(s)G_2(s)}{1+G_1(s)H_1(s)}$
