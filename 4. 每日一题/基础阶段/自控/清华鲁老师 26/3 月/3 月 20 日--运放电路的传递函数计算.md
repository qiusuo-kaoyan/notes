### **题目：** 运算放大器（运放）电路的传递函数计算 (电路分析，传递函数)

**给定一个运算放大器电路图，求其传递函数 $G(s) = U_{out}(s) / U_{in}(s)$。**

#### **一、 核心概念：理想运算放大器**

分析运放电路的基础是理解理想运放的两个重要特性：“虚短”和“虚断”。

1.  **虚短 (Virtual Short):**
    *   **定义:** 在负反馈配置下，运放的两个输入端（同相输入端 $U^+$ 和反相输入端 $U^-$）的电压近似相等。
    *   **数学表示:** $U^+ = U^-$
    *   **原因:** 理想运放具有无穷大的开环增益。在负反馈稳定工作时，输出电压 $U_{out} = A_{OL}(U^+ - U^-)$，其中 $A_{OL} \to \infty$。要使输出 $U_{out}$ 为有限值，必须有 $U^+ - U^- \to 0$，即 $U^+ \approx U^-$。
    *   **本题应用:** 同相输入端 $U^+$ 接地，所以 $U^+ = 0$。根据虚短特性，反相输入端电压 $U^- = U^+ = 0$。

2.  **虚断 (Virtual Open):**
    *   **定义:** 流入运放两个输入端的电流近似为零。
    *   **数学表示:** $I^+ = 0$, $I^- = 0$
    *   **原因:** 理想运放具有无穷大的输入阻抗。
    *   **本题应用:** 流入反相输入端 $U^-$ 的电流为零。这意味着从输入部分流向节点的电流必须等于从节点流向反馈部分的电流。

#### **二、 核心概念：元件的 S 域阻抗**

为了在 S 域（拉普拉斯域）中分析电路并求解传递函数，需要将时域元件模型转换为 S 域阻抗模型（假设初始条件为零）。

| 元件     | 时域关系 ($u(t), i(t)$)           | S 域关系 ($U(s), I(s)$)         | S 域阻抗 $Z(s) = U(s)/I(s)$ |
| :------- | :------------------------------ | :--------------------------- | :-------------------------- |
| **电阻 R** | $u(t) = R \cdot i(t)$         | $U(s) = R \cdot I(s)$        | $$Z_R(s) = R$$              |
| **电容 C** | $u(t) = \frac{1}{C}\int i(t)dt$ | $U(s) = \frac{1}{C} \frac{I(s)}{s}$ | $$Z_C(s) = \frac{1}{Cs}$$     |
| **电感 L** | $u(t) = L \frac{di(t)}{dt}$     | $U(s) = L \cdot sI(s)$       | $$Z_L(s) = Ls$$             |

**注意:** 传递函数的定义是在 **零初始条件** 下，因此拉氏变换时不需要考虑初始值项。

#### **三、 电路分析步骤**

1.  **识别输入和输出:** 确定电路的输入信号 $U_{in}$ 和输出信号 $U_{out}$。
2.  **应用虚短和虚断:** 利用 $U^+=U^-$ 和 $I^+=I^-=0$ 来简化电路分析。
3.  **节点分析 (KCL):** 选择关键节点（通常是运放的输入端），应用基尔霍夫电流定律（KCL）：流入节点的电流之和等于流出节点的电流之和。
4.  **写出 S 域方程:** 将电路中的元件替换为其 S 域阻抗，并使用节点电压法写出关于 $U_{in}(s)$ 和 $U_{out}(s)$ 的代数方程。
5.  **求解传递函数:** 整理代数方程，求解 $G(s) = U_{out}(s) / U_{in}(s)$。

#### **四、 本题具体分析**

1.  **输入阻抗 $Z_{in}(s)$:** 输入部分是 $R_0$ 和 $C_0$ 并联。
    $$Z_{in}(s) = R_0 || Z_{C0}(s) = \frac{R_0 \cdot \frac{1}{C_0s}}{R_0 + \frac{1}{C_0s}} = \frac{\frac{R_0}{C_0s}}{\frac{R_0C_0s+1}{C_0s}} = \frac{R_0}{1 + R_0C_0s}$$

2.  **反馈阻抗 $Z_f(s)$:** 反馈部分是电阻 $R_1$。
    $$Z_f(s) = R_1$$

3.  **节点分析 (KCL @ $U^-$):** 根据虚断，$I^- = 0$。因此，流过 $Z_{in}$ 的电流 $I_{in}(s)$ 等于流过 $Z_f$ 的电流 $I_f(s)$。
    *   电流方向定义：从 $U_{in}$ 流向 $U^-$，从 $U^-$ 流向 $U_{out}$。
    *   $I_{in}(s) = \frac{U_{in}(s) - U^-(s)}{Z_{in}(s)}$
    *   $I_f(s) = \frac{U^-(s) - U_{out}(s)}{Z_f(s)}$
    *   由 KCL ($I_{in}(s) = I_f(s)$) 和虚短 ($U^-(s) = U^+(s) = 0$) 可得：
        $$\frac{U_{in}(s) - 0}{Z_{in}(s)} = \frac{0 - U_{out}(s)}{Z_f(s)}$$

4.  **求解传递函数:**
    $$\frac{U_{in}(s)}{Z_{in}(s)} = -\frac{U_{out}(s)}{Z_f(s)}$$
    $$U_{out}(s) = - \frac{Z_f(s)}{Z_{in}(s)} U_{in}(s)$$
    $$G(s) = \frac{U_{out}(s)}{U_{in}(s)} = - \frac{Z_f(s)}{Z_{in}(s)} = - \frac{R_1}{\frac{R_0}{1 + R_0C_0s}}$$
    $$G(s) = - \frac{R_1 (1 + R_0C_0s)}{R_0} = -\frac{R_1}{R_0} (1 + R_0C_0s)$$

#### **五、 结果**

该运放电路的传递函数为：
$$ G(s) = \frac{U_{out}(s)}{U_{in}(s)} = -\frac{R_1}{R_0}(1 + R_0C_0s) $$
这是一个 **比例微分 (PD)**环节。

#### **六、 举一反三 (例子)**

*   **反相比例放大器:** 如果将电容 $C_0$ 去掉（或视为断路），则 $Z_{in}(s) = R_0$。传递函数变为 $G(s) = -R_1/R_0$，这是一个常数增益。
*   **反相积分器:** 如果将 $R_0$ 去掉（或视为断路），输入阻抗变为 $Z_{in}(s) = Z_{C0}(s) = 1/(C_0s)$。传递函数变为 $G(s) = -R_1 / (1/(C_0s)) = -R_1C_0s$。*更正：应该是输入为R，反馈为C才是积分器，即 $Z_{in}=R, Z_f = 1/(Cs)$，则 $G(s) = - \frac{1/(Cs)}{R} = -\frac{1}{RCs}$。*
*   **反相微分器:** 如果输入阻抗为电容 $C_{in}$，反馈阻抗为电阻 $R_f$。则 $Z_{in}(s) = 1/(C_{in}s)$，$Z_f(s) = R_f$。传递函数变为 $G(s) = -R_f / (1/(C_{in}s)) = -R_fC_{in}s$。
*   **同相放大器:** 如果输入信号加在 $U^+$ 端，并将 $R_0$ (或 $Z_{in}$) 的一端接地， $R_1$ (或 $Z_f$) 连接在 $U_{out}$ 和 $U^-$ 之间。则 $U^- = U_{out} \frac{Z_{in}}{Z_{in}+Z_f}$。由虚短 $U^+ = U^-$，即 $U_{in} = U_{out} \frac{Z_{in}}{Z_{in}+Z_f}$。传递函数 $G(s) = \frac{U_{out}(s)}{U_{in}(s)} = 1 + \frac{Z_f(s)}{Z_{in}(s)}$。

#### **七、 总结**

1.  **核心方法:** 理想运放的“虚短”、“虚断”特性是分析基础。
2.  **S域分析:** 将电路元件转换为 S 域阻抗 $Z(s)$。
3.  **分析工具:** 节点电压法（KCL）是列写方程的主要手段。
4.  **传递函数:** 最终目标是求解输出 $U_{out}(s)$ 与输入 $U_{in}(s)$ 的比值（零初始条件下）。
5.  **常见电路:** 熟悉常见的运放电路（比例、积分、微分）及其传递函数有助于快速解题。
