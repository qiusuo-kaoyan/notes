### **题目：** 已知控制系统结构图，采用结构图化简的方法求系统的闭环传递函数 (结构图化简)
![[3-31-1.png]]
![[3-31-2.png]]
**给定：** 一个包含交叉反馈的复杂控制系统结构图。

**要求：** 采用结构图化简的方法，求系统的闭环传递函数，并写出详细的化简过程。

#### **一、 结构图化简的基本思路与技巧**

1.  **化简目标：** 将复杂的结构图逐步简化为标准形式（如单个反馈回路），最终得到输入到输出的等效传递函数。
2.  **处理交叉：** 当结构图中存在交叉连接时，首要任务是消除交叉，使反馈回路和前向通路更加清晰。
3.  **信号等效移动：**
    *   **移动求和点/比较点：**
        *   求和点后移过一个方框 $G(s)$：原先未经过 $G(s)$ 的支路需要串联一个 $1/G(s)$。
        *   求和点前移过一个方框 $G(s)$：原先未经过 $G(s)$ 的支路需要串联一个 $G(s)$。
    *   **移动分支点/引出点：**
        *   分支点后移过一个方框 $G(s)$：原先未经过 $G(s)$ 的支路需要串联一个 $G(s)$。
        *   **分支点前移过一个方框 $G(s)$：原先未经过 $G(s)$ 的支路需要串联一个 $1/G(s)$。 (本题核心技巧)**
4.  **识别基本连接形式并化简：**
    *   **串联：** $G_1(s)G_2(s)$
    *   **并联：** $G_1(s) \pm G_2(s)$
    *   **负反馈：** $\frac{G(s)}{1+G(s)H(s)}$
    *   **正反馈：** $\frac{G(s)}{1-G(s)H(s)}$

**数学公式表达 (反馈回路)：**
* 负反馈: $C(s)/R(s) = \frac{G(s)}{1+G(s)H(s)}$
* 正反馈: $C(s)/R(s) = \frac{G(s)}{1-G(s)H(s)}$

**举例 (移动分支点前移):**
* 假设原分支点在 $G_1(s)$ 之后，引出信号为 $X(s)$，则 $X(s) = \text{Input} \cdot G_1(s)$。
* 若将分支点移到 $G_1(s)$ 之前，引出信号仍需为 $X(s)$。此时，该分支点引出的信号为 $\text{Input}$。
* 为了保持信号等效，需要在移动后的分支点引出支路上串联一个 $G_1(s)$。
* 反之，若分支点从 $G_1(s)$ 之后移动到 $G_1(s)$ 之前，为了保持引出点后面的信号不变，需要在引出支路上乘以 $1/G_1(s)$。（本题中H2反馈支路的处理方式，原反馈点在G3后，现在要从总输出引出，相当于把分支点从G3后移到G3前，所以H2要乘以1/G3；同理，反馈到总输入前，相当于把分支点从G1前移到G1后，所以H2要乘以G1）

**更正演讲稿中的思路:** 演讲稿中关于信号B和A，以及C和D的描述，以及如何移动H2反馈的思路有些绕。更直接的理解是：
*   **目标：** 将H2的反馈从G3的输出端引出，反馈到G1的输入端。
*   **H2反馈输出端的移动：**
    *   原H2反馈从G3的输出端引出。
    *   现在要从总输出端（即G3的输出端）引出，这一步没有变化。
*   **H2反馈输入端的移动：**
    *   原H2反馈到G1和G2之间。
    *   现在要反馈到总输入端（即G1的输入端）。
    *   相当于将H2的求和点**前移**过了G1。
    *   因此，H2的反馈通路上需要乘以 $1/G_1(s)$。

    **演讲稿中提到的 $H_2 \cdot \frac{1}{G_3} \cdot G_1$ 实际上是 $H_2 \cdot G_1/G_3$。** 这个理解可能与演讲稿的表述有所出入，但是根据图和常见的移动规则是这样的。
    *  **视频中的思路是：** 将H2的反馈点从G3的输出端引出，反馈到G1的输入端。
        *   从G3输出端引出，**前移**过G3，则反馈支路乘以 $1/G_3$。
        *   反馈到G1输入端，相当于反馈点从G1的输入端**后移**过G1，则反馈支路乘以 $G_1$。
        *   所以等效的H2反馈通路上的传递函数为 $H_2 \cdot G_1 / G_3$。 **(演讲稿的最终结果是 $H_2 / (G_1 G_3)$，这与常见的移动规则有出入，需要以标准移动规则为准，或者理解演讲稿中可能有其特定的等效逻辑。) **

    **以演讲稿最终化简的 $H_2 / (G_1 G_3)$ 为准进行后续分析，假设其移动方式正确。**

#### **二、 本题化简步骤 (根据演讲稿的思路)**

1.  **消除交叉 - 移动 H2 反馈支路：**
    *   **目标：** 将 H2 的反馈从 G3 的输出端引出，并反馈到 G1 的输入端，形成一个大的外层反馈。
    *   **等效变换：** (根据演讲稿的结论)
        *   H2 的反馈引出点从 G3 输出端前移过 G3，反馈支路乘以 $1/G_3$。
        *   H2 的反馈输入点从 G1 输入端前移过 G1，反馈支路乘以 $1/G_1$。
        *   因此，新的等效反馈通路为 $H_{eq}(s) = \frac{H_2(s)}{G_1(s)G_3(s)}$。

    **(化简后的第一步示意图，参照演讲稿)**
    ```
          R(s) -->O---->[ G1 ]---->[ G2 ]---->[ G3 ]----> C(s)
                -^ -           -^           -^
                 | |            |            |
                 | +<---[ H1 ]<--+            +<---[ H3 ]<--+
                 |
                 +<--------------------[ H2/(G1*G3) ]<-------------------+
    ```

2.  **化简内部两个独立的负反馈回路：**
    *   **左侧反馈 (G1, H1):** $G_{L}(s) = \frac{G_1(s)}{1+G_1(s)H_1(s)}$
    *   **右侧反馈 (G3, H3):** $G_{R}(s) = \frac{G_3(s)}{1+G_3(s)H_3(s)}$

    **(化简后的第二步示意图)**
    ```
          R(s) -->O---->[ G_L(s) ]---->[ G2(s) ]---->[ G_R(s) ]----> C(s)
                -^
                 |
                 +<--------------------[ H2/(G1*G3) ]<-------------------+
    ```

3.  **计算新的前向通路总增益 $G_{open}(s)$：**
    *   $G_{open}(s) = G_L(s) \cdot G_2(s) \cdot G_R(s)$
    *   $G_{open}(s) = \frac{G_1}{1+G_1H_1} \cdot G_2 \cdot \frac{G_3}{1+G_3H_3}$

4.  **化简最外层的大负反馈回路：**
    *   前向通路：$G_{open}(s)$
    *   反馈通路：$H_{eq}(s) = \frac{H_2}{G_1G_3}$
    *   闭环传递函数 $\frac{C(s)}{R(s)} = \frac{G_{open}(s)}{1+G_{open}(s)H_{eq}(s)}$
    *   $\frac{C(s)}{R(s)} = \frac{\frac{G_1G_2G_3}{(1+G_1H_1)(1+G_3H_3)}}{1 + \frac{G_1G_2G_3}{(1+G_1H_1)(1+G_3H_3)} \cdot \frac{H_2}{G_1G_3}}$
    *   $\frac{C(s)}{R(s)} = \frac{G_1G_2G_3}{(1+G_1H_1)(1+G_3H_3) + G_2H_2}$

**(最终化简的示意图)**
    ```
          R(s) -->[  C(s)/R(s)  ]----> C(s)
    ```

#### **三、总结**

1.  **核心方法：** 结构图等效变换规则，特别是分支点和求和点的移动。
2.  **解题关键：**
    *   准确识别交叉点，并选择合适的移动策略消除交叉。
    *   正确应用移动规则，保证信号的等效性。
    *   逐步化简，将复杂结构分解为简单的串联、并联、反馈。
3.  **注意事项：**
    *   每一步化简都要画出清晰的等效结构图。
    *   代数运算要仔细，避免计算错误。
    *   对于要求结构图化简的题目，必须展示详细的化简步骤，不能直接用梅森公式。
4.  **与梅森公式的对比：**
    *   结构图化简更直观，但步骤可能较多，尤其对于复杂交叉。
    *   梅森公式更通用，尤其对于复杂图，但需要准确找出所有前向通路和回路。
