### **题目：** 已知系统各变量间的关系方程，绘制系统结构框图并求传递函数 (结构框图绘制与化简)
![[3-29.png]]
**已知：** 系统各变量之间的关系方程 (S域形式)：

1.  $X_1(s) = R(s) - C(s)$
2.  $X_2(s) = X_1(s) + H_1(s)X_4(s)$
3.  $X_3(s) = G_1(s)X_2(s) - H_2(s)C(s)$
4.  $X_4(s) = G_2(s)X_3(s)$
5.  $C(s) = G_3(s)X_4(s)$

**要求：**

1.  画出系统的结构框图。
2.  求出系统的传递函数 $\frac{C(s)}{R(s)}$。

#### **一、 绘制系统结构框图**

1.  **基本思路/技巧：倒着画** (从输出端开始，逐步向前推)。

2.  **绘制步骤 (按照倒着画的思路)：**

    *   **从 $C(s)$ 开始：** $C(s) = G_3(s)X_4(s)$
        *   画出输出 $C(s)$。
        *   画一个传递函数为 $G_3(s)$ 的方框，其输入为 $X_4(s)$。
    *   **$X_4(s)$：** $X_4(s) = G_2(s)X_3(s)$
        *   在 $G_3(s)$ 前面画一个传递函数为 $G_2(s)$ 的方框，其输入为 $X_3(s)$。
    *   **$X_3(s)$：** $X_3(s) = G_1(s)X_2(s) - H_2(s)C(s)$
        *   在 $G_2(s)$ 前面画一个求和点，其输出为 $X_3(s)$。
        *   从求和点引出一条支路，经过传递函数为 $G_1(s)$ 的方框，其输入为 $X_2(s)$ (正向输入到求和点)。
        *   从输出 $C(s)$ 引出一条反馈支路，经过传递函数为 $H_2(s)$ 的方框，负向输入到 $X_3(s)$ 处的求和点。
    *   **$X_2(s)$：** $X_2(s) = X_1(s) + H_1(s)X_4(s)$
        *   在 $G_1(s)$ 前面画一个求和点，其输出为 $X_2(s)$。
        *   从求和点引出一条支路，其输入为 $X_1(s)$ (正向输入到求和点)。
        *   从 $X_4(s)$ (即 $G_3(s)$ 的输入端) 引出一条反馈支路，经过传递函数为 $H_1(s)$ 的方框，正向输入到 $X_2(s)$ 处的求和点。
    *   **$X_1(s)$：** $X_1(s) = R(s) - C(s)$
        *   在 $X_2(s)$ 处的求和点的前面画一个求和点，其输出为 $X_1(s)$。
        *   画出系统输入 $R(s)$，正向输入到 $X_1(s)$ 处的求和点。
        *   从输出 $C(s)$ 引出一条反馈支路 (单位反馈)，负向输入到 $X_1(s)$ 处的求和点。

3.  **注意事项：**
    *   明确信号流向 (箭头)。
    *   正确表示求和点的正负号。
    *   如果给出的是时域微分方程，需要先进行拉氏变换。

#### **二、 求解系统传递函数**

1.  **方法选择：**
    *   **结构框图化简：** 适用于简单、无交叉的框图。对于复杂或有交叉的框图，化简过程可能繁琐易错。
    *   **梅森增益公式 ([[Mason's Gain Formula]])：** 适用于各种复杂程度的结构框图，尤其是有交叉回路的情况。

2.  **梅森增益公式应用步骤：**

    *   **识别前向通路及其增益 ($P_k$)：**
        *   **本题：** 只有一条前向通路 $P_1 = G_1(s)G_2(s)G_3(s)$。
    *   **识别所有独立回路及其增益 ($L_i$)：**
        *   $L_1 = -G_2(s)G_3(s)H_2(s)$ (围绕 $X_3, X_4, C$ 的负反馈回路)
        *   $L_2 = G_1(s)G_2(s)H_1(s)$ (围绕 $X_2, X_3, X_4$ 的正反馈回路)
        *   $L_3 = -G_1(s)G_2(s)G_3(s)$ (整个系统的大负反馈回路，单位反馈)
    *   **计算特征行列式 ($\Delta$)：**
        *   $\Delta = 1 - \sum L_i + \sum L_i L_j - \sum L_i L_j L_k + ...$
        *   $\sum L_i$ 是所有独立回路增益之和。
        *   $\sum L_i L_j$ 是所有两两互不接触的回路增益乘积之和。
        *   **本题：** 三个回路均相互接触，所以 $\sum L_i L_j = 0$, $\sum L_i L_j L_k = 0$。
        *   $\Delta = 1 - (L_1 + L_2 + L_3)$
        *   $\Delta = 1 - (-G_2G_3H_2 + G_1G_2H_1 - G_1G_2G_3)$
        *   $\Delta = 1 + G_2G_3H_2 - G_1G_2H_1 + G_1G_2G_3$
    *   **计算余因子式 ($\Delta_k$)：**
        *   $\Delta_k$ 是在特征行列式 $\Delta$ 中，去掉与第 $k$ 条前向通路相接触的回路后，剩余部分的特征行列式。
        *   **本题：** $P_1$ 与所有三个回路都接触，因此去掉这些回路后，没有剩余回路。
        *   $\Delta_1 = 1$
    *   **应用梅森公式：**
        *   $\frac{C(s)}{R(s)} = \frac{\sum P_k \Delta_k}{\Delta}$
        *   **本题：** $\frac{C(s)}{R(s)} = \frac{P_1 \Delta_1}{\Delta} = \frac{G_1(s)G_2(s)G_3(s) \cdot 1}{1 + G_2(s)G_3(s)H_2(s) - G_1(s)G_2(s)H_1(s) + G_1(s)G_2(s)G_3(s)}$
        *   **视频中结果：** (分母项符号与上述推导略有出入，需仔细核对回路定义时的正负号，但基本思路一致)
            $\frac{C(s)}{R(s)} = \frac{G_1G_2G_3}{1 + G_1G_2G_3 + G_2G_3H_2 - G_1G_2H_1}$

**梅森公式数学表达：**
$\Phi(s) = \frac{\sum_{k} P_k \Delta_k}{\Delta}$
其中：
*   $P_k$: 第 $k$ 条前向通路的增益。
*   $\Delta = 1 - \sum_i L_i + \sum_{i,j} L_i L_j - \sum_{i,j,k} L_i L_j L_k + \dots$
    *   $L_i$: 第 $i$ 个独立回路的增益。
    *   $\sum L_i L_j$: 所有两两互不接触的回路增益乘积之和。
*   $\Delta_k$: $\Delta$ 中去掉与 $P_k$ 相接触的回路后得到的余因子式。

**举例:**

*   **简化法适用场景:** $C(s) = \frac{G(s)}{1+G(s)H(s)}R(s)$ 这样的简单负反馈系统.
*   **梅森公式中不接触回路:** 假设有两个回路 $L_a$ 和 $L_b$ 它们之间没有共同的节点或支路, 则在计算 $\Delta$ 时会有 $L_a L_b$ 这一项.
*   **多条前向通路:** 如果系统有多条从输入到输出的路径 $P_1, P_2, ...$, 则分子为 $P_1\Delta_1 + P_2\Delta_2 + ...$.

#### **三、总结**

1.  **结构框图绘制技巧：** 对于给定的变量关系方程，**倒着画** 是一个非常有效的策略，可以使绘制过程更清晰、简单。
2.  **传递函数求解方法：**
    *   **结构框图化简：** 适用于简单系统。
    *   **梅森增益公式：** 通用性强，尤其适用于复杂、有交叉回路的系统，步骤清晰，不易出错。
3.  **梅森公式关键点：**
    *   准确识别前向通路和所有独立回路。
    *   正确判断回路之间是否接触，以确定 $\Delta$ 和 $\Delta_k$ 的计算。
    *   注意回路增益的正负号。
4.  **考试应对：** 此类题目是考研常见题型，掌握绘制结构框图的技巧和熟练运用梅森公式至关重要。
