### **题目：** 已知系统结构图（含扰动），求系统输出 C(s) 的表达式 (系统数学模型 - 传递函数求解)

**给定一个包含输入 $R(s)$ 和扰动 $N(s)$ 的系统结构图，求输出 $C(s)$ 关于 $R(s)$ 和 $N(s)$ 的表达式。**
![[3-23.png]]
#### **一、 核心原理：叠加原理 (Superposition Principle)**

*   对于线性系统，当存在多个输入信号时，系统的总输出等于 **每个输入信号单独作用时** 产生的输出之和。
*   **计算步骤:**
    1.  计算输入 $R(s)$ 单独作用下的输出：令 $N(s) = 0$，求传递函数 $\Phi_R(s) = \frac{C(s)}{R(s)}\Big|_{N(s)=0}$，则 $C_R(s) = \Phi_R(s) \cdot R(s)$。
    2.  计算扰动 $N(s)$ 单独作用下的输出：令 $R(s) = 0$，求传递函数 $\Phi_N(s) = \frac{C(s)}{N(s)}\Big|_{R(s)=0}$，则 $C_N(s) = \Phi_N(s) \cdot N(s)$。
    3.  总输出： $C(s) = C_R(s) + C_N(s) = \Phi_R(s) \cdot R(s) + \Phi_N(s) \cdot N(s)$。

#### **二、 求解传递函数的方法**

1.  **方法一：[[方框图]]简化 (Block Diagram Reduction)**
    *   通过串联、并联、反馈等基本规则逐步简化方框图，最终得到输入到输出的等效传递函数。
    *   对于多输入系统，需要分别针对每个输入进行简化（将其他输入置零）。
    *   讲座中提到此方法对于本题相对简单，但未详细演示。
        *   求 $\Phi_R(s)$ 时， $N(s)=0$。系统简化为 $G_1$ 与 内反馈回路 ($G_2, H_1$) 串联，再进行外层单位负反馈。
        *   求 $\Phi_N(s)$ 时， $R(s)=0$。 $N(s)$ 作用点之后的部分 ($G_2$) 到输出 $C(s)$ 形成一个包含两个反馈（内反馈 $H_1$ 和外反馈 $G_1$）的复杂反馈结构。

2.  **方法二：梅森增益公式 ([[Mason's Gain Formula]])**
    *   适用于**信号流图**，但也可直接应用于方框图。
    *   **公式:** $\Phi(s) = \frac{\sum_{k} P_k \Delta_k}{\Delta}$
        *   $\Phi(s)$: 从指定输入到指定输出的总传递函数。
        *   $P_k$: 从输入到输出的第 $k$ 条 **前向通路** 的增益。
        *   $\Delta$: 系统的 **特征式 (Determinant)**
            $\Delta = 1 - (\sum L_i) + (\sum L_i L_j) - (\sum L_i L_j L_k) + \dots$
            *   $\sum L_i$: 所有 **单独回路增益** 之和。
            *   $\sum L_i L_j$: 所有 **两两互不接触** 回路增益乘积之和。
            *   $\sum L_i L_j L_k$: 所有 **三三互不接触** 回路增益乘积之和。
            *   ...以此类推。
        *   $\Delta_k$: $\Delta$ 中 **除去与第 $k$ 条前向通路 $P_k$ 相接触** 的回路后的剩余部分计算得到的特征式。

#### **三、 使用梅森公式求解本题 (讲座演示)**

*   **符号约定 (讲座):** $G_1=J_1$, $G_2=J_2$, $H_1=J_p$。外反馈通路假设为单位反馈 ($H_2=1$, 讲座中用 $J_s$ 可能代表 $H_2=1$ 或输出点本身，按单位反馈理解）。
*   **Step 1: 识别回路 (Loops)** - 回路对 R(s) 和 N(s) 是共用的
    *   内回路: $L_1 = -G_2 H_1$ (讲座: $-J_2 J_p$)
    *   外回路: $L_2 = -G_1 G_2$ (讲座: $-J_1 J_2 J_s$, 假设 $J_s=1$)
*   **Step 2: 计算系统特征式 $\Delta$**
    *   $L_1$ 和 $L_2$ 是否接触？是 (都经过 $G_2$)。
    *   因此，没有互不接触的回路对。
    *   $\Delta = 1 - (L_1 + L_2) = 1 - (-G_2 H_1 - G_1 G_2) = 1 + G_2 H_1 + G_1 G_2$
    *   (讲座: $1 + J_2 J_p + J_1 J_2 J_s$)
*   **Step 3: 计算 $\Phi_R(s) = \frac{C(s)}{R(s)}\Big|_{N(s)=0}$**
    *   **前向通路 (Forward Path) $P_{R1}$:** $R(s) \rightarrow G_1 \rightarrow G_2 \rightarrow C(s)$。 $P_{R1} = G_1 G_2$ (讲座: $J_1 J_2 J_s$)
    *   **计算 $\Delta_{R1}$:** $P_{R1}$ 接触 $L_1$ 和 $L_2$ 吗？是 (都经过 $G_2$)。
    *   除去所有接触回路，$\Delta_{R1} = 1$。
    *   **应用公式:** $\Phi_R(s) = \frac{P_{R1} \Delta_{R1}}{\Delta} = \frac{G_1 G_2 \times 1}{1 + G_2 H_1 + G_1 G_2}$
    *   (讲座: $\frac{J_1 J_2 J_s}{1 + J_2 J_p + J_1 J_2 J_s}$)
*   **Step 4: 计算 $\Phi_N(s) = \frac{C(s)}{N(s)}\Big|_{R(s)=0}$**
    *   **前向通路 $P_{N1}$:** $N(s) \rightarrow G_2 \rightarrow C(s)$。 $P_{N1} = G_2$ (讲座: $J_2$)
    *   **计算 $\Delta_{N1}$:** $P_{N1}$ 接触 $L_1$ 和 $L_2$ 吗？是 (都经过 $G_2$)。
    *   除去所有接触回路，$\Delta_{N1} = 1$。
    *   **应用公式:** $\Phi_N(s) = \frac{P_{N1} \Delta_{N1}}{\Delta} = \frac{G_2 \times 1}{1 + G_2 H_1 + G_1 G_2}$
    *   (讲座: $\frac{J_2}{1 + J_2 J_p + J_1 J_2 J_s}$)
*   **Step 5: 应用叠加原理得到最终输出 $C(s)$**
    *   $C(s) = \Phi_R(s) \cdot R(s) + \Phi_N(s) \cdot N(s)$
    *   $$ C(s) = \left( \frac{G_1 G_2}{1 + G_2 H_1 + G_1 G_2} \right) R(s) + \left( \frac{G_2}{1 + G_2 H_1 + G_1 G_2} \right) N(s) $$
    *   (讲座最终表达式结构一致)

#### **四、 总结与关键点**

1.  求解多输入系统的输出，**叠加原理** 是基础。
2.  计算各输入到输出的传递函数，可以使用 **方框图简化** 或 **梅森增益公式**。
3.  **梅森公式** 提供了一种系统化的方法，特别适用于复杂系统：
    *   关键在于正确识别 **所有前向通路** 和 **所有回路**。
    *   准确判断通路与回路之间、回路与回路之间的 **接触关系**。
    *   严格按照公式计算 $\Delta$ 和 $\Delta_k$。
4.  在最终的输出表达式中，**不要遗漏任何输入项** (包括扰动)。
5.  注意讲座中使用了 **非标准符号** ($J_1, J_2, J_s, J_p$)，理解其对应关系 ($G_1, G_2, 1, H_1$) 很重要。

#### **五、 补充示例**

*   **例1 (简单负反馈):** $R(s) \rightarrow (+) \rightarrow G(s) \rightarrow C(s)$, $C(s) \rightarrow (-) \rightarrow H(s)$ 回到输入。
    *   $L_1 = -G(s)H(s)$
    *   $P_1 = G(s)$
    *   $\Delta = 1 - L_1 = 1 + G(s)H(s)$
    *   $\Delta_1 = 1$ (通路接触回路)
    *   $\Phi(s) = \frac{P_1 \Delta_1}{\Delta} = \frac{G(s)}{1+G(s)H(s)}$
*   **例2 (两个并联通路):** $R(s)$ 分别经过 $G_1(s)$ 和 $G_2(s)$ 到达输出 $C(s)$ (无回路)。
    *   $L_i = 0$ (无回路)
    *   $P_1 = G_1(s)$, $P_2 = G_2(s)$
    *   $\Delta = 1$
    *   $\Delta_1 = 1$, $\Delta_2 = 1$
    *   $\Phi(s) = \frac{P_1 \Delta_1 + P_2 \Delta_2}{\Delta} = \frac{G_1(s) \times 1 + G_2(s) \times 1}{1} = G_1(s) + G_2(s)$

这个总结详细解释了叠加原理和梅森公式，并结合讲座内容逐步演示了求解过程，最后给出了总结和补充示例。
