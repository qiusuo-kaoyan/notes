### **题目：** 已知系统结构图，求系统传递函数 (系统框图化简/梅森增益公式)
![[1. 控制科学备考/4. 每日一题/自控/清华鲁老师 26/Picture Set/3-26.png]]
**考点类型:** 系统框图化简 / 梅森增益公式

#### **一、 核心知识点回顾**

1.  **系统框图求传递函数的方法:**
    *   **方法一：框图简化 (等效变换法)**
        *   通过串联、并联、反馈等基本连接的等效变换规则，逐步简化框图，最终得到输入输出间的传递函数。
        *   **适用场景:** 结构相对简单，交叉较少的框图。
        *   **本题分析:** 讲稿中提到，由于前向通路与反馈通路存在交叉，导致简化过程复杂，不推荐使用此方法。
    *   **方法二：梅森增益公式 ([[Mason's Gain Formula]])**
        *   一种直接根据信号流图或框图计算传递函数的系统方法。
        *   **适用场景:** 结构复杂，存在较多交叉回路的框图或信号流图。
        *   **本题推荐:** 讲稿中明确指出，对于存在反馈和前向通路交叉的复杂框图，梅森公式更为简便。

2.  **梅森增益公式:**
    $P(s) = \frac{\sum_{k} P_k \Delta_k}{\Delta}$
    其中：
    *   $P(s)$: 系统的总传递函数。
    *   $P_k$: 第 $k$ 条前向通路的增益。
    *   $\Delta$: 系统的特征式 (行列式)，计算公式为：
        $\Delta = 1 - \sum L_i + \sum L_i L_j - \sum L_i L_j L_k + ...$
        *   $\sum L_i$: 所有独立回路增益之和。
        *   $\sum L_i L_j$: 所有两两互不接触的回路增益乘积之和。
        *   $\sum L_i L_j L_k$: 所有三三互不接触的回路增益乘积之和，以此类推。
    *   $\Delta_k$: 第 $k$ 条前向通路的余子式，即在系统中除去与第 $k$ 条前向通路相接触的回路后，剩余部分的特征式 (计算方法同 $\Delta$)。如果前向通路与所有回路都接触，则 $\Delta_k = 1$。

**数学公式表达：**
*   **串联:** $G(s) = G_1(s)G_2(s)$
*   **并联:** $G(s) = G_1(s) \pm G_2(s)$
*   **负反馈:** $G(s) = \frac{G_f(s)}{1+G_f(s)H(s)}$ (其中 $G_f(s)$ 为前向通路，$H(s)$ 为反馈通路)
*   **正反馈:** $G(s) = \frac{G_f(s)}{1-G_f(s)H(s)}$

#### **二、 题目解答步骤 (梅森增益公式法)**

1.  **找出所有前向通路及其增益 ($P_k$):**
    *   **前向通路1 ($P_1$):** R → G1 → G4 → C
        $P_1 = G_1 G_4$
    *   **前向通路2 ($P_2$):** R → G1 → G2 → G3 → C
        $P_2 = G_1 G_2 G_3$

2.  **找出所有独立回路及其增益 ($L_i$):**
    *   **回路1 ($L_1$):** G1 → G2 → G3 → H1 (负反馈)
        $L_1 = -G_1 G_2 G_3 H_1$
    *   **回路2 ($L_2$):** G1 → G2 → G3 → H2 (负反馈)
        $L_2 = -G_1 G_2 G_3 H_2$
    *   **回路3 ($L_3$):** G1 → G4 → H2 (负反馈)
        $L_3 = -G_1 G_4 H_2$

3.  **判断有无互不接触的回路:**
    *   观察框图，所有三个回路都共用 G1 节点，因此不存在两两互不接触的回路。
    *   所以，$\sum L_i L_j = 0$, $\sum L_i L_j L_k = 0$, ...

4.  **计算系统特征式 ($\Delta$):**
    *   $\Delta = 1 - (L_1 + L_2 + L_3)$
    *   $\Delta = 1 - (-G_1 G_2 G_3 H_1 - G_1 G_2 G_3 H_2 - G_1 G_4 H_2)$
    *   $\Delta = 1 + G_1 G_2 G_3 H_1 + G_1 G_2 G_3 H_2 + G_1 G_4 H_2$

5.  **计算各前向通路的余子式 ($\Delta_k$):**
    *   **对于 $P_1 = G_1 G_4$:**
        *   $P_1$ 与 $L_1$ 接触 (通过 G1)。
        *   $P_1$ 与 $L_2$ 接触 (通过 G1)。
        *   $P_1$ 与 $L_3$ 接触 (通过 G1, G4, H2)。
        *   由于 $P_1$ 与所有回路都接触，所以 $\Delta_1 = 1$。
    *   **对于 $P_2 = G_1 G_2 G_3$:**
        *   $P_2$ 与 $L_1$ 接触 (通过 G1, G2, G3, H1)。
        *   $P_2$ 与 $L_2$ 接触 (通过 G1, G2, G3, H2)。
        *   $P_2$ 与 $L_3$ 接触 (通过 G1)。
        *   由于 $P_2$ 与所有回路都接触，所以 $\Delta_2 = 1$。

6.  **代入梅森增益公式计算总传递函数 $P(s)$:**
    *   $P(s) = \frac{P_1 \Delta_1 + P_2 \Delta_2}{\Delta}$
    *   $P(s) = \frac{G_1 G_4 \times 1 + G_1 G_2 G_3 \times 1}{1 + G_1 G_2 G_3 H_1 + G_1 G_2 G_3 H_2 + G_1 G_4 H_2}$
    *   $P(s) = \frac{G_1 G_4 + G_1 G_2 G_3}{1 + G_1 G_2 G_3 H_1 + G_1 G_2 G_3 H_2 + G_1 G_4 H_2}$

**举例：**

*   **简单反馈系统:** 对于一个简单负反馈 $G(s)/(1+G(s)H(s))$
    *   $P_1 = G(s)$
    *   $L_1 = -G(s)H(s)$
    *   $\Delta = 1 - L_1 = 1 + G(s)H(s)$
    *   $\Delta_1 = 1$ (前向通路与回路接触)
    *   $P(s) = \frac{G(s) \times 1}{1+G(s)H(s)}$
*   **包含不接触回路的系统:** (假设一个更复杂的系统)
    *   如果存在两个不接触的回路 $L_a$ 和 $L_b$，则 $\Delta$ 中会包含 $L_a L_b$ 项。
    *   如果某条前向通路 $P_x$ 不接触回路 $L_c$，则 $\Delta_x$ 的计算会包含 $1 - L_c$ 项。

#### **三、总结**

1.  **方法选择:**
    *   框图简单、交叉少：优先考虑框图简化法。
    *   框图复杂、交叉多（尤其是前向通路与反馈通路交叉）：强烈推荐梅森增益公式。
2.  **梅森公式步骤:** 熟练掌握找前向通路、找回路、判断回路接触情况、计算 $\Delta$ 和 $\Delta_k$ 的步骤。
3.  **符号注意:** 计算回路增益时，要注意反馈的符号（正反馈为正，负反馈为负）。
4.  **“不接触”的判断:** 准确判断回路之间以及前向通路与回路之间是否“不接触”（即没有共用任何节点或支路）。

该总结强调了在不同情况下选择合适方法的重要性，并详细分解了使用梅森增益公式的步骤，有助于理解和应用。
