### **题目：** 已知系统结构图，求传递函数 $Y(s)/R(s)$ (系统框图化简与梅森增益公式应用题)
![[3-24.png]]
**已知：** 系统的结构框图。
**要求：** 求系统的闭环传递函数 $Y(s)/R(s)$。

#### **一、 方法一：结构图化简法 (逐步化简)**

1.  **识别基本连接方式：**
    *   串联：$G_1(s)G_2(s)$
    *   并联：$G_1(s) \pm G_2(s)$
    *   反馈：$\frac{G(s)}{1 \mp G(s)H(s)}$ (负反馈为“+”，正反馈为“-”)

2.  **化简步骤 (针对本题)：**
    *   **步骤1：内部并联环节化简**
        *   识别出 $G_2$ 和 $G_3$ 串联后与 $G_4$ 并联的结构。
        *   化简为：$G_A = G_4 + G_2G_3$
        *   此时结构图变为：$R(s) \rightarrow [G_1] \rightarrow [G_A] \rightarrow Y(s)$，其中 $G_A$ 有一个负反馈 $H$。

    *   **步骤2：内部反馈环节化简**
        *   $G_A$ 和 $H$ 构成一个负反馈回路。
        *   化简为：$G_B = \frac{G_A}{1 + G_A H} = \frac{G_4 + G_2G_3}{1 + (G_4 + G_2G_3)H}$
        *   此时结构图变为：$R(s) \rightarrow [G_1] \rightarrow [G_B] \rightarrow Y(s)$，并且整体有一个单位负反馈。
        *   **注意：** 视频中讲解老师将 $G_B$ 视为了一个整体的“开环传递函数” $G_{open}$，这在应用整体单位负反馈公式时是正确的，但从逐步化简的角度，可以先算内部反馈。

    *   **步骤3：整体反馈环节化简**
        *   $G_1$ 与 $G_B$ 串联，形成新的前向通路 $G_{open} = G_1 G_B$。
        *   系统整体是一个单位负反馈。
        *   闭环传递函数：$\frac{Y(s)}{R(s)} = \frac{G_{open}}{1 + G_{open}} = \frac{G_1 G_B}{1 + G_1 G_B}$
        *   代入 $G_B$ 并化简：
            $\frac{Y(s)}{R(s)} = \frac{G_1 \frac{G_4 + G_2G_3}{1 + (G_4 + G_2G_3)H}}{1 + G_1 \frac{G_4 + G_2G_3}{1 + (G_4 + G_2G_3)H}} = \frac{G_1(G_4 + G_2G_3)}{1 + (G_4 + G_2G_3)H + G_1(G_4 + G_2G_3)}$
            $\frac{Y(s)}{R(s)} = \frac{G_1G_4 + G_1G_2G_3}{1 + G_4H + G_2G_3H + G_1G_4 + G_1G_2G_3}$

    *   **老师视频中的思路 (等效开环传递函数法):**
        *   将内部的 $G_4 + G_2G_3$ 与反馈 $H$ 构成的回路看作一个新的模块，其传递函数为 $G_{inner} = \frac{G_4 + G_2G_3}{1 + (G_4 + G_2G_3)H}$
        *   这个 $G_{inner}$ 与 $G_1$ 串联构成了整个系统在单位负反馈前的“开环传递函数” $G_{open} = G_1 \cdot G_{inner}$。
        *   然后应用单位负反馈公式：$\frac{Y(s)}{R(s)} = \frac{G_{open}}{1 + G_{open}}$，代入后结果与上述一致。

#### **二、 方法二：梅森增益公式**

1.  **梅森增益公式：**
    $P(s) = \frac{1}{\Delta} \sum_{k} P_k \Delta_k$
    其中：
    *   $P(s)$：系统总传递函数。
    *   $P_k$：第 $k$ 条前向通路的增益。
    *   $\Delta$：系统的特征式，$\Delta = 1 - \sum L_i + \sum L_i L_j - \sum L_i L_j L_k + ...$
        *   $\sum L_i$：所有单个回路增益之和。
        *   $\sum L_i L_j$：所有两两互不接触回路增益乘积之和。
        *   $\sum L_i L_j L_k$：所有三三互不接触回路增益乘积之和。
    *   $\Delta_k$：与第 $k$ 条前向通路不接触的回路所构成的子图的特征式 (计算方法同 $\Delta$ )。如果前向通路与所有回路都接触，则 $\Delta_k = 1$。

2.  **应用步骤 (针对本题)：**
    *   **步骤1：找出所有前向通路及其增益 ($P_k$)**
        *   $P_1$ (走 $G_4$): $G_1 G_4$
        *   $P_2$ (走 $G_2G_3$): $G_1 G_2 G_3$

    *   **步骤2：找出所有独立回路及其增益 ($L_i$)**
        *   $L_1$: $-(G_2G_3)H$ (内部小回路，涉及 $G_2, G_3, H$)
        *   $L_2$: $-G_1G_2G_3$ (大回路，涉及 $G_1, G_2, G_3$ 和外部单位反馈)
        *   $L_3$: $-G_4H$ (内部小回路，涉及 $G_4, H$)
        *   $L_4$: $-G_1G_4$ (大回路，涉及 $G_1, G_4$ 和外部单位反馈)

    *   **步骤3：判断回路之间是否接触**
        *   本题中，所有四个回路两两之间都是接触的 (它们共享了求和点或 $G_1$ 或 $H$ 等)。
        *   因此，$\sum L_i L_j = 0$, $\sum L_i L_j L_k = 0$, ...

    *   **步骤4：计算系统特征式 ($\Delta$)**
        *   $\Delta = 1 - (L_1 + L_2 + L_3 + L_4)$
        *   $\Delta = 1 - (-(G_2G_3)H - G_1G_2G_3 - G_4H - G_1G_4)$
        *   $\Delta = 1 + G_2G_3H + G_1G_2G_3 + G_4H + G_1G_4$

    *   **步骤5：计算每条前向通路的余子式 ($\Delta_k$)**
        *   **对于 $P_1 = G_1G_4$:**
            *   去掉 $G_1$ 和 $G_4$ 及其相连的路径后，观察剩余部分是否构成回路。
            *   视频中指出，去掉 $P_1$ 后，不再形成任何完整的回路。因此 $\Delta_1 = 1$。
            *   **关键点理解：** 当 $P_1$ (包含 $G_1, G_4$) 被移除时，原先的 $L_1 (-(G_2G_3)H)$ 的一部分 ($H$) 可能还在，但 $G_2G_3$ 路径断了，无法形成闭环。原先的 $L_2 (-G_1G_2G_3)$ 由于 $G_1$ 被移除而断开。原先的 $L_3 (-G_4H)$ 由于 $G_4$ 被移除而断开。原先的 $L_4 (-G_1G_4)$ 由于 $G_1, G_4$ 被移除而断开。
        *   **对于 $P_2 = G_1G_2G_3$:**
            *   去掉 $G_1, G_2, G_3$ 及其相连的路径后，观察剩余部分是否构成回路。
            *   视频中指出，去掉 $P_2$ 后，也不再形成任何完整的回路。因此 $\Delta_2 = 1$。
            *   **关键点理解：** 当 $P_2$ (包含 $G_1, G_2, G_3$) 被移除时，原先的 $L_1 (-(G_2G_3)H)$ 由于 $G_2G_3$ 被移除而断开。原先的 $L_2 (-G_1G_2G_3)$ 由于 $G_1, G_2, G_3$ 被移除而断开。原先的 $L_3 (-G_4H)$ 的一部分 ($H$) 可能还在，但 $G_4$ 路径还在，但前向通路断了，无法形成闭环。原先的 $L_4 (-G_1G_4)$ 由于 $G_1$ 被移除而断开。
            *   **老师强调的难点：** 正确判断移除前向通路后是否还存在回路。如果错误地认为还存在回路，$\Delta_k$ 的计算就会出错。

    *   **步骤6：代入梅森增益公式**
        *   $P(s) = \frac{P_1 \Delta_1 + P_2 \Delta_2}{\Delta}$
        *   $P(s) = \frac{G_1G_4 \cdot 1 + G_1G_2G_3 \cdot 1}{1 + G_2G_3H + G_1G_2G_3 + G_4H + G_1G_4}$
        *   $P(s) = \frac{G_1G_4 + G_1G_2G_3}{1 + G_4H + G_2G_3H + G_1G_4 + G_1G_2G_3}$

3.  **最终结果校验：** 两种方法得到的结果应一致。
    (视频中老师给出的梅森公式最后结果的分母项顺序与化简法略有不同，但各项都存在，是等价的)

#### **三、总结**

1.  **核心方法：**
    *   **结构图化简法：** 适用于结构相对简单，易于分步化简的系统。关键是熟练掌握串联、并联、反馈的化简规则。
    *   **梅森增益公式：** 适用于复杂交联系统，不易直接化简的情况。关键是正确找出所有前向通路、所有独立回路，并准确判断它们之间的接触关系以及计算 $\Delta_k$。

2.  **关键点 (梅森公式)：**
    *   **前向通路 $P_k$ 的识别。**
    *   **独立回路 $L_i$ 的识别及其增益符号 (负反馈回路增益为负)。**
    *   **回路间不接触的判断。**
    *   **$\Delta_k$ 的计算：** 移除第 $k$ 条前向通路后，剩余部分形成的回路特征式。如果移除后无回路，则 $\Delta_k = 1$。这是本题的难点和易错点。

3.  **计算准确性：** 无论哪种方法，代数运算的准确性都非常重要。
