#### **一、 控制系统基本概念**

1.  **[[控制系统的分类]]:**
    *   **开环控制系统 (Open-Loop Control System):**
        *   **特点:** 输出对输入没有影响，没有反馈通路。控制作用是预先设定的，不受输出结果的影响。
        *   **优点:** 结构简单、成本低、易于实现。
        *   **缺点:** 抗干扰能力差、精度不高，无法修正偏差。
        *   **基本框图结构:**
            ```
            [输入信号] -> [控制器] -> [执行器] -> [被控对象] -> [输出信号]
            ```
        *   **例子:** 定时洗衣机、面包机、交通信号灯（固定时序）。

    *   **闭环控制系统 (Closed-Loop Control System) / 反馈控制系统:**
        *   **特点:** 输出量通过测量（检测）装置反馈回输入端，与输入信号（期望值）进行比较，产生偏差信号，控制器根据偏差信号产生控制作用，以减小或消除偏差。
        *   **优点:** 抗干扰能力强、控制精度高。
        *   **缺点:** 结构较复杂、成本较高、可能存在稳定性问题。
        *   **基本框图结构:**
            ```
                      +                      (扰动)
            [输入信号] -> (比较点) -> [控制器] -> [执行器] -> [被控对象] ----> [输出信号]
                       ^    -                                    |
                       |------------ [测量(检测)装置] -------------| (反馈信号)
            ```
        *   **例子:** 家用空调（恒温控制）、汽车巡航定速、人体体温调节、本题的炉温控制。

2.  ==**闭环控制的核心:** **负反馈**。通过比较期望值和实际值，根据偏差进行调节。==

#### **二、 炉温控制系统分析与框图绘制**

1.  **系统目标:** 将炉子内部的实际温度维持在指定的温度值。

2.  **系统类型:** 这是一个典型的**闭环（反馈）控制系统**。

3.  **识别主要环节及其功能:**

    *   **输入 (给定/期望值):** **指定温度** (Setpoint)。这是我们希望炉子达到的温度。
    *   **给定变换装置 (电位器/电阻电压装置):** 将“指定温度”这个设定值转换为一个电信号（电压 $U_R$）。 *(可以看作是给定环节的一部分)*
    *   **比较点 (Summing Point):** 将代表指定温度的电压 $U_R$ 与代表实际温度的反馈电压 $U_C$ 进行比较，产生**偏差电压 $\Delta U = U_R - U_C$**。
    *   **控制器 (电压功率放大器):** 接收偏差电压 $\Delta U$，将其放大，提供足够的能量去驱动执行器（电机）。
    *   **执行器 (电机 + 煤气开关阀):** 电机根据放大后的偏差信号转动，调节煤气开关阀的开度。
    *   **被控对象 (混合器 + 炉子):** 煤气与空气混合后在炉内燃烧，产生热量，改变炉子的温度。
    *   **输出 (被控量):** **实际炉温**。
    *   **测量/检测装置 (热电偶 + 放大器):** 热电偶测量实际炉温，并将其转换为电信号，经过放大器处理后得到反馈电压 $U_C$。
    *   **反馈通路:** 实际炉温 -> 热电偶 -> 放大器 -> 比较点。
    *   **扰动:** 放入炉内的**工件**会吸收或放出热量，影响炉温，是一种主要的扰动信号。

4.  **绘制系统框图:**

    ```mermaid
    graph LR
        subgraph 控制系统
            R(指定温度) --> Comp(比较点);
            Comp -- "ΔU = UR - UC" --> Amp1(电压功率放大器);
            Amp1 --> M(电机);
            M --> V(煤气阀);
            V --> Furnace(炉子);
            Furnace -- 实际温度 --> Output(输出: 实际温度);
            Output --> Thermocouple(热电偶 + 放大器);
            Thermocouple -- "反馈电压 UC" --> Comp;
        end
        Setpoint_Device(给定变换装置) -- "UR" --> Comp;
        R --> Setpoint_Device;

        Disturbance(扰动: 工件) --> Furnace;

     style Comp shape circle
    ```

    *   **说明:**
        *   指定温度 R 经过“给定变换装置”（如电位器）变成电压信号 $U_R$ 输入到比较点。
        *   实际温度（输出）由“热电偶+放大器”测量并转换为反馈电压 $U_C$。
        *   比较点计算偏差 $\Delta U = U_R - U_C$。
        *   偏差信号 $\Delta U$ 经“电压功率放大器”放大后驱动“电机”。
        *   “电机”控制“煤气阀”的开度，调节进入“炉子”的燃料量。
        *   “炉子”是被控对象，其输出为“实际温度”。
        *   “工件”作为“扰动”直接作用于“炉子”。

5.  **工作过程简述:**
    当实际温度低于指定温度时，$U_C < U_R$，偏差 $\Delta U > 0$，放大器输出信号使电机驱动阀门开大，增加燃料供应，炉温升高。反之，当实际温度高于指定温度时，$\Delta U < 0$，阀门关小，减少燃料，炉温下降。最终系统趋向于使 $\Delta U \approx 0$，即实际温度接近指定温度。扰动（如放入冷工件）会引起温度变化，反馈机制会使其重新调节。

6.  **例子与思考:**
    *   **改变设定温度:** 调节电位器，改变 $U_R$，系统将自动调节到新的平衡点。
    *   **放入冷工件:** 扰动使炉温下降，$U_C$ 减小，$\Delta U$ 增大，阀门开大，增加热量以补偿扰动。
    *   **热电偶老化/精度下降:** 测量不准，导致反馈信号 $U_C$ 错误，系统无法精确控制温度，可能产生==稳态误差==。
    *   **放大器增益调整:** 改变放大器增益会影响系统的==响应速度==和==稳定性==。

#### **三、 总结**

1.  **核心任务:** 理解物理系统的组成和工作原理，识别出控制系统中的各个基本环节（给定、比较、控制、执行、被控对象、测量反馈、扰动）。
2.  **关键步骤:**
    *   确定输入量（期望值）和输出量（被控量）。
    *   识别测量反馈环节和通路。
    *   识别控制和执行环节。
    *   识别被控对象。
    *   确定比较点和信号流向。
    *   考虑主要的扰动及其作用点。
3.  **框图意义:** 将复杂的物理系统抽象为信号传递关系的图形表示，是进行系统分析和设计的基础。
4.  **现阶段重点:** 理解闭环控制思想和基本组成，能够正确绘制简单系统的框图，为后续引入数学模型（传递函数）打下基础。

这个总结不仅涵盖了开环和闭环的基本概念，还详细分析了炉温控制系统的各个环节，并给出了绘制框图的步骤和结果，最后强调了框图的意义和现阶段的学习重点。
