#### **一、 预备知识 & 方法**

1.  **劳斯判据 (稳定性判据):**

    *   **基本原理:** 通过分析特征方程的系数，判断系统特征根在复平面上的位置，从而判断系统稳定性。
    *   **稳定条件:** 劳斯表中第一列元素均为正数。
    *   **劳斯表构建:**
        *   对于 n 阶特征方程：$a_0s^n + a_1s^{n-1} + ... + a_{n-1}s + a_n = 0$
        *   劳斯表格式：

            ```
            s^n   | a_0   a_2   a_4   ...
            s^{n-1} | a_1   a_3   a_5   ...
            s^{n-2} | b_1   b_2   b_3   ...
            s^{n-3} | c_1   c_2   c_3   ...
            ...    | ...   ...   ...
            s^0   | h_1
            ```
            其中：$b_1 = \frac{a_1a_2 - a_0a_3}{a_1}$, $b_2 = \frac{a_1a_4 - a_0a_6}{a_1}$, ...
            $c_1 = \frac{b_1a_3 - a_1b_2}{b_1}$, ...
            依次类推。

2.  **特征根实部与系统稳定性:**

    *   特征根实部均小于 0：系统稳定（特征根位于 s 平面左半平面）。
    *   特征根实部存在大于 0 的：系统不稳定。
    *   特征根实部均小于 -σ (σ > 0)：系统特征根均位于直线 s = -σ 左侧。

3.  **坐标平移 (关键):**

    *   **目的:** 将“特征根实部小于 -1” 转化为 “新特征方程的根实部小于 0”。
    *   **方法:**  令 $z = s + 1$ (或 $s = z - 1$)，将原特征方程中的 s 替换为 z - 1，得到新的特征方程。  (左加右减, 这里是左移)
    *   **原理:**  相当于将坐标轴向左平移 1 个单位。 原来要求根在 s = -1 左侧, 现在变为要求新方程的根在 z = 0 (虚轴) 左侧。

#### **二、 题目解答**

1.  **坐标平移:**

    *   令 $z = s + 1$, 则 $s = z - 1$。
    *   将 $s = z - 1$ 代入原特征方程：
        $(z-1+1)(z-1+1.5)(z-1+2) + K = 0$
        $z(z+0.5)(z+1) + K = 0$
        $z^3 + 1.5z^2 + 0.5z + K = 0$

2.  **构造劳斯表:**

    ```
    z^3 | 1     0.5
    z^2 | 1.5   K
    z^1 | (1.5*0.5 - 1*K) / 1.5 = (0.75 - K) / 1.5
    z^0 | K
    ```

3.  **应用劳斯判据:**

    *   为使新特征方程的根实部均小于 0 (对应原特征方程根实部小于 -1)，劳斯表第一列元素需均为正数：
        *   $1.5 > 0$ (已满足)
        *   $\frac{0.75 - K}{1.5} > 0 \Rightarrow 0.75 - K > 0 \Rightarrow K < 0.75$
        *   $K > 0$

4.  **确定 K 的范围:**

    *   综合上述条件：$0 < K < 0.75$

    因此，使系统所有闭环特征根的实部均小于 -1 的最大 K 值为 0.75。

**举例:**
*   **实部小于-2:** 如果要求实部小于-2, 则令$z = s+2$, 即$s = z-2$, 代入原特征方程.
*   **更高阶系统**: 对于更高阶的特征方程, 劳斯表的构建和判别方法相同.
* **不稳定情况**: 如果通过劳斯表发现第一列有负数, 则说明存在特征根实部大于-1(变换后大于0), 系统不稳定.
* **临界稳定**: 如果劳斯表中第一列出现0, 则可能存在纯虚根或重根, 需要进一步分析.

#### **三、总结**

1.  **核心知识点:** 劳斯判据、特征根实部与稳定性的关系、坐标平移。
2.  **解题步骤:** 坐标平移 (得到新特征方程) -> 构建劳斯表 -> 应用劳斯判据 (第一列元素均为正) -> 确定参数范围。
3.  **关键技巧:** 利用坐标平移将特征根实部约束问题转化为常规的稳定性判别问题。
4. **考试常见题型**: 此类题目是考试中的常见题型, 务必掌握。

这个总结清晰地阐述了利用劳斯判据解决特征根实部约束问题的知识点、方法和解题步骤，并强调了坐标平移的重要性。
