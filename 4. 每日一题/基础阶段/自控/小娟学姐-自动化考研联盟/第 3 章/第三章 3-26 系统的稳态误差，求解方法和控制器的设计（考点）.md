
### **题目：** 多输入下系统稳态误差分析与控制器设计 (稳态误差计算与控制器设计)
![[1. 控制科学备考/4. 每日一题/自控/清华鲁老师 26/Picture Set/3-26.png]]

#### **一、 预备知识 & 方法**

1.  **稳态误差定义:** 系统稳定后，输出量的实际值与期望值之间的偏差。

2.  **多输入系统稳态误差:** (线性系统) 各个输入分别作用下产生的稳态误差的叠加。

3.  **系统稳定性:** 存在稳态误差的前提是系统稳定。

4.  **稳态误差求解方法:**
    *   **静态误差系数法:** (适用于单位负反馈系统)
        *   [[控制系统的分类|开环传递函数]]：$G(s)H(s)$
        *   位置误差系数：$K_p = \lim_{s \to 0} G(s)H(s)$ (0 型系统)
        *   速度误差系数：$K_v = \lim_{s \to 0} sG(s)H(s)$ (I 型系统)
        *   加速度误差系数：$K_a = \lim_{s \to 0} s^2G(s)H(s)$ (II 型系统)
        *   稳态误差计算公式 (根据输入信号类型和系统类型选择):
            *   阶跃输入：$e_{ss} = \frac{R_0}{1+K_p}$ (0 型)
            *   斜坡/速度输入：$e_{ss} = \frac{R_0}{K_v}$ (I 型)
            *   加速度输入：$e_{ss} = \frac{R_0}{K_a}$ (II 型)
           *  $R_0$是输入的幅值.
    *   **终值定理法:** (通用方法，适用于各种反馈形式)
        *   误差定义：$E(s) = R(s) - C(s)$ (或根据具体框图确定)
        *   稳态误差：$e_{ss} = \lim_{s \to 0} sE(s)$
    * **注意:** 使用静态误差系数法,需要判断是否为单位负反馈.

5.  **[[Routh Criterion|劳斯判据]]:** 判断系统稳定性 (特征方程所有根均位于 s 平面左半平面)。

6.  **PD 控制器:**
    *   传递函数：$G_c(s) = 1 + T_d s$ (比例-微分)
    *   作用：
        *   不改变系统的自然频率 $\omega_n$。
        *   增大系统的阻尼比 $\zeta$。
        *   可以通过调整开环增益和微分时间常数，在减小斜坡输入稳态误差的同时，保证阶跃响应具有满意的动态性能。
    * **注意**: PD控制器会引入一个零点.

7. **[[Mason's Gain Formula|梅森公式]]:**用于求解信号流图的传递函数.

**数学公式：**

*   静态误差系数法 (单位负反馈, I 型系统, 斜坡输入): $e_{ss} = \frac{R_0}{K_v}$, 其中 $K_v = \lim_{s \to 0} sG(s)H(s)$
*   终值定理法: $e_{ss} = \lim_{s \to 0} sE(s)$
*   PD 控制器: $G_c(s) = 1 + T_d s$

#### **二、 题目解答**

**第一问：**

1.  **判稳:**
    *   求开环传递函数 (输入单独作用): $G(s) = \frac{2K}{s(s+3)(s+1)}$
    *   求[[传递函数的特征方程|特征方程]]: $D(s) = s^3 + 4s^2 + 3s + 2K = 0$
    *   劳斯表:
        ```
        s^3 | 1   3
        s^2 | 4   2K
        s^1 | (12-2K)/4  0
        s^0 | 2K
        ```
    *   稳定条件: $0 < K < 6$

2.  **输入引起的稳态误差 (静态误差系数法):**
    *   系统为 I 型系统。
    *   $K_v = \lim_{s \to 0} sG(s) = \frac{2K}{3}$
    *   $r(t) = 2t$, $e_{ssR} = \frac{2}{K_v} = \frac{3}{K}$
    *   ![[3-26--1(输入误差).png]]

3.  **扰动引起的稳态误差 (终值定理法):**
    *   求误差传递函数 (输入置零):
        $\Phi_e(s) = \frac{E(s)}{D(s)} = \frac{s(s+1)(s+3)}{s(s+1)(s+3) + 2K}$ (利用梅森公式或者方框图化简)
    *   $d(t) = e(t)$, $D(s) = \frac{1}{s}$
    *   $E(s) = \Phi_e(s)D(s) = \frac{s(s+1)(s+3)}{s[s(s+1)(s+3) + 2K]}\cdot \frac{1}{s}$
    *   $e_{ssD} = \lim_{s \to 0} sE(s) = \frac{1}{2K}$
    *   ![[3-26--1(扰动误差).png]]

4.  **总稳态误差:**
    $e_{ss} = e_{ssR} + e_{ssD} = \frac{3}{K} + \frac{1}{2K} = \frac{7}{2K}$

**第二问：**

1.  **分析:**
    *   $e_{ss} = \frac{7}{2K} < 0.1 \Rightarrow K > 35$
    *   $K > 35$ 与稳定条件 $0 < K < 6$ 矛盾。
    *   不能仅通过调节开环增益 K 满足要求。
2.  **控制器设计 (PD 控制器):**
      *   在系统的前向通道,扰动作用点之前，加入 PD 控制器 $G_c(s) = 1 + T_d s$。
3.  **加入 PD 控制器后重新分析:**
    *   新的开环传递函数: $G'(s) = (1+T_ds) \cdot \frac{2K}{s(s+1)(s+3)}$
    *   新的特征方程: $s^3 + 4s^2 + (3+2KT_d)s + 2K = 0$
    *   劳斯表:
        ```
          s^3 | 1      3+2KT_d
          s^2 | 4      2K
          s^1 | (12 + 8KT_d - 2K)/4   0
          s^0 | 2K
        ```

    *    稳定条件, 除了$K>0$外, 还需要满足$12 + 8KT_d - 2K >0$, 即$K(1-4T_d) < 6$.
4. **参数选择:**
    * 需要同时满足$K>35$和$K(1-4T_d) < 6$.
    * 选择合适的K和$T_d$,使得在满足稳态误差要求的同时,保证系统稳定.
    * $T_d$的范围：$T_d > \frac{K-6}{4K}$
     * 具体参数需要根据实际需求折中选择。

**举例:**

*   **不加控制器,只调整K:** 假设要求稳态误差小于0.05, 需要$K>70$, 依然超出稳定范围.
*    **PD控制器的作用:** 可以想象, PD控制器引入了一个零点, 改变了系统的根轨迹, 从而可以在增大K的同时, 保持系统稳定.
* **不同的控制器:** 除了PD控制器, 还可以考虑其他类型的控制器, 例如PI控制器, PID控制器等, 但需要根据具体情况进行分析和设计.

#### **三、总结**

1.  **核心知识点:** 稳态误差定义、多输入系统稳态误差计算、静态误差系数法、终值定理法、劳斯判据、PD 控制器作用。
2.  **解题步骤:** 判稳 -> 分别计算各输入引起的稳态误差 -> 叠加 -> 控制器设计 (若需要)。
3.  **技巧:** 熟练运用静态误差系数法和终值定理法，理解 PD 控制器的作用。
4.  **考试常见题型:** 稳态误差计算和简单的控制器设计是考试中的常见题型。
