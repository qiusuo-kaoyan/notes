### **题目：** 已知单位反馈系统开环传递函数，求系统持续等幅振荡时的参数 (劳斯判据和根轨迹与虚轴交点)

![[第三章 3-23 系统持续振荡时，劳斯表和虚轴交点的应用（考点）.png]]

#### **一、 预备知识 & 理论基础**

1.  **持续等幅振荡:**
    *   对应于二阶系统，阻尼比 $\zeta = 0$ 的情况。
    *   系统特征方程有一对纯虚根：$s_{1,2} = \pm j\omega_n$。
    *   在根轨迹上，表现为根轨迹与虚轴相交。

2.  **根轨迹与虚轴交点:**
    *   交点对应系统闭环特征方程的一对纯虚根。
    *   可用于确定系统产生持续等幅振荡时的参数。

3.  **求解方法:**

    *   **方法一 (代数法):**
        1.  求出闭环特征方程。
        2.  令 $s = j\omega$ 代入特征方程。
        3.  分离实部和虚部，并令它们分别等于零。
        4.  解方程组，求出参数。
    *   **方法二 (劳斯判据):**
        1.  求出闭环特征方程。
        2.  列出劳斯表。
        3.  利用劳斯表中出现全零行的情况 (特殊情况) 确定参数。
        4.  全零行对应的辅助方程的根即为纯虚根。

4. **劳斯判据特殊情况：**
    *   劳斯表中出现全零行，表明特征方程存在绝对值相同、符号相反的根（包括纯虚根和关于原点对称的实根）。
    *   全零行上一行的系数构成辅助方程。

**数学公式表达：**
* 单位反馈系统闭环特征方程： $1+G(s)H(s)=0$, 因为$H(s)=1$所以, $1+G(s) = 0$
* 二阶系统特征方程(标准形式): $s^2 + 2\zeta\omega_ns + \omega_n^2 = 0$

**举例:**

*   **二阶系统阻尼比为0:**  对于特征方程$s^2 + 4 = 0$, 解为$s = \pm 2j$, 对应频率为2的等幅振荡.
*   **劳斯表全零行:** 如果劳斯表某一行全为0, 则说明存在关于原点对称的根.

#### **二、 题目解答**

*   **方法一 (代数法):**

    1.  **闭环特征方程:**
        $1 + G(s) = 0 \Rightarrow s^3 + As^2 + 2s + 1 + K(s+1) = 0$
        $s^3 + As^2 + (2+K)s + (K+1) = 0$

    2.  **令 $s = j\omega = j2$:**
        $(j2)^3 + A(j2)^2 + (2+K)(j2) + (K+1) = 0$
        $-8j - 4A + 2j(2+K) + K + 1 = 0$

    3.  **分离实部和虚部:**
        *   实部: $-4A + K + 1 = 0$
        *   虚部: $-8 + 4 + 2K = 0$

    4.  **解方程组:**
        $K = 2$
        $A = 0.75$

	-   ![[3-23 法一.png]]

*   **方法二 (劳斯判据):**

    1.  **闭环特征方程:** (同上)
        $s^3 + As^2 + (2+K)s + (K+1) = 0$

    2.  **列劳斯表:**

        | $s^3$ | 1     | 2 + K |
        | ----- | ----- | ----- |
        | $s^2$ | A     | K + 1 |
        | $s^1$ |  $\frac{A(2+K)-(K+1)}{A}$     | 0     |
        | $s^0$ | K + 1 |       |

    3.  **全零行:** 令 $s^1$ 行的元素为 0：
         $A(2+K) - (K+1) = 0$
         则$A = \frac{K+1}{K+2}$

    4.  **辅助方程:**  $s^2$ 行的系数构成辅助方程：
        $As^2 + (K+1) = 0$
        $s^2 = -\frac{K+1}{A}$

    5. **利用已知频率 $\omega = 2$:**
    因为$s = \pm j\omega$, 所以$s^2 = -\omega^2 = -4$
    即: $-\frac{K+1}{A} = -4$
    则: $K+1 = 4A$

    6. **解方程:** 将$A = \frac{K+1}{K+2}$代入,并结合$K+1=4A$, 解得：
        $K = 2$
        $A = 0.75$

	-   ![[3-23 法二.png]]

**注意:** 视频中一开始使用的直接假设特征方程有根为-b的方法,然后进行多项式乘法,是不严谨的.

#### **三、 总结**

1.  **核心知识点:** 持续等幅振荡与纯虚根的关系、根轨迹与虚轴交点、劳斯判据特殊情况、代数法。
2.  **解题关键:**
    *   理解持续等幅振荡的条件。
    *   熟练掌握代数法和劳斯判据两种方法。
3.  **方法选择:**
    *   低阶系统 (如本题的三阶) 可用代数法，计算较简便。
    *   高阶系统推荐用劳斯判据，更系统、不易出错。
4. **考试常见题型:** 给出系统结构和振荡频率,求参数, 是常考题型.

该总结全面地概括了系统持续等幅振荡时的参数求解问题，重点强调了两种方法的原理、步骤和适用情况，并指出了视频讲解中的不严谨之处。 同时加入了对理论知识的补充和举例说明.
