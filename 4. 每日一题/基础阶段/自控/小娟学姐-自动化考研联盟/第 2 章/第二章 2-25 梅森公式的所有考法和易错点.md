### 题目：利用梅森公式求传递函数 (梅森公式综合应用题)

**已知：** 给定一个信号流图（如下图所示）。
![[2-25 美森公式所有考点和易错点.png]]
**求：**

1.  $Y_6/Y_1$ (输出节点/输入节点)
2.  $Y_2/Y_1$ (混合节点/输入节点)
3.  $Y_5/Y_2$ (混合节点/混合节点)

**信号流图：**

```
     A       C       E
Y1 --> Y2 --> Y3 --> Y4 --> Y5 --> Y6
     ^       ^       ^       ^
     |       |       |       |
     B       D       F       (无)
     |       |       |
     v       v       v
     Y2 <-- Y3 <-- Y4 <-- Y5
         -H      -I      -G
```

#### 一、预备知识

1.  **信号流图基本概念:**
    *   **输入节点 (源节点):** 只有输出支路的节点 (如 Y1)。
    *   **输出节点 (阱节点/紧节点):** 只有输入支路的节点 (如 Y6)。
    *   **混合节点:** 既有输入支路，又有输出支路的节点 (如 Y2, Y3, Y4, Y5)。
    *   **支路:** 连接两个节点的有向线段，表示信号的传递方向。
    *   **支路增益:** 支路上的数值或函数，表示信号通过该支路时的放大或衰减倍数。
    *   **前向通路:** 从输入节点到输出节点，且每个节点只通过一次的通路。
    *   **回路:** 起点和终点在同一节点，且信号通过每一节点不多于一次的闭合通路。
    *   **不接触回路:** 两个或多个回路之间没有任何公共节点。

2.  **梅森公式 (核心):**

    $P = \frac{1}{\Delta} \sum_{k=1}^{N} P_k \Delta_k$

    *   $P$: 系统总增益 (传递函数)。
    *   $N$: 前向通路总数。
    *   $P_k$: 第 k 条前向通路的增益。
    *   $\Delta$: 系统特征式。
        $\Delta = 1 - \sum L_a + \sum L_b L_c - \sum L_d L_e L_f + ...$
        *   $\sum L_a$: 所有单回路增益之和。
        *   $\sum L_b L_c$: 所有两两互不接触回路的增益乘积之和。
        *   $\sum L_d L_e L_f$: 所有三三互不接触回路的增益乘积之和。
        *   ... (通常最多考虑到三三互不接触)
    *   $\Delta_k$: 第 k 条前向通路的余子式 (特征式中除去与第 k 条前向通路接触的回路增益后的余项)。

3.  **混合节点变输出节点:** 从混合节点引出一条增益为 1 的支路，即可将混合节点变为输出节点。

4.  **梅森公式适用范围 (重要):**

    *   **适用:**
        *   输出节点 / 输入节点
        *   混合节点 / 输入节点 (需将混合节点变为输出节点)
    *   **不适用:**
        *   混合节点 / 混合节点 (需采用其他方法)

5.  **特征式 ($\Delta$) 的不变性:** 对于给定的信号流图，无论求哪个节点之间的传递函数，特征式 $\Delta$ 都是相同的。

#### 二、解题步骤 & 技巧

1.  **求特征式 Δ (公共步骤，只需计算一次):**

    *   找出所有单回路。
    *   找出所有两两互不接触回路。
    *   找出所有三三互不接触回路 (本题没有)。
    *   计算 Δ。

2.  **求各传递函数:**

    *   **$Y_6/Y_1$ (输出/输入):**
        *   找出所有前向通路。
        *   计算每条前向通路的增益 $P_k$。
        *   计算每条前向通路的余子式 $\Delta_k$。
        *   代入梅森公式计算。
    *   **$Y_2/Y_1$ (混合/输入):**
        *   从 Y2 引出一条增益为 1 的支路，将 Y2 变为输出节点。
        *   找出所有前向通路。
        *   计算每条前向通路的增益 $P_k$。
        *   计算每条前向通路的余子式 $\Delta_k$。
        *   代入梅森公式计算。
    *   **$Y_5/Y_2$ (混合/混合) - 两种方法:**
        *   **方法一 (代数法 - 推荐):**
            $Y_5/Y_2 = (Y_5/Y_1) / (Y_2/Y_1)$ (分别求 $Y_5/Y_1$ 和 $Y_2/Y_1$, Y1作为中间变量)
        *   **方法二 (梅森公式 - 需变换信号流图):**
            *   断开所有反馈到 Y2 的支路，使 Y2 成为输入节点。
            *   从 Y5 引出一条增益为 1 的支路，将 Y5 变为输出节点。
            *   重新计算特征式 Δ' (因为信号流图已改变)。
            *   找出所有前向通路。
            *   计算每条前向通路的增益 $P_k$。
            *   计算每条前向通路的余子式 $\Delta_k$。
            *   代入梅森公式计算。

#### 三、具体解答

1.  **求特征式 Δ:**

    *   **单回路:**
        *   $L_1 = -CG$
        *   $L_2 = -EH$
        *   $L_3 = -CDEI$
        *   $L_4 = -BEI$
    *   **两两互不接触回路:**
        *   $L_1 L_2 = CGEH$
    *   **三三互不接触回路:** 无
    *   **特征式:**
        $\Delta = 1 - (-CG - EH - CDEI - BEI) + CGEH = 1 + CG + EH + CDEI + BEI + CGEH$

2.  **求各传递函数:**

    *   **$Y_6/Y_1$:**
        *   $P_1 = ACDEF$, $\Delta_1 = 1$
        *   $P_2 = ABEF$, $\Delta_2 = 1$
        *   $Y_6/Y_1 = (ACDEF + ABEF) / \Delta$
    *   **$Y_2/Y_1$:**
        *   将 Y2 变为输出节点。
        *   $P_1 = A$, $\Delta_1 = 1 + EH$ (与回路 -EH 不接触)
        *   $Y_2/Y_1 = A(1 + EH) / \Delta$
    *   **$Y_5/Y_2$:**
        *   **方法一 (代数法):**
            *   $Y_5/Y_1$:
                *   $P_1 = ACDE$, $\Delta_1 = 1$
                *   $P_2 = ABE$, $\Delta_2 = 1$
                *   $Y_5/Y_1 = (ACDE + ABE) / \Delta$
            *   $Y_5/Y_2 = (Y_5/Y_1) / (Y_2/Y_1) = \frac{ACDE + ABE}{A(1+EH)} = \frac{CDE + BE}{1 + EH}$
        *   **方法二 (梅森公式):**
            *   断开反馈到 Y2 的支路，将 Y2 变为输入，Y5 变为输出。
            *   新特征式: $\Delta' = 1 - (-EH) = 1 + EH$
            *   $P_1 = CDE$, $\Delta_1 = 1$
            *   $P_2 = BE$, $\Delta_2 = 1$
            *   $Y_5/Y_2 = (CDE + BE) / (1 + EH)$

#### 四、易错点 & 注意事项

*   **回路和前向通路判断:** 确保每个节点只通过一次。
*   **混合节点/混合节点:** 不能直接用梅森公式，需采用代数法或变换信号流图。
* **信号流图变换:** 使用方法二求混合节点间传递函数时，变换信号流图后，特征式会改变。
*   **余子式:** 注意是“除去”与前向通路“接触”的回路，而不是“不接触”的。反向记忆:保留住互不接触的回路。
*   **梅森公式适用范围:** 牢记梅森公式只适用于输出节点/输入节点，或混合节点/输入节点 (需转换)。
*  **代数法更推荐**: 除非遇到原题或对信号流图变换非常熟练, 否则混合节点间传递函数求解更推荐代数法.

#### 五、补充说明

*   视频中提到，如果尝试用梅森公式法 (方法二) 求 $Y_6/Y_3$, $Y_5/Y_3$ 等，结果可能与代数法不同，原因可能与 Y2 连接的反馈较多有关，变换信号流图时需要更谨慎。因此，更推荐使用代数法。
