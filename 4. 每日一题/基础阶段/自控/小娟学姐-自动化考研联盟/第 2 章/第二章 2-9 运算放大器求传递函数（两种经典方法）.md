#自动控制原理 #每日一题 
### **题目：** 由[[运算放大器]]组成的有源网络，求传递函数 (运放电路传递函数计算)

***
#### **一、 预备知识 & 方法**

1.  **[[运算放大器]] (运放) 特性:**

    *   **理想运放假设:**
        *   输入阻抗无穷大 (虚断)
        *   开环放大倍数无穷大 (虚短)

    *   **虚短:** 运放正负输入端电压近似相等 (通常正端接地，负端也近似为 0V)。
        *   解释：由于开环放大倍数无穷大, 要使得输出为有限值, 则两输入端电压差必须无限接近于0.
    *   **虚断:** 运放输入电流近似为零。
       * 解释: 由于输入阻抗无穷大, 流入运放的电流无限接近于0.
   * ![[2-9 运放的虚短虚断.png]]

2.  **复阻抗法:**

    *   **基本思想:** 将电路中的电阻、电容、电感用复阻抗表示，然后利用电路分析方法 (如节点电压法、回路电流法) 列写方程，求解传递函数。

    *   **元件的复阻抗:**
        *   电阻: $Z_R = R$
        *   电容: $Z_C = \frac{1}{Cs}$
        *   电感: $Z_L = Ls$

    *   **阻抗的串并联:**
        *   串联: $Z_{eq} = Z_1 + Z_2 + ... + Z_n$
        *   并联: $\frac{1}{Z_{eq}} = \frac{1}{Z_1} + \frac{1}{Z_2} + ... + \frac{1}{Z_n}$ 或 $Z_{eq} = \frac{Z_1Z_2}{Z_1 + Z_2}$ (两个阻抗并联)

3. **反相放大器模型(重要)**
    *   如果运放电路可以等效为反相放大器模型(负反馈接在负输入端,正输入端接地),则可以直接利用公式:
    $\frac{U_o(s)}{U_i(s)} = -\frac{Z_f(s)}{Z_i(s)}$, 其中$Z_f(s)$是反馈回路的等效阻抗,$Z_i(s)$是输入回路的等效阻抗.

4.  **基尔霍夫电流定律 (KCL):** 在任一时刻，流入任一节点的电流的代数和为零。
    * 解释: 流入为正, 流出为负.

**数学公式表达：**

*   反相放大器传递函数：$\frac{U_o(s)}{U_i(s)} = -\frac{Z_f(s)}{Z_i(s)}$
*   KCL：$\sum_{k=1}^n i_k = 0$ (对节点 n)

#### **二、 解题方法 (两种)**

##### **方法一：直接复阻抗法 (利用反相放大器模型)**

1.  **等效电路:** 将运放电路等效为反相放大器模型，识别输入回路和反馈回路。
2.  **求等效阻抗:** 分别求出输入回路的等效阻抗 $Z_i(s)$ 和反馈回路的等效阻抗 $Z_f(s)$。
3.  **应用公式:** 利用公式 $\frac{U_o(s)}{U_i(s)} = -\frac{Z_f(s)}{Z_i(s)}$ 计算传递函数。

##### **方法二：节点电流法 (结合 KCL)**

1.  **确定节点:** 选择合适的节点 (通常选择运放负输入端和输出端)。
2.  **应用虚短虚断:** 运放负输入端电压为 0V，流入运放的电流为 0A。
3.  **列写 KCL 方程:** 对所选节点应用 KCL 定律，列写电流方程。
    * **注意电流方向:** 根据假定的电流方向，确定电流的正负号。(通常假设电流流入节点为正)
    * **电压降方向:** 电压降的方向与电流方向一致, 在列写方程的时候注意电位差. 例如,电流从A点流向B点, A点电位为$U_A$, B点电位为$U_B$, 则电压降为$U_A-U_B$.
4.  **消去中间变量:** 如果引入了中间变量 (如节点电压)，需要通过方程组消去。
5.  **求解传递函数:** 从方程组中解出输出电压与输入电压的比值，即传递函数。

#### **三、 题目解答 (两种方法)**
![[2-9 答案.png]]
##### **电路 1**
![[2-9 运放求传递函数.png]]
*   **方法一 (直接复阻抗法):**
    1.  $Z_i(s) = R_1$
    2.  $Z_f(s) = (R_2 + \frac{1}{Cs}) // R_3 = \frac{(R_2 + \frac{1}{Cs})R_3}{R_2 + \frac{1}{Cs} + R_3} = \frac{(R_2Cs + 1)R_3}{R_2Cs + 1 + R_3Cs} = \frac{R_3(R_2Cs+1)}{(R_2+R_3)Cs + 1}$
    3.  $\frac{U_c(s)}{U_r(s)} = -\frac{Z_f(s)}{Z_i(s)} = -\frac{R_3(R_2Cs + 1)}{R_1[(R_2+R_3)Cs + 1]} = -\frac{R_3}{R_1}\cdot\frac{R_2Cs+1}{(R_2+R_3)Cs+1}$

##### **电路 2**
![[2-9 运放求传递函数 2.png]]
*   **方法一 (直接复阻抗，较复杂):**
    1. 需要先计算反馈回路的等效阻抗,需要一定的电路分析能力.
    2. 不推荐.

*   **方法二 (节点电流法):**
    1.  **节点:** 运放负输入端 (0V), 输出端 ($U_c$), 引入中间节点 ($U$)。
    2.  **KCL 方程:**
        *   运放负输入端: $\frac{U_r - 0}{R_1} = \frac{0 - U}{R_2} \Rightarrow U = -\frac{R_2}{R_1}U_r$ (方程1)
        *   节点 U: $\frac{0 - U}{R_2} = \frac{U - 0}{\frac{1}{Cs}} + \frac{U - U_c}{R_3}$ (方程2)
    3.  **消去 U:** 将方程 1 代入方程 2, 并化简:
        $-\frac{U}{R_2} = CsU + \frac{U}{R_3} - \frac{U_c}{R_3} \Rightarrow (\frac{1}{R_2} + Cs + \frac{1}{R_3})U = -\frac{U_c}{R_3}$
        $(\frac{1}{R_2} + Cs + \frac{1}{R_3})(-\frac{R_2}{R_1}U_r) = -\frac{U_c}{R_3}$

        $\Rightarrow \frac{U_c}{U_r} = -\frac{R_3}{R_1}(\frac{1}{R_3}+\frac{1}{R_2}+Cs)R_2 = -\frac{R_3}{R_1}(1 + \frac{R_3}{R_2} + R_3Cs)R_2= - \frac{R_2+R_3+R_2R_3Cs}{R_1}$

**举例:**

* **虚短虚断的应用**: 在分析运放电路时,首先标出虚短点(通常为0V)和虚断(无电流流入).
* **KCL方程列写**: 对于一个节点,流入的电流之和等于流出的电流之和.
* **不同电路结构**: 对于不同的电路结构,需要灵活运用复阻抗法和节点电流法.
* **简化电路**: 可以通过戴维南等效等方法简化电路, 方便计算.

#### **四、总结**

1.  **核心知识点:** 运放虚短虚断、复阻抗法、KCL 定律、反相放大器模型。
2.  **解题方法:** 直接复阻抗法 (适用于反相放大器模型)、节点电流法 (结合 KCL)。
3.  **技巧:** 灵活运用虚短虚断简化分析，正确列写 KCL 方程，注意电流方向和电压降方向。
4. **易错点**: 符号错误, 电流方向和电压降方向弄反.
