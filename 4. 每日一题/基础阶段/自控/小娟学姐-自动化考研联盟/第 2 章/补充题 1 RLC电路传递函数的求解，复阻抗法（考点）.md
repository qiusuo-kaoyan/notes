
### **题目：** RLC 电路传递函数求解 (复阻抗法)
![[补充题 1 RLC 电路传递函数的求解, 复阻抗法.png]]
***
#### **一、 预备知识 & 方法**

1.  **复阻抗法:**
    *   **核心思想:** 将电路中的电阻、电容、电感都用复阻抗表示，然后像分析纯电阻电路一样分析 RLC 电路。
    *   **优点:** 简化计算，避免复杂的微分方程。

2.  **元件的复阻抗:**
    *   电阻 R:  $Z_R = R$
    *   电容 C:  $Z_C = \frac{1}{Cs}$
    *   电感 L:  $Z_L = Ls$  (本题未涉及)

3.  **串并联复阻抗:**
    *   **串联:** $Z_{串} = Z_1 + Z_2 + ... + Z_n$
    *   **并联:** $\frac{1}{Z_{并}} = \frac{1}{Z_1} + \frac{1}{Z_2} + ... + \frac{1}{Z_n}$  或  $Z_{并} = \frac{Z_1Z_2}{Z_1+Z_2}$(两个阻抗并联时)

4.  **分压原理 (复阻抗形式):**
    对于串联的复阻抗 $Z_1$ 和 $Z_2$，总电压为 $U$，则 $Z_1$ 两端电压为：
    $U_1 = \frac{Z_1}{Z_1 + Z_2}U$

#### **二、 解题思路 (关键)**

1.  **复阻抗表示:** 将电路中的电容用复阻抗表示：$C_1 \rightarrow \frac{1}{C_1s}$, $C_2 \rightarrow \frac{1}{C_2s}$。

2.  **阻抗分组 (巧妙之处):**
    *   将电路左侧部分 (包含 $R$, $C_1$, 和一个 $R$) 视为一个整体复阻抗 $Z_1$。
    *   将电路右侧的 $C_2$ 视为复阻抗 $Z_2$。
    *  **好处:** 这样，整个电路可以看作是 $Z_1$ 和 $Z_2$ 的串联，流过它们的电流相同，便于利用分压原理。

3.  **计算 $Z_1$ 和 $Z_2$:**
    *   $Z_2 = \frac{1}{C_2s}$
    *   $Z_1$:  先计算 R 与 C1 的串联: $R + \frac{1}{C_1s}$, 再计算与 R 的并联: $Z_1 = \frac{R(R+\frac{1}{C_1s})}{R + R + \frac{1}{C_1s}}$

4.  **表示 $U_i(s)$:** $U_i(s) = (Z_1 + Z_2)I(s)$

5.  **表示 $U_o(s)$:**
    *   $U_o(s)$ 为 R 和 $Z_2$ 两端的电压之和。
    *   R 两端电压: 利用分压原理, $\frac{R}{R+\frac{1}{C_1s}} \cdot Z_1 \cdot I(s)$
    *   $Z_2$ 两端电压: $Z_2 \cdot I(s)$
    *    $U_o(s) = [\frac{R}{R+\frac{1}{C_1s}} \cdot Z_1 + Z_2]I(s)$

6.  **求传递函数:**
    $\frac{U_o(s)}{U_i(s)} = \frac{[\frac{R}{R+\frac{1}{C_1s}} \cdot Z_1 + Z_2]I(s)}{(Z_1 + Z_2)I(s)} = \frac{\frac{R}{R+\frac{1}{C_1s}} \cdot Z_1 + Z_2}{Z_1 + Z_2}$

7.  **代入化简:** 将 $Z_1$ 和 $Z_2$ 的表达式代入，化简得到最终结果。

#### **三、 详细计算**

1.  $Z_1 = \frac{R(R+\frac{1}{C_1s})}{2R + \frac{1}{C_1s}} = \frac{R(RC_1s + 1)}{2RC_1s + 1} = \frac{R^2C_1s+R}{2RC_1s+1}$

2.  $Z_2 = \frac{1}{C_2s}$

3.  $U_i(s) = (Z_1+Z_2)I(s)$

4.   $U_o(s) = [\frac{R}{R + \frac{1}{C_1s}}Z_1 + Z_2]I(s) = [\frac{RC_1s}{RC_1s+1}Z_1+Z_2]I(s)$

5. $\frac{U_o(s)}{U_i(s)} = \frac{\frac{RC_1s}{RC_1s+1}Z_1+Z_2}{Z_1+Z_2}$

6.  代入$Z_1$和$Z_2$:
    $\frac{U_o(s)}{U_i(s)} = \frac{\frac{RC_1s}{RC_1s+1}\cdot\frac{R^2C_1s+R}{2RC_1s+1} + \frac{1}{C_2s}}{\frac{R^2C_1s+R}{2RC_1s+1} + \frac{1}{C_2s}}$

7. 化简（上下同乘$(RC_1s+1)(2RC_1s+1)C_2s$）:
化简后得：
$\frac{U_o(s)}{U_i(s)} = \frac{R^2C_1C_2s^2 + 2RC_1s + 1}{R^2C_1C_2s^2 + (2C_1 + C_2)Rs + 1}$

#### **四、总结**

1.  **核心知识点:** 复阻抗法、复阻抗的串并联、分压原理。
2.  **解题关键:** 将电路巧妙地分解为 $Z_1$ 和 $Z_2$ 两部分，利用串联电路电流相等的特点，简化计算。
3.  **优点:** 避免直接列写微分方程，使问题转化为代数运算。
4.  **与传统方法比较:** 比起胡寿松教材中的分流法，复阻抗法思路更清晰，计算更简便。
5. **举例**:
    * 如果电路中包含电感, 则电感的复阻抗为Ls, 分析方法相同.
    * 如果是其他形式的RLC电路, 也可以尝试用复阻抗法, 将电路分解为若干个复阻抗的串并联组合.

该总结详细地展示了用复阻抗法求解 RLC 电路传递函数的步骤和技巧，强调了复阻抗法的优势，以及与传统方法的比较。
