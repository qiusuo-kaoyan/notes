# **题目：** 非零初始条件下，已知传递函数和输入，求系统输出 (微分方程求解)
#自动控制原理 #每日一题 #basic_conception 

***
![[2-7 非零初始条件解微分方程.png]]
**已知：**

*   系统传递函数：$C(s)/R(s) = \frac{2}{s^2 + 3s + 2}$
*   初始条件：$c(0) = 1$, $\dot{c}(0) = 0$
*   输入信号：$r(t) = 1(t)$ (单位阶跃信号)

**求：** 系统输出 $c(t)$。

#### **一、 预备知识 & 方法**

1.  **传递函数的局限性:** 传递函数是在零初始条件下定义的，不能直接用于非零初始条件的情况。

2.  **求解步骤 (核心):**

    1.  **传递函数 → 微分方程:** 将传递函数转换为微分方程形式。
    2.  **考虑初始条件的拉氏变换:** 利用拉氏变换的微分定理，对微分方程的每一项进行拉氏变换，并将初始条件代入。
    3.  **求解 C(s):** 从变换后的代数方程中解出输出 $C(s)$ 的表达式。
    4.  **拉氏反变换:** 对 $C(s)$ 进行拉氏反变换，得到输出 $c(t)$。

3.  **拉氏变换的微分定理 (重点):**

    *   **一阶导数:**
        $\mathcal{L}\{\frac{df(t)}{dt}\} = sF(s) - f(0)$
        ,其中$F(s) = \mathcal{L}\{f(t)\}$
    *   **二阶导数:**
        $\mathcal{L}\{\frac{d^2f(t)}{dt^2}\} = s^2F(s) - sf(0) - \dot{f}(0)$
    *   **n 阶导数:**
    $\mathcal{L}\{\frac{d^nf(t)}{dt^n}\} = s^nF(s) - s^{n-1}f(0) - s^{n-2}\dot{f}(0) - ... - f^{(n-1)}(0)$

**微分定理的推导（分部积分法）**
* 根据拉普拉斯变换定义：$\mathcal{L}\{\frac{df(t)}{dt}\} = \int_0^\infty \frac{df(t)}{dt}e^{-st}dt$
* 利用分部积分：$\int_0^\infty \frac{df(t)}{dt}e^{-st}dt = \int_0^\infty e^{-st}df(t) = [e^{-st}f(t)]_0^\infty - \int_0^\infty f(t)de^{-st}$
* 计算得到结果: $= (0 - f(0)) - (-s)\int_0^\infty f(t)e^{-st}dt = sF(s) - f(0)$

**高阶导数微分定理的规律**
* $s$的阶次与$f(t)$的导数阶次之和为$n$ (总阶次)
* 系数中,$s$的阶次逐次递减,$f(0)$及其导数的阶次逐次递增.

1. **零初始条件下的简化:** 若 $f(0)$, $\dot{f}(0)$, ..., $f^{(n-1)}(0)$ 均为 0，则 $\mathcal{L}\{\frac{d^nf(t)}{dt^n}\} = s^nF(s)$。

2. **单位阶跃信号的拉氏变换:** $\mathcal{L}\{e(t)\} = \mathcal{L}\{1\} = \frac{1}{s}$

**数学公式表达：**

*   微分方程 (以二阶为例): $a_2\ddot{c}(t) + a_1\dot{c}(t) + a_0c(t) = b_1\dot{r}(t) + b_0r(t)$
*   考虑初始条件的拉氏变换:
    $a_2[s^2C(s) - sc(0) - \dot{c}(0)] + a_1[sC(s) - c(0)] + a_0C(s) = b_1[sR(s) - r(0)] + b_0R(s)$

#### **二、 题目解答**

1.  **传递函数 → 微分方程:**

    *   $C(s)(s^2 + 3s + 2) = 2R(s)$
    *   $s^2C(s) + 3sC(s) + 2C(s) = 2R(s)$
    *   $\ddot{c}(t) + 3\dot{c}(t) + 2c(t) = 2r(t)$  (利用微分定理反推)
    **错误做法**: 直接将$R(s) = 1/s$代入传递函数, 然后对$C(s)$进行拉氏反变换. 这是因为传递函数是在零初始条件下定义的.

2.  **考虑初始条件的拉氏变换:**

    *   $[s^2C(s) - sc(0) - \dot{c}(0)] + 3[sC(s) - c(0)] + 2C(s) = 2R(s)$
    *   代入初始条件 $c(0) = 1$, $\dot{c}(0) = 0$ 和 $R(s) = \frac{1}{s}$:
        $s^2C(s) - s - 0 + 3sC(s) - 3 + 2C(s) = \frac{2}{s}$

3.  **求解 C(s):**

    *   $(s^2 + 3s + 2)C(s) = \frac{2}{s} + s + 3$
    *   $C(s) = \frac{2 + s^2 + 3s}{s(s^2 + 3s + 2)} = \frac{s^2 + 3s + 2}{s(s+1)(s+2)} = \frac{1}{s} - \frac{4}{s+1} + \frac{4}{s+2}$ (部分分式展开，上一题已讲)
     更正：$C(s) = \frac{2 + s^2 + 3s}{s(s^2 + 3s + 2)} = \frac{s^2 + 3s + 2}{s(s+1)(s+2)} = \frac{1}{s} - \frac{4}{s+1} + \frac{4}{s+2}$应该更正为：
     $C(s) = \frac{s^2 + 3s + 2}{s(s+1)(s+2)} = \frac{1}{s} - \frac{4}{s+1} + \frac{4}{s+2}$ 部分分式分解后为：$C(s) = \frac{1}{s} - \frac{4}{s + 1} + \frac{4}{s+2}$

4.  **拉氏反变换:**

    *   $c(t) = 1 - 4e^{-t} + 4 e^{-2t}$
    更正后答案:$c(t) = 1 - 4e^{-t} + 4e^{-2t}$

**举例:**

*   **不同初始条件:** 如果初始条件变为$c(0)=0, \dot{c}(0) = 1$, 则拉氏变换后的方程会不同, 最终结果也不同.
*   **不同输入**: 如果输入变为$r(t) = t$, 则$R(s) = 1/s^2$, 求解过程类似, 但$C(s)$的表达式会不同.
*   **一阶系统**: 对于一阶系统, 只需要用到一阶导数的微分定理.
* **高阶系统**: 对于更高阶的系统, 需要用到更高阶导数的微分定理, 但思路是一样的.

#### **三、总结**

1.  **核心知识点:** 传递函数局限性、拉氏变换微分定理、非零初始条件下微分方程求解步骤。
2.  **解题关键:** 正确应用微分定理，将初始条件代入拉氏变换后的方程。
3.  **易错点:** 不能直接将 $R(s)$ 代入传递函数求解，必须先转换为微分方程，再考虑初始条件进行拉氏变换。
4. **考试常见题型:** 这类题目是考试中的常见题型，务必掌握。
