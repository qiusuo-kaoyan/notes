### **题目：** 求下列函数的拉普拉斯反变换 (拉氏反变换计算题)
#自动控制原理 #每日一题 #basic_conception 
***
**给定一组函数 $F(s)$，求其对应的拉普拉斯反变换 $f(t)$。**

![[2-1 拉氏反变换.png]]

#### **一、 预备知识 & 方法**

1.  **[[Laplace Transform|拉氏反变换定义]] (概念):** 将函数从复频域 ($s$ 域) 变换回时域 ($t$ 域)。

2.  **标准形式与部分分式:**
    *   **标准形式:** $F(s) = \frac{B(s)}{A(s)}$, 其中 $B(s)$ 和 $A(s)$ 为 $s$ 的多项式。
    *   **部分分式:** 将 $F(s)$ 展开为若干个简单分式之和，便于查表或利用常用公式进行反变换。

3.  **求解方法:**
    *   **留数法:** (理论基础，适用于各种情况，计算可能较繁琐)
        *   无重根情况
        *   有重根情况
    *   **部分分式法:** (实战常用，结合待定系数法)

4.  **部分分式法展开的两种情况:**
    *   **无重根:** $A(s) = 0$ 的根均为单根。
    *   **有重根:** $A(s) = 0$ 存在重根。

5.  **待定系数法:** (部分分式法中的一种技巧)
    *   将 $F(s)$ 展开为部分分式形式，但系数待定。
    *   通分，使分母与原 $F(s)$ 相同。
    *   比较分子，令对应项系数相等，建立方程组。
    *   解方程组，求出待定系数。

6. **解题步骤(非常重要)**
    1.  **判断是否为真分式：** 只有真分式才能直接应用部分分式法。如果不是真分式,需要先化为真分式+多项式的形式.
    2.  **找出 $A(s) = 0$ 的所有根：** 确定根的类型（单根、重根），以便选择合适的展开形式。
    3.  **展开为部分分式之和：** 根据根的类型，将 $F(s)$ 展开为部分分式。
    4.  **计算系数：** 利用待定系数法（通分比较）或留数法计算系数。
    5.  **拉氏反变换：** 查表或利用常用公式，求出每个部分分式的原函数，并将它们相加。

#### **二、 常用拉氏变换对 (必须熟记)**

| $f(t)$                 | $F(s)$                       | 备注                             |
| ---------------------- | ---------------------------- | -------------------------------- |
| $\delta(t)$            | 1                            | 单位冲激函数                     |
| $1$  (阶跃函数u(t))          | $\frac{1}{s}$                 |                                  |
| $t$                    | $\frac{1}{s^2}$               |                                  |
| $t^n$ ($n$ 为正整数)   | $\frac{n!}{s^{n+1}}$         |                                  |
| $e^{-at}$              | $\frac{1}{s+a}$               |                                  |
| $t e^{-at}$             | $\frac{1}{(s+a)^2}$          |                                  |
| $t^n e^{-at}$           | $\frac{n!}{(s+a)^{n+1}}$     |                                   |
| $\sin(\omega t)$        | $\frac{\omega}{s^2 + \omega^2}$ |                                  |
| $\cos(\omega t)$        | $\frac{s}{s^2 + \omega^2}$     |                                  |
| $e^{-at}\sin(\omega t)$ | $\frac{\omega}{(s+a)^2 + \omega^2}$ | 不一定常考,但要知道思路|
| $e^{-at}\cos(\omega t)$ | $\frac{s+a}{(s+a)^2 + \omega^2}$   | 不一定常考,但要知道思路|

**补充(未出现, 但很常用)**

*   $\frac{1}{s(s+a)} \rightarrow \frac{1-e^{-at}}{a}$
*   $\frac{1}{(s+a)(s+b)} \rightarrow \frac{e^{-at}-e^{-bt}}{b-a}$

**数学公式表达：**

*   真分式：$F(s) = \frac{B(s)}{A(s)}$, 且 $B(s)$ 的阶数小于 $A(s)$ 的阶数。
*   部分分式展开（无重根）：$F(s) = \frac{B(s)}{(s-s_1)(s-s_2)...(s-s_n)} = \frac{A_1}{s-s_1} + \frac{A_2}{s-s_2} + ... + \frac{A_n}{s-s_n}$
* 部分分式展开(有 $r$ 重根): $F(s) = \frac{B(s)}{(s-s_1)^r(s-s_2)...(s-s_n)} = \frac{A_{11}}{(s-s_1)^r} + \frac{A_{12}}{(s-s_1)^{r-1}} + ... + \frac{A_{1r}}{s-s_1} +  \frac{A_2}{s-s_2} + ... + \frac{A_n}{s-s_n}$
* 留数法公式( $r$ 重根):$A_{1k}=\frac{1}{(k-1)!}\lim_{s \to s_1}\frac{d^{k-1}}{ds^{k-1}}[(s-s_1)^rF(s)]$

#### **三、 题目解答 (关键步骤)**

*   **$F1(s)$:**
    *   **部分分式:** $\frac{s+1}{s(s^2+s+1)} = \frac{A}{s} + \frac{Bs+C}{s^2+s+1}$
    *   **待定系数法:** $A=1, B=-1, C=0$
    *   **化简:** $\frac{1}{s} - \frac{s}{s^2+s+1}$  (配方，凑成常用形式)
    *   **反变换:** 利用 $e^{-at}\sin(\omega t)$ 和 $e^{-at}\cos(\omega t)$ 对应的公式 (较复杂,不一定常考)

![[2-1-1 正余弦.png]]

*   **$F2(s)$:**
    *   **部分分式:** $\frac{6s+3}{s^2} = \frac{A}{s^2} + \frac{B}{s}$
    *   **待定系数/留数法** $A = 3$, $B=6$.
    *   **反变换:** $3t + 6$

![[2-1-2 重根(简单).png]]

*   **$F3(s)$:**
    *   **部分分式:**  $\frac{3(s^2+6s+16)}{(s+1)(s+2)^3} = \frac{A}{s+1} + \frac{B}{(s+2)^3} + \frac{C}{(s+2)^2} + \frac{D}{s+2}$
    *   **留数法：** $A, B, C, D$
        * $A = 3$
        * $B = 8$
        * $C = 3$
        * $D = -3$
    *   **反变换:** 利用 $t^n e^{-at}$ 对应的公式。

![[2-1-3 重根(复杂) 注意公式.png]]

*   **$F4(s)$:**
    *   **部分分式:** $\frac{1}{s^2(s^2+\omega^2)} = \frac{A}{s^2} + \frac{B}{s^2+\omega^2}$ (利用待定系数，避免交叉项)
    *   **待定系数:** $A = \frac{1}{\omega^2}, B = -\frac{1}{\omega^2}$
    *   **反变换:**  $\frac{1}{\omega^2}(t - \frac{1}{\omega}\sin(\omega t))$

![[2-1-4.png]]

**举例:**
* **待定系数法应用**: 对于$\frac{2s+1}{(s+1)(s+2)}$, 可以设为$\frac{A}{s+1}+\frac{B}{s+2}$, 通分后得到$2s+1 = A(s+2)+B(s+1)$, 对比系数可得: $A+B=2$, $2A+B=1$, 从而解出A和B.
* **不同分母形式**: 对于分母为$s^2+4$的形式, 可以联想到$\sin(2t)$和$\cos(2t)$的拉氏变换.
* **三重根**: 对于$\frac{1}{(s+1)^3}$, 可以直接利用公式得到$\frac{1}{2}t^2e^{-t}$.
* **真分式判断**: $\frac{s^3+1}{s^2+1}$不是真分式, 需要先化为$s + \frac{1-s}{s^2+1}$的形式.

#### **四、总结**

1.  **核心知识点：** 拉氏反变换定义、部分分式法（待定系数法、留数法）、常用拉氏变换对。
2.  **解题步骤：** 真分式判断 -> 找根 -> 部分分式展开 -> 计算系数 -> 反变换。
3.  **技巧：** 待定系数法（通分比较），熟记常用拉氏变换对。
