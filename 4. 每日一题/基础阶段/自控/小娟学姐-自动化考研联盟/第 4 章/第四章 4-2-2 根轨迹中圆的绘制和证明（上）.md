### **题目：** 概略绘制闭环根轨迹图 (根轨迹绘制，圆的证明)


![[1. 控制科学备考/4. 每日一题/自控/小娟学姐-自动化考研联盟/Picture Set/4-2.png]]

#### **一、 预备知识 & 方法**

1.  **根轨迹定义:** 开环传递函数中某个参数（通常是开环增益 $K_G$）从 0 到 $\infty$ 变化时，闭环特征方程的根在 s 平面上的变化轨迹。
2.  **根轨迹类型:**
    *   **180° 根轨迹 (常规根轨迹):** 负反馈系统。
    *   **0° 根轨迹:** 正反馈系统。
3.  **绘制根轨迹的基本步骤 (盟主独家 6+1 步骤):**

    1.  **判断根轨迹类型:** 180° 还是 0°。(**非常重要，影响整个绘图**)
    2.  **标出零、极点:** 确定零点、极点个数及位置。
    3.  **确定实轴上的根轨迹:** 根据零、极点分布判断。并由此判断是否存在分离点。
    4.  **求分离点、分离角:** 若存在分离点，则计算。
    5.  **求渐近线:** 计算渐近线与实轴的交点和夹角。
    6.   **求起始角、终止角、与虚轴的交点**：若存在复数极点或零点，则计算。与虚轴的交点非常重要, 不能省略。
    7.  **(验证):** 利用根之和、对称性进行验证。
    - ![[4-2 解题方法.png]]
4.  **实轴上的根轨迹:**
    *   **判断规则:** 实轴上某一段，若其右侧的开环零、极点个数之和为奇数，则该段为根轨迹。
    *  **起点与终点：**根轨迹起始于开环极点,终止于开环零点(包括无穷远零点)。

5.  **分离点:**
    *   **定义:** 根轨迹在实轴上相遇并分离的点。
    *   **计算公式:** $\sum_{i=1}^{m} \frac{1}{d-z_i} = \sum_{i=1}^{n} \frac{1}{d-p_i}$, 其中 d 为分离点，$z_i$ 为零点，$p_i$ 为极点。
6.  **渐近线:**
    *   **定义:** 当 s 趋于无穷远时，根轨迹趋近的直线。
    *   **与实轴交点:** $\sigma_a = \frac{\sum_{i=1}^{n} p_i - \sum_{i=1}^{m} z_i}{n-m}$, 其中 n 为极点数，m 为零点数。
    *   **与实轴夹角:** $\phi_a = \frac{(2k+1)\pi}{n-m}$ (180° 根轨迹) 或 $\frac{2k\pi}{n-m}$ (0° 根轨迹), 其中 k = 0, 1, 2, ..., n-m-1.
7. **复数极点的起始角:**
     * **定义:**从复数极点出发的根轨迹切线与正实轴的夹角。
     * **计算公式:**$\theta_{p_i}=180^\circ + \sum_{j=1}^m \angle(p_i - z_j) - \sum_{j=1,j\ne i}^n\angle(p_i - p_j)$
     * **口诀:**同减异加(同为极点则相减,一极点一零点则相加).
8. **根轨迹对称性:** 根轨迹关于实轴对称。
9. **分母有理化(技巧):**
    对于$\frac{1}{a+jb} + \frac{1}{a-jb}$形式,可以化简为$\frac{2a}{a^2+b^2}$

#### **二、 根轨迹为圆弧的证明（拓展）**

*   **适用情况:** 由两个极点（实数或复数）和一个有限零点组成的开环系统，且有限零点不在两个实数极点之间。
*   **结论:** 当开环增益从 $0$ 变化到 $∞$ 时，闭环根轨迹的复数部分是以有限零点为圆心，以有限零点到分离点的距离为半径的圆（或圆的一部分）。

**证明方法 (反证法):**

1.  **确定圆心、半径:** 根据绘制的根轨迹，确定圆弧的圆心和半径。
2.  **列出圆的方程:**  以实部为 $x$，虚部为 $y$，写出圆的方程。
3.  **求[[传递函数的特征方程|特征方程]]的根:** 根据闭环特征方程 ($1 + G(s)H(s) = 0$)，求出闭环特征根。
4.  **将根的实部、虚部代入圆的方程:** 将求出的根的实部和虚部分别代入圆的方程，验证等式是否成立。若成立，则证明根轨迹为圆弧。

![[4-2 拓展.png]]

**数学公式表达：**

*   圆的方程（假设圆心为 (a, 0), 半径为 r）：$(σ - a)^2 + ω^2 = r^2$
*   闭环特征方程：$1 + G(s)H(s) = 0$
* 开环传递函数化简:$\frac{K_G(s+2)}{(s+1)^2+2} = \frac{K_G(s+2)}{(s+1+j\sqrt{2})(s+1-j\sqrt{2})}$

#### **三、 题目解答**

1.  **判断类型:** 180° 根轨迹 (负反馈)。

2.  **零、极点:**
    *   零点: $z_1 = -2$
    *   极点: $p_1 = -1 + j\sqrt{2}$, $p_2 = -1 - j\sqrt{2}$
    * 零点数m=1; 极点数n=2

3.  **实轴上的根轨迹:** $(-\infty, -2]$

4.  **分离点:**
    *   判断存在分离点
    *   $\frac{1}{d+1+j\sqrt{2}} + \frac{1}{d+1-j\sqrt{2}} = \frac{1}{d+2}$
    *   化简: $d^2 + 4d + 1 = 0$
    *   求解: $d = -2 \pm \sqrt{3}$
    *   舍去: $d = -2 + \sqrt{3}$ (不在实轴根轨迹上)
    *   分离点: $d = -2 - \sqrt{3}$

5.  **渐近线:**
    *   $n - m = 1$, 只有一条渐近线。
    *   $\phi_a = \frac{(2k+1)\pi}{n-m} = \pi$ (负实轴)

6.  **起始角:**
    *   $\theta_{p_1} = 180^\circ + \angle(p_1 - z_1) - \angle(p_1 - p_2)$
        * $\angle(p_1-z_1)$：连接$p_1$和$z_1$,该向量与实轴正方向夹角.
        * $\angle(p_1 - p_2) = 90^\circ$
    * 计算:$\theta_{p_1} = 180^\circ + \arctan(\frac{\sqrt{2}}{-1-(-2)}) - 90^\circ = 144.7^\circ$
    *   $\theta_{p_2} = -\theta_{p_1} = -144.7^\circ$ (对称性)

7. **绘制根轨迹**

![[4-2 答案.png]]

**证明根轨迹为圆弧：**

1.  **圆心、半径:** 圆心 $(-2, 0)$, 半径 $\sqrt{3}$

2.  **圆的方程:** $(\sigma + 2)^2 + \omega^2 = 3$

3.  **特征方程的根:**
    *   $1 + \frac{K_G(s+2)}{(s+1)^2 + 2} = 0$
    *   $s^2 + 2s + 3 + K_G(s+2) = 0$
    *   $s^2 + (2+K_G)s + 3 + 2K_G = 0$
    *   $s = \frac{-(2+K_G) \pm \sqrt{(2+K_G)^2 - 4(3+2K_G)}}{2}$
      由于是复数部分, 开根号部分为负数, 提出负号,得到:
    *   $s = \frac{-(2+K_G)}{2} \pm j\frac{\sqrt{4(3+2K_G) - (2+K_G)^2}}{2}$
    * $\sigma = -\frac{2+K_G}{2}$
    * $\omega^2 =  \frac{4(3+2K_G) - (2+K_G)^2}{4}$

4.  **代入验证:** 将 $\sigma$ 和 $\omega^2$ 代入圆的方程 $(\sigma + 2)^2 + \omega^2 = 3$, 化简后等式成立。
