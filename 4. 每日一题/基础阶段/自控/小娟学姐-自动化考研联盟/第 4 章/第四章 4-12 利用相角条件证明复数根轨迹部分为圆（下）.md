
### **题目：** 利用相角条件证明根轨迹为圆 (根轨迹证明题)
![[4-12.png]]
**已知：** 单位负反馈系统的开环传递函数 $G(s)H(s) = \frac{1}{s(s+1)(s+2)}$

**证明：** 系统的复数根轨迹部分是以 (-2, j0) 为圆心，以 $\sqrt{2}$ 为半径的圆。

#### **一、 预备知识 & 方法**

1.  **根轨迹的相角条件:** 根轨迹上任意一点的相角满足：
    $\angle G(s)H(s) = (2k+1)180^\circ$, 其中 $k = 0, \pm1, \pm2, ...$

2.  **证明思路:**
    1.  令 $s = \sigma + j\omega$ (复数形式)。
    2.  将 $s$ 代入相角条件，得到关于 $\sigma$ 和 $\omega$ 的方程。
    3.  利用反正切函数的和角公式，化简方程。
    4.  将方程整理为圆的标准方程形式。

3.  **反正切函数的和角公式 (重要):**

    *   $\tan(A \pm B) = \frac{\tan A \pm \tan B}{1 \mp \tan A \tan B}$
    *   $\arctan A \pm \arctan B = \arctan \frac{A \pm B}{1 \mp AB}$ (本题主要用这个)

4.  **圆的标准方程:** $(x - a)^2 + (y - b)^2 = r^2$, 其中 $(a, b)$ 为圆心，$r$ 为半径。

5.  **零点和极点:**
    *   零点：使 $G(s)H(s)$ 的分子为 0 的点。
    *   极点：使 $G(s)H(s)$ 的分母为 0 的点。

6. **tan函数的性质**: $\tan(k\pi + \alpha) = \tan \alpha$, 其中k为整数.

**数学公式表达：**

*   相角条件: $\sum_{i=1}^{m} \angle(s - z_i) - \sum_{j=1}^{n} \angle(s - p_j) = (2k+1)180^\circ$, 其中 $z_i$ 为零点，$p_j$ 为极点。

#### **二、 证明步骤**

1.  **确定零点和极点:**
    *   零点：$z_1 = -2$
    *   极点：$p_1 = 0$, $p_2 = -1$

2.  **写出相角条件:**
    $\angle(s+2) - \angle(s) - \angle(s+1) = (2k+1)180^\circ$

3.  **令 $s = \sigma + j\omega$ 并代入:**
    $\angle(\sigma + j\omega + 2) - \angle(\sigma + j\omega) - \angle(\sigma + j\omega + 1) = (2k+1)180^\circ$

4.  **利用反正切函数表示相角:**
    $\arctan(\frac{\omega}{\sigma+2}) - \arctan(\frac{\omega}{\sigma}) - \arctan(\frac{\omega}{\sigma+1}) = (2k+1)180^\circ$

5.  **利用反正切函数的和角公式化简:**
    *   将 $\arctan(\frac{\omega}{\sigma+2}) - \arctan(\frac{\omega}{\sigma})$ 移到等式左边.
    *  令$A = \arctan(\frac{\omega}{\sigma+2}), B= \arctan(\frac{\omega}{\sigma})$
    * 利用公式得到: $\arctan\frac{\frac{\omega}{\sigma+2}-\frac{\omega}{\sigma}}{1+\frac{\omega^2}{\sigma(\sigma+2)}} =  \arctan(\frac{\omega}{\sigma+1}) + (2k+1)180^\circ$
    * 化简左边: $\arctan\frac{-2\omega}{\sigma(\sigma+2)+\omega^2}$
    * 对等式两边同时取tan:
      $\frac{-2\omega}{\sigma(\sigma+2)+\omega^2} = \tan[\arctan(\frac{\omega}{\sigma+1}) + (2k+1)180^\circ]$
    *   利用 $\tan(k\pi + \alpha) = \tan \alpha$:
         $\frac{-2\omega}{\sigma(\sigma+2)+\omega^2} = \tan[\arctan(\frac{\omega}{\sigma+1})] = \frac{\omega}{\sigma+1}$

6.  **化简并整理为圆的方程:**

    *   $-2\omega(\sigma+1) = \omega[\sigma(\sigma+2) + \omega^2]$ (交叉相乘)
    *   $-2(\sigma+1) = \sigma^2 + 2\sigma + \omega^2$ (两边消去 $\omega$, 假设 $\omega \ne 0$, 因为题目要求证的是复数根轨迹)
    *   $\sigma^2 + 4\sigma + \omega^2 + 2 = 0$
    *   $(\sigma + 2)^2 + \omega^2 = 2$ (配方)
    *   $(\sigma + 2)^2 + \omega^2 = (\sqrt{2})^2$

7.  **结论:** 方程为圆的标准方程，圆心为 $(-2, 0)$，半径为 $\sqrt{2}$。


#### **三、总结**

1.  **核心知识点:** 根轨迹相角条件、反正切函数和角公式、圆的标准方程。
2.  **证明步骤:** 写相角条件 -> 代入复数 -> 反正切表示 -> 和角公式化简 -> 整理为圆方程。
3. **易错点:** 反正切函数的和角公式, 化简过程中的正负号和消元.
4. **技巧:** 化简时, 注意利用$\tan(k\pi + \alpha) = \tan \alpha$.
