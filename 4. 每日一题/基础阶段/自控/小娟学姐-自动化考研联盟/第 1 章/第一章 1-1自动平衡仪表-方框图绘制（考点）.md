#自动控制原理 #每日一题

![[1-1 方框图绘制.png]]
#### **一、 方框图绘制**

1.  **确定输入输出：**
    *   输入：电压 $U_1$
    *   输出：指针位置（位移 $X$）

2.  **识别主要元件：**
    *   放大器
    *   电动机（$M$）
    *   电位器

3.  **分析系统类型：**
    *   根据==元件连接关系==，判断为闭环控制系统。

4.  **绘制方框图步骤：**
    *   从输入端 $U_1$ 开始，画出比较器（求和点）。
    *   连接放大器。
    *   连接电动机。
    *   ==**注意：** 电动机输出为角度（$θ$），需经过“机械传动”环节转换为位移 $X$（指针位置）。==
    *   从角度（$θ$）引出反馈，经过电位器，连接回比较器，形成闭环。
    *   补充参数：反馈电压 $U_{ba}$, 偏差电压 $ΔU$ (非必须,但加上更清晰).

**方框图如下：**

```
[U1] -->(+)--> [放大器] --> [电动机] --> [角度θ] --> [机械传动] --> [指针位置X]
         ^                                                 |
         |                                                 |
         |---- [电位器] <-- [反馈电压Uba] <------------------
```

数学公式表达(仅供参考,非必须):

*   偏差电压：  $ΔU = U1 - U{ba}$
*   电动机输入电压可以认为是 $ΔU$, 也可以在放大器后另设一个变量.
*   电动机输出角度 $θ$ 与 $ΔU$ 的关系，可用传递函数 $G(s)$ 表示（本题未涉及具体传递函数）。
*   机械传动环节，可简单表示为比例环节 $K$

**举例：**

*   **不同输入电压$U_1$:** 若 $U_1$ 为阶跃信号,代表目标位置突变; 若 $U_1$ 为斜坡信号,代表目标位置匀速变化. 系统都会调整指针位置 $X$ 跟踪 $U1$
*   **不同反馈系数:** 如果电位器的反馈系数可调,则会影响系统的灵敏度和稳定性. 反馈系数越大,系统越灵敏,但也越容易震荡.
*   **有扰动的情况:** 如果在指针位置 $X$ 处施加一个外力(扰动), 系统也会自动调节,使指针回到 $U_1$ 指定的位置.
*   **加入积分环节:**如果要求消除稳态误差,可以在放大器后加入积分环节,构成 $PI$ 控制器.

#### **二、 原理分析**

1.  **系统定性：** 闭环控制系统。

2.  **关键要素确定：**
    *   给定量：输入电压 $U_1$
    *   被控对象：电动机
    *   ==被控量：电动机输出角度 $θ$ (注意不是指针位置 $X$)==

3. **不同工况下的原理描述 （模板化）**
    可将原理描述分为几个情况,每个情况都按如下模板进行:

    **==模板：==**
    > 当【给定量】发生【变化】时，【被控对象】随之【动作】，并带动其他元件【动作】，直到【某个条件满足】，系统达到【新的平衡状态】，最终实现【结果】。

*   **情况一：初始状态**
    > 当输入电压 $U_1 = 0$ 时，反馈电压 $U_{ba} = 0$，偏差电压 $ΔU = 0$，记录仪指针位于零位，电动机不转动。

*   **情况二：达到工作点**
    > 当输入电压 $U_1$ 上升到某一工作点时，被控对象（电动机）随之转动，并带动电位器电刷和记录仪指针同步移动，直到反馈电压 $U_{ba} = U_1$（即 $ΔU = 0$），电动机停止转动，此时记录仪指针停留在 $U_1$ 对应的位置。

*    **情况三：工作点附近波动**
    >   当输入电压 $U_1$ 在工作点附近上下波动时，电动机受 $U_1$ 与 $U_{ba}$ 的偏差电压 $ΔU$ 的影响（经放大器放大后）而转动，带动仪表指针向 $U_1$ 要求的位置移动，直到 $U_{ba} = U_1$，系统达到新的平衡状态，电动机停止转动，实现了电压的自动跟踪记录。

**举例:**

*   **$U_1$ 阶跃变化**: 假设 $U_1$ 从 0 突然变为 5V, 电动机将转动,带动指针移动,直到 $U_b$ 也变为 5V,指针停在 5V 对应的刻度.
*   **U1缓慢变化**: 假设 $U_1$ 从 0V 缓慢上升,指针将跟随 $U_1$ 的上升而缓慢移动,始终保持 $U_{ba}=U1$.
*    **负载变化**: 假设在指针上施加一个阻力, 指针会偏离目标位置, 但由于$U_b$ 变化, 产生偏差电压, 电动机将再次转动,克服阻力,使指针回到目标位置.
*   **电位器损坏**: 如果电位器接触不良,反馈电压 $U_b$ 不稳定,会导致指针摆动或无法准确指示.

#### **三、 总结**

1.  **审题：** 找出输入、输出量。
2.  **元件识别：** 记住常见元件名称（放大器、电动机、电位器等）。
3.  **系统连接：** 在原理图上直接连线，再绘制方框图。
4.  **参数补充：** 补充关键参数（$U_{ba}$, $ΔU$）。
5.  **==原理分析：==**
    *   确定给定量、被控对象、被控量。
    *   分情况描述工作原理（套用模板）。
