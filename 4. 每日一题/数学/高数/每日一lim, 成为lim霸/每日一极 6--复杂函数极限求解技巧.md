#每日一题 #高等数学 
***
## 1. 题目分析
![[4. 每日一题/数学/高数/每日一lim, 成为lim霸/Picture Set/6.png]] ![[6.1--双拉格朗日.png]] ![[6.2--两角和差.png]]

求解极限：
$$
\lim_{x \to 0} \frac{\cos(\sin x) - e^{\cos x - 1}}{\tan^2 x - \sin^2 x}
$$

*   **难点：** 分子形式复杂，涉及复合函数和指数函数。
*   **直接[[知识枢纽：泰勒公式与等价无穷小|泰勒展开]]：** 计算量巨大，容易出错。
*   **突破口：** 分母形式相对简单，可以先估计分母阶数。

## 2. 方法一：先估计分母阶数，再拆分分子

### 2.1. 估计分母阶数

*   利用平方差公式：
    $$
    \tan^2 x - \sin^2 x = (\tan x - \sin x)(\tan x + \sin x)
    $$
*   **泰勒展开估计阶数：**
    *   $\tan x = x + \frac{x^3}{3} + o(x^3)$
    *   $\sin x = x - \frac{x^3}{6} + o(x^3)$
    *   $\tan x - \sin x = (x + \frac{x^3}{3}) - (x - \frac{x^3}{6}) + o(x^3) = \frac{x^3}{2} + o(x^3)$
    *   $\tan x + \sin x = (x + \frac{x^3}{3}) + (x - \frac{x^3}{6}) + o(x^3) = 2x + o(x)$
*   **分母阶数：**
    $$
    (\tan x - \sin x)(\tan x + \sin x) \sim (\frac{x^3}{2})(2x) = x^4
    $$
    分母是 $x$ 的 **4 阶**无穷小。

### 2.2. 利用洛必达降低阶数

*   **目的：** 4 阶无穷小不易直接配凑，先用[[洛必达法则原始笔记]]将分母降阶。
*   **应用洛必达：**
    $$
    \lim_{x \to 0} \frac{(\cos(\sin x) - e^{\cos x - 1})'}{(\tan^2 x - \sin^2 x)'} = \lim_{x \to 0} \frac{-\sin(\sin x)\cos x - e^{\cos x - 1}(-\sin x)}{4x^3} \quad \text{(这里视频中分母求导直接写了 } 4x^3 \text{，实际过程更复杂，但最终等价于 } 4x^3 \text{)}
    $$
    * 注: 视频中分母求导直接写了 $4x^3$ , 这是因为 $(\tan^2 x - \sin^2 x) \sim x^4$, 对 $x^4$ 求导即 $4x^3$. 实际计算时, 对 $(\tan x - \sin x)(\tan x + \sin x)$ 求导会比较复杂, 但利用等价无穷小可以简化.
    $$
    \lim_{x \to 0} \frac{-\sin(\sin x)\cos x + e^{\cos x - 1}\sin x}{4x^3}
    $$
*   **分母降为 3 阶。**

### 2.3. 拆分分子

*   **目标：** 将分子拆分成已知极限形式，且与分母的 3 阶匹配。
*   **寻找 3 阶无穷小：**
    *   已知 $\sin x - x \sim -\frac{x^3}{6}$
    *   $\sin x - \sin(\sin x) = (\sin x) - \sin(\sin x)$，令 $u = \sin x \sim x$
    *   则 $\sin x - \sin(\sin x) \sim u - \sin u \sim \frac{u^3}{6} \sim \frac{(\sin x)^3}{6} \sim \frac{x^3}{6}$
*   **配凑思路：**
    *   分子中有 $\sin(\sin x)$ 项，考虑配凑出 $\sin x - \sin(\sin x)$ 形式。
    *   由于原式有 $\cos x$ 因子，添加项也应包含 $\cos x$。
    *   **拆分：** 添加并减去 $\cos x \sin x$
        $$
        \text{分子} = -\sin(\sin x)\cos x + \cos x \sin x - \cos x \sin x + e^{\cos x - 1}\sin x
        $$
        $$
        = \cos x (\sin x - \sin(\sin x)) + \sin x (e^{\cos x - 1} - \cos x)
        $$

### 2.4. 分别求极限

*   **第一部分：**
    $$
    \lim_{x \to 0} \frac{\cos x (\sin x - \sin(\sin x))}{4x^3} = \lim_{x \to 0} \frac{1 \cdot (\frac{x^3}{6})}{4x^3} = \frac{1}{24}
    $$
    (利用了 $\lim_{x\to 0} \cos x = 1$ 和 $\sin x - \sin(\sin x) \sim \frac{x^3}{6}$)

*   **第二部分：**
    $$
    \lim_{x \to 0} \frac{\sin x (e^{\cos x - 1} - \cos x)}{4x^3} = \lim_{x \to 0} \frac{x (e^{\cos x - 1} - \cos x)}{4x^3} = \lim_{x \to 0} \frac{e^{\cos x - 1} - \cos x}{4x^2}
    $$
    (利用了 $\sin x \sim x$)
    *   **进一步拆分：** 添加并减去 1
        $$
        \lim_{x \to 0} \frac{(e^{\cos x - 1} - 1) - (\cos x - 1)}{4x^2}
        $$
    *   利用等价无穷小：
        *   $e^{\cos x - 1} - 1 \sim \cos x - 1$
        *   $\cos x - 1 \sim -\frac{x^2}{2}$
        $$
        \lim_{x \to 0} \frac{(\cos x - 1) - (\cos x - 1)}{4x^2} = \lim_{x \to 0} \frac{-\frac{x^2}{2} - (-\frac{x^2}{2})}{4x^2} = 0
        $$
        * 注: 视频中讲解时写成了 `e^(\cos x - 1) - \cos x - 1` 和 `1 - \cos x` 可能是口误或者笔误, 按照 $e^u - 1 \sim u$ 和 $1 - \cos x \sim \frac{x^2}{2}$ 推导, 最终两项会抵消.
        * **严格计算**
        $$
        \lim_{x \to 0} \frac{e^{\cos x - 1} - 1}{4x^2} = \lim_{x \to 0} \frac{\cos x - 1}{4x^2} = \lim_{x \to 0} \frac{-\frac{x^2}{2}}{4x^2} = -\frac{1}{8}
        $$
        $$
        \lim_{x \to 0} \frac{-(\cos x - 1)}{4x^2} = \lim_{x \to 0} \frac{-(-\frac{x^2}{2})}{4x^2} = \frac{1}{8}
        $$
        两部分相加为 0.

### 2.5. 最终结果

$$
\text{原极限} = \frac{1}{24} + 0 = \frac{1}{24}
$$

## 3. 方法二：直接拆分分子，利用拉格朗日中值定理

### 3.1. 拆分思路

*   分母阶数为 4 阶。
*   直接将分子拆分为两部分，使得每部分与分母阶数匹配。
*   **拆分：**
    $$
    \text{分子} = (\cos(\sin x) - \cos x) + (\cos x - e^{\cos x - 1})
    $$

### 3.2. 处理第一部分：$\cos(\sin x) - \cos x$

*   **利用拉格朗日中值定理：**
    令 $f(t) = \cos t$，则 $f(\sin x) - f(x) = f'(\xi)(\sin x - x)$，其中 $\xi$ 在 $x$ 和 $\sin x$ 之间。
    $$
    \cos(\sin x) - \cos x = (-\sin \xi)(\sin x - x)
    $$
*   **极限计算：**
    $$
    \lim_{x \to 0} \frac{\cos(\sin x) - \cos x}{\tan^2 x - \sin^2 x} = \lim_{x \to 0} \frac{(-\sin \xi)(\sin x - x)}{x^4}
    $$
    *   $\lim_{x \to 0} \xi = 0$ （由夹逼定理，$\xi$ 在 $x$ 和 $\sin x$ 之间）
    *   $\lim_{x \to 0} (-\sin \xi) = 0$
    *   $\sin x - x \sim -\frac{x^3}{6}$
    *   此路不通，因为 $(-\sin \xi)$ 趋于 0，导致结果为 0，阶数估计错误。

*   **修正：利用泰勒展开处理 $\cos(\sin x) - \cos x$**
    * $\cos u = 1 - \frac{u^2}{2!} + \frac{u^4}{4!} + o(u^4)$
    * $\cos(\sin x) = 1 - \frac{\sin^2 x}{2} + \frac{\sin^4 x}{24} + o(x^4)$
      $= 1 - \frac{(x - x^3/6 + ...)^2}{2} + \frac{(x - x^3/6 + ...)^4}{24} + o(x^4)$
      $= 1 - \frac{x^2 - x^4/3 + ...}{2} + \frac{x^4 + ...}{24} + o(x^4)$
      $= 1 - \frac{x^2}{2} + \frac{x^4}{6} + \frac{x^4}{24} + o(x^4) = 1 - \frac{x^2}{2} + \frac{5x^4}{24} + o(x^4)$
    * $\cos x = 1 - \frac{x^2}{2} + \frac{x^4}{24} + o(x^4)$
    * $\cos(\sin x) - \cos x = (\frac{5x^4}{24} - \frac{x^4}{24}) + o(x^4) = \frac{4x^4}{24} + o(x^4) = \frac{x^4}{6} + o(x^4)$
    * **第一部分极限：**
    $$
    \lim_{x \to 0} \frac{\cos(\sin x) - \cos x}{\tan^2 x - \sin^2 x} = \lim_{x \to 0} \frac{\frac{x^4}{6}}{x^4} = \frac{1}{6}
    $$

### 3.3. 处理第二部分：$\cos x - e^{\cos x - 1}$

*   **利用等价无穷小：**
    *   令 $u = \cos x - 1 \sim -\frac{x^2}{2}$
    *   $\cos x - e^{\cos x - 1} = (1 + (\cos x - 1)) - e^{\cos x - 1} = 1 + u - e^u$
    *   已知 $e^u - 1 - u \sim \frac{u^2}{2}$
    *   $1 + u - e^u = -(e^u - 1 - u) \sim -\frac{u^2}{2}$
    *   $\cos x - e^{\cos x - 1} \sim -\frac{(\cos x - 1)^2}{2} \sim -\frac{(-\frac{x^2}{2})^2}{2} = -\frac{x^4}{8}$
*   **第二部分极限：**
    $$
    \lim_{x \to 0} \frac{\cos x - e^{\cos x - 1}}{\tan^2 x - \sin^2 x} = \lim_{x \to 0} \frac{-\frac{x^4}{8}}{x^4} = -\frac{1}{8}
    $$

### 3.4. 最终结果

$$
\text{原极限} = \frac{1}{6} - \frac{1}{8} = \frac{4 - 3}{24} = \frac{1}{24}
$$

## 4. 总结与技巧

*   **先易后难：** 先处理相对简单的部分（如分母），估计阶数。
*   ==**阶数匹配：** 根据分母阶数，选择合适的拆分方法和等价无穷小/泰勒展开阶数。==
*   ==**洛必达降阶：** 当直接配凑困难时，可尝试用[[洛必达法则原始笔记]]降低阶数，简化问题。==
*   ==**灵活拆分：** 根据分子结构和已知等价无穷小/泰勒公式，灵活添加、减去项进行配凑。==
*   **常用等价无穷小/泰勒展开：**
    *   $\tan x - \sin x \sim \frac{x^3}{2}$
    *   $\tan x + \sin x \sim 2x$
    *   $\sin x - \sin(\sin x) \sim \frac{x^3}{6}$
    *   $e^u - 1 - u \sim \frac{u^2}{2}$
    *   $\cos(\sin x) - \cos x \sim \frac{x^4}{6}$
    *   $\cos x - e^{\cos x - 1} \sim -\frac{x^4}{8}$
*   **多种思路：** 同一道题可能有多种解法，选择计算量小、思路清晰的方法。
*   **拉格朗日中值定理** 在某些涉及函数差值的极限中有应用，但需注意阶数匹配问题。
