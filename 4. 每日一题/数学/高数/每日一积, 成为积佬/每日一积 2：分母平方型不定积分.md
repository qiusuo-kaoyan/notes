## 1. 题目

求不定积分：
$$
\int \frac{x + \sin x \cos x}{(\cos x - x \sin x)^2} dx
$$

## 2. 难点分析

*   **分式形式：** 分子分母关系不明显，难以直接凑微分。
*   **分母平方：** 提示了一种可能的解题思路。

## 3. 解题思路：==原函数猜测法== + 系数配凑

### 3.1. 核心思想

1.  **分母平方的提示：**
    *   分式求导会产生分母平方： $(\frac{f(x)}{g(x)})' = \frac{f'(x)g(x) - f(x)g'(x)}{g^2(x)}$
    *   猜测原函数形式：$\int \frac{f(x)}{g^2(x)} dx$ 的原函数可能是 $\frac{F(x)}{g(x)}$ 的形式。

2.  **原函数猜测：**
    *   设原函数为 $F(x) = \frac{f(x)}{\cos x - x \sin x}$。

3.  **求导验证：**
    *   对 $F(x)$ 求导，看能否与被积函数对应。
    $$
    F'(x) = \frac{f'(x)(\cos x - x \sin x) - f(x)(-\sin x - \sin x - x \cos x)}{(\cos x - x \sin x)^2}
    $$

4.  **系数配凑：**
    *   化简 $F'(x)$ 的分子，使其与被积函数的分子 $x + \sin x \cos x$ 对应。

5. **转化为微分方程**
    将求导验证的式子和原被积函数对应, 转化为求解 $f(x)$

### 3.2. 具体步骤

1.  **猜测原函数：**
    $$
    F(x) = \frac{f(x)}{\cos x - x \sin x}
    $$

2.  **求导：**
    $$
    F'(x) = \frac{f'(x)(\cos x - x \sin x) - f(x)(-2\sin x - x \cos x)}{(\cos x - x \sin x)^2}
    $$

3.  **化简分子：**
    $$
    \begin{aligned}
    & f'(x)(\cos x - x \sin x) - f(x)(-2\sin x - x \cos x) \\
    &= f'(x)\cos x - xf'(x)\sin x + 2f(x)\sin x + xf(x)\cos x  \\
    &= [xf(x)\cos x - xf'(x)\sin x] + [2f(x)\sin x + f'(x)\cos x]
    \end{aligned}
    $$

4.  **系数配凑：**
    *   目标：使化简后的分子等于 $x + \sin x \cos x$。
    *   观察：
        *   第一部分 $[xf(x)\cos x - xf'(x)\sin x]$ 对应 $x$。
        *   第二部分 $[2f(x)\sin x + f'(x)\cos x]$ 对应 $\sin x \cos x$。

5.  **构造方程组：**
    $$
    \begin{cases}
    xf(x)\cos x - xf'(x)\sin x = x \\
    2f(x)\sin x + f'(x)\cos x = \sin x \cos x
    \end{cases}
    $$
    化简：
    $$
    \begin{cases}
    f(x)\cos x - f'(x)\sin x = 1  \quad &(1)\\
    2f(x)\sin x + f'(x)\cos x = \sin x \cos x \quad &(2)
    \end{cases}
    $$

6.  **求解 $f(x)$：**
    *   **观察 (1) 式：** 联想到 $\cos^2 x + \sin^2 x = 1$。
    *   **猜测：** $f(x) = \cos x$
    *   **验证：**
        *   将 $f(x) = \cos x$ 代入 (1) 式，成立。
        *   将 $f(x) = \cos x$ 代入 (2) 式，成立。

7.  **得到原函数：**
    $$
    F(x) = \frac{\cos x}{\cos x - x \sin x} + C
    $$

## 4. 倒推验证（凑微分）

虽然通过猜测和验证得到了原函数，但考试时需要写出推导过程（凑微分）。

1.  **目标：** 将被积函数凑成 $\frac{d(\cos x - x \sin x)}{(\cos x - x \sin x)^2}$ 的形式。

2.  **将被积函数的分母化简:**
$$
\frac{x + \sin x \cos x}{(\cos x - x \sin x)^2}  = \frac{x + \sin x \cos x}{\cos^2 x (1 - x \tan x)^2}
$$

3. **凑微分：**
    *   将被积函数分子中的 $\cos^2 x$ 提到前面：
        $$
        \frac{x + \sin x \cos x}{\cos^2 x (1 - x \tan x)^2} = \frac{1}{\cos^2 x} \cdot \frac{x + \sin x \cos x}{(1 - x \tan x)^2}
        $$
        把$1/cos^2x$ 拿到分子:
        $$
           \frac{x \cdot (1/\cos^2 x) + \tan x }{(1 - x \tan x)^2}
        $$
        由于$(1-x \cdot tanx)' = -tanx - x(1/cos^2x)$

    *   凑微分：
       $$
       \begin{aligned}
       \int \frac{x + \sin x \cos x}{(\cos x - x \sin x)^2} dx &= \int \frac{1}{\cos^2 x} \cdot \frac{x + \sin x \cos x}{(1 - x \tan x)^2} dx \\
        &= \int \frac{x \sec^2 x + \tan x}{(1 - x \tan x)^2} dx \\
        &= -\int \frac{d(1 - x \tan x)}{(1 - x \tan x)^2} \\
        &= \frac{1}{1 - x \tan x} + C \\
        &= \frac{\cos x}{\cos x - x \sin x} + C
       \end{aligned}
       $$

## 5. 总结

*   ==**分母平方型不定积分：** 尝试猜测原函数为 $\frac{f(x)}{g(x)}$ 的形式，其中 $g(x)$ 为分母去除平方后的部分。==
*   **原函数猜测法：** 通过求导、化简、系数配凑，将问题转化为求解微分方程。
*   **系数配凑：** 寻找 $f(x)$，使求导后的分子与被积函数的分子对应。
*   **倒推验证：** 通过凑微分，将解题过程完整地呈现出来。
*    **核心思想**： 将求不定积分, 转化为求一个微分方程, 再利用观察法, 结合三角恒等式, 求解 $f(x)$。

## 6. 技巧

*   **化简：** 将被积函数化简为更易于观察和凑微分的形式。
*   **观察：** 观察分子分母的结构，寻找可能的联系和凑微分方向。
*   **尝试：** 大胆猜测原函数形式，并通过求导验证。
*   **联想：** 联想已知公式和恒等式（如 $\cos^2 x + \sin^2 x = 1$）。
