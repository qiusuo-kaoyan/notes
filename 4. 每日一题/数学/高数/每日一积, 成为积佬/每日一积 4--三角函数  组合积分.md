## 1. 问题引入

求解形如 $\int \frac{a\sin x + b\cos x}{c\sin x + d\cos x} dx$ 的不定积分。

**例题：**
$$
\int \frac{2\sin x + \cos x}{3\sin x - \cos x} dx
$$

## 2. 常见方法及其局限性

*   **万能代换法 (令 $t = \tan \frac{x}{2}$):**
    *   **优点：** 理论上可以将任何三角函数的有理式积分转化为有理函数积分，是通用方法。
    *   **缺点：** 往往导致计算量非常大，过程繁琐，不推荐作为首选方法，除非万不得已。

## 3. 待定系数法：一种更巧妙的思路

### 3.1. 核心思想

观察被积函数 $\frac{a\sin x + b\cos x}{c\sin x + d\cos x}$ 的结构，特别是分母 $D(x) = c\sin x + d\cos x$ 与其导数 $D'(x) = c\cos x - d\sin x$ 的关系。

==**基本假设：** 我们可以将被积函数的**分子**表示为**分母**及其**导数**的线性组合。==即：
$$
a\sin x + b\cos x = A (c\sin x + d\cos x) + B (c\cos x - d\sin x)
$$
其中 $A$ 和 $B$ 是待定常数。

### 3.2. 方法步骤

1.  **写出分母及其导数：**
    *   分母：$D(x) = c\sin x + d\cos x$
    *   分母的导数：$D'(x) = c\cos x - d\sin x$

2.  **建立待定系数方程：**
    令分子等于 $A \times (\text{分母}) + B \times (\text{分母的导数})$：
    $$
    a\sin x + b\cos x = A(c\sin x + d\cos x) + B(c\cos x - d\sin x)
    $$

3.  **合并同类项：**
    将右侧按 $\sin x$ 和 $\cos x$ 合并：
    $$
    a\sin x + b\cos x = (Ac - Bd)\sin x + (Ad + Bc)\cos x
    $$

4.  **求解待定系数：**
    通过比较等式两边 $\sin x$ 和 $\cos x$ 的系数，建立关于 $A$ 和 $B$ 的方程组：
    $$
    \begin{cases}
    Ac - Bd = a \\
    Ad + Bc = b
    \end{cases}
    $$
    解这个线性方程组，求出 $A$ 和 $B$ 的值。

5.  **改写并求解积分：**
    将被积函数代入原积分：
    $$
    \int \frac{A(c\sin x + d\cos x) + B(c\cos x - d\sin x)}{c\sin x + d\cos x} dx
    $$
    拆分为两部分：
    $$
    = \int \left( A \frac{c\sin x + d\cos x}{c\sin x + d\cos x} + B \frac{c\cos x - d\sin x}{c\sin x + d\cos x} \right) dx
    $$
    $$
    = \int A dx + \int B \frac{D'(x)}{D(x)} dx
    $$
    计算积分：
    $$
    = Ax + B \ln|D(x)| + C'
    $$
    即：
    $$
    = Ax + B \ln|c\sin x + d\cos x| + C'
    $$

### 3.3. 例题详解

求解 $\int \frac{2\sin x + \cos x}{3\sin x - \cos x} dx$

1.  **分母与导数：**
    *   $D(x) = 3\sin x - \cos x$
    *   $D'(x) = 3\cos x - (-\sin x) = 3\cos x + \sin x$

2.  **建立方程：**
    $$
    2\sin x + \cos x = A(3\sin x - \cos x) + B(3\cos x + \sin x)
    $$

3.  **合并同类项：**
    $$
    2\sin x + \cos x = (3A + B)\sin x + (-A + 3B)\cos x
    $$

4.  **求解系数：**
    $$
    \begin{cases}
    3A + B = 2 \\
    -A + 3B = 1
    \end{cases}
    $$
    *   由第二式得 $A = 3B - 1$。
    *   代入第一式：$3(3B - 1) + B = 2 \implies 9B - 3 + B = 2 \implies 10B = 5 \implies B = \frac{1}{2}$。
    *   代回求 $A$：$A = 3(\frac{1}{2}) - 1 = \frac{3}{2} - 1 = \frac{1}{2}$。
    *   所以，$A = \frac{1}{2}$, $B = \frac{1}{2}$。
    *(注：视频中讲解时设定的关系略有不同，导致中间系数选择看似不一致，但最终结果是等价的。这里按标准的待定系数法流程求解)*

5.  **改写并积分：**
    $$
    \int \frac{2\sin x + \cos x}{3\sin x - \cos x} dx = \int \left( \frac{1}{2} \frac{3\sin x - \cos x}{3\sin x - \cos x} + \frac{1}{2} \frac{3\cos x + \sin x}{3\sin x - \cos x} \right) dx
    $$
    $$
    = \int \frac{1}{2} dx + \int \frac{1}{2} \frac{d(3\sin x - \cos x)}{3\sin x - \cos x}
    $$
    $$
    = \frac{1}{2}x + \frac{1}{2}\ln|3\sin x - \cos x| + C'
    $$

## 4. 总结

*   ==待定系数法是求解形如 $\int \frac{a\sin x + b\cos x}{c\sin x + d\cos x} dx$ 不定积分的一种**高效技巧**。==
*   **核心**是将分子表达为分母与分母导数的线性组合。
*   相比万能代换法，此方法通常**计算量更小**，不易出错。
*   掌握此方法需要理解其背后的**构造思想**，并熟练求解线性方程组。
