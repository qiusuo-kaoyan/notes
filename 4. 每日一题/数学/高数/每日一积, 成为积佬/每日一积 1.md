#高等数学 #每日一题 
![[4. 每日一题/数学/高数/每日一积, 成为积佬/Picture Set/1.png]]

**难点：** 分母根号下复杂表达式的处理。

## 2. 解题思路

### 2.1. 核心思想：凑微分

*   观察分子 $\ln x$ 和分母中的 $x\ln x - x$。
*   发现 $(\ln x - x)' = \frac{1}{x} - 1 = \frac{1-x}{x}$, 和 $\ln x$ 没有直接关系.
*   发现 $(x\ln x - x)' = \ln x + x \cdot \frac{1}{x} - 1 = \ln x$

### 2.2. 凑微分

*   利用 $(x\ln x - x)' = \ln x$，构造微分形式：
    $$
    d(x\ln x - x) = \ln x \, dx
    $$

### 2.3. 换元

*   令 $m = x\ln x - x$。
*   原不定积分转化为：
    $$
    \int \frac{dm}{\sqrt{1 + m^2}}
    $$

## 3. 求解 $\int \frac{dm}{\sqrt{1 + m^2}}$

### 3.1. 方法一：直接写出原函数（推荐）

*   **公式：**
    $$
    \int \frac{dm}{\sqrt{1 + m^2}} = \ln|m + \sqrt{1 + m^2}| + C
    $$
    需要记住这个常见的不定积分形式。

### 3.2. 方法二：三角换元

1.  **换元：** 令 $m = \tan t$。
2.  **微分：** $dm = \sec^2 t \, dt = \frac{1}{\cos^2 t} dt$
3.  **代入：**
    $$
    \int \frac{dm}{\sqrt{1 + m^2}} = \int \frac{\frac{1}{\cos^2 t}}{\sqrt{1 + \tan^2 t}} dt = \int \frac{1}{\cos^2 t \cdot |\sec t|} dt = \int \frac{1}{\cos t} dt
    $$
    因为 $\sqrt{1+tan^2t} = \sqrt{sec^2t}=|sec\ t|$。 假设 $sect > 0$.
4.  **求解 $\int \frac{1}{\cos t} dt$：** (也需要记忆)
$$\int \frac{1}{\cos t} dt = ln|sect+tant|+C$$

5.  **反代换：**
    *   作直角三角形，令 $\tan \angle B = m = \frac{m}{1}$。
    *   则 $\sin t = \sin \angle B = \frac{m}{\sqrt{1+m^2}}$，$\cos t = \cos \angle B = \frac{1}{\sqrt{1+m^2}}$, $sec\ t = \frac{1}{cos\ t} = \sqrt{1+m^2}$.
    *   代入得到：
        $$
        \int \frac{1}{\cos t} dt = \ln|\sqrt{1+m^2} + m| + C
        $$

### 3.3. 方法三：$\int \frac{1}{\cos t} dt$ 的另一种求解

1.  **凑微分：**
    $$
    \int \frac{1}{\cos t} dt = \int \frac{\cos t}{\cos^2 t} dt = \int \frac{d(\sin t)}{1 - \sin^2 t}
    $$

2.  **裂项：**
    $$
    \int \frac{d(\sin t)}{1 - \sin^2 t} = \frac{1}{2} \int \left( \frac{1}{1 + \sin t} + \frac{1}{1 - \sin t} \right) d(\sin t)
    $$

3.  **求解：**
    $$
    \frac{1}{2} \int \left( \frac{1}{1 + \sin t} + \frac{1}{1 - \sin t} \right) d(\sin t) = \frac{1}{2} \ln \left| \frac{1 + \sin t}{1 - \sin t} \right| + C
    $$

4.  **化简：**
分子分母同时乘以 $(1+sint)$:
    $$
    \frac{1}{2} \ln \left| \frac{1 + \sin t}{1 - \sin t} \right| = \frac{1}{2} \ln \left| \frac{(1 + \sin t)^2}{1 - \sin^2 t} \right| = \frac{1}{2} \ln \left| \frac{(1 + \sin t)^2}{\cos^2 t} \right| = \ln \left| \frac{1 + \sin t}{\cos t} \right| +C =  \ln \left| \frac{1}{\cos t} + \tan t \right| + C
    $$

## 4. 回代

*   将 $m = x\ln x - x$ 代回：
    $$
    \int \frac{\ln x}{ \sqrt{1 + (x\ln x - x)^2}} dx = \ln|x\ln x - x + \sqrt{1 + (x\ln x - x)^2}| + C
    $$

## 5. 总结

*   **核心技巧：** 观察到 $\ln x$ 与 $x\ln x - x$ 的导数关系，利用凑微分和换元简化积分。
*   **关键公式：**
    *   $d(x\ln x - x) = \ln x \, dx$
    *   $\int \frac{dm}{\sqrt{1 + m^2}} = \ln|m + \sqrt{1 + m^2}| + C$
    *    $\int \frac{1}{cos\ t}dt = ln|sect+tant|+C$
*   **方法拓展：**
    *   三角换元
    *   分式裂项
* 这道题的难点在于第一步的凑微分, 需要多做题积累经验.
