#basic_conception 
***
### 一、定义与核心特征
指示代词（Demonstrative Pronouns）是一类具有指代功能的代词，核心作用是通过空间或时间上的距离差异来**标识或区分人或事物**，常译为“这个/些”“那个/些”。其核心特征包括：
1. **指定性**：与定冠词、人称代词类似，具有明确的指向性，可替代前文提及的名词。
2. **距离区分**：通过近指/远指概念构建指示关系。

> 示例：  
> - This is our newly developed product.（近指）  
> - I like these shoes more than those.（近指与远指对比）

### 二、形式与分类
#### （一）基础形式

| 距离   | 单数      | 复数       |
|--------|-----------|------------|
| **近指** | this（这） | these（这些） |
| **远指** | that（那） | those（那些） |


**功能区别**：  
- 单数（this/that）修饰单数名词（如 *this book*）  
- 复数（these/those）修饰复数名词（如 *those apples*）

#### （二）扩展分类（基于语言类型）
1. **空间距离**（核心分类）：  
   - 近指（this/these）：说话人身边或刚提及的事物（*This gift is for you.*）  
   - 远指（that/those）：远离说话人或对比性事物（*I don't like those games.*）  
2. **时间距离**：  
   - this/these 指现在或近期（*This week is busy.*）  
   - that/those 指过去或远期（*That year was difficult.*）  
3. **方言与特殊语言**：  
   - 临沂方言分近指（“这”）、远指（“那/捏”）、统指（“恁”）  
   - 霍佩赫语分远指（*qa*）、近指（*qo*）、中指（*qe*）、虚指（*qu*）  

### 三、语法功能与示例
指示代词在句子中可担任多种成分：  

| 语法角色   | 示例（英文→中文）                          | 引用     |
|------------|---------------------------------------------|----------|
| **主语**   | *This is the way to do it.*（这事儿就该这么做）    |  |
| **宾语**   | *I like this better than that.*（我喜欢这个多于那个） |  |
| **表语**   | *What I want is that.*（我想要的是那个）           |      |
| **定语**   | *Those books are hers.*（那些书是她的）          |  |
| **介词宾语** | *Don’t think too much of that.*（别想太多）     |        |


### 四、特殊用法与注意事项
1. **副词化用法**：  
   this/that 可修饰形容词或副词，相当于 *so*（如此/这么）：  
   > - *Is your daughter always this busy?*（你女儿总这么忙吗？）  
   > - *I’ve done only that much.*（我只做了这么多）  

2. **避免重复**：  
   - 用 **that/those** 代替前文提到的名词（*The weather here is colder than that in Shanghai.*）  
   - **this/these** 不可用于替代前文名词。  

3. **电话用语规则**：  
   - 自我介绍用 *This is...*（不可用 *That is...* 或 *I am...*）  
   - 询问对方用 *Is that...?*（不可用 *Are you...?*）  

4. **强调功能**：  
   将指示代词置于句首突出对象：  
   > - *This is the book I mentioned!*（这就是我提过的那本书！）  

### 五、易混淆点解析
1. **指代不明**：  
   滥用指示代词可能导致歧义（如“*Give that to me*”未明确“that”所指）。  
2. **中英文差异**：  
   古汉语中“是”可作指示代词（如“*是知也*”中“是”指“这”），但现代汉语无此用法。  
3. **限定词 vs 代词**：  
   - **指示代词**独立替代名词（*I want this.*）  
   - **指示限定词**修饰名词（*I want this book.*）  

### 六、总结
指示代词的核心价值在于**通过空间/时间距离实现精准指代**，其形式、功能及特殊规则需结合语境灵活运用。学习者应重点掌握近远指区分（this/that）、语法角色转换及电话用语等特殊场景，避免指代不明导致的沟通障碍。
