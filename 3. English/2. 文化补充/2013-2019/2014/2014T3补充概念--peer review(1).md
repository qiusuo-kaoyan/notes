# 一、什么是 peer review（同行评审）？

**同行评审**是学术界用来评估研究质量、可行性与可靠性的核心制度。简单说：当一篇论文、一个研究计划或申请经费的项目提交时，相关领域的“同行专家”会审阅它，给出独立意见，编辑或资助机构据此决定是否接受/资助/发表并提出修改建议。

用途上主要有三类场景：

- **期刊论文审稿（pre-publication journal review）**：决定是否把论文发表在学术期刊上。
    
- **经费/项目评审（grant review）**：决定哪些研究项目能拿到资助。
    
- **人事/晋升评审（hiring, tenure, promotion）**：学术职位、终身教职等的评估也常依赖同行评语或外部评审信。
    

# 二、同行评审通常是怎样运作的？（以期刊为例）

1. 作者投稿 → 编辑室做初筛（editorial triage，判断是否适合本刊）。
    
2. 编辑邀请 2–4 位领域内专家作为审稿人（reviewers）。
    
3. 审稿人读稿，写评语：包括原创性、方法是否严谨、结论是否合理、是否需要重大/小改动，或直接拒稿。
    
4. 编辑根据审稿意见作决定（接受、要求修改后再审、或拒绝）。
    
5. 作者按意见修改，可能再一轮审稿，直至接受或拒绝。
    

常见的**审稿模型**：

- **单盲（single-blind）**：审稿人知道作者，作者不知道审稿人（很常见）。
    
- **双盲（double-blind）**：双方互相匿名（有助减少名气偏见）。
    
- **开放式（open review）**：审稿人身份公开，审稿报告有时会公开发表。
    

# 三、同行评审的功能（为什么科学界重视它）

- **质量控制**：过滤明显错误、低质量或不严谨的研究。
    
- **改进研究**：审稿人的建议常能显著提升论文/项目的严谨性与可读性。
    
- **学术认证**：发表在同行评审期刊上，通常被视为对工作达到学界接受标准的一种“认证”。
    
- **资源分配的合理化**：经费评审把有限资金分配给被同行认为有价值的研究。
    

# 四、同行评审的局限与批评（重要）

文章中对“新奖可能会‘扭曲’同行评审主导的研究体系”的忧虑，正是基于这些局限：

1. **偏见问题**
    
    - 名望偏见（prestige bias）：名校/名人的稿子更容易被优先接纳或得到宽容。
        
    - 性别/种族/地域偏见：研究显示女性、某些少数族裔或非西方机构的稿件常遭更严格审查。
        
2. **保守性（conservatism）**
    
    - 审稿人倾向于认可主流/熟悉的方法，可能排斥非常新颖或跨学科的工作（容易扼杀“激进创新”）。
        
3. **可靠性与一致性低**
    
    - 不同审稿人对同一稿件的意见常常差异很大（互评一致性差）。
        
4. **时间与效率问题**
    
    - 审稿周期长（几个月到几年），科研传播与迭代慢。
        
5. **激励不足和利益冲突**
    
    - 审稿人多为无偿劳动，动力不足；作者与审稿人在学术圈中互为竞争者，存在利益冲突。
        
6. **并非“真理保证”**
    
    - 通过同行评审并不等于研究可复现；可重复性危机说明即便是同行评审的研究也会出问题。
        

# 五、同行评审 vs. 奖项（为什么文章说“they do not fund peer-reviewed research”）

文章指出那些互联网富豪设立的大额奖项 **“do not fund peer-reviewed research”**，意思并非字面“从未经过同行评审的人不能领奖”，而是在强调两点：

1. **功能不同**：
    
    - 期刊和资助机构通过同行评审来判断研究质量并决定是否发表/资助未来工作；而私人大奖通常是对**过去成就**的表彰（reward/recognition），奖金发给个人或团队，不是通过标准化的同行评审程序来决定未来研究的资助流向。
        
2. **选择机制不同**：
    
    - 私人奖可能由小范围评委会、创始人或名人提名/决定，评选标准和程序不一定透明或基于严格的同行评审标准。这可能**强化名望效应**、突出“个人英雄”而忽视团队和长期同行评议所确认的成果。
        

因此，文章担忧的是：如果巨额资金与公共资助脱钩，且奖励机制绕过或弱化同行审查，就可能 **改变学术激励结构**（研究者为赢取各种曝光或明星式奖励去追逐“亮眼”而非“稳健”的科研），或使“英雄科学家”神话更流行，而忽视大规模合作与长期的同行评审过程。

# 六、改进同行评审与替代/补充机制（学界正在尝试的）

为了修补同行评审的缺陷，学术界尝试了多种改良：

- **双盲审稿**：减少名声/机构偏见。
    
- **开放同行评审**：审稿意见公开，提高透明度与问责。
    
- **注册报告（registered reports）**：先评审研究方案，接受后再进行研究，减少发表偏见。
    
- **预印本（preprints）与后发表评议（post-publication review）**：研究先上预印本，社区评价并修正，速度快且更民主。
    
- **数据与代码共享、可重复性检查**：强化研究可验证性。
    
- **多样化审稿人池与培训**：减少偏见、提升审稿质量。
    

# 七、实战：遇到新闻或文章说“peer-reviewed”你怎么判别？（小清单）

1. **看来源**：研究是否发表在同行评审期刊？期刊主页会写明审稿流程。
    
2. **查期刊性质**：是否为学术期刊（有 editorial board / ISSN / DOI / 被索引）还是流行媒体？
    
3. **看稿件类型**：是研究论文（research article）还是 opinion/commentary/editorial（通常非实验性研究或未同行评审）？
    
4. **查预印本与最终版**：有时预印本先发布、后在同行评审期刊发表——注意两者版本差异。
    
5. **关注资助来源和评审透明度**：资助机构是否说明其评审程序（例如 NSF、NIH 通常有公开的同行评审机制）。
    
6. **对大奖**：查清楚奖项的评选流程：是公开同行评审、广泛提名，还是小范围内部决定？透明度越高，越可信。
    

# 八、回到文章：为什么同行评审对“奖励体系”很重要？

- **维持学术评判的公正性**：同行评审（尽管有缺点）提供了比单一富豪判断更具制度化的质量判断。
    
- **保护长期、协作型科研**：同行评审与经费评估往往支持长期团队合作研究；巨额私人奖若偏爱“耀眼个人”，可能弱化对长期基础研究/团队贡献的支持。
    
- **但也别忽视现实**：诺贝尔等传统奖项并非完美，且本身也源自私人资金；关键在于——**透明、公正、多元的评选流程**与对科研生态的平衡。
    

# 简短结论（便于记忆）

- **Peer review = 学术的“质量把关与认证”机制**，关键在于同行专家对研究的独立评估。
    
- 它有明显价值（质量、改进、认证），但也有偏见、保守与效率问题。
    
- 新兴私人大奖“绕开”或“弱化”同行评审机制，可能带来资金与声誉的短期好处，但也可能改变学术激励、放大名望偏见、强化“个人英雄”叙事。
    
- 最理想的情形是：私有资金与传统同行评审机制互补——奖金与环境应当透明、公正，并鼓励对可重复性、合作性研究的认可。
