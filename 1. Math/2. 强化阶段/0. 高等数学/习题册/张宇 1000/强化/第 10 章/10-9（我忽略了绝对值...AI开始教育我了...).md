> [!question] 题目描述
> 9. 设函数 $f(x)$ 在 $[0,1]$ 上连续，在 $(0,1)$ 内可导，且满足 $xf'(x) = f(x) + x^2$。已知曲线 $y=f(x)$ 与 $x=0, x=1, y=0$ 所围的图形 $S$ 面积为 2。求 $f(x)$ 的表达式，以及图形 $S$ 绕 $x$ 轴旋转一周所得旋转体的体积。

## 💡 战前推演 (Pre-battle Analysis)

> [!tip] 核心解题策略
> 求解面积时必须考虑被积函数的正负，对含参函数使用分类讨论法处理绝对值积分。

*   **初步解题思路**:
    1.  **识别题眼**:
        *   $xf'(x) = f(x) + x^2 \implies f'(x) - \frac{1}{x}f(x) = x$，这是一阶线性非齐次微分方程。
        *   “面积为2”是用来确定常数 $C$ 的关键条件，但**面积的定义是 $\int |f(x)|dx$，而非 $\int f(x)dx$**。
        *   $f(x)$ 解出来会含参，这意味着它的正负在积分区间 $[0,1]$ 内是不确定的。
    2.  **匹配战术**:
        *   使用一阶线性微分方程的通解公式求出 $f(x)$。
        *   **核心战术**：对面积积分 $\int_0^1 |f(x)| dx = 2$ 进行**分类讨论**。讨论的依据是 $f(x)$ 在 $[0,1]$ 上的零点位置。
        *   旋转体体积直接套公式 $V = \pi \int_0^1 [f(x)]^2 dx$。由于是平方，这里的正负号问题自然消失。
    3.  **制定步骤**:
        *   (a) 解微分方程，得到含 $C$ 的通解 $f(x)$。
        *   (b) 分析 $f(x)$ 的零点，发现它是一个含参的二次函数，其零点位置决定了讨论的边界。
        *   (c) 根据参数 $C$ 的不同取值范围，分情况去掉绝对值，建立关于 $C$ 的方程，解出所有可能的 $C$ 值。
        *   (d) 确定 $f(x)$ 的最终表达式（可能有多个）。
        *   (e) 将求出的 $f(x)$ 代入体积公式，计算出最终结果。

## ✍️ 我的原始解答 (My Original Attempt)

> [!caution] **【第一手犯罪现场资料】**
> 我完全没有考虑面积定义中被积函数需要加绝对值，错误地将“面积为2”理解为“定积分为2”。
> 
> 我的错误列式是：
> $$ \int_0^1 f(x) dx = 2 $$
> 解出 $f(x) = x^2+Cx$ 后，我直接计算：
> $$ \int_0^1 (x^2+Cx) dx = \left[ \frac{x^3}{3} + C\frac{x^2}{2} \right]_0^1 = \frac{1}{3} + \frac{C}{2} = 2 $$
> 这导致我算出了 $C = 10/3$。这个推理过程是完全错误的，因为它**默认了 $f(x)$ 在 $[0,1]$ 上恒为非负**，从而遗漏了另一种可能性，并且没有对解的适用范围进行验证。

## 🔥 反思熔炉 (The Reflective Crucible)

> [!abstract] 在这里，将“错误”这块“煤炭”，炼化为“能力”这颗“钻石”。

### **Part 1: 错误精准定位 (Pinpoint the Failure)**

> [!danger]- 一句话错误本质
> **我混淆了“函数的定积分”与“图形的面积”这两个概念的根本区别。面积是几何概念，恒为非负，其计算必须对函数取绝对值；而定积分是函数在区间上的“代数和”，可正可负。**

> [!example]- 错误诊断报告 (Automated Diagnostics)
> *   思路错误
> *   条件遗漏

### **Part 2: 正确路径重演 (Replay the Correct Path)**

1.  **求解通解 $f(x)$**:
    $f'(x) - \frac{1}{x}f(x) = x$
    $P(x) = -\frac{1}{x}$, $Q(x) = x$
    $f(x) = e^{-\int P(x)dx} \left( \int Q(x)e^{\int P(x)dx} dx + C \right)$
    $f(x) = e^{\ln x} \left( \int x e^{-\ln x} dx + C \right) = x \left( \int x \cdot \frac{1}{x} dx + C \right) = x(x+C) = x^2+Cx$

2.  **建立正确的面积积分方程**:
    $S = \int_0^1 |f(x)| dx = \int_0^1 |x^2 + Cx| dx = 2$

3.  **分类讨论**:
    被积函数 $f(x) = x(x+C)$ 的零点为 $x=0$ 和 $x=-C$。我们在 $[0,1]$ 区间内讨论。![[10-9过程.png]]

    *   **Case 1: $f(x)$ 在 $[0,1]$ 上非负**
        *   **条件**: 这要求另一个零点 $-C$ 不在 $(0,1)$ 的右侧，即 $-C \le 0 \implies C \ge 0$。
        *   **计算**: 此时 $|f(x)| = f(x)$。
          $\int_0^1 (x^2+Cx)dx = \frac{1}{3} + \frac{C}{2} = 2 \implies C = \frac{10}{3}$。
        *   **验证**: $C = 10/3 \ge 0$，满足条件。**所以 $f(x) = x^2 + \frac{10}{3}x$ 是一个解**。

    *   **Case 2: $f(x)$ 在 $[0,1]$ 上非正**
        *   **条件**: 这要求另一个零点 $-C$ 在 $1$ 的右侧或恰好为1，即 $-C \ge 1 \implies C \le -1$。
        *   **计算**: 此时 $|f(x)| = -f(x)$。
          $\int_0^1 -(x^2+Cx)dx = -(\frac{1}{3} + \frac{C}{2}) = 2 \implies \frac{1}{3} + \frac{C}{2} = -2 \implies C = -\frac{14}{3}$。
        *   **验证**: $C = -14/3 \approx -4.67 \le -1$，满足条件。**所以 $f(x) = x^2 - \frac{14}{3}x$ 是另一个解**。

    *   **Case 3: $f(x)$ 在 $[0,1]$ 上变号**
        *   **条件**: 这要求零点 $-C$ 位于区间 $(0,1)$ 内部，即 $0 < -C < 1 \implies -1 < C < 0$。
        *   **计算**: 此时积分需要分段。在 $[0, -C]$ 上 $f(x) \le 0$，在 $[-C, 1]$ 上 $f(x) \ge 0$。
          $\int_0^1 |f(x)| dx = \int_0^{-C} -(x^2+Cx)dx + \int_{-C}^1 (x^2+Cx)dx = 2$
          计算可得：$-\left[\frac{x^3}{3}+\frac{Cx^2}{2}\right]_0^{-C} + \left[\frac{x^3}{3}+\frac{Cx^2}{2}\right]_{-C}^1 = -\frac{C^3}{6} + \frac{C}{2} + \frac{1}{3} = 2$。
        *   **验证**: 令 $g(C) = -\frac{C^3}{6} + \frac{C}{2} + \frac{1}{3}$，研究它在 $(-1,0)$ 上的值域。
          $g'(C) = -\frac{C^2}{2} + \frac{1}{2}$。在 $(-1,0)$ 上，$C^2 < 1$，所以 $g'(C) > 0$，函数单调递增。
          $g(-1) = \frac{1}{6}-\frac{1}{2}+\frac{1}{3} = 0$。$g(0) = \frac{1}{3}$。
          所以 $g(C)$ 在此区间的值域为 $(0, 1/3]$，不可能等于 2。此情况无解。

4.  **最终结论**:
    *   $f(x)$ 有两个可能的表达式:
        $$ f_1(x) = x^2 + \frac{10}{3}x \quad \text{或} \quad f_2(x) = x^2 - \frac{14}{3}x $$
    *   计算体积 $V = \pi \int_0^1 (x^2+Cx)^2 dx = \pi \int_0^1 (x^4+2Cx^3+C^2x^2) dx = \pi(\frac{1}{5} + \frac{C}{2} + \frac{C^2}{3})$
        *   当 $C=10/3$: $V_1 = \pi(\frac{1}{5} + \frac{5}{3} + \frac{100}{27}) = \frac{752\pi}{135}$
        *   当 $C=-14/3$: $V_2 = \pi(\frac{1}{5} - \frac{7}{3} + \frac{196}{27}) = \frac{752\pi}{135}$
        (体积结果相同是由于平方项和一次项系数在计算中相互抵消影响)

### **Part 3: 由点及面-知识簇拓展**

#### 1.  **核心知识点回顾 (Core Concepts Review)**
*   **定积分的几何意义**: 定积分 $\int_a^b f(x)dx$ 的值是 $x$ 轴上方图形面积减去 $x$ 轴下方图形面积的**代数和**。
*   **面积的计算**: 由 $y=f(x), y=0, x=a, x=b$ 围成的面积是 $A = \int_a^b |f(x)|dx$，恒为非负。

#### 2.  **易混淆辨析 (Comparison of Confusing Concepts)**
| 概念 | 数学表达式 | 物理/几何意义 |
| :--- | :--- | :--- |
| **定积分** | $\int_a^b f(x)dx$ | 有向面积，可正可负 |
| **图形面积**| $\int_a^b \mid(f(x)\mid dx$| 实际几何面积，恒为非负 |

### **Part 4: 可迁移的启发 (The Transferable Takeaway)**

1.  **决策模型优化**: **凡是题目中出现“面积”二字，我的大脑必须立刻拉响一级警报，自动将积分表达式套上【绝对值】`|...|`。** 如果被积函数含有参数，必须立即启动**【分类讨论】**的子程序，根据参数对函数零点位置的影响来划分区间，去绝对值。

2.  **新技巧GET**: 巩固了通过分析含参二次函数零点分布，来对定积分进行分类讨论的完整流程。这是处理含参积分问题的核心通用技巧。

3.  **关联相似题**: 这道题的陷阱模式与所有“求面积”而函数图像可能穿过x轴的问题都一样。例如，求 $y=\sin x$ 在 $[0, 2\pi]$ 上与x轴围成的面积，如果直接积分为0就大错特错了，必须分成 $[0,\pi]$ 和 $[\pi, 2\pi]$ 两段计算 $\int_0^\pi \sin x dx + \int_\pi^{2\pi} | \sin x | dx$。本题只是将显式函数换成了含参的、需要自己求解的函数，增加了迷惑性。
