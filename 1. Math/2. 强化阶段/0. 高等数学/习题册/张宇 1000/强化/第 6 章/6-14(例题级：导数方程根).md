> [!question] 题目描述
> 设函数 $f(x)$ 在 $[0,1]$ 上连续且 $f(0)=f(1)=0$，在 $(0,1)$ 内二阶可导且 $f''(x)<0$。记 $M = \max_{0 \le x \le 1} f(x)$。
> (1) 证明对任意正整数 $n$，存在唯一的 $x_n \in (0,1)$，使得 $f'(x_n) = \frac{M}{n}$。
> (2) 对(1)中得到的 $\{x_n\}$，证明 $\lim_{n \to \infty} x_n$ 存在，且 $\lim_{n \to \infty} f(x_n) = M$。

---

### Path 1-A 罗尔定理辅助函数法
> [!tip]- **展开查看详细推演**
> **思考过程 (Mental Process):**
> **第一问的核心是证明 $f'(x_n) - \frac{M}{n} = 0$ 有唯一解。** 这是典型的“导数方程根”问题。看到这种形式，我的第一反应就是使用**罗尔定理**。罗尔定理的结论是 $F'(\xi)=0$，所以我需要构造一个辅助函数 $F(x)$，使得它的导数恰好是 $f'(x) - \frac{M}{n}$。
>
> 逆向思考，什么函数求导后是它？显然是 $F(x) = f(x) - \frac{M}{n}x$。
>
> 辅助函数已定，下一步就是罗尔定理的第二个关键：**找两个点 $a, b$，使得 $F(a)=F(b)$**。
> *   **初步试探**：我首先会检查题干给出的信息点，即区间端点0和1。计算 $F(0)=f(0)-0=0$。但 $F(1)=f(1)-\frac{M}{n} = -\frac{M}{n} \ne 0$。这条路走不通。
> *   **策略升级**：既然端点不行，我必须利用其他条件创造一个新的点。题干中最重要的信息是“$M$是最大值”。我必须将这个抽象的“最大值”**具体化**。
>     **【关键一步】**: 设最大值在 $x=c$ 处取得，即 $M=f(c)$。
>     现在我有了三个点：0, 1, c。我需要在这三个点以及它们的函数值 $F(0), F(1), F(c)$ 中寻找机会。
>     *   $F(0)=0$
>     *   $F(c) = f(c) - \frac{M}{n}c = M(1-\frac{c}{n}) > 0$ (因为 $c \in (0,1), n \ge 1$)
>     *   $F(1) = -\frac{M}{n} < 0$
>     我发现 $F(c)$ 和 $F(1)$ 异号！根据**连续函数零点定理**，在 $(c, 1)$ 之间必有一点 $d$，使得 $F(d)=0$。
>     现在我终于找到了两个使辅助函数值为0的点：$F(0)=0, F(d)=0$。罗尔定理的条件满足了！
>
> **唯一性证明**：通常通过函数的单调性来证明。$F'(x) = f'(x) - \frac{M}{n}$，其导数为 $F''(x) = f''(x)$。因为 $f''(x)<0$，所以 $F'(x)$ 严格单调递减，因此 $F'(x)=0$ 的根至多一个。
> **具体执行 (Execution):**
> 1.  **具体化最大值**: 设 $M = f(c)$，其中 $c \in (0,1)$。
> 2.  **构造辅助函数**: 令 $F(x) = f(x) - \frac{M}{n}x$。
> 3.  **寻找等值点**:
>     *   计算 $F(c) = f(c) - \frac{M}{n}c = M(1-\frac{c}{n}) > 0$。
>     *   计算 $F(1) = f(1) - \frac{M}{n} = -\frac{M}{n} < 0$。
>     *   由于 $F(x)$ 在 $[c,1]$ 上连续，且 $F(c) \cdot F(1) < 0$，由**零点定理**，存在 $d \in (c,1)$ 使得 $F(d)=0$。
> 4.  **应用罗尔定理**:
>     *   已知 $F(0)=0, F(d)=0$。
>     *   由于 $F(x)$ 在 $[0,d]$ 上连续，在 $(0,d)$ 内可导，由**罗尔定理**，存在 $x_n \in (0,d) \subset (0,1)$，使得 $F'(x_n)=0$。
>     *   即 $f'(x_n) - \frac{M}{n} = 0 \implies f'(x_n) = \frac{M}{n}$。存在性证毕。
> 5.  **证明唯一性**:
>     *   令 $h(x) = f'(x)$。因为 $h'(x) = f''(x) < 0$，所以 $f'(x)$ 在 $(0,1)$ 内严格单调递减。
>     *   因此，方程 $f'(x) = \frac{M}{n}$ 最多只有一个根。结合存在性，根是唯一的。

### Path 1-B 导数介值定理几何直观法
> [!tip]- **展开查看详细推演**
> **思考过程 (Mental Process):**
> 这条路径的起点是几何直观。$f'(x_n) = \frac{M}{n}$ 的左边是**切线斜率**，右边是一个**常数**。我要证明存在一个点的切线斜率恰好等于 $\frac{M}{n}$。**导数介值定理（达布定理）** 是解决这类问题的利器：如果我能找到两个点，其切线斜率一个大于 $\frac{M}{n}$，一个小于 $\frac{M}{n}$，那么在这两点之间，必有一点的切线斜率等于 $\frac{M}{n}$。
>
> *   **寻找一个小的斜率**：最容易想到的就是最大值点 $c$。因为 $f(x)$ 在 $c$ 点取到最大值，由**费马引理**，$f'(c)=0$。由于 $\frac{M}{n}>0$，我们已经找到了一个比它小的斜率：$f'(c) = 0 < \frac{M}{n}$。
> *   **寻找一个大的斜率**：我需要在 $(0,c)$ 区间内找一个点，其斜率要足够大。**拉格朗日中值定理**是构造导数值的绝佳工具。我对 $f(x)$ 在区间 $[0,c]$ 上使用拉格朗日中值定理：
>     存在 $\xi \in (0,c)$，使得 $f'(\xi) = \frac{f(c)-f(0)}{c-0} = \frac{M-0}{c} = \frac{M}{c}$。
>     因为 $c \in (0,1), n \ge 1$，所以 $c < 1 \le n \implies \frac{1}{c} > \frac{1}{n} \implies \frac{M}{c} > \frac{M}{n}$。
>     我成功找到了一个比它大的斜率：$f'(\xi) = \frac{M}{c} > \frac{M}{n}$。
>
> 现在，在 $[\xi, c]$ 区间上，我有了 $f'(\xi) > \frac{M}{n}$ 和 $f'(c) < \frac{M}{n}$。根据**导数介值定理**，必然存在 $x_n \in (\xi, c) \subset (0,1)$，使得 $f'(x_n) = \frac{M}{n}$。
> **唯一性证明**同方法一。
>
> **具体执行 (Execution):**
> 1.  **具体化最大值**: 设 $M=f(c)$，其中 $c \in (0,1)$。
> 2.  **寻找两个导数值**:
>     *   在最大值点 $c$，由**费马引理**，$f'(c)=0$。
>     *   在 $[0,c]$ 上应用**拉格朗日中值定理**，存在 $\xi \in (0,c)$，使得 $f'(\xi) = \frac{f(c)-f(0)}{c-0} = \frac{M}{c}$。
> 3.  **比较大小**:
>     *   因为 $c \in (0,1), n \ge 1$，所以 $c<n$，故 $\frac{M}{c} > \frac{M}{n}$。
>     *   我们有 $f'(\xi) > \frac{M}{n}$ 且 $f'(c) = 0 < \frac{M}{n}$。
> 4.  **应用导数介值定理**:
>     *   由于 $f'(x)$ 连续（因为二阶可导），在区间 $[\xi, c]$ 上应用介值定理（这里是导数介值定理，不要求连续），存在 $x_n \in (\xi, c) \subset (0,1)$，使得 $f'(x_n) = \frac{M}{n}$。存在性证毕。
> 5.  **唯一性证明**同方法一。

### Path 2-A 方程分析法
> [!tip]- **展开查看详细推演**
> **思考过程 (Mental Process):**
> 要证明数列 $\{x_n\}$ 极限存在，标准方法是**单调有界准则**。有界性是显然的，因为 $x_n \in (0,1)$。关键在于证明其**单调性**。
>
> 我没有 $x_n$ 的显式表达式，但我有两个关于 $x_n$ 和 $x_{n+1}$ 的方程：
> *   $f'(x_n) = \frac{M}{n}$
> *   $f'(x_{n+1}) = \frac{M}{n+1}$
>
> 比较 $x_n$ 和 $x_{n+1}$ 的大小，等价于比较它们的函数值 $f'(x_n)$ 和 $f'(x_{n+1})$ 的大小，再利用 $f'(x)$ 的单调性。
>
> *   **比较函数值**：显然，$\frac{M}{n} > \frac{M}{n+1}$，所以 $f'(x_n) > f'(x_{n+1})$。
> *   **利用单调性**：我们已经证明了 $f'(x)$ 是**严格单调递减**的。对于一个减函数，自变量越大，函数值越小。所以从 $f'(x_n) > f'(x_{n+1})$ 可以推出 $x_n < x_{n+1}$。
>
> 这样，我们就证明了 $\{x_n\}$ 是一个单调递增的有界数列，因此极限必然存在。
> **具体执行 (Execution):**
> 1.  **证明单调性**:
>     *   已知 $f'(x_n) = \frac{M}{n}$ 和 $f'(x_{n+1}) = \frac{M}{n+1}$。
>     *   由于 $n < n+1$，所以 $\frac{M}{n} > \frac{M}{n+1}$，即 $f'(x_n) > f'(x_{n+1})$。
>     *   又因 $f''(x)<0$，故 $f'(x)$ 严格单调递减。
>     *   因此，从 $f'(x_n) > f'(x_{n+1})$ 可得 $x_n < x_{n+1}$。数列 $\{x_n\}$ 单调递增。
> 2.  **证明有界性**:
>     *   由第一问知，$x_n \in (0,1)$，所以 $\{x_n\}$ 有上界1。
> 3.  **应用单调有界准则**:
>     *   $\{x_n\}$ 是单调递增的有上界数列，故 $\lim_{n \to \infty} x_n$ 存在。设其为 $A$。
> 4.  **求极限值**:
>     *   对等式 $f'(x_n) = \frac{M}{n}$ 两边取极限：$\lim_{n \to \infty} f'(x_n) = \lim_{n \to \infty} \frac{M}{n} = 0$。
>     *   因为 $f'(x)$ 连续，所以 $\lim_{n \to \infty} f'(x_n) = f'(\lim_{n \to \infty} x_n) = f'(A)$。
>     *   故 $f'(A)=0$。由于 $f'(x)$ 严格单调递减，等于0的点是唯一的，即最大值点 $c$。所以 $A=c$。
>     *   因此，$\lim_{n \to \infty} x_n = c$。
> 5.  **计算第二个极限**:
>     *   $\lim_{n \to \infty} f(x_n) = f(\lim_{n \to \infty} x_n)$ (因为 $f(x)$ 连续) $= f(c) = M$。

### Path 2-B 反函数分析法
> [!tip]- **展开查看详细推演**
> **思考过程 (Mental Process):**
> 这是一个更抽象但更直接的方法。由于 $f'(x)$ 是严格单调函数，它必然存在**反函数**。我们记这个反函数为 $g(y)$。
>
> 从 $f'(x_n) = \frac{M}{n}$，我们可以直接“解出”$x_n$：$x_n = g(\frac{M}{n})$。
>
> 这样，我们就得到了 $x_n$ 的“准”表达式。现在判断 $\{x_n\}$ 的单调性，就等价于判断函数 $g(y)$ 的单调性，以及数列 $\{\frac{M}{n}\}$ 的单调性。
>
> *   **反函数单调性**：原函数 $f'(x)$ 是严格单调递减的，其反函数 $g(y)$ 也必定是**严格单调递减**的。
> *   **数列单调性**：数列 $\{\frac{M}{n}\}$ 显然是严格单调递减的。
> *   **复合函数单调性**：一个减函数作用在一个减数列上，得到的新数列是**单调递增**的。
>
> 这样，我们同样证明了 $\{x_n\}$ 单调递增。后续步骤与方法A完全相同。
> **具体执行 (Execution):**
> 1.  **证明反函数存在**:
>     *   由于 $f'(x)$ 在 $(0,1)$ 上严格单调递减，其反函数 $g(y)$ 存在，且也是严格单调递减的。
> 2.  **写出 $x_n$ 表达式**:
>     *   由 $f'(x_n) = \frac{M}{n}$，得 $x_n = g(\frac{M}{n})$。
> 3.  **证明单调性**:
>     *   数列 $u_n = \frac{M}{n}$ 是严格单调递减的。
>     *   函数 $g(u)$ 是严格单调递减的。
>     *   所以复合数列 $x_n = g(u_n)$ 是严格单调递增的。
> 4.  **后续步骤**同方法A。
