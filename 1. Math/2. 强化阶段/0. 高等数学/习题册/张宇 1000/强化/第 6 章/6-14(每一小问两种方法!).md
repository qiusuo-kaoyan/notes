# 题目 :: 来源：`$= dv.current().source` | 题号：`$= dv.current().id` | 难度：`$= "⭐".repeat(dv.current().difficulty)` | 重要性：`$= "🔥".repeat(dv.current().importance)`

> [!question] 题目描述
> 设函数 $f(x)$ 在 $[0,1]$ 上连续且 $f(0)=f(1)=0$，在 $(0,1)$ 内二阶可导且 $f''(x)<0$。记 $M = \max_{0 \le x \le 1}\{f(x)\}$。
> (1) 证明：对任意正整数 $n$，存在唯一的 $x_n \in (0,1)$，使得 $f'(x_n) = \frac{M}{n}$；
> (2) 对(1)中得到的 $\{x_n\}$，证明：$\lim_{n \to \infty} x_n$ 存在，且 $\lim_{n \to \infty} f(x_n) = M$。

---

## 💡 知识点与思路分析 (解题前的深度准备)

这道题的每一步都建立在对条件的深刻理解之上。

1.  **榨干条件**:
    *   **$f(0)=f(1)=0$ 与 $f''(x)<0$**: 这两个条件共同描绘了一幅**严格上凸的、过 $(0,0)$ 和 $(1,0)$ 的拱形桥**的图像。
    *   **最大值 $M$**: 对于这座“拱桥”，其最大值 $M$ 必然在**开区间 $(0,1)$** 内部的唯一一点 $c$ 处取得。我们必须立即“具体化”这个 $M$：存在唯一的 $c \in (0,1)$，使得 $f(c)=M$。
    *   **费马引理**: 由于 $c$ 是内部极值点且函数可导，必有 **$f'(c)=0$**。
    *   **导数单调性**: $f''(x)<0$ 意味着 **$f'(x)$ 严格单调递减**。这是证明**唯一性**和**数列单调性**的核心武器。
2.  **战略规划**:
    *   **第一问 (存在唯一性)**:
        *   **唯一性**: 由 $f'(x)$ 的严格单调性直接得出。
        *   **存在性**: 核心是证明 $M/n$ 这个值在导函数 $f'(x)$ 的值域内。
            *   **路径1 (罗尔定理)**: 构造一个新函数，其导数包含 $f'(x)-M/n$，然后证明这个新函数有两个等值点。
            *   **路径2 (介值定理)**: 直接考察导函数 $f'(x)$，找到两个点，其导数值分别大于和小于 $M/n$，然后用**导数介值定理**。
    *   **第二问 (数列极限)**:
        *   **收敛性**: 证明数列单调有界。有界性已知，关键是**单调性**。
        *   **极限值**: 对关系式 $f'(x_n)=M/n$ 取极限，解出 $\lim x_n$，再代入 $f(x)$ 求极限。

## ✍️ 第一问：存在唯一性证明

### **方法一：构造函数 + 零点定理 + 罗尔定理 (前一总结的方法)**

![[6-14-1.png]]![[6-14-2.png]]

这种方法更“程序化”，是解决此类存在性问题的通用模板。
*   **唯一性**: 因 $f'(x)$ 严格单调递减，故方程 $f'(x)=C$ 最多一解。
*   **存在性**:
    1.  构造辅助函数 $\varphi(x) = f(x) - \frac{M}{n}x$。
    2.  找到三个关键点的函数值：
        *   $\varphi(0)=0$
        *   $\varphi(c) = f(c) - \frac{M}{n}c = M(1-\frac{c}{n}) > 0$
        *   $\varphi(1) = f(1) - \frac{M}{n} = -\frac{M}{n} < 0$
    3.  由 $\varphi(c)>0$ 和 $\varphi(1)<0$，据**零点定理**，存在 $\eta \in (c,1)$ 使 $\varphi(\eta)=0$。
    4.  在 $[0,\eta]$ 上，我们有 $\varphi(0)=\varphi(\eta)=0$。据**罗尔定理**，存在 $x_n \in (0,\eta)$ 使 $\varphi'(x_n)=0$，即 $f'(x_n)=M/n$。

### **方法二：导数介值定理直接打击 (本视频老师的方法)**

这种方法更依赖于对函数图像和导数几何意义的深刻洞察，更为直接。

1.  **唯一性**: 同方法一，由 $f'(x)$ 的**严格单调递减**性保证。

2.  **存在性 (Darboux's Theorem)**:
    *   **目标**: 证明 $M/n$ 介于导函数 $f'(x)$ 在某两点的值之间。
    *   **寻找两个关键的导数值**:
        *   **值一 (最小值)**: 我们知道在最大值点 $c$ 处，**$f'(c)=0$**。
        *   **值二 (一个正值)**: 我们需要找到一个导数值比 $M/n$ 大。我们可以利用**拉格朗日中值定理**来创造一个导数值。对函数 $f(x)$ 在区间 $[0,c]$ 上应用拉格朗日中值定理，存在 $\eta \in (0,c)$ 使得：
            $$
            f'(\eta) = \frac{f(c)-f(0)}{c-0} = \frac{M-0}{c} = \frac{M}{c}
            $$
    *   **构建介值区间**:
        *   我们现在有两个导数值：$f'(c)=0$ 和 $f'(\eta)=M/c$。
        *   因为 $c \in (0,1)$ 且 $n \ge 1$，所以 $c/n < 1 \implies c < n \implies M/c > M/n$。
        *   因此，我们有 $0 < \frac{M}{n} < \frac{M}{c}$。
        *   即：$f'(c) < \frac{M}{n} < f'(\eta)$。
    *   **应用导数介值定理**:
        *   导函数 $f'(x)$ 在区间 $[\eta, c]$ 上满足介值定理（因为二阶可导，一阶导必连续）。
        *   既然目标值 $M/n$ 介于 $f'(\eta)$ 和 $f'(c)$ 之间，那么必然存在一点 $x_n \in (\eta, c) \subset (0,1)$，使得 $f'(x_n) = \frac{M}{n}$。存在性得证。

## ✍️ 第二问：数列极限证明

![[6-14-1.png]]![[6-14-2.png]]

### **方法一：利用导数单调性比较 (前一总结的方法)**

*   **单调性**:
    1.  已知 $f'(x_n) = M/n$，$f'(x_{n+1})=M/(n+1)$。
    2.  易知 $f'(x_{n+1}) < f'(x_n)$。
    3.  因为 $f'(x)$ **严格单调递减**，函数值的大小关系与自变量的大小关系相反。
    4.  故 $x_{n+1} > x_n$，数列 $\{x_n\}$ 单调递增。

### **方法二：利用反函数性质比较 (本视频老师的方法)**

这种方法更加形式化，将逻辑隐藏在反函数的性质中。

*   **单调性**:
    1.  因为 $f'(x)$ 在 $(0,c)$ 或 $(c,1)$ 等区间上是严格单调的，所以它的反函数存在。我们记 $f'(x)$ 的反函数为 $g(y)$。
    2.  因为 $f'(x)$ 严格单调递减，所以其**反函数 $g(y)$ 也严格单调递减**。
    3.  从 $f'(x_n)=M/n$ 可以反解出 $x_n = g(M/n)$。同理，$x_{n+1}=g(M/(n+1))$。
    4.  比较反函数的输入：$M/n > M/(n+1)$。
    5.  因为 $g(y)$ **严格单调递减**，函数值的大小关系与自变量的大小关系相反。
    6.  故 $g(M/n) < g(M/(n+1))$，即 $x_n < x_{n+1}$。数列 $\{x_n\}$ 单调递g增。

*(后续求极限的步骤，两种方法完全相同，都是对 $f'(x_n)=M/n$ 取极限，得出 $\lim x_n = c$，再求 $\lim f(x_n) = f(c) = M$。)*

> [!danger]- 错题复盘
> ### 1. 一句话错误本质
> > 未能掌握解决**导数方程根存在性**问题的多重武器（特别是**导数介值定理**），以及未能将**函数单调性**灵活运用于**数列单调性**的判断中（特别是通过**反函数**这一视角）。
>
> ### 2. 两种方法的深度对比
> *   **第一问 (存在性)**:
>     *   **罗尔定理法**: 更具**构造性**和**普适性**。它的优点是不需要事先知道导函数的值域，通过构造一个必有水平切线的函数来“强行”找到解。这是解决中值定理证明题的“重武器”。
>     *   **介值定理法**: 更具**分析性**和**技巧性**。它的优点是更直接，但前提是你需要能找到两个导数值，将目标值“夹在”中间。这要求对函数的性质有更深的洞察。
> *   **第二问 (单调性)**:
>     *   **直接比较法**: 更**直观**。直接利用单调性的定义（自变量大则函数值小，反之亦然）。
>     *   **反函数法**: 更**形式化**和**理论化**。它将问题转化为研究反函数的性质，逻辑上同样严谨，但需要对反函数理论有更清晰的认识。
>
> ### 4. 可迁移的启发 (The Takeaway)
> > 1.  **丰富你的“武器库”**: 面对“证明存在$\xi$使$f'(\xi)=k$”这类问题，你的脑海中应至少有两套方案：【方案A：构造辅助函数用罗尔定理】和【方案B：找到导数的值域区间用介值定理】。
> > 2.  **具体化是解题的第一步**: 题目中出现的抽象符号，如最大值 $M$，一定要第一时间把它具体化，即“设 $f(c)=M$”。这会立刻解锁一系列可用条件，如 $f'(c)=0$。
> > 3.  **单调性是核心枢纽**: 在这道题中，由 $f''(x)<0$ 得到的 $f'(x)$ 的**严格单调性**是绝对的核心。它既保证了第一问的**唯一性**，又成为了第二问判断**数列单调性**的桥梁。抓住这个核心，整道题的逻辑链就清晰了。
> > 4.  **反函数是“照妖镜”**: 当你需要从函数值的关系反推自变量的关系时，引入反函数的概念会让逻辑链条非常清晰。虽然本质上和直接用单调性定义一样，但它提供了一个更高级、更系统的视角。
