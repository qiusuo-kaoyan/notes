# 题目 :: 来源：`$= dv.current().source` | 题号：`$= dv.current().id` | 难度：`$= "⭐".repeat(dv.current().difficulty)` | 重要性：`$= "🔥".repeat(dv.current().importance)`

> [!question] 题目描述
> 设函数 $f(x,y)$ 具有二阶连续偏导数，且在点 $(x_0, y_0)$ 处取得极大值。记 $a = \frac{\partial^2 f}{\partial x^2}\bigg|_{(x_0,y_0)}$, $b = \frac{\partial^2 f}{\partial y^2}\bigg|_{(x_0,y_0)}$，则 ( )
> (A) $a>0, b>0$
> (B) $a \ge 0, b \ge 0$
> (C) $a<0, b<0$
> (D) $a \le 0, b \le 0$

---

## 💡 知识点与思路分析

*   **考察知识点**:: `$= dv.current().knowledge_points`
*   **初步解题思路 (The Trap & The Insight)**:
    1.  **第一反应 (陷阱)**: 看到二元函数极大值，立刻想到充分条件：$\Delta = AC-B^2 > 0$ 且 $A<0$。所以 $a<0, b$ 呢？好像没法直接判断 $b$。这条路似乎走不通，因为它是一个**充分条件**，而不是**必要条件**。题目说“已知是极大值”，是让我们**反推**性质，而不是用性质去**判断**。
    2.  **正确的思路 (降维打击)**:
        *   **核心思想**: 将二元函数的问题，转化为我们更熟悉的一元函数问题来分析。
        *   **几何直觉 (切西瓜)**: 如果一个西瓜的最高点（极大值点）在 $(x_0, y_0)$，那么我沿着 $y=y_0$ 这条线竖着切一刀，得到的切面曲线（一个一元函数）的最高点，也必然在 $x=x_0$ 处。
        *   **代数转化**: 这个“切一刀”的操作，在代数上就是**固定一个变量**。
        *   **应用一元结论**: 将一元函数取得极大值的**必要条件**（特别是二阶导数的性质）应用到这个降维后的问题上。

## ✍️ 我的解答

### **第一步：降维——从二元到一元**

1.  **分析 $a = f''_{xx}(x_0, y_0)$**:
    *   根据偏导数的定义，$f''_{xx}(x_0, y_0)$ 是对函数 $f(x,y)$ 先后两次对 $x$ 求偏导。在整个求导过程中，**变量 $y$ 被固定为常数 $y_0$**。
    *   因此，我们可以构造一个**一元函数** $\phi(x) = f(x, y_0)$。这个函数就是老师口中“切西瓜”得到的切面曲线。
    *   那么，$f'_{x}(x, y_0) = \phi'(x)$，$f''_{xx}(x, y_0) = \phi''(x)$。
    *   所以，$a = f''_{xx}(x_0, y_0) = \phi''(x_0)$。

2.  **建立联系**:
    *   题目已知 $f(x,y)$ 在 $(x_0, y_0)$ 取得极大值。根据极大值的定义，在 $(x_0,y_0)$ 的一个邻域内，对于所有的 $(x,y)$，都有 $f(x,y) \le f(x_0,y_0)$。
    *   这个性质对于邻域内 $y=y_0$ 这条直线上的点当然也成立。即，对于所有 $x$ 使得 $(x,y_0)$ 在该邻域内，都有 $f(x,y_0) \le f(x_0,y_0)$。
    *   换成我们构造的一元函数语言，就是 $\phi(x) \le \phi(x_0)$。
    *   这说明，我们的一元函数 $\phi(x)$ 在点 $x=x_0$ 处也取得了**极大值**。

3.  **分析 $b = f''_{yy}(x_0, y_0)$**:
    *   同理，我们可以构造另一个一元函数 $\psi(y) = f(x_0, y)$。
    *   $f(x,y)$ 在 $(x_0,y_0)$ 取得极大值，意味着 $\psi(y)$ 在 $y=y_0$ 处也取得极大值。
    *   $b = f''_{yy}(x_0, y_0) = \psi''(y_0)$。

### **第二步：应用一元函数极值的二阶必要条件 (本题灵魂)**

![[22-结论.png]]

现在问题转化为了：已知二阶可导的一元函数 $\phi(x)$ 在 $x_0$ 处取极大值，问 $\phi''(x_0)$ 的符号是什么？

> [!warning] 核心辨析：充分条件 vs 必要条件
> *   **充分条件**: 若 $g'(c)=0$ 且 $g''(c)<0$，则 $c$ 是极大值点。
> *   **必要条件**: 若 $c$ 是极大值点且 $g(x)$ 二阶可导，则 $g'(c)=0$ 且 **$g''(c) \le 0$**。
>
> **为什么必须带等号？**
>
> 这是一个至关重要的结论。因为当二阶导数为零时，判别法会失效，但该点**仍有可能是极大值点**。
>
> **经典反例**: 考虑一元函数 $g(x) = -x^4$。
> *   $g'(x)=-4x^3 \implies g'(0)=0$。
> *   $g''(x)=-12x^2 \implies g''(0)=0$。
> *   此时二阶导判别法失效。但根据定义，$g(0)=0$，而在 $x=0$ 的邻域内，$-x^4 < 0$。所以 $x=0$ **确实是一个极大值点**。
>
> 这个反例雄辩地证明了：一个点是极大值点，只能保证其二阶导数**非正**($\le 0$)，而不能保证其**严格为负**($<0$)。

### **第三步：得出结论**

1.  因为 $\phi(x)$ 在 $x_0$ 处取得极大值，所以其二阶导数必然满足 $\phi''(x_0) \le 0$。即 **$a \le 0$**。
2.  因为 $\psi(y)$ 在 $y_0$ 处取得极大值，所以其二阶导数必然满足 $\psi''(y_0) \le 0$。即 **$b \le 0$**。

综合起来，正确答案是 (D) $a \le 0, b \le 0$。

> [!danger]- 错题复盘
> ### 1. 一句话错误本质
> > 混淆了二元函数极值的**充分条件**与**必要条件**，并且对一元函数极大值的二阶导数必要条件**未能准确记忆（忘记了可以取等号）**。
>
> ### 2. 错误诊断清单
> - [x] **思路错误**: 强行套用 $AC-B^2>0$ 的充分条件，而不是从“已知极大值”这个前提反推其性质。
> - [x] **概念不清**: 对一元函数极大值的二阶必要条件是 $f''(x_0) \le 0$ 而非 $f''(x_0) < 0$ 理解不深，或被反例 $f(x)=-x^4$ 的知识点遗忘。
> - [x] **技巧不熟**: 未能掌握将二元函数问题**“降维”**到一元函数进行分析的思考方法。
>
> ### 4. 可迁移的启发 (The Takeaway)
> > 1.  **“降维”是处理多元问题的基本法宝**: 当你对一个多元函数问题感到困惑时，尝试**固定其他变量，只让一个变量变化**，将其转化为你更熟悉的一元函数问题来思考。这种“降维打击”的思想，是解决多元微积分问题的核心策略之一。
> > 2.  **深刻理解“充分”与“必要”**: 在数学中，这两个词有天壤之别。
> >    *   “A 是 B 的**充分条件**” (`A => B`)：用来**判断**。你知道 A 成立，就能断定 B 也成立。
> >    *   “A 是 B 的**必要条件**” (`B => A`)：用来**反推性质**。你知道 B 成立，就能断定 A 必然满足。
> >    本题是典型的“已知 B，问 A”，必须使用必要条件。
> > 1.  **用反例加深记忆**: 对于那些容易混淆的数学结论（尤其是带不带等号的问题），最好的记忆方法就是记住一个**经典的反例**。函数 $f(x)=-x^4$ 是判断极大值时二阶导能否为零的“黄金反例”，必须牢记于心。
