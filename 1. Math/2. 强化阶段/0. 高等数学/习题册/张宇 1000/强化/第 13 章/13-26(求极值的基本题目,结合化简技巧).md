# 题目 :: 来源：`$= dv.current().source` | 题号：`$= dv.current().id` | 难度：`$= "⭐".repeat(dv.current().difficulty)` | 重要性：`$= "🔥".repeat(dv.current().importance)`

> [!question] 题目描述
> 求由方程 $2x^2+2y^2+z^2+8xz-z+8=0$ 所确定的函数 $z=z(x,y)$ 的极值，并指出是极大值还是极小值。

---

## 💡 知识点与思路分析

*   **考察知识点**:: `$= dv.current().knowledge_points`
*   **初步解题思路 (The Standard Procedure)**:
    1.  **求驻点 (必要条件)**:
        *   将方程两边分别对 $x$ 和 $y$ 求偏导，得到含有 $\frac{\partial z}{\partial x}$ 和 $\frac{\partial z}{\partial y}$ 的两个方程。
        *   令 $\frac{\partial z}{\partial x}=0$ 和 $\frac{\partial z}{\partial y}=0$，解出满足条件的 $(x,y,z)$ 坐标。这些就是可能的极值点（驻点）。
    2.  **求二阶偏导数 (准备充分条件)**:
        *   将第一步得到的两个偏导数方程，再次分别对 $x$ 和 $y$ 求偏导，得到关于二阶偏导数 $A=\frac{\partial^2 z}{\partial x^2}$, $B=\frac{\partial^2 z}{\partial x \partial y}$, $C=\frac{\partial^2 z}{\partial y^2}$ 的方程组。
        *   **核心技巧**: 在求二阶偏导的过程中，要利用**驻点处一阶偏导为零**的性质来极大简化计算。
    3.  **判断极值 (充分条件)**:
        *   将驻点的坐标代入，解出 $A,B,C$ 的具体数值。
        *   计算判别式 $\Delta = AC-B^2$，根据其符号和 $A$ 的符号来判断极值类型。

## ✍️ 我的解答

### **第一步：求驻点**

令 $F(x,y,z) = 2x^2+2y^2+z^2+8xz-z+8=0$。

1.  **方程两边对 $x$ 求偏导**: (将 $y$ 视为常数，$z$ 是 $x,y$ 的函数)
    $$
    4x + 2z\frac{\partial z}{\partial x} + 8z + 8x\frac{\partial z}{\partial x} - \frac{\partial z}{\partial x} = 0
    $$
    整理得：
    $$
    (2z+8x-1)\frac{\partial z}{\partial x} + 4x + 8z = 0 \quad \cdots(\text{式1})
    $$

2.  **方程两边对 $y$ 求偏导**: (将 $x$ 视为常数，$z$ 是 $x,y$ 的函数)
    $$
    4y + 2z\frac{\partial z}{\partial y} + 8x\frac{\partial z}{\partial y} - \frac{\partial z}{\partial y} = 0
    $$
    整理得：
    $$
    (2z+8x-1)\frac{\partial z}{\partial y} + 4y = 0 \quad \cdots(\text{式2})
    $$

3.  **令一阶偏导为零，解驻点**:
    令 $\frac{\partial z}{\partial x}=0$ 且 $\frac{\partial z}{\partial y}=0$。
    *   从 **(式 1)** 得到：$4x+8z=0 \implies x=-2z$。
    *   从 **(式 2)** 得到：$4y=0 \implies y=0$。
    *   将 $x=-2z$ 和 $y=0$ **代回原方程**，求解 $z$：
        $2(-2z)^2 + 2(0)^2 + z^2 + 8(-2z)z - z + 8 = 0$
        $8z^2 + z^2 - 16z^2 - z + 8 = 0$
        $-7z^2 - z + 8 = 0 \implies 7z^2 + z - 8 = 0$
        $(7z+8)(z-1) = 0$
        解得 $z=1$ 或 $z=-8/7$。

4.  **得到两个驻点**:
    *   **驻点 1**: 当 $z=1$ 时，$x=-2(1)=-2$，$y=0$。即点 $(-2, 0)$，对应函数值 $z=1$。
    *   **驻点 2**: 当 $z=-8/7$ 时，$x=-2(-8/7)=16/7$，$y=0$。即点 $(16/7, 0)$，对应函数值 $z=-8/7$。

### **第二步：求二阶偏导数 (核心技巧的应用)**

我们不需要解出 $\frac{\partial z}{\partial x}$ 和 $\frac{\partial z}{\partial y}$ 的复杂表达式再求导，而是直接对 **(式 1)** 和 **(式 2)** 进行求导。

1.  **对 (式 1) 两边再对 $x$ 求偏导**:
    $$
    \frac{\partial}{\partial x}\left[(2z+8x-1)\frac{\partial z}{\partial x}\right] + \frac{\partial}{\partial x}(4x+8z) = 0
    $$
    使用乘积法则：
    $$
    \left(2\frac{\partial z}{\partial x}+8\right)\frac{\partial z}{\partial x} + (2z+8x-1)\frac{\partial^2 z}{\partial x^2} + 4 + 8\frac{\partial z}{\partial x} = 0
    $$

2.  **对 (式 1) 两边再对 $y$ 求偏导**:
    $$
    \frac{\partial}{\partial y}\left[(2z+8x-1)\frac{\partial z}{\partial x}\right] + \frac{\partial}{\partial y}(4x+8z) = 0
    $$
    $$
    \left(2\frac{\partial z}{\partial y}\right)\frac{\partial z}{\partial x} + (2z+8x-1)\frac{\partial^2 z}{\partial x \partial y} + 8\frac{\partial z}{\partial y} = 0
    $$

3.  **对 (式 2) 两边再对 $y$ 求偏导**:
    $$
    \frac{\partial}{\partial y}\left[(2z+8x-1)\frac{\partial z}{\partial y}\right] + \frac{\partial}{\partial y}(4y) = 0
    $$
    $$
    \left(2\frac{\partial z}{\partial y}\right)\frac{\partial z}{\partial y} + (2z+8x-1)\frac{\partial^2 z}{\partial y^2} + 4 = 0
    $$

> [!tip] 降维打击技巧
> 我们的目标是在**驻点**处求二阶偏导数。在驻点，**所有的一阶偏导数都等于零**！
> 将 $\frac{\partial z}{\partial x}=0$ 和 $\frac{\partial z}{\partial y}=0$ 代入上述三个求导后的式子，它们会瞬间被净化：
> *   $(0+8)\cdot 0 + (2z+8x-1)A + 4 + 8\cdot 0 = 0 \implies \boldsymbol{A = \frac{-4}{2z+8x-1}}$
> *   $(0)\cdot 0 + (2z+8x-1)B + 8\cdot 0 = 0 \implies \boldsymbol{B = 0}$
> *   $(0)\cdot 0 + (2z+8x-1)C + 4 = 0 \implies \boldsymbol{C = \frac{-4}{2z+8x-1}}$

### **第三步：判断极值**

现在，我们把两个驻点的坐标代入 $A,B,C$ 的表达式来判断极值。

#### **对于驻点 1: $(-2,0)$，此时 $z=1$**
*   $A = \frac{-4}{2(1)+8(-2)-1} = \frac{-4}{2-16-1} = \frac{-4}{-15} = \frac{4}{15}$
*   $B=0$
*   $C = \frac{-4}{2(1)+8(-2)-1} = \frac{4}{15}$
*   **判别**:
    *   $\Delta = AC - B^2 = (\frac{4}{15})(\frac{4}{15}) - 0^2 = \frac{16}{225} > 0$。
    *   $A = \frac{4}{15} > 0$。
    *   **结论**: 函数在该点取得**极小值**，极小值为 $z=1$。

#### **对于驻点 2: $(16/7,0)$，此时 $z=-8/7$**
*   $A = \frac{-4}{2(-8/7)+8(16/7)-1} = \frac{-4}{-16/7+128/7-1} = \frac{-4}{112/7-1} = \frac{-4}{16-1} = -\frac{4}{15}$
*   $B=0$
*   $C = \frac{-4}{2(-8/7)+8(16/7)-1} = -\frac{4}{15}$
*   **判别**:
    *   $\Delta = AC - B^2 = (-\frac{4}{15})(-\frac{4}{15}) - 0^2 = \frac{16}{225} > 0$。
    *   $A = -\frac{4}{15} < 0$。
    *   **结论**: 函数在该点取得**极大值**，极大值为 $z=-8/7$。

> [!danger]- 错题复盘
> ### 1. 一句话错误本质
> > 在计算隐函数的二阶偏导数时，走了“先解出一阶偏导数表达式，再求导”的弯路，而没有利用**“驻点处一阶偏导为零”**这一核心性质，直接在求导后的方程中进行代入化简，导致计算过程异常繁琐。
>
> ### 2. 错误诊断清单
> - [x] **思路低效**: 没有掌握在驻点处计算二阶偏导的“捷径”，做了大量无用功。
> - [ ] **计算失误**: 在求解驻点或计算二阶偏导数值时出错。
> - [ ] **概念不清**: 对二元函数极值的判别法（$\Delta$ 和 A 的符号）记忆混淆。
>
> ### 4. 可迁移的启发 (The Takeaway)
> > 1.  **“先置零，再求导” vs “先求导，再置零”**: 在处理隐函数极值问题时，求二阶偏导的正确思维是：**先对一阶偏导的隐式方程求导，得到含有各种一阶、二阶偏导数的复杂方程，然后再利用驻点处一阶偏导为零的条件，将这个复杂方程瞬间“净化”**。千万不要试图先解出一阶偏导的显式表达式。
> > 2.  **抓住问题的核心**: 我们的目标不是求出二阶偏导数在**任意点**的表达式，而是求出它们在**驻点**的**数值**。这个目标决定了我们的解题策略可以大大简化。
> > 3.  **流程化、模块化**: 隐函数求极值的过程是一个高度流程化的“三部曲”：① 求一阶偏导方程组；② 令一阶偏导为零，代入原方程解驻点；③ 对一阶偏导方程组再求导，并代入“一阶偏导为零”和驻点坐标，解出 A, B, C，最后判别。将这个流程刻在脑子里，就能以不变应万变。
