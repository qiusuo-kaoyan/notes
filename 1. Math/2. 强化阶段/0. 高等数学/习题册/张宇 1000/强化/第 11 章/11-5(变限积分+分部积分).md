# 题目 :: 来源：`$= dv.current().source` | 题号：`$= dv.current().id` | 难度：`$= "⭐".repeat(dv.current().difficulty)` | 重要性：`$= "🔥".repeat(dv.current().importance)`

> [!question] 题目描述
> 设函数 $f(x), g(x)$ 在区间 $[a,b]$ 上连续，对任意的 $x \in [a,b]$，满足 $\int_a^x g(t)dt \le \int_a^x f(t)dt$，且 $\int_a^b g(t)dt = \int_a^b f(t)dt$。证明：$\int_a^b xf(x)dx \le \int_a^b xg(x)dx$。

---

## 💡 知识点与思路分析

*   **考察知识点**:: `$= dv.current().knowledge_points`
*   **初步解题思路**:
    1.  **识别题眼**:
        *   **条件**: 题目给出的条件不是关于 $f(x), g(x)$ 本身，而是关于它们的**变上限积分**（即它们的一个**原函数**）的大小关系。
        *   **结论**: 要证明的结论是关于 $xf(x), xg(x)$ 的积分大小关系。
    2.  **匹配战术**: 如何将关于 $f,g$ 的积分，与关于它们**原函数**的已知条件联系起来？**分部积分法**是连接函数与其原函数（或导函数）的最强有力的桥梁。

---

## ✍️ 方法一：直接分部积分法 (视频讲解思路)



这种方法更加直观，直接对要证明的结论左右两边同时“开刀”。

1.  **核心思想**：将题目中的**变上限积分**看作是 $f(x)$ 和 $g(x)$ 的**原函数**。
    *   令 $F(x) = \int_a^x f(t)dt$ 和 $G(x) = \int_a^x g(t)dt$。
    *   那么 $F'(x)=f(x)$，$G'(x)=g(x)$。

2.  **翻译题目条件**：
    *   $\int_a^x g(t)dt \le \int_a^x f(t)dt \implies G(x) \le F(x)$，对任意 $x \in [a,b]$。
    *   $\int_a^b g(t)dt = \int_a^b f(t)dt \implies G(b) = F(b)$。
    *   根据定义，显然 $G(a) = \int_a^a g(t)dt = 0$，$F(a) = \int_a^a f(t)dt = 0$。

3.  **对结论进行变形 (关键步骤)**：
    我们要证明 $\int_a^b xf(x)dx \le \int_a^b xg(x)dx$。
    分别对两边的积分使用**分部积分法**：
    *   **左边**:
        $$
        \int_a^b xf(x)dx = \int_a^b x d(F(x)) = \left[x F(x)\right]_a^b - \int_a^b F(x)dx = bF(b) - aF(a) - \int_a^b F(x)dx
        $$
        因为 $F(a)=0$，所以左边 $= bF(b) - \int_a^b F(x)dx$。
    *   **右边**:
        $$
        \int_a^b xg(x)dx = \int_a^b x d(G(x)) = \left[x G(x)\right]_a^b - \int_a^b G(x)dx = bG(b) - aG(a) - \int_a^b G(x)dx
        $$
        因为 $G(a)=0$，所以右边 $= bG(b) - \int_a^b G(x)dx$。

4.  **利用条件完成证明**：
    原不等式等价于证明：
    $$
    bF(b) - \int_a^b F(x)dx \le bG(b) - \int_a^b G(x)dx
    $$
    因为条件告诉我们 $F(b)=G(b)$，所以 $bF(b)$ 和 $bG(b)$ 两项可以**直接消去**！不等式变为：
    $$
    - \int_a^b F(x)dx \le - \int_a^b G(x)dx
    $$
    两边同乘 $-1$，不等号反向：
    $$
    \int_a^b F(x)dx \ge \int_a^b G(x)dx \quad \text{或} \quad \int_a^b (F(x)-G(x))dx \ge 0
    $$
    根据我们翻译的条件，$F(x) \ge G(x)$，所以被积函数 $F(x)-G(x) \ge 0$。一个非负函数在区间上的定积分必然大于等于零。
    因此，上式成立，原命题得证。

## ✍️ 方法二：构造辅助函数法 (答案解析思路)

![[11-5-解析.png]]

这种方法更加“套路化”，是证明此类问题的标准格式。它首先将不等式移项，构造出一个函数，然后证明这个函数的某个性质。

1.  **核心思想**：将要证的不等式移项，构造一个辅助函数，然后利用分部积分证明与这个辅助函数相关的某个积分的符号。

2.  **构造辅助函数**：
    *   要证 $\int_a^b xf(x)dx \le \int_a^b xg(x)dx$，即证 $\int_a^b x(f(x)-g(x))dx \le 0$。
    *   令 $\phi(x) = f(x)-g(x)$。
    *   令 $\Phi(x) = \int_a^x \phi(t)dt = \int_a^x (f(t)-g(t))dt$。

3.  **翻译题目条件**：
    *   $\int_a^x g(t)dt \le \int_a^x f(t)dt \implies \Phi(x) = \int_a^x (f(t)-g(t))dt \ge 0$。
    *   $\int_a^b g(t)dt = \int_a^b f(t)dt \implies \Phi(b) = \int_a^b (f(t)-g(t))dt = 0$。
    *   根据定义，$\Phi(a) = 0$。
    *   并且，$\Phi'(x) = \phi(x) = f(x)-g(x)$。

4.  **对目标积分进行变形**：
    我们的目标是证明 $\int_a^b x\phi(x)dx \le 0$。
    $$
    \int_a^b x\phi(x)dx = \int_a^b x \Phi'(x)dx = \int_a^b x d(\Phi(x))
    $$
    使用**分部积分法**：
    $$
    = \left[x \Phi(x)\right]_a^b - \int_a^b \Phi(x)dx
    $$

5.  **利用条件完成证明**：
    *   计算第一项：$\left[x \Phi(x)\right]_a^b = b\Phi(b) - a\Phi(a)$。
    *   因为 $\Phi(a)=0$ 且 $\Phi(b)=0$，所以这一项等于 $b \cdot 0 - a \cdot 0 = 0$。
    *   因此，目标积分就等于：
        $$
        \int_a^b x\phi(x)dx = - \int_a^b \Phi(x)dx
        $$
    *   根据我们翻译的条件，$\Phi(x) \ge 0$ 在整个区间 $[a,b]$ 上恒成立。
    *   所以，$\int_a^b \Phi(x)dx \ge 0$。
    *   那么，$-\int_a^b \Phi(x)dx \le 0$。
    *   原命题得证。

> [!danger]- 错题复盘
> ### 1. 一句话错误本质
> > 未能将题目中给出的**变上限积分条件**与**原函数**的核心概念联系起来，从而没有想到使用**分部积分法**这一关键工具，在 `f(x)` 的积分与 `F(x)` 的积分之间建立联系。
>
> ### 2. 错误诊断清单
> - [x] **思路错误**: 可能尝试对条件求导，但正如视频中所说，由积分的大小关系无法直接推出被积函数（或其导数）的大小关系。
> - [x] **技巧不熟**: 对**分部积分法**的战略意义理解不深。它不仅仅是计算不定积分的工具，更是证明题中进行**表达式转换**的利器。
> - [ ] **概念不清**: 对变上限积分函数的基本性质（如 $\Phi(a)=0$）不熟悉。
>
> ### 4. 可迁移的启发 (The Takeaway)
> > 1.  **决策模型优化 (建立条件反射)**: 
> >    *   **看到 `变上限积分` $\implies$ 想到 `原函数`**。立刻在脑海中建立 $F(x)=\int_a^x f(t)dt$, 并写下其性质: $F'(x)=f(x), F(a)=0$。
> >    *   **看到 `xf(x)` 的积分 $\implies$ 想到 `分部积分`**。这个结构是使用分部积分的强烈信号，因为它可以将 `xf(x)` 的积分问题，转化为其原函数 `F(x)` 的积分问题，从而利用上题目给的条件。
> > 2.  **方法论对比**:
> >    *   **方法一（直接法）** 更适合在考场上思维灵活时使用，它直接作用于结论，思路清晰。
> >    *   **方法二（构造法）** 是更“标准化”的证明流程。当不知道如何下手时，可以尝试将要证的不等式移项，构造 `h(x)=f(x)-g(x)`，然后分析 `h(x)` 及其原函数的性质。这是保底的、更具普适性的解题策略。
> > 3.  **积累“母题”**: 本题就是一道完美的“母题”。它揭示了一类问题的核心解法。务必将这个“**原函数 + 分部积分**”的组合拳牢记于心，下次遇到类似问题时，就能迅速识别并破解。
