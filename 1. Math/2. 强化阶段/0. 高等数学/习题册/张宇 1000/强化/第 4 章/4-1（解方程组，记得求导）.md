> [!question] 题目描述
> 设 $f(x)$ 为三次多项式，且 $f(x)+1$ 能被 $(x-1)^2$ 整除，$f(x)-1$ 能被 $(x+1)^2$ 整除，则 $f(x) = \_\_\_\_\_\_\_\_\_\_$.

---

## 💡 知识点与思路分析

*   **考察知识点**:: `$= dv.current().knowledge_points || "尚未关联"`
*   **初步解题思路**:
    1.  **识别题眼**: 题目核心是“三次多项式”和“能被 $(x-a)^2$ 整除”。后者是题目的关键突破口。
    2.  **匹配战术**: 联想到**重根 (multiple root)** 的性质。一个多项式 $P(x)$ 如果能被 $(x-a)^2$ 整除，意味着 $x=a$ 是 $P(x)$ 的一个二重根（或更高次）。这不仅意味着 $P(a)=0$，更重要的是，它的导数在该点也为零，即 $P'(a)=0$。
    3.  **制定步骤**:
        *   设 $P_1(x) = f(x)+1$，由题意 $P_1(x)$ 被 $(x-1)^2$ 整除，可得 $P_1(1)=0$ 和 $P_1'(1)=0$。
        *   设 $P_2(x) = f(x)-1$，由题意 $P_2(x)$ 被 $(x+1)^2$ 整除，可得 $P_2(-1)=0$ 和 $P_2'(-1)=0$。
        *   将上述四个条件转化为关于 $f(x)$ 和 $f'(x)$ 的四个方程。
        *   设出三次多项式的一般形式 $f(x) = ax^3 + bx^2 + cx + d$，并求出其导数 $f'(x)$。
        *   利用四个方程和四个未知数($a,b,c,d$)，建立一个线性方程组。
        *   解该方程组，求出所有系数，从而确定 $f(x)$。

## ✍️ 我的解答

#### **第一步：从整除条件导出函数及其导数的方程**

1.  **对于条件 "$f(x)+1$ 能被 $(x-1)^2$ 整除"**:
    令 $P(x) = f(x)+1$。根据重根的性质，我们有：
    *   $P(1) = 0 \implies f(1)+1=0 \implies f(1)=-1$
    *   $P'(1) = 0 \implies f'(1)=0$

2.  **对于条件 "$f(x)-1$ 能被 $(x+1)^2$ 整除"**:
    令 $Q(x) = f(x)-1$。根据重根的性质，我们有：
    *   $Q(-1) = 0 \implies f(-1)-1=0 \implies f(-1)=1$
    *   $Q'(-1) = 0 \implies f'(-1)=0$

我们得到了四个关键条件：
$f(1)=-1$, $f'(1)=0$, $f(-1)=1$, $f'(-1)=0$.

#### **第二步：建立并求解线性方程组**

设三次多项式为 $f(x) = ax^3 + bx^2 + cx + d$。
其导数为 $f'(x) = 3ax^2 + 2bx + c$。

将四个条件代入：
1.  $f(1) = a+b+c+d = -1$
2.  $f'(1) = 3a+2b+c = 0$
3.  $f(-1) = -a+b-c+d = 1$
4.  $f'(-1) = 3a-2b+c = 0$

开始解这个方程组：
*   **由方程 (2) 和 (4)**:
    $(3a+2b+c) - (3a-2b+c) = 0 - 0 \implies 4b=0 \implies \boldsymbol{b=0}$
*   将 $b=0$ 代回方程 (2) 或 (4)，得到：
    $3a+c=0 \implies \boldsymbol{c = -3a}$ **(5)**
*   **由方程 (1) 和 (3)** (代入 $b=0$):
    $a+c+d = -1$ **(1')**
    $-a-c+d = 1$ **(3')**
*   将方程 (1') 和 (3') 相加：
    $(a+c+d) + (-a-c+d) = -1+1 \implies 2d=0 \implies \boldsymbol{d=0}$
*   将 $b=0, d=0$ 代回方程 (1')：
    $a+c = -1$ **(6)**
*   **联立方程 (5) 和 (6)**:
    将 $c = -3a$ 代入 (6) 中，得到 $a + (-3a) = -1 \implies -2a = -1 \implies \boldsymbol{a = \frac{1}{2}}$。
    再代回答 (5)，得到 $\boldsymbol{c = -3 \cdot \frac{1}{2} = -\frac{3}{2}}$。

#### **第三步：写出最终多项式**
我们求得了所有系数：$a=\frac{1}{2}, b=0, c=-\frac{3}{2}, d=0$。
所以，该三次多项式为：
$$
f(x) = \frac{1}{2}x^3 - \frac{3}{2}x
$$

> [!danger]- 错题复盘 (如果做错)
> ### 1. 一句话错误本质
> > *未能将“被 $(x-a)^2$ 整除”这一代数语言，准确翻译为“函数在该点的导数值为零”这一微积分语言。*
> 
> ### 2. 错误诊断清单
> - [x] **概念不清**: 核心错误。只知道被 $(x-a)$ 整除意味着 $f(a)=...$，但不知道被 $(x-a)^2$ 整除还额外给出了一个关于**导数**的条件 $f'(a)=...$。
> - [ ] **计算失误**: 在解四元一次线性方程组时出现加减错误。
> - [ ] **思路错误**: 可能采用了更复杂但低效的方法，例如设 $f(x)+1 = (x-1)^2(Ax+B)$，然后进行繁琐的多项式乘法和比较系数，这会大大增加计算量和出错风险。
> - [x] **条件遗漏**: 只用了两个或三个条件，导致无法唯一确定四个未知系数。
> 
> ### 3. 正确解法与反思
> 本题是多项式理论和基础微分学的完美结合。最优解法就是利用“重根处导数为零”这一性质。它能迅速、简洁地将一个看似复杂的整除问题，转化为一个程序化的、必能解出的线性方程组问题。这体现了微积分作为工具的强大威力。
> 
> ### 4. 可迁移的启发 (The Takeaway)
> > *这道题给了我什么超越题目本身的启发？我能提炼出什么通用的技巧或思维模式，应用到其他题目中？*
