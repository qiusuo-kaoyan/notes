> [!question] 题目描述
> 设 $0 \le x_1 \le \sqrt{c}$, $x_{n+1} = \frac{c(1+x_n)}{c+x_n}$, $n \in \mathbb{Z}^+, c > 1$。证明数列 $\{x_n\}$ 收敛，并求其值。

## 💡 知识点与思路分析

*   **考察知识点**:: `$= dv.current().knowledge_points || "尚未关联"`
*   **初步解题思路**:
    1.  **识别题眼**: 题目给出递推关系式 $x_{n+1} = f(x_n)$，要求证明收敛并求极限。这是**单调有界准则**的典型应用场景。
    2.  **匹配战术 (定性分析)**: 在动手证明前，先进行“侦察”。
        *   **寻找不动点**: 假设数列收敛于 $L$，则 $L$ 必满足 $L = f(L)$。
            $L = \frac{c(1+L)}{c+L} \implies L(c+L) = c(1+L) \implies cL + L^2 = c + cL \implies L^2 = c$。
            由于 $x_n \ge 0$，所以 $L = \sqrt{c}$。
        *   **确定方向**: 我们找到了**候选极限** $\sqrt{c}$。题目给出 $x_1 \le \sqrt{c}$。基于考研题目“必然单调”的特点，我们可以大胆猜测：该数列将**单调递增**，并以 $\sqrt{c}$ 为**上界**。这个猜测将指导我们后续的证明。
    3.  **制定证明方案**: 讲座中提供了多种证明路径。
        *   **方法一 (标准流程)**:
            *   **证明有界性**: 使用**数学归纳法**证明 $0 \le x_n \le \sqrt{c}$。
            *   **证明单调性**: 使用**作差法** ($x_{n+1} - x_n$) 证明数列递增。
        *   **方法二 (函数性质法)**:
            *   **构造递推函数**: 定义 $f(x) = \frac{c(1+x)}{c+x}$。
            *   **证明单调性**: 通过计算导数 $f'(x)$ 的符号来判断递推函数的单调性，进而判断数列 $\{x_n\}$ 的单调性。
            *   **证明有界性**: 可结合数学归纳法或观察法。
        *   **方法三 (变形观察法)**:
            *   对递推式进行恒等变形，直接观察出数列的有界性。

## ✍️ 我的解答

#### **预备工作：不动点分析**
假设 $\lim_{n \to \infty} x_n = L$ 存在，对递推式 $x_{n+1} = \frac{c(1+x_n)}{c+x_n}$ 两边取极限得：
$L = \frac{c(1+L)}{c+L} \implies Lc + L^2 = c + cL \implies L^2 = c$
因为显然 $x_n > 0$ 对所有 $n$ 成立，所以 $L \ge 0$。故极限的唯一可能值为 $L=\sqrt{c}$。

### **方法一：标准法 (数学归纳法 + 作差法)**

#### **第一步：证明有界性 ($0 \le x_n \le \sqrt{c}$)**
我们使用**数学归纳法**。
*   **基础**: 当 $n=1$ 时，由题设 $0 \le x_1 \le \sqrt{c}$，命题成立。
*   **归纳**: 假设当 $n=k$ 时命题成立，即 $0 \le x_k \le \sqrt{c}$。
    当 $n=k+1$ 时，
    $x_{k+1} = \frac{c(1+x_k)}{c+x_k}$。由于 $c>1, x_k \ge 0$，显然 $x_{k+1} > 0$。
    接下来是关键，证明 $x_{k+1} \le \sqrt{c}$。
    **==技巧==**: 不直接作差，而是利用递推函数的单调性。
    令 $f(x) = \frac{c(1+x)}{c+x}$。求导得 $f'(x) = \frac{c(c+x) - c(1+x)}{(c+x)^2} = \frac{c^2-c}{(c+x)^2}$。
    因为 $c > 1$，所以 $c^2 > c$，故 $f'(x) > 0$。因此 $f(x)$ 是一个严格增函数。
    由归纳假设 $x_k \le \sqrt{c}$，根据 $f(x)$ 的单调性，可得：
    $x_{k+1} = f(x_k) \le f(\sqrt{c})$
    而 $\sqrt{c}$ 是 $f(x)$ 的不动点，即 $f(\sqrt{c}) = \sqrt{c}$。
    所以，$x_{k+1} \le \sqrt{c}$。
*   **结论**: 由数学归纳法，对所有 $n \ge 1$，$0 \le x_n \le \sqrt{c}$ 成立。数列有界。

#### **第二步：证明单调性 (递增)**
使用**作差法**：
$x_{n+1} - x_n = \frac{c(1+x_n)}{c+x_n} - x_n = \frac{c+cx_n - cx_n - x_n^2}{c+x_n} = \frac{c - x_n^2}{c+x_n}$
*   分母 $c+x_n > 0$。
*   分子 $c - x_n^2$，因为我们已证明 $x_n \le \sqrt{c}$，所以 $x_n^2 \le c$，即 $c-x_n^2 \ge 0$。
因此，$x_{n+1} - x_n \ge 0$，即 $x_{n+1} \ge x_n$。数列 $\{x_n\}$ 单调不减。

#### **第三步：结论**
数列 $\{x_n\}$ 单调不减且有上界 $\sqrt{c}$，根据**单调有界准则**，该数列必定收敛。
其极限值必为不动点 $\sqrt{c}$。

### **方法二：函数法 (利用递推函数性质)**
#### **第一步：构造函数并证明其单调性**
令递推函数为 $f(x) = \frac{c(1+x)}{c+x}$。
如方法一所示，我们已计算出 $f'(x) = \frac{c^2-c}{(c+x)^2} > 0$。
因为递推函数 $f(x)$ 是增函数，所以数列 $\{x_n\}$ 必然是**单调**的。

#### **第二步：确定单调方向**
我们只需比较 $x_2$ 和 $x_1$ 的大小：
$x_2 - x_1 = f(x_1) - x_1 = \frac{c - x_1^2}{c+x_1}$
由题设 $0 \le x_1 \le \sqrt{c}$，可知 $x_1^2 \le c$，故 $c-x_1^2 \ge 0$。
所以 $x_2 \ge x_1$。
由于数列是单调的，且前两项是增加的关系，所以整个数列 $\{x_n\}$ 是单调递增的。

#### **第三步：证明有界性并得出结论**
有界性的证明同方法一（使用数学归纳法）。
得到数列单调递增有上界 $\sqrt{c}$，因此收敛于 $\sqrt{c}$。
**注**: 讲座中还提到，可以用**拉格朗日中值定理**更严谨地说明 $f'(x)>0$ 如何保证单调性：
$x_{n+2}-x_{n+1} = f(x_{n+1}) - f(x_n) = f'(\eta)(x_{n+1}-x_n)$ (其中 $\eta \in (x_n, x_{n+1})$)
因为 $f'(\eta)>0$，所以 $x_{n+2}-x_{n+1}$ 与 $x_{n+1}-x_n$ 同号，从而证明了数列的单调性。

### **方法三：变形观察法**

#### **第一步：恒等变形**
对递推关系式进行变形，类似于将假分式化为带分式：
$x_{n+1} = \frac{c(1+x_n)}{c+x_n} = \frac{c(c+x_n) - c^2 + c}{c+x_n} = c + \frac{c - c^2}{c+x_n} = c - \frac{c(c-1)}{c+x_n}$

#### **第二步：观察有界性**
*   由于 $c>1, x_n \ge 0$，分母 $c+x_n > 0$，分数项 $\frac{c(c-1)}{c+x_n} > 0$。
*   因此，$x_{n+1} = c - (\text{正数}) < c$。
*   这直接给出了一个上界 $c$。虽然不如 $\sqrt{c}$ 精确，但也足以证明数列有上界。

#### **第三步：后续步骤**
得到有界性后，再用方法一或方法二的手段证明其单调性，最终得出收敛的结论。

> ### 1. 多种解法总结与对比
> 
> *   **方法一：标准法 (不动点分析 + 数学归纳法 + 作差法)**
>     *   **优点**: 逻辑最清晰，步骤最标准，是解决此类问题的“通用公式”。只要按部就班，几乎总能解决问题。
>     *   **缺点**: 有时作差或数学归纳的某一步代数运算可能比较复杂。
>     *   **核心**: **不动点分析是灵魂**。先算出候选极限，用它来指导后续证明，事半功倍。
> 
> *   **方法二：函数法 (利用递推函数性质)**
>     *   **优点**: 非常强大和优雅。将数列的单调性问题，转化为函数的单调性问题，利用导数这一强力工具，可以迅速判断。
>     *   **缺点**: 需要抽象出递推函数并求导，对函数思想要求较高。
>     *   **核心**: 理解 $x_{n+1}=f(x_n)$ 中，$f(x)$ 的性质（特别是单调性）决定了 $\{x_n\}$ 的性质。
> 
> *   **方法三：变形观察法**
>     *   **优点**: 如果能成功变形，可能可以非常迅速地看出有界性，极具技巧性。
>     *   **缺点**: 对代数变形能力要求高，不具有普适性，很多题目无法用此法。
>

> ### 4. 可迁移的启发 (The Takeaway)
> > 1.  **不动点优先原则**: 面对 $x_{n+1}=f(x_n)$ 类型的收敛性证明题，第一步永远是解方程 $L=f(L)$ 找出不动点。这个不动点就是你的“靶子”，它告诉你数列的极限可能是多少，上/下界可能是多少。
> > 2.  **函数思想是降维打击**: 将离散的数列问题，转化为连续的函数问题（研究递推函数 $f(x)$），往往能使问题变得更简单。特别是用导数 $f'(x)$ 判断单调性，是一个必须掌握的高级技巧。
> > 3.  **一题多解，触类旁通**: 同一道题，尝试用不同的方法去解决，可以极大地加深对知识点内在联系的理解。本题就是绝佳的例子，它将单调有界、不动点、函数性质、数学归纳法完美地串联在了一起。
