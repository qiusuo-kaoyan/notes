# [[3. 知识枢纽：一元函数微分学的概念]]

> [!abstract] 核心战略思想 (The Grand Strategy)
> **一元微分学的核心，是围绕“导数”这一概念，建立一套从“点”到“邻域”，从“函数”到“导函数”的精密逻辑体系。所有概念辨析的根源，在于深刻理解“点的导数”是一个由极限定义的、独立的==局部==性质，而“导函数的极限”则描述了其==邻域== 内的变化趋势。除非有“导数极限定理”（洛必达法则的基石）作为桥梁，否则两者之间并无必然联系。==回归导数定义==是破解一切迷惑性命题的最高法院，而[[#2. 不可导点判定终极法则|快速判定法则]]则是将这一定义的精髓，提炼为可用于闪电战的精确制导武器。**
