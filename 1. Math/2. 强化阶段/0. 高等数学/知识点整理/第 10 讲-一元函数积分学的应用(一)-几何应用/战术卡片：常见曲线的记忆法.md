# [[战术卡片：常见曲线的记忆法]]

***
> [!abstract] 战术概览 (Tactical Overview)
> 
> > [!summary]- 战术A：摆线 (Cycloid)
> > *   **一句话作用：** 快速判定摆线方向，并瞬时记忆其面积与弧长。
> > *   **核心识别码：** 参数方程中出现 `t - sin(t)` 的结构。
> > *   **核心记忆法：** **“凭依轴”判定法 + “三块方砖”预算法。**
> 
> > [!summary]- 战术B：星形线 (Astroid)
> > *   **一句话作用：** 通过类比，秒画标准星形线。
> > *   **核心识别码：** $x^{2/3} + y^{2/3} = a^{2/3}$。
> > *   **核心记忆法：** **“凹凸”类比法。**
> 
> > [!summary]- 战术C：心形线 (Cardioid)
> > *   **一句话作用：** 根据极坐标方程，精确定位心形线的朝向。
> > *   **核心识别码：** $r = a(1 \pm \cos\theta)$ 或 $r = a(1 \pm \sin\theta)$。
> > *   **核心记忆法：** **“桃心定位”四步法。**
>
> > [!summary]- 战术D：双纽线 (Lemniscate)
> > *   **一句话作用：** 根据方程的对称性，确定双纽线的对称轴。
> > *   **核心识别码：** $(x^2+y^2)^2 = \dots$ 或 $r^2 = \dots$。
> > *   **核心记忆法：** **“对称轴”判定法。**

## 深度拆解 (In-depth Deconstruction)

> [!example]- 战术一：摆线 (“三块方砖”记忆法)
> 
> > [!todo]
> > *   **标准方程① (卧式):** $\begin{cases} x = a(t-\sin t) \\ y = a(1-\cos t) \end{cases} \quad (0 \le t \le 2\pi)$
> > *   **标准方程② (立式):** $\begin{cases} x = a(1-\cos t) \\ y = a(t-\sin t) \end{cases} \quad (0 \le t \le 2\pi)$
> > *   **核心考点：** 与坐标轴围成的面积、一拱的弧长。
> 
> > [!TIP] **战术决策与执行**
> > 1.  **判定朝向 (“凭依轴”法):**
> >     *   观察方程组，找到含有 `t - sin(t)` 的那一部分。
> >     *   **该部分对应的变量，就是摆线这个“月牙”所凭依、所“躺在”的坐标轴。**
> >     *   **例①：** `x` 的表达式是 `a(t-sint)`，故摆线“躺”在x轴上。
> >     *   **例②：** `y` 的表达式是 `a(t-sint)`，故摆线“躺”在y轴上。
> > 2.  **计算面积与弧长 (“三块方砖”法):**
> >     *   **构建模型：** 想象有三块边长为 `a` 的正方形，并排摆放。
> >     
> >     *   **计算面积：**
> >         *   三块方砖的总面积是 $3a^2$。
> >         *   摆线天然与 `π` 有关，给它乘上一个 `π`。
> >         *   **结论：一拱的面积 $S = 3\pi a^2$。**
> >     *   **计算弧长：**
> >         *   数出这个“三连方砖”模型外轮廓的周长（不含内部分割线）。
> >         *   上边 `a+a+a=3a`，下边 `a+a+a=3a`，左边 `a`，右边 `a`。
> >         *   总长为 `3a+3a+a+a = 8a`。
> >         *   **结论：一拱的弧长 $L = 8a$。**

> [!example]- 战术二：星形线 (“凹凸”类比法)
> 
> > [!todo]
> > *   **标准方程:** $x^{2/3} + y^{2/3} = a^{2/3}$
> > *   **核心考点：** 图像绘制、围成面积、弧长。
> 
> > [!TIP] **战术决策与执行**
> > 1.  **建立参照系:**
> >     *   $x^1 + y^1 = a^1$ 是一条直线。
> >     *   $x^2 + y^2 = a^2$ 是一个向外**凸**的圆。
> > 2.  **应用类比:**
> >     *   星形线的指数是 `2/3`，**小于1**。
> >     *   既然指数大于1的圆是向外“凸”的，那么指数小于1的星形线，就必须是向内**“凹”**的。
> >     *   以第一象限为例，它“凹”在直线 $x+y=a$ 的内部。
> > 3.  **四方复制:** 在每个象限内，都画出这个向内凹陷的曲线，即可得到完整的星形线。

> [!example]- 战术三：心形线 (“桃心定位”四步法)
> 
> > [!todo]
> > *   **标准方程:** $r = a(1 \pm \cos\theta)$ 或 $r = a(1 \pm \sin\theta)$
> > *   **核心考点：** 图像绘制、围成面积、弧长。
> 
> > [!TIP] **战术决策与执行 (定位桃心尖点)**
> > 1.  **定轴 (看函数):**
> >     *   方程中出现 **`cosθ`** $\implies$ “桃心”的对称轴是 **X轴**。
> >     *   方程中出现 **`sinθ`** $\implies$ “桃心”的对称轴是 **Y轴**。
> > 2.  **定向 (看符号):**
> >     *   方程中是 **`+`** 号 $\implies$ “桃心尖”朝向对应坐标轴的**正半轴**。
> >     *   方程中是 **`-`** 号 $\implies$ “桃心尖”朝向对应坐标轴的**负半轴**。
> > 3.  **组合应用:**
> >     *   $r=a(1+\cos\theta)$: `cos`->X轴，`+`->正向。尖点在X轴正半轴。
> >     *   $r=a(1+\sin\theta)$: `sin`->Y轴，`+`->正向。尖点在Y轴正半轴。
> >     *   $r=a(1-\cos\theta)$: `cos`->X轴，`-`->负向。尖点在X轴负半轴。
> >     *   $r=a(1-\sin\theta)$: `sin`->Y轴，`-`->负向。尖点在Y轴负半轴。

> [!example]- 战术四：双纽线 (“对称轴”判定法)
> 
> > [!todo]
> > *   **方程① (cos型):** $r^2 = a^2 \cos(2\theta)$ 或 $(x^2+y^2)^2 = a^2(x^2-y^2)$
> > *   **方程② (sin型):** $r^2 = a^2 \sin(2\theta)$ 或 $(x^2+y^2)^2 = 2a^2xy$
> > *   **核心考点：** 图像绘制、围成面积。
> 
> > [!TIP] **战术决策与执行**
> > 1.  **识别方程① (cos型):**
> >     *   **核心特征:** 笛卡尔方程为 $(x^2+y^2)^2 = a^2(x^2-y^2)$。
> >     *   **分析约束:** 右侧必须 $\ge 0$，故 $x^2 \ge y^2$，即 $|x| \ge |y|$。
> >     *   **判定图像:** 这意味着图像被限制在由直线 $y=x$ 和 $y=-x$ 所形成的、包含X轴的两个对顶区域内。
> >     *   **结论:** 图像是横躺的“∞”形，关于 **X轴对称**。
> > 2.  **识别方程② (sin型):**
> >     *   **核心特征:** 笛卡尔方程为 $(x^2+y^2)^2 = 2a^2xy$。
> >     *   **分析约束:** 右侧必须 $\ge 0$，故 $xy \ge 0$。这意味着 $x, y$ 同号。
> >     *   **判定图像:** 图像被限制在第一、三象限。观察方程，将 $x, y$ 互换，方程不变，说明图像关于直线 $y=x$ 对称。
> >     *   **结论:** 图像是斜躺的“∞”形，关于 **直线 $y=x$ 对称**。
