# 题目 :: 例6.16 - 利用二阶可导证明一阶导数有界

> [!question] 题目描述
> 设 `f(x)` 在 `[0, 1]` 上二阶可导，`f(0) = f(1)`，且 `|f''(x)| ≤ 2`。证明：对任意 `x ∈ (0, 1)`，有 `|f'(x)| ≤ 1`。
> *(注：音频中条件为 `|f''(x)| ≤ 2`，结论为 `|f'(x)| ≤ 1`，此处按讲座中的数值书写，其解题思想完全一致。)*

---

## 💡 知识点与思路分析

-   **题眼识别**: `二阶可导`, `f(0)=f(1)`, `|f''(x)|≤M` 型, 证明 `|f'(x)|` 有界。
-   **匹配战术**:
    1.  **[[解题方法论：细节处理]]** -> **脱胎换骨**：看到 `二阶可导`，立刻想到 **[[深度解析：由导数条件挖掘隐含信息]]**。
    2.  **核心武器**：泰勒公式（带拉格朗日余项）是必然选择，因为它能同时联系 `f(x)`, `f'(x)`, `f''(x)`。
-   **解题路径**:
    -   **易错路径**: 将 `f(x)` 在 `x=0` 和 `x=1` 处展开。这无法在同一个式子中引入 `f'(x)`，导致无法证明。
    -   **正确路径 (移花接木)**: 将**端点值** `f(0)` 和 `f(1)` 分别在**内部点 `x`** 处进行泰勒展开。这样就能构造出含有 `f'(x)` 的等式，为求解 `f'(x)` 创造条件。

---

## ✍️ 我的解答

#### 1. 挖掘隐含条件并选择展开策略
根据“脱胎换骨”原则，`二阶可导` 意味着我们可以使用带拉格朗日余项的一阶泰勒公式。
为证 `f'(x)`，我们将 `f(0)` 和 `f(1)` 在 `x` 点展开：
$$f(0) = f(x) + f'(x)(0-x) + \frac{f''(\xi_1)}{2!}(0-x)^2, \quad \xi_1 \in (0, x) \quad (1)$$
$$f(1) = f(x) + f'(x)(1-x) + \frac{f''(\xi_2)}{2!}(1-x)^2, \quad \xi_2 \in (x, 1) \quad (2)$$

#### 2. 构造核心等式 (移花接木)
用 `(2) - (1)` 式，并利用已知条件 `f(0) = f(1)`：
$$0 = [f'(x)(1-x) - (-x \cdot f'(x))] + [\frac{f''(\xi_2)}{2}(1-x)^2 - \frac{f''(\xi_1)}{2}x^2]$$
$$0 = f'(x) + \frac{1}{2}[f''(\xi_2)(1-x)^2 - f''(\xi_1)x^2]$$

#### 3. 求解并放缩
移项得到 `f'(x)` 的表达式：
$$f'(x) = \frac{1}{2}[f''(\xi_1)x^2 - f''(\xi_2)(1-x)^2]$$
两边取绝对值，利用三角不等式 `|a-b| ≤ |a| + |b|`：
$$|f'(x)| = \frac{1}{2}|f''(\xi_1)x^2 - f''(\xi_2)(1-x)^2| \le \frac{1}{2}[|f''(\xi_1)|x^2 + |f''(\xi_2)|(1-x)^2]$$
利用余项放缩条件 `|f''(t)| ≤ 2`：
$$|f'(x)| \le \frac{1}{2}[2 \cdot x^2 + 2 \cdot (1-x)^2] = x^2 + (1-x)^2$$

#### 4. 求二次函数最值
令 $g(x) = x^2 + (1-x)^2 = 2x^2 - 2x + 1$，`x ∈ (0, 1)`。
这是一个开口向上的抛物线，对称轴为 $x=0.5$。在 `x=0` 或 `x=1` 处取最大值 `1`。
因此，`|f'(x)| ≤ x^2 + (1-x)^2 ≤ 1`。证毕。

> [!danger]- 错题复盘

#### 1. 一句话错误本质
> 陷入思维定势，习惯于在已知点（0或1）展开泰勒公式，而没有根据待证目标 `f'(x)` 灵活选择 `x` 作为展开中心。

#### 2. 错误诊断清单
-   [ ] 概念不清
-   [x] 技巧不熟 (泰勒公式的灵活应用)
-   [x] 思路错误 (没有围绕目标构造表达式)
-   [ ] 条件遗漏
-   [ ] 计算失误

#### 3. 正确解法与反思
-   **核心洞察**：解题的关键是“移花接木”，将两个独立的泰勒展开式巧妙地联立，消去未知项 `f(x)`，从而解出目标 `f'(x)`。
-   **反思**：当标准方法（如在端点展开）走不通时，必须回到问题的本源：我要证什么？我该如何构造它？这正是“脱胎换骨”思想的体现。

#### 4. 可迁移的启发
> 1.  **决策模型优化**: 当中值定理证明题同时涉及 `f, f', f''` 且有端点条件时，要优先考虑**“将端点在内部点展开”**这一高级战术。
> 2.  **新技巧 get**: 掌握二次函数 $h(x) = x^2 + (1-x)^2$ 在 `[0, 1]` 上的最大值为 `1` 这个常用小结论，可以加速解题。
> 3.  **关联相似题**: 此题是中值定理与泰勒公式结合的典范，其思想可迁移至所有涉及高阶导数有界性的证明题。
