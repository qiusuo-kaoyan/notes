# 题目 :: 2020年真题 - 利用命题逻辑证明中值定理问题

> [!question] 题目描述
> 已知 $f(x)$ 在 $[0,2]$ 上具有连续导数，$f(0)=f(2)=0$, $M=\max_{x\in[0,2]}|f(x)|$。
> (1) 证明：存在 $\xi\in(0,2)$，使得 $|f'(\xi)| \ge M$。
> (2) 若对任意 $x\in(0,2)$，有 $|f'(x)| \le M$，证明：$M=0$。

---

## 💡 知识点与思路分析

-   **题眼识别**:
    -   **(1)问**: `存在ξ` (特称命题)，`[0,2]` 大区间，`f(0)=f(2)=0` 端点信息，`M` (最值)。
    -   **(2)问**: `任意x` (全称命题)，`若...则...` (蕴含式命题)。
-   **匹配战术**:
    -   **(1)问**: [[拉格朗日中值定理]] + **区间划分思想**。直接在[0,2]上用定理无法引入M，必须找到M的取值点x₀，并以x₀为界划分区间[0,x₀]和[x₀,2]进行讨论。
    -   **(2)问**: **命题逻辑分析**。第二问的条件 `∀x∈(0,2), |f'(x)|≤M` 与第一问的结论 `∃ξ∈(0,2), |f'(ξ)|≥M` 存在深刻的逻辑关联。
-   **解题路径**:
    1.  **(1)问**: 设 $M=|f(x_0)|, x_0 \in [0,2]$。对 $x_0$ 的位置进行分情况讨论（核心是利用区间长度小于等于1）。
        -   若 $x_0 \in (0,1]$，在 $[0, x_0]$ 上对 $f(x)$ 用拉格朗日中值定理，有 $f'(\xi_1) = \frac{f(x_0)-f(0)}{x_0-0} = \frac{f(x_0)}{x_0}$。取绝对值 $|f'(\xi_1)| = \frac{|f(x_0)|}{x_0} = \frac{M}{x_0} \ge M$ (因 $0 < x_0 \le 1$)。
        -   若 $x_0 \in (1,2)$，在 $[x_0, 2]$ 上用拉格朗日中值定理，有 $f'(\xi_2) = \frac{f(2)-f(x_0)}{2-x_0} = \frac{-f(x_0)}{2-x_0}$。取绝对值 $|f'(\xi_2)| = \frac{|f(x_0)|}{2-x_0} = \frac{M}{2-x_0} > M$ (因 $0 < 2-x_0 < 1$)。
        -   综合得证。
    2.  **(2)问**: 核心是**利用(1)的结论反过来约束(2)的条件**。
        -   (1)证明了：只要 $x_0 \ne 1$，总能找到一个子区间，使得该区间内 `∃ξ` 满足 $|f'(\xi)| > M$ (严格大于，因为分母小于1)。这是一个蕴含式命题 `P: (x_0 \ne 1) ⇒ (∃ξ, |f'(\xi)| > M)`。
        -   (2)给的条件是：`Q: ∀x∈(0,2), |f'(x)| ≤ M`。
        -   `Q` 等价于 `¬(∃x, |f'(x)| > M)`。
        -   这两个命题 `P` 和 `Q` 要同时为真，唯一的可能性就是 `P` 的前提 `(x_0 \ne 1)` 为假。这就是纯粹的逻辑推演。
        -   因此，必有 $x_0 = 1$，即得到关键隐含条件：$M=|f(1)|$。
        -   再利用 $f(1)$ 是最值点且在区间内部，由[[费马定理]]知 $f'(1)=0$。
        -   构造辅助函数或利用(2)的条件和 $f(1)$ 的信息证明 $M=0$。

---

## ✍️ 我的解答

#### 错误解法（保留）

-   **第一问**: 直接在 `[0,2]` 上用罗尔定理，得到 `f'(c)=0`，无法与M建立联系。或用拉格朗日中值定理，得到 `f(x)-f(0) = f'(c)x`，推出 $|f'(c)| = |f(x)/x|$，无法保证 `≥M`。
-   **第二问**: 完全看不懂条件 `|f'(x)|≤M` 和第一问的关系，陷入死循环，无法推进。

#### 正确解法

如“解题路径”所述，核心在于通过命题逻辑的分析，由第一问和第二问的条件对冲，推导出 $M=|f(1)|$ 这一至关重要的“题眼”。

> [!danger]- 错题复盘

#### 1. 一句话错误本质
> **只看到了微积分（中值定理），没看到其下的命题逻辑骨架，导致无法理解第二问的条件是从何而来、向何而去。**

#### 2. 错误诊断清单
-   [ ] 概念不清
-   [x] 技巧不熟 (区间划分、命题逻辑分析)
-   [ ] 计算失误
-   [ ] 条件遗漏
-   [x] 思路错误 (未能将两问关联，未能从条件对冲中榨取信息)

#### 3. 正确解法与反思
-   **第一问的反思**: 大区间问题，优先考虑切分。切分的依据通常是极值点、最值点或特殊点。
-   **第二问的反思**: 这不是一个孤立的证明题，而是对第一问结论的深度应用。出题人通过给出一个与第一问“几乎”矛盾的条件，来逼迫我们找出那个唯一的“不矛盾”的特殊情况 ($x_0=1$)。
-   **逻辑的力量**: `(1)`的结论可以表述为 `P: (x₀≠1) ⇒ (∃ξ, |f'(ξ)|>M)`。`(2)`的条件是 `Q: ∀x, |f'(x)|≤M`，它的等价形式是 `¬(∃x, |f'(x)|>M)`。当 `P` 和 `Q` 同时为真时，只能是 `P` 的前提 `(x₀≠1)` 为假。

#### 4. 可迁移的启发
> 1.  **决策模型优化**: 遇到多问压轴题，默认各问之间存在强关联。后一问的条件很可能是对前一问结论的“应用”、“特例化”或“逻辑反转”。
> 2.  **新技巧 get**: 学会“反向利用结论”：用已证的结论去“审判”新给的条件，从中榨取出隐藏信息。
> 3.  **关联相似题**: 所有“先证存在，再给全体”模式的题目，都可能隐藏着类似的逻辑陷阱。
