> [!abstract] 战术概览 (Tactical Overview)
> 
> > [!summary]+ 战术 A：常规链式求导 (阵地消耗战)
> > *   **一句话作用：** 对一阶导数表达式 $y' = -\frac{F'_x}{F'_y}$ 再次关于 `x` 求导，通过复杂的商式和链式法则计算二阶导。
> > *   **核心识别码：** 隐函数求二阶导的标准流程。
> > *   **战术定位：** 常规陆军。理论上万能，但在计算极值问题时，过程冗长、极易出错，效率低下。
> 
> > [!summary]+ 战术 B：驻点秒杀公式 (外科手术打击)
> > *   **一句话作用：** 在隐函数的**驻点**（即 $y'=0$ 的点），利用一个“神仙公式”直接秒杀二阶导数的值，极大简化计算。
> > *   **核心识别码：** 题目要求**判断隐函数极值**，或在已知 $y'=0$ 的前提下求 $y''$。
> > *   **战术定位：** 为极值判定量身定制的“刺客匕首”。其威力源于对驻点特性 ($y'=0 \implies F'_x=0$) 的深刻洞察。

## 深度拆解：外科手术打击法的作战原理与流程

### 核心原理：驻点条件下的极限简化
- **常规武器库：** $y' = -\frac{F'_x}{F'_y}$
- **驻点情报：** 在驻点处，我们有 $y' = 0$。假设 $F'_y \ne 0$，则必然有 $F'_x = 0$。
- **武器升级：** 对 $y'$ 进行常规求导，会得到一个极其复杂的表达式，其分子中含有 $F'_x$ 因子。
  $$ y'' = -\frac{(F''_{xx} + F''_{xy}y')F'_y - F'_x(\dots)}{(F'_y)^2} $$
- **降维打击：** 将驻点情报 ($y'=0$ 和 $F'_x=0$) 代入上述复杂表达式：
  *   $F''_{xy}y'$ 项变为 0。
  *   $-F'_x(\dots)$ 整个项变为 0。
  *   表达式瞬间坍缩，只剩下一项非零部分。
- **秒杀公式诞生：**
  > [!tip] 隐函数驻点二阶导终极武器
  > 若由方程 $F(x,y)=0$ 确定的隐函数 $y=y(x)$ 在某点 $(x_0, y_0)$ 处为**驻点** (即 $y'(x_0)=0$)，则该点的二阶导数为：
  > $$ y''(x_0) = - \frac{F''_{xx}(x_0, y_0)}{F'_y(x_0, y_0)} $$

### “三步必杀”作战流程
1.  **锁定目标 (寻找驻点)：**
    *   计算 $F'_x$ 和 $F'_y$。
    *   令 $y' = -\frac{F'_x}{F'_y} = 0$，解出 $F'_x = 0$。
    *   将 $F'_x=0$ 与原方程 $F(x,y)=0$ 联立，解出所有驻点的坐标 $(x_0, y_0)$。
2.  **准备弹药 (计算偏导)：**
    *   计算二阶偏导 $F''_{xx}(x,y)$。
    *   将驻点坐标 $(x_0, y_0)$ 代入，得到 $F''_{xx}(x_0, y_0)$ 和 $F'_y(x_0, y_0)$ 的值。
3.  **执行秒杀 (代入公式)：**
    *   将计算出的数值代入秒杀公式，直接得到 $y''(x_0)$ 的值。

> [!example]- 【经典战例】隐函数求极值
> 
> > [!todo]
> > **题目 (2014 数一)：** 设函数 $y=f(x)$ 由方程 $y^3+xy^2+x^2y-x^3-3=0$ 确定，求 $f(x)$ 的极值。
> 
> > [!TIP] **战术决策与执行**
> > 1.  **锁定目标 (寻找驻点)：**
> >     *   令 $F(x,y) = y^3+xy^2+x^2y-x^3-3$。
> >     *   $F'_x = y^2+2xy-3x^2$。
> >     *   $F'_y = 3y^2+2xy+x^2$。
> >     *   令 $F'_x = y^2+2xy-3x^2 = (y+3x)(y-x)=0$，得到 $y=x$ 或 $y=-3x$。
> >     *   将 $y=x$ 代入原方程得 $x=-1$，故驻点为 $(-1, -1)$。
> >     *   将 $y=-3x$ 代入原方程得 $x=1$，故驻点为 $(1, -3)$。
> > 2.  **准备弹药 (计算偏导)：**
> >     *   $F''_{xx} = 2y-6x$。
> >     *   **在驻点 `(1, -3)` 处：**
> >         *   $F''_{xx}(1, -3) = 2(-3)-6(1) = -12$。
> >         *   $F'_y(1, -3) = 3(-3)^2+2(1)(-3)+1^2 = 27-6+1=22$。
> > 3.  **执行秒杀 (代入公式)：**
> >     *   $y''(1) = - \frac{F''_{xx}(1, -3)}{F'_y(1, -3)} = - \frac{-12}{22} = \frac{6}{11} > 0$。
> > 4.  **得出结论：** 在 $x=1$ 处取得**极小值** $f(1)=-3$。(另一驻点的判断同理)
> > > [!attention] **大题书写技巧**
> > > 在答题纸上，无需写出此公式。只需写出 $F'_x, F'_y, F''_{xx}$，找到驻点，然后**轻描淡写地**写一句：“在驻点处，有 $y' = 0$，将原方程两边对 x 求导两次，并将 $y'=0$ 代入，得 $y'' = ...$”，直接填入你在草稿纸上用公式算出的结果 $\frac{6}{11}$。
