# [[战术卡片：高阶导数求导的四大核心战法]]

![[求高阶导数方法总结.pdf]]

***
> [!abstract] 战术概览 (Tactical Overview)
> 
> > [!summary]+ 战法A：归纳法 (基础侦察)
> > *   **一句话作用：** 通过计算前几阶导数，洞察规律，推广至n阶导数。是处理**基础函数**的基石。
> > *   **核心识别码：** 函数结构单一，适合直接求导，如 $e^{ax+b}, \sin(ax+b), \frac{1}{ax+b}, \ln(ax+b)$。
> > *   **战术定位：** 所有战法的基础，必须熟练掌握五个常用公式的推导。
> 
> > [!summary]+ 战法B：莱布尼茨公式 (协同作战)
> > *   **一句话作用：** 精准处理**两函数乘积**的高阶导数，特别是“幂函数 × 其他函数”的王牌战术。
> > *   **核心识别码：** $f(x) = u(x)v(x)$，且其中一个函数（通常是幂函数）求有限次导后为0。
> > *   **战术定位：** 结构化的协同打击，又名“u(x)与v(x)的热血组合技”。
>
> > [!summary]+ 战法C：泰勒公式 (降维打击)
> > *   **一句话作用：** 通过将函数展开为幂级数，利用系数比对关系，间接秒杀在某点（尤其是 $x=0$）的n阶导数值。
> > *   **核心识别码：** 题目要求 $f^{(n)}(x_0)$，且 $f(x)$ 是能快速写出泰勒展开式的常见函数。
> > *   **战术定位：** 将微分问题转化为代数问题，是最高效的降维打击手段。
> > 核心公式:
> > $$f^{(n)}(x_0) = n! \cdot a_n$$
>
> > [!summary]+ 战法D：函数性质法 (情报战)
> > *   **一句话作用：** 利用函数的**奇偶性、周期性**等先验情报，在特定点（如 $x=0, x=2\pi$）直接判定导数值，实现不战而屈人之兵。
> > *   **核心识别码：** 函数是明显的奇/偶/周期函数，且求导点特殊。
> > *   **战术定位：** 最顶级的战术，依赖于对战场情报的敏锐洞察。

## 深度拆解 (In-depth Deconstruction)

> [!example]- 战法A：归纳法 (附常用n阶导数公式)
> 
> > [!TIP] **核心战术：先求三阶，再看规律。五个公式必须做到能随时手推。**
> > 1.  $(e^{ax+b})^{(n)} = a^n e^{ax+b}$
> > 2.  $[\sin(ax+b)]^{(n)} = a^n \sin(ax+b + \frac{n\pi}{2})$
> > 3.  $[\cos(ax+b)]^{(n)} = a^n \cos(ax+b + \frac{n\pi}{2})$
> > 4.  $(\frac{1}{ax+b})^{(n)} = \frac{(-1)^n a^n n!}{(ax+b)^{n+1}}$ (推导技巧：先写成 `(ax+b)^(-1)` 再求导)
> > 5.  $[\ln(ax+b)]^{(n)} = \frac{(-1)^{n-1} a^n (n-1)!}{(ax+b)^n}$ (推导技巧：求一阶导后，转化为公式4的 `n-1` 阶导问题)
> 
> > [!todo]
> > *   **【例1】(2007数二)** 设函数 $y = \frac{1}{2x+3}$，则 $y^{(n)}(0) = ?$
> > *   **【例2】(2010数二)** 函数 $y=\ln(1-2x)$ 在 $x=0$ 处的n阶导数 $y^{(n)}(0) = ?$
> 
> > [!TIP] **战术决策与执行**
> > *   **【例1】解析：**
> >     1.  **识别与套用：** 此为公式4的直接应用，其中 $a=2, b=3$。
> >     2.  **写出通式：** $y^{(n)}(x) = \frac{(-1)^n 2^n n!}{(2x+3)^{n+1}}$。
> >     3.  **代入求值：** $y^{(n)}(0) = \frac{(-1)^n 2^n n!}{3^{n+1}}$。
> > *   **【例2】解析：**
> >     1.  **识别与套用：** 此为公式5的直接应用，其中 $a=-2, b=1$。
> >     2.  **写出通式：** $y^{(n)}(x) = \frac{(-1)^{n-1} (-2)^n (n-1)!}{(1-2x)^n}$。
> >     3.  **化简符号：** $(-1)^{n-1}(-2)^n = (-1)^{n-1}(-1)^n 2^n = (-1)^{2n-1} 2^n = -2^n$。
> >     4.  **代入求值：** $y^{(n)}(0) = \frac{-2^n (n-1)!}{1^n} = -2^n(n-1)!$。

> [!example]- 战法B：莱布尼茨公式 (“热血组合技”)
> 
> > [!todo]
> > *   **【例3】(2024数二)** 已知函数 $f(x) = x^2(e^x-1)$，则 $f^{(5)}(1) = ?$
> > *   **【例4】(2015数二)** 函数 $f(x)=x^2 2^x$ 在 $x=0$ 处的n阶导数 $f^{(n)}(0) = ?$
> 
> > [!TIP] **战术决策与执行**
> > *   **【例3】解析：**
> >     1.  **战术简化：** $f(x) = x^2 e^x - x^2$。由于 $(-x^2)$ 从三阶导开始就为0，故 $f^{(5)}(x) = (x^2 e^x)^{(5)}$。
> >     2.  **设定角色：** 令 $u(x)=x^2, v(x)=e^x$。$u$ 求导三次后为0，只需展开三项。
> >     3.  **展开公式：** $(uv)^{(5)} = C_5^0 u v^{(5)} + C_5^1 u' v^{(4)} + C_5^2 u'' v^{(3)}$
> >     4.  **代入计算：** $f^{(5)}(x) = 1 \cdot x^2 e^x + 5 \cdot (2x) e^x + 10 \cdot (2) e^x = e^x(x^2+10x+20)$。
> >     5.  **代入求值：** $f^{(5)}(1) = e^1(1+10+20) = 31e$。
> > *   **【例4】解析：**
> >     1.  **设定角色：** 令 $u(x)=x^2, v(x)=2^x$。$u$ 求导三次后为0，只需展开三项。
> >     2.  **展开公式：** $f^{(n)}(x) = C_n^0 u v^{(n)} + C_n^1 u' v^{(n-1)} + C_n^2 u'' v^{(n-2)}$。
> >     3.  **关键情报：** 题目求的是 **$f^{(n)}(0)$**！
> >     4.  **战术优化 (先代值再计算)：**
> >         *   第一项含 $u(x)=x^2$，代入 $x=0$ 后为0。
> >         *   第二项含 $u'(x)=2x$，代入 $x=0$ 后为0。
> >         *   **只有第三项存活！**
> >     5.  **精算第三项：** $f^{(n)}(0) = C_n^2 \cdot u''(0) \cdot v^{(n-2)}(0) = \frac{n(n-1)}{2} \cdot 2 \cdot [2^x(\ln 2)^{n-2}]_{x=0}$。
> >     6.  **最终结果：** $f^{(n)}(0) = n(n-1) (\ln 2)^{n-2}$。

> [!example]- 战法C：泰勒公式 (系数比对)
> 
> > [!todo]
> > *   **【例5】(2010数二)** (同例2) 函数 $y=\ln(1-2x)$ 在 $x=0$ 处的n阶导数 $y^{(n)}(0) = ?$
> > *   **【例6】(2000数二)** 求函数 $f(x)=x^2\ln(1+x)$ 在 $x=0$ 处的n阶导数 $f^{(n)}(0) = ?$ ($n \ge 3$)
> 
> > [!TIP] **战术决策与执行**
> > *   **【例5】解析：**
> >     1.  **调用泰勒公式：** $\ln(1+u) = \sum_{k=1}^\infty \frac{(-1)^{k-1}}{k}u^k$。
> >     2.  **整体代换：** 令 $u=-2x$，则 $y = \sum_{n=1}^\infty \frac{(-1)^{n-1}}{n}(-2x)^n = \sum_{n=1}^\infty \frac{(-1)^{n-1}(-1)^n 2^n}{n}x^n = \sum_{n=1}^\infty -\frac{2^n}{n}x^n$。
> >     3.  **锁定系数：** $x^n$ 项的系数 $a_n = -\frac{2^n}{n}$。
> >     4.  **应用核心公式：** $y^{(n)}(0) = n! \cdot a_n = n! \cdot (-\frac{2^n}{n}) = -2^n(n-1)!$。
> > *   **【例6】解析：**
> >     1.  **展开基础函数：** $\ln(1+x) = x - \frac{x^2}{2} + \dots + \frac{(-1)^{k-1}}{k}x^k + \dots$
> >     2.  **乘以因子：** $f(x) = x^2 \ln(1+x) = x^3 - \frac{x^4}{2} + \dots + \frac{(-1)^{k-1}}{k}x^{k+2} + \dots$
> >     3.  **定位目标 (核心陷阱)：** 我们要找的是 $x^n$ 项的系数 $a_n$。在 $f(x)$ 的展开式中，这意味着 $k+2=n$，即 $k=n-2$。
> >     4.  **逆向追踪系数：** 所以，$x^n$ 项是由 $\ln(1+x)$ 展开式中的 $x^{n-2}$ 项乘以 $x^2$ 得到的。$\ln(1+x)$ 中 $x^{n-2}$ 项的系数是 $\frac{(-1)^{(n-2)-1}}{n-2} = \frac{(-1)^{n-3}}{n-2}$。
> >     5.  **锁定系数：** $a_n = \frac{(-1)^{n-3}}{n-2}$。
> >     6.  **应用核心公式：** $f^{(n)}(0) = n! \cdot a_n = n! \cdot \frac{(-1)^{n-3}}{n-2} = (-1)^{n-3} n(n-1)(n-3)!$。

> [!example]- 战法D：函数性质法 (情报战)
> 
> > [!todo]
> > *   **【例7】(2017数一)** 已知函数 $f(x) = \frac{1}{1+x^2}$，则 $f^{(3)}(0) = ?$
> > *   **【例8】(2022数三)** 已知函数 $f(x) = e^{\sin x} + e^{-\sin x}$，则 $f'''(2\pi) = ?$
> 
> > [!TIP] **战术决策与执行**
> > *   **【例7】解析：**
> >     1.  **情报分析：** $f(-x) = \frac{1}{1+(-x)^2} = f(x)$，函数为**偶函数**。
> >     2.  **调用性质：** 偶函数的奇数阶导数是奇函数。故 $f^{(3)}(x)$ 是一个奇函数。
> >     3.  **直接结论：** 任何奇函数在 $x=0$ 点的值（若有定义）都为0。所以 $f^{(3)}(0)=0$。
> > *   **【例8】解析：**
> >     1.  **情报分析 (双重性质)：**
> >         *   **周期性：** $\sin(x+2\pi) = \sin x$，故 $f(x)$ 是以 $2\pi$ 为周期的函数。
> >         *   **奇偶性：** $f(-x) = e^{\sin(-x)} + e^{-\sin(-x)} = e^{-\sin x} + e^{\sin x} = f(x)$，函数为**偶函数**。
> >     2.  **调用性质 (组合拳)：**
> >         *   根据周期性，函数及其各阶导数都以 $2\pi$ 为周期。故 $f'''(2\pi) = f'''(0)$。
> >         *   根据奇偶性，偶函数 $f(x)$ 的三阶导数 $f'''(x)$ 是奇函数。
> >     3.  **直接结论：** $f'''(0)=0$。因此，$f'''(2\pi)=0$。
