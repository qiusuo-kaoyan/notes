### **第三部分：核心绝技——导数法快速确定等价无穷小阶数**

> **导语**：在处理反常积分时，最关键也最容易卡壳的一步，就是在**非零点**（如 $x \to a, a \neq 0$）进行等价无穷小替换。传统方法（换元、凑项、诱导公式）步骤繁琐且极易出错。本部分将详细讲解一个革命性的技巧：**利用求导来降维打击等价无穷小阶数的判断问题**。此方法原理简单、操作机械、速度极快，是拉开差距的关键。

#### **一、 核心技巧：导数法定阶法**

**问题场景**：需要判断函数 $f(x)$ 在 $x \to a$ 时的无穷小阶数，即寻找 $n$ 使得 $f(x) \sim C(x-a)^n$。

**传统方法回顾**：
1.  令 $t = x-a$ 进行换元。
2.  将被积函数中的 $x$ 替换为 $a+t$。
3.  利用三角函数的诱导公式、和差角公式或泰勒公式展开，化简为关于 $t$ 的表达式。
4.  找到 $t$ 的最低次幂，确定阶数。

**全新技巧：导数法定阶**
此方法的思想是：**函数无穷小的阶数，由其在瑕点处的第一个非零导数所决定。**

**操作流程**:
1.  确认函数在瑕点 $a$ 处为无穷小，即 $f(a)=0$。
2.  计算一阶导数 $f'(x)$，并代入瑕点 $a$ 计算 $f'(a)$。
    *   如果 $f'(a) \neq 0$，则 $f(x)$ 为**一阶**无穷小，即 $f(x) \sim f'(a)(x-a)$。
3.  如果 $f'(a) = 0$，则继续计算二阶导数 $f''(x)$，并代入瑕点 $a$ 计算 $f''(a)$。
    *   如果 $f''(a) \neq 0$，则 $f(x)$ 为**二阶**无穷小，即 $f(x) \sim \frac{f''(a)}{2!}(x-a)^2$。
4.  以此类推，如果 $f(a)=f'(a)=\dots=f^{(n-1)}(a)=0$，而 $f^{(n)}(a) \neq 0$，那么 $f(x)$ 就是**n阶**无穷小。

> **判敛实战应用**: 在判断敛散性时，我们只关心阶数 $n$，**可以完全忽略前面的非零常数系数**（如 $f'(a)$ 或 $\frac{f''(a)}{2!}$），因为常数不影响敛散性。我们只需要问自己一个问题：**“我对函数求导几次后，代入瑕点的值才不等于零？”** —— 求了 $n$ 次，它就是 $n$ 阶无穷小。

#### **二、 技巧原理：泰勒公式的降维打击**

这个“绝技”的背后，是我们早已学过的**泰勒公式**。一个函数 $f(x)$ 在 $x=a$ 处的泰勒展开式为：
$$
f(x) = f(a) + \frac{f'(a)}{1!}(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \dots + \frac{f^{(n)}(a)}{n!}(x-a)^n + o((x-a)^n)
$$
当我们处理无穷小量时，有 $f(a)=0$。此时：
*   若 $f'(a) \neq 0$，则展开式为 $f(x) = f'(a)(x-a) + O((x-a)^2)$。低阶无穷小是主要部分，高阶可以忽略。因此，$f(x) \sim f'(a)(x-a)$，为**一阶**无穷小。
*   若 $f'(a) = 0$ 但 $f''(a) \neq 0$，则展开式为 $f(x) = \frac{f''(a)}{2!}(x-a)^2 + O((x-a)^3)$。此时，二阶项成为主要部分。因此，$f(x) \sim \frac{f''(a)}{2!}(x-a)^2$，为**二阶**无穷小。

这个方法将一个需要凑项和变形的**等价替换问题**，转化成了一个机械的**求导问题**，实现了“降维打击”。

#### **三、 专题例题精讲**

**题目标注：例题5**
*   **题目**: $\int_0^{\pi/2} \frac{\tan^p x}{(x+1)^{1-p}} dx$ 收敛，求 $p$ 的范围。
*   **知识点类型**: 导数法定阶，多瑕点分析。
*   **核心思路**: 分别处理 $x=0$ 和 $x=\pi/2$ 两个瑕点。
*   **逐步推导**:
    1.  **瑕点 $x \to 0^+$**:
        *   $\tan x \sim x$，$(x+1)^{1-p} \to 1$。
        *   被积函数 $\sim x^p$。这是无界函数（当$p<0$）或正常函数。为收敛，需 $p<1$ (当$p>0$时)，或指数 $-p<1$（当$p<0$时）。综合为 $p<1$。**修正：讲座中此处推导有口误**，实际应为$\frac{\tan^p x}{(x+1)^{1-p}} \sim \frac{x^p}{1} = \frac{1}{x^{-p}}$。Q积分收敛要求 $-p < 1 \implies p>-1$。
    2.  **瑕点 $x \to (\pi/2)^-$**:
        *   $(x+1)^{1-p} \to (\pi/2+1)^{1-p}$ (非零常数，忽略)。
        *   关键是分析 $\tan x$ 这个无穷大的阶数。$\tan x = \frac{\sin x}{\cos x}$，其中 $\sin x \to 1$ (忽略)。问题转化为分析 $\frac{1}{\cos x}$ 的阶数。
        *   **使用导数法判断 $\cos x$ 在 $x=\pi/2$ 处的阶数**:
            *   令 $f(x) = \cos x$。$f(\pi/2) = 0$。
            *   $f'(x) = -\sin x$。$f'(\pi/2) = -1 \neq 0$。
            *   求导**1**次不为零 $\implies$ 它是**1**阶无穷小。即 $\cos x \sim C(x-\pi/2)$。
        *   所以 $\tan^p x \sim \frac{1}{[\cos x]^p} \sim \frac{C}{(x-\pi/2)^p}$。
        *   根据 Q-积分判据，要求指数 $p < 1$。
    3.  **综合结论**: 联立 $p>-1$ 和 $p<1$，得到 $-1 < p < 1$。**（讲座中对$x=0$的分析有误，此处为修正后结果）**

**题目标注：例题6 (张宇十八讲)**
*   **题目**: $\int_0^{\pi/2} \left[ \ln(\sin x) + \frac{1}{\ln(\sin x)} \right] dx$ 的敛散性。
*   **知识点类型**: 导数法定阶，对数性质。
*   **核心思路**: 分别处理 $x=0$ 和 $x=\pi/2$ 两个瑕点。
*   **逐步推导**:
    1.  **瑕点 $x \to 0^+$**:
        *   $\sin x \sim x$。被积函数等价于 $\ln x + \frac{1}{\ln x}$。
        *   $\int \ln x dx$ 和 $\int \frac{1}{\ln x} dx$ 在 $x=0$ 附近都是收敛的（前者可分部积分，后者被积函数有界）。故此瑕点处收敛。==(无论指数是多少, $ln^q$ 在 0 必收敛,  因为它变大的速度太慢了! 在♾️处必发散, 因为 $\frac{1}{ln^q}$ 变小的速度太慢了!)==
    2.  **瑕点 $x \to (\pi/2)^-$**:
        *   $\sin x \to 1$，则 $\ln(\sin x) \to \ln 1 = 0$。
        *   第一部分 $\ln(\sin x)$ 是有界的，积分存在。
        *   关键看第二部分 $\frac{1}{\ln(\sin x)}$。我们需要判断分母 $\ln(\sin x)$ 的无穷小阶数。
        *   因为 $\ln(1+u) \sim u$，所以 $\ln(\sin x) = \ln(1 + (\sin x - 1)) \sim \sin x - 1$。
        *   **使用导数法判断 $f(x) = \sin x - 1$ 在 $x=\pi/2$ 处的阶数**:
            *   $f(\pi/2)=0$。
            *   $f'(x)=\cos x$。$f'(\pi/2) = 0$。
            *   $f''(x)=-\sin x$。$f''(\pi/2) = -1 \neq 0$。
            *   求导**2**次不为零 $\implies$ 它是**2**阶无穷小。
        *   所以 $\ln(\sin x) \sim C(x-\pi/2)^2$。
        *   被积函数第二部分 $\sim \frac{1}{C(x-\pi/2)^2}$。
        *   根据 Q-积分，指数 $2 \ge 1$，故发散。
    3.  **综合结论**: 只要有一个瑕点发散，整个积分就**发散**。

**题目标注：例题8 (李永乐/李艳芳模拟题)**
*   **题目**: 设 $\alpha>0, \beta>0$，若 $\int_{\pi/2}^{\pi} \frac{1}{(\cos x)^\alpha (1+\cos x)^\beta} dx$ 收敛，求 $\alpha, \beta$ 范围。
*   **知识点类型**: 导数法定阶在三角函数中的高级应用。
*   **核心思路**: 分别处理瑕点 $x=\pi/2$ 和 $x=\pi$。
*   **逐步推导**:
    1.  **瑕点 $x \to (\pi/2)^+$**:
        *   $1+\cos x \to 1$ (忽略)。被积函数 $\sim \frac{1}{(\cos x)^\alpha}$。
        *   判断 $\cos x$ 在 $x=\pi/2$ 的阶数：$f(x)=\cos x$, $f'(\pi/2)=-1 \neq 0$，**1**阶。
        *   函数 $\sim \frac{1}{C(x-\pi/2)^\alpha}$。Q-积分收敛要求 $\alpha < 1$。
    2.  **瑕点 $x \to \pi^-$**:
        *   $\cos x \to -1$ (忽略)。被积函数 $\sim \frac{1}{(1+\cos x)^\beta}$。
        *   判断 $1+\cos x$ 在 $x=\pi$ 的阶数：$g(x)=1+\cos x$, $g'(\pi)=0$, $g''(\pi)= -(-\cos \pi) = 1 \neq 0$，**2**阶。
        *   函数 $\sim \frac{1}{C(x-\pi)^{2\beta}}$。Q-积分收敛要求 $2\beta < 1 \implies \beta < 1/2$。
    3.  **综合结论**: 联立 $\alpha<1$ 和 $\beta < 1/2$，再结合题设 $\alpha>0, \beta>0$。
        *   最终范围：$0 < \alpha < 1, 0 < \beta < 1/2$。**(选项C)**
