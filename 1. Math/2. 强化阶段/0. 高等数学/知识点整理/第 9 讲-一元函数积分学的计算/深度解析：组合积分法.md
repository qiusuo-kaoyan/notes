## 核心理论：组合积分法的战略蓝图

> [!abstract] 核心思想 (一句话总结)
> 当目标积分 `I` 难以直接求解时，通过人为构造一个与其“对偶”或“互补”的积分 `J`，转而求解它们的线性组合（如 `I+J` 与 `I-J`），从而建立关于 `I` 和 `J` 的方程组，最终解出目标积分 `I`。该方法在三角函数积分中尤为强大。

### **作战纲领：组合积分法的四步决策流程**

1.  **【侦察】识别战机：** 什么时候应该考虑使用组合积分法？
    *   **信号一：** 被积函数是三角函数的有理分式。
    *   **信号二：** 分母通常是 `a*sin(x) + b*cos(x)` 或其幂次的形式，结构较为稳定。
    *   **信号三：** 分子是 `sin(x)`, `cos(x)` 或它们的幂次，存在“配对”的可能性。

2.  **【配对】构造搭档：** 如何找到与 `I` 相配对的积分 `J`？（**首要问题**）
    *   **策略一（互换）：** 如果 `I` 的分子是 `sin(x)`，则 `J` 的分子可设为 `cos(x)`。
    *   **策略二（幂次互补）：** 如果 `I` 的分子是 `sin^n(x)`，则 `J` 的分子可设为 `cos^n(x)`。
    *   **策略三（内部拆分）：** 有时 `I` 本身就可以拆分为两个具有配对潜力的积分。

3.  **【组合】设计战术：** 如何对 `I` 和 `J` 进行组合？（**核心问题**）
    *   **目标A（代数化简）：** 通过组合（如 `I+J`）使分子分母能够约分，或利用 `sin^2(x)+cos^2(x)=1`、立方和/差等公式，使被积函数变为常数或简单函数。
    *   **目标B（导数关联）：** 通过组合（如 `I-J` 或带权重的 `aJ-bI`）使分子恰好是分母的导数（或其常数倍），从而构造出 `∫(f'(x)/f(x))dx = ln|f(x)|+C` 或 `∫(f'(x)/f(x)^n)dx` 的形式。
    *   **权重选择：** 当分母为 `a*sin(x)+b*cos(x)` 时，组合系数往往与 `a`, `b` 相关，需要以实现上述两大目标为导向来决定。

4.  **【求解】解方程组：**
    *   求解由两个组合积分构成的线性方程组，得到 `I` 的表达式。

> [!TIP]+ 辅助武器库：不可或缺的三角函数公式
> *   **辅助角公式：** $a\sin x + b\cos x = \sqrt{a^2+b^2} \sin(x+\varphi)$, 其中 $\tan\varphi = \frac{b}{a}$. 这是处理分母的利器。
> *   **平方关系：** $( \sin x \pm \cos x )^2 = \sin^2x + \cos^2x \pm 2\sin x\cos x = 1 \pm \sin(2x)$. 这是建立 `sin(x)±cos(x)` 与 `sin(x)cos(x)` 之间联系的桥梁。
> *   **立方和/差公式：**
>     *   $a^3+b^3 = (a+b)(a^2-ab+b^2)$
>     *   $a^3-b^3 = (a-b)(a^2+ab+b^2)$

## 例题解析

> [!example]- 【例题 1 & 2】基础组合与带权组合
> ### **【例题 1】 $\int \frac{\sin x}{\sin x + \cos x} dx$**
> 
> *   **战略分析：** 分子是`sin(x)`，分母是`sin(x)+cos(x)`。这是最典型的无权重组合模型。配对积分`J`的分子应为`cos(x)`。组合策略预计为`I+J`（化为常数1）和`I-J`（构造`f'/f`）。
> *   **步骤一：构造配对积分 I & J**
>     $I = \int \frac{\sin x}{\sin x + \cos x} dx$, $\quad J = \int \frac{\cos x}{\sin x + \cos x} dx$
> *   **步骤二：计算组合**
>     1.  **求和：**
>         $$I+J = \int \frac{\sin x + \cos x}{\sin x + \cos x} dx = \int 1 dx = x + C_1$$
>     2.  **求差：**
>         $$I-J = \int \frac{\sin x - \cos x}{\sin x + \cos x} dx$$
>         注意到分母的导数是 `cos(x) - sin(x)`，分子是其相反数。
>         $$I-J = -\int \frac{\cos x - \sin x}{\sin x + \cos x} dx = -\int \frac{d(\sin x + \cos x)}{\sin x + \cos x} = -\ln|\sin x + \cos x| + C_2$$
> *   **步骤三：求解原积分**
>     解方程组，$(I+J) + (I-J) = 2I$，所以 $I = \frac{1}{2}[(I+J) + (I-J)]$。
>     $$I = \frac{1}{2}(x - \ln|\sin x + \cos x|) + C$$
> 
> ### **【例题 2】 $\int \frac{\sin x}{a\sin x + b\cos x} dx$**
> 
> *   **战略分析：** 例1的推广，分母带有权重`a`和`b`。直接`I+J`和`I-J`无法有效化简。组合时必须引入权重，目标依然是“代数化简”和“导数关联”。
> *   **步骤一：构造配对积分 I & J**
>     $I = \int \frac{\sin x}{a\sin x + b\cos x} dx$, $\quad J = \int \frac{\cos x}{a\sin x + b\cos x} dx$
> *   **步骤二：计算带权组合**
>     1.  **组合A（凑分母，代数化简）：** 目标是使分子凑成 `a*sin(x) + b*cos(x)`。
>         $$aI+bJ = \int \frac{a\sin x + b\cos x}{a\sin x + b\cos x} dx = \int 1 dx = x + C_1 \quad \cdots (1)$$
>     2.  **组合B（凑分母导数）：** 分母 `a*sin(x) + b*cos(x)` 的导数是 `a*cos(x) - b*sin(x)`。
>         $$aJ-bI = \int \frac{a\cos x - b\sin x}{a\sin x + b\cos x} dx = \int \frac{d(a\sin x + b\cos x)}{a\sin x + b\cos x} = \ln|a\sin x + b\cos x| + C_2 \quad \cdots (2)$$
> *   **步骤三：求解原积分（解线性方程组）**
>     将 I, J 视为变量，解关于I, J的方程组。为求I，需消去J。
>     $(1) \times a \implies a^2I + abJ = ax + aC_1$
>     $(2) \times b \implies abJ - b^2I = b\ln|...| + bC_2$
>     两式相减：
>     $(a^2 - (-b^2))I = ax - b\ln|a\sin x + b\cos x| + C'$
>     $$(a^2+b^2)I = ax - b\ln|a\sin x + b\cos x| + C'$$
>     最终：
>     $$I = \frac{1}{a^2+b^2} (ax - b\ln|a\sin x + b\cos x|) + C$$

> [!example]- 【例题 3 & 4】分子为二次幂的组合
> ### **【例题 3】 $\int \frac{\sin^2 x}{\sin x + \cos x} dx$**
> 
> *   **战略分析：** 分子是二次幂。配对`J`的分子应为`cos^2(x)`。`I+J`可利用`sin^2+cos^2=1`简化。`I-J`可利用平方差公式`sin^2-cos^2`化简。
> *   **步骤一：构造配对积分 I & J**
>     $I = \int \frac{\sin^2 x}{\sin x + \cos x} dx$, $\quad J = \int \frac{\cos^2 x}{\sin x + \cos x} dx$
> *   **步骤二：计算组合**
>     1.  **求和：**
>         $$I+J = \int \frac{\sin^2 x + \cos^2 x}{\sin x + \cos x} dx = \int \frac{1}{\sin x + \cos x} dx$$
>         使用辅助角公式：
>         $$I+J = \int \frac{1}{\sqrt{2}\sin(x+\frac{\pi}{4})} dx = \frac{1}{\sqrt{2}}\int \csc(x+\frac{\pi}{4}) d(x+\frac{\pi}{4})$$
>         $$I+J = \frac{1}{\sqrt{2}} \ln|\csc(x+\frac{\pi}{4}) - \cot(x+\frac{\pi}{4})| + C_1$$
>     2.  **求差：**
>         $$I-J = \int \frac{\sin^2 x - \cos^2 x}{\sin x + \cos x} dx = \int \frac{(\sin x - \cos x)(\sin x + \cos x)}{\sin x + \cos x} dx$$
>         $$I-J = \int (\sin x - \cos x) dx = -\cos x - \sin x + C_2$$
> *   **步骤三：求解原积分**
>     $I = \frac{1}{2}[(I+J) + (I-J)]$
>     $$I = \frac{1}{2} \left( \frac{1}{\sqrt{2}} \ln|\csc(x+\frac{\pi}{4}) - \cot(x+\frac{\pi}{4})| - \cos x - \sin x \right) + C$$
> 
> ### **【例题 4】 $\int \frac{\sin^2 x}{a\sin x + b\cos x} dx$**
> 
> *   **战略分析：** 例3的带权推广。`I+J`与例3类似。关键在于`I-J`的组合，直接相减无法约分，需要设计权重，使得分子在使用平方差后可以与分母约分。
> *   **步骤一：构造配对积分 I & J**
>     $I = \int \frac{\sin^2 x}{a\sin x + b\cos x} dx$, $\quad J = \int \frac{\cos^2 x}{a\sin x + b\cos x} dx$
> *   **步骤二：计算组合**
>     1.  **求和 (无权重)：**
>         $$I+J = \int \frac{1}{a\sin x + b\cos x} dx = \frac{1}{\sqrt{a^2+b^2}} \ln|\csc(x+\varphi) - \cot(x+\varphi)| + C_1$$
>         (其中 $\tan\varphi = b/a$)
>     2.  **带权组合 (构造可约分的分子)：** 目标是构造出 `(b*cos)^2 - (a*sin)^2` 以便和分母的 `b*cos + a*sin` 形式关联。
>         $$b^2 J - a^2 I = \int \frac{b^2\cos^2 x - a^2\sin^2 x}{a\sin x + b\cos x} dx$$
>         $$= \int \frac{(b\cos x - a\sin x)(b\cos x + a\sin x)}{a\sin x + b\cos x} dx$$
>         $$= \int (b\cos x - a\sin x) dx = b\sin x + a\cos x + C_2$$
> *   **步骤三：求解原积分**
>     解关于I, J的方程组，过程略，最终可得到I的表达式。

> [!example]- 【例题 5 & 6】高次幂与内部分解
> ### **【例题 5】 $\int \frac{\cos^3 x}{\sin x + \cos x} dx$**
> 
> *   **战略分析：** 分子为三次方。配对`J`的分子为`sin^3(x)`。`I+J`可利用立方和公式化简。`I-J`较为复杂，需要利用 `sin(x)cos(x)` 和 `sin(x)+cos(x)` 的关系进行代换。
> *   **步骤一：构造配对积分 I & J**
>     $I = \int \frac{\cos^3 x}{\sin x + \cos x} dx$, $\quad J = \int \frac{\sin^3 x}{\sin x + \cos x} dx$
> *   **步骤二：计算组合**
>     1.  **求和 (I+J)：**
>         $$I+J = \int \frac{\cos^3 x + \sin^3 x}{\sin x + \cos x} dx$$
>         利用立方和 $a^3+b^3=(a+b)(a^2-ab+b^2)$：
>         $$I+J = \int (\sin^2 x - \sin x\cos x + \cos^2 x) dx = \int(1-\sin x\cos x)dx$$
>         $$I+J = x - \frac{1}{2}\int \sin(2x)dx = x + \frac{1}{4}\cos(2x) + C_1$$
>         *(或者 $\int \sin x \cos x dx = \int \sin x d(\sin x) = \frac{1}{2}\sin^2 x + C'$)*
>     2.  **求差 (I-J)：**
>         $$I-J = \int \frac{\cos^3 x - \sin^3 x}{\sin x + \cos x} dx$$
>         利用立方差公式并化简：
>         $$I-J = \int \frac{(\cos x - \sin x)(1+\sin x\cos x)}{\sin x + \cos x} dx$$
>         此形式不易直接积分。**核心技巧：** 统一变量。将 `sin(x)cos(x)` 用 `sin(x)+cos(x)` 表示。
>         令 $t = \sin x + \cos x$, 则 $t^2 = 1+2\sin x\cos x \implies \sin x\cos x = \frac{t^2-1}{2}$。
>         同时, `(cos(x)-sin(x))dx` 可以凑成 `-d(sin(x)+cos(x)) = -dt`。
>         但这里分母不是 `cos(x)-sin(x)`，不能直接约分。
>         **讲座中的思路：** 将分子 `(cos(x)-sin(x))dx` 凑成 `d(sin(x)+cos(x))`。
>         $$I-J = \int \frac{1+\sin x\cos x}{\sin x + \cos x} d(\sin x+\cos x)$$
>         令 $t = \sin x + \cos x$:
>         $$I-J = \int \frac{1+\frac{t^2-1}{2}}{t} dt = \int \frac{\frac{t^2+1}{2}}{t} dt = \frac{1}{2}\int (t+\frac{1}{t})dt$$
>         $$= \frac{1}{2} (\frac{1}{2}t^2 + \ln|t|) + C_2 = \frac{1}{4}(\sin x+\cos x)^2 + \frac{1}{2}\ln|\sin x + \cos x| + C_2$$
> *   **步骤三：求解原积分**
>     $I = \frac{1}{2}[(I+J)+(I-J)]$，代入即可。
> 
> ### **【例题 6】 $\int \frac{\sin x + 2\cos x}{(\sqrt{3}\sin x + \cos x)^2} dx$**
> 
> *   **战略分析：** 分母是平方项，分子是线性组合。**核心思路：** 将原积分直接拆分为`I`和`2J`，而不是构造全新的`J`。
> *   **步骤一：内部分解构造 I & J**
>     设 $I = \int \frac{\sin x}{(\sqrt{3}\sin x + \cos x)^2} dx$, $\quad J = \int \frac{\cos x}{(\sqrt{3}\sin x + \cos x)^2} dx$.
>     则原积分为 $I+2J$。
> *   **步骤二：计算带权组合**
>     1.  **组合A (凑分母括号内项)：**
>         $$\sqrt{3}I+J = \int \frac{\sqrt{3}\sin x + \cos x}{(\sqrt{3}\sin x + \cos x)^2} dx = \int \frac{1}{\sqrt{3}\sin x + \cos x} dx$$
>         使用辅助角公式，分母为 $2\sin(x+\pi/6)$。
>         $$\sqrt{3}I+J = \frac{1}{2}\int\csc(x+\pi/6)dx = \frac{1}{2}\ln|\csc(x+\pi/6)-\cot(x+\pi/6)|+C_1 \quad \cdots (1)$$
>     2.  **组合B (凑分母括号内项的导数)：**
>         分母括号内 `(sqrt(3)sin(x)+cos(x))` 的导数是 `sqrt(3)cos(x)-sin(x)`。
>         $$\sqrt{3}J-I = \int \frac{\sqrt{3}\cos x - \sin x}{(\sqrt{3}\sin x + \cos x)^2} dx = \int \frac{d(\sqrt{3}\sin x + \cos x)}{(\sqrt{3}\sin x + \cos x)^2}$$
>         这是一个 `∫ du/u^2` 的形式。
>         $$\sqrt{3}J-I = -\frac{1}{\sqrt{3}\sin x + \cos x} + C_2 \quad \cdots (2)$$
> *   **步骤三：求解原积分**
>     解出 I 和 J，再计算 I+2J。

## 🧠 三步精炼复盘区 (Refinement & Review Zone)

### **第一步：知识晶体萃取 (Knowledge Crystal)**
> [!summary]+ 公式/概念卡片 (Core Card)
> *   **核心对象：** 难以直接积分的三角函数不定积分，特别是分式形式。
> *   **一句话作用：** 通过构造并组合一个“配对”积分，将复杂的原积分问题转化为一个更易求解的代数方程组问题。
> *   **关键前提/约束：** 能够找到一个合适的配对积分 `J`，且 `I` 和 `J` 的某种线性组合能被显著化简。
> *   **证明/推导精髓：** 利用三角函数的对称性、互补性以及代数恒等式，进行巧妙的结构重组。

### **第二步：横向辨析 (Comparison & Connection)**
> [!compare]- 横向辨析：与易混淆概念的对决
> | 对比维度 | **组合积分法** | **万能代换法 ($t=\tan\frac{x}{2}$)** |
> | :--- | :--- | :--- |
> | **适用范围** | 针对特定结构的三角函数积分，技巧性强。 | 理论上适用于所有三角函数有理式积分。 |
> | **计算过程** | 巧妙，依赖观察和构造，计算量可能很小。 | 流程化，但代换后通常导致复杂的多项式/有理函数积分，计算量大。 |
> | **核心思想** | **结构对称性**与**代数化简**。 | **统一变量**，将三角问题完全转化为代数问题。 |
> | **优缺点** | 优点：高效、优雅。缺点：不具有普适性，依赖经验。 | 优点：普适、可靠。缺点：计算繁琐、易错。 |

### **第三步：出题人视角 (Examiner's Perspective)**
> [!multi-column]
> > [!example|icon-name: "Target"] **题型母题 (Archetype)**
> > > 1.  分母为 `a*sin(x)+b*cos(x)`，分子为`sin(x)`或`cos(x)`。
> > > 2.  分母为 `a*sin(x)+b*cos(x)`，分子为`sin^2(x)`或`cos^2(x)`。
> > > 3.  分母为 `(a*sin(x)+b*cos(x))^n`，分子与分母括号内的导数或其本身有关。
> 
> > [!danger|icon-name: "Skull"] **命题陷阱 (Common Traps)**
> > > 4.  **权重错误：** 对带权分母的积分使用无权重组合（如直接`I+J`, `I-J`），导致无法化简。
> > > 5.  **目标不明：** 盲目组合，不清楚组合的目标是“代数化简”还是“导数关联”。
> > > 6.  **代数失误：** 在最后解关于 `I`, `J` 的线性方程组时出现计算错误。
> > > 7.  **基础遗忘：** 忘记辅助角公式、立方和/差等关键的三角或代数恒等式。
