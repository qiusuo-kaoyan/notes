# [[战术卡片：定积分计算技巧——对称性、周期性与区间变换]]

[[定积分计算技巧.pdf]]

***
> [!abstract] 战术概览 (Tactical Overview)
> 
> > [!summary]+ 战术A：对称区间积分 (阵地战核心)
> > *   **一句话作用：** 快速处理在对称区间 `[-a, a]` 上的积分，是利用函数对称性进行降维打击的经典战术。
> > *   **核心前提：** 积分区间必须是关于原点对称的 `[-a, a]`。
> > *   **证明/推导精髓：** 将被积函数 `f(x)` 拆分为偶部 `½[f(x)+f(-x)]` 和奇部 `½[f(x)-f(-x)]`。在对称区间上，奇部积分为0，偶部积分为2倍半区间积分。**核心公式：** $\int_{-a}^a f(x)dx = \int_0^a [f(x)+f(-x)]dx$。
> 
> > [!summary]+ 战术B：周期函数积分 (运动战精髓)
> > *   **一句话作用：** 处理周期函数在长区间或任意单周期长度区间上的积分问题。
> > *   **核心前提：** 被积函数 `f(x)` 必须是周期为 `T` 的周期函数。
> > *   **证明/推导精髓：** 1. **(周期叠加)** $\int_0^{nT} f(x)dx = n \int_0^T f(x)dx$。 2. **(平移不变)** $\int_a^{a+T} f(x)dx = \int_0^T f(x)dx$，即积分值只与**单周期长度**有关，与起点无关。
>
> > [!summary]+ 战术C：区间变换 & 几何意义 (战略机动与可视化)
> > *   **一句话作用：** 作为最基础、最灵活的战略机动，通过平移积分区间来简化被积函数，或主动创造出对称/周期结构。几何意义则能对特定函数进行“降维打击”。
> > *   **核心前提：** 适用于任何定积分。
> > *   **证明/推导精髓：** 换元法的直接体现。**口诀：区间减 c，变量加 c；区间加 c，变量减 c**。几何意义则是将积分计算转化为面积计算，对 `√r²-x²` (圆)、`|x|` (V形) 等图形有奇效。

## 深度拆解 (In-depth Deconstruction)

> [!example]- 【例4.1】战术C：几何意义的妙用
> 
> > [!todo]
> > *   **(1) $\int_0^a \sqrt{2ax-x^2}dx$**
> > *   **(2) $\int_{-1}^x (1-|t|)dt$ (x≥-1)**
> 
> > [!TIP] **战术决策与执行**
> > *   **问题(1)解析：**
> >     1.  **识别特征：** 被积函数 $\sqrt{2ax-x^2}$ 看到根号下有 $x$ 的二次式，立刻警惕是否能配方成圆。
> >     2.  **配方变形：** $y = \sqrt{2ax-x^2} \implies y^2 = -(x^2-2ax) \implies y^2 = -( (x-a)^2 - a^2 ) \implies (x-a)^2 + y^2 = a^2$。
> >     3.  **几何降维：** 该方程表示圆心在 `(a, 0)`、半径为 `a` 的圆。由于 $y \ge 0$，故为上半圆。积分区间 `[0, a]` 恰好是该上半圆的左半部分，即四分之一圆。
> >     4.  **直接秒杀：** 面积为 $\frac{1}{4}\pi a^2$。
> > *   **问题(2)解析：**
> >     1.  **识别特征：** 被积函数含绝对值 `|t|`，且积分上限 `x` 是变量，需分类讨论。直接积分复杂，考虑几何意义。
> >     2.  **图像分析：** 画出 `y = 1-|t|` 的图像，是一个顶点在 `(0,1)`，与x轴交于 `(-1,0)` 和 `(1,0)` 的等腰三角形。
> >     3.  **分类讨论 (按 `x` 的位置)：**
> >         *   **Case 1: `-1 ≤ x < 0`**：此时积分求的是 `[-1, x]` 区间内图形面积。这是一个上底为 `1-|x|=1+x`，下底为 `1-|-1|=0`，高为 `x-(-1)=x+1` 的直角梯形。面积为 $\frac{1}{2}(1+x)(x+1) = \frac{1}{2}(x+1)^2$。
> >         *   **Case 2: `x ≥ 0`**：此时积分求的是 `[-1, 0]` 和 `[0, x]` 两部分面积之和。左侧 `[-1, 0]` 是一个底和高均为1的三角形，面积为 $\frac{1}{2}$。右侧 `[0, x]` 是一个梯形，上底 `1-|x|=1-x`，下底 `1`，高 `x`。面积为 $\frac{1}{2}((1-x)+1)x = x - \frac{1}{2}x^2$。总面积为 $\frac{1}{2} + x - \frac{1}{2}x^2$。

> [!example]- 【例4.2】战术A：对称区间的三重境界
> 
> > [!todo]
> > *   **(1) $\int_{-1}^1 [(x+1)e^x + (x-1)e^{-x}]dx$** (显式奇偶性)
> > *   **(2) $\int_{-\pi/4}^{\pi/4} |\tan x|\arctan(e^x)dx$** (隐式奇偶性)
> > *   **(3) $\int_0^{10\pi} [e^{\arctan(x-5\pi)} - e^{\arctan(5\pi-x)} + x] |\cos(\frac{x}{2})| dx$** (构造对称性)
> 
> > [!TIP] **战术决策与执行**
> > *   **问题(1) - 显式奇偶性:**
> >     *   **法一 (拆解法):** $f(x) = (xe^x + e^x) + (xe^{-x} - e^{-x}) = x(e^x+e^{-x}) + (e^x-e^{-x})$。前者是 **偶×偶=偶**，后者是奇函数。等一下，`x`是奇函数，`e^x+e^-x`是偶函数，所以第一项是**奇函数**。第二项 `e^x-e^-x` 是奇函数。故整个被积函数是**奇函数**，在对称区间积分为 **0**。
> >     *   **法二 (公式法):** 计算 $f(x)+f(-x)$。$f(-x) = (-x+1)e^{-x} + (-x-1)e^{x}$。$f(x)+f(-x) = (x+1)e^x - (x+1)e^x + (x-1)e^{-x} - (x-1)e^{-x} = 0$。故积分为 $\int_0^1 0 dx = 0$。
> > *   **问题(2) - 隐式奇偶性:**
> >     1.  **识别特征:** 被积函数 `f(x)` 本身无奇偶性。但符合 `f(x)+f(-x)` 可能化简的模式。
> >     2.  **应用公式:** $I = \int_0^{\pi/4} [f(x)+f(-x)]dx$。
> >     3.  $f(x)+f(-x) = |\tan x|\arctan(e^x) + |-\tan x|\arctan(e^{-x}) = |\tan x|(\arctan(e^x) + \arctan(e^{-x}))$。
> >     4.  **调用恒等式:** $\arctan u + \arctan(1/u) = \pi/2$ (当 $u>0$ 时)。因为 $e^x > 0$，所以括号内为 $\pi/2$。
> >     5.  **化简积分:** $I = \int_0^{\pi/4} \tan x \cdot \frac{\pi}{2} dx = \frac{\pi}{2} [-\ln|\cos x|]_0^{\pi/4} = -\frac{\pi}{2}(\ln(\frac{\sqrt{2}}{2}) - \ln(1)) = \frac{\pi}{4}\ln 2$。
> > *   **问题(3) - 构造对称性:**
> >     1.  **识别特征:** 区间 `[0, 10π]` 不对称，但被积函数中反复出现 `x-5π`，强烈暗示以 `5π` 为中心进行变换。
> >     2.  **战略机动 (区间变换):** 令 $t = x-5\pi$，则 $x=t+5\pi$，积分区间变为 `[-5π, 5π]`。
> >     3.  **代换被积函数:**
> >         *   $e^{\arctan(t)} - e^{\arctan(-t)} + (t+5\pi)$
> >         *   $|\cos(\frac{t+5\pi}{2})| = |\cos(\frac{t}{2}+\frac{5\pi}{2})| = |\cos(\frac{t}{2}+\frac{\pi}{2})| = |-\sin(\frac{t}{2})| = |\sin(\frac{t}{2})|$
> >     4.  **分解与歼灭:** 原积分变为 $\int_{-5\pi}^{5\pi} \underbrace{[ (e^{\arctan t} - e^{-\arctan t}) + t ]}_{\text{奇函数}} \underbrace{|\sin(\frac{t}{2})|}_{\text{偶函数}} dt + \int_{-5\pi}^{5\pi} \underbrace{5\pi |\sin(\frac{t}{2})|}_{\text{偶函数}} dt$。
> >     5.  第一部分（奇×偶=奇）积分为0。第二部分（偶函数）积分为 $2 \int_0^{5\pi} 5\pi |\sin(\frac{t}{2})| dt = 10\pi \int_0^{5\pi} |\sin(\frac{t}{2})| dt$。
> >     6.  **应用周期性:** $|\sin(t/2)|$ 的周期为 $T = \pi / (1/2) = 2\pi$。$5\pi$ 是 $2.5$ 个周期。
> >     7.  $\int_0^{5\pi} = \int_0^{4\pi} + \int_{4\pi}^{5\pi} = 2 \int_0^{2\pi} |\sin(t/2)|dt + \int_0^{\pi} |\sin(u/2)|du$。计算可得 $10\pi \cdot (2 \cdot 4 + 2) = 100\pi$。

> [!example]- 【例4.3】核心战例：周期性与对称性的巅峰配合
> 
> > [!todo]
> > *   **计算 $\int_0^{2\pi} \frac{1}{1+\cos^2 x}dx$**
> 
> > [!danger] **核心陷阱：牛顿-莱布尼茨公式的误用**
> > 该积分的不定积分为 `(1/√2)arctan(tan(x)/√2)`。但 `tan(x)` 在 `x=π/2, 3π/2` 处有间断点，导致原函数不连续，**严禁直接使用牛顿-莱布尼茨公式**。必须先通过技巧处理积分区间，避开间断点。
> 
> > [!TIP] **六路重装突击策略**
> > **第一阶段：从 `[0, 2π]` 压缩至 `2 * ∫[0, π]`**
> > *   **路径1 (周期性):** 被积函数 $f(x) = 1/(1+\cos^2x)$ 周期为 `π`。故 `∫[0,2π] = 2 * ∫[0,π]`。
> > *   **路径2 (区间变换):** 令 `t=x-π`，将 `[0, 2π]` 变为 `[-π, π]`。因被积函数是偶函数，故 `∫[-π,π] = 2 * ∫[0,π]`。
> > 
> > **第二阶段：从 `∫[0, π]` 压缩至 `2 * ∫[0, π/2]`**
> > *   **路径A (函数对称性):** $1/(1+\cos^2x) = 1/(2-\sin^2x)$，是 $g(\sin x)$ 的形式，其图像关于 `x=π/2` 对称。故 `∫[0,π] = 2 * ∫[0,π/2]`。
> > *   **路径B (区间变换):** 令 `t=x-π/2`，区间变为 `[-π/2, π/2]`。被积函数变为 `1/(1+\sin²t)`，是偶函数。故 `∫[-π/2,π/2] = 2 * ∫[0,π/2]`。
> > *   **路径C (区间再现):** $\int_0^\pi f(x)dx = \int_0^\pi f(\pi-x)dx$。此例中 $f(\pi-x)=f(x)$，无化简效果，但可引出路径A的思考。
> > 
> > **最终决战:**
> > 组合后得到 `I = 2 * 2 * ∫[0, π/2] dx/(1+cos²x) = 4 * ∫[0, π/2] dx/(1+cos²x)`。在此安全区间内，上下同除 `cos²x`：
> > $$ I = 4 \int_0^{\pi/2} \frac{\sec^2 x}{\sec^2 x+1}dx = 4 \int_0^{\pi/2} \frac{d(\tan x)}{\tan^2 x+2} $$
> > 令 $u=\tan x$，积分变为 $4 \int_0^\infty \frac{du}{u^2+2} = 4 [\frac{1}{\sqrt{2}}\arctan(\frac{u}{\sqrt{2}})]_0^\infty = 4 \cdot \frac{1}{\sqrt{2}} \cdot \frac{\pi}{2} = \sqrt{2}\pi$。

> [!example]- 【例4.4】高维打击：在二重积分中应用定积分技巧
> 
> > [!todo]
> > *   **计算 $I = \iint_D |y-x^2|dxdy$, 其中 $D: |x|≤1, 0≤y≤2$**
> 
> > [!TIP] **战术决策与执行**
> > 1.  **区域对称性:** 积分区域 `D` 关于 `y` 轴对称，被积函数 `|y-x²|` 关于 `x` 是偶函数，故 `I = 2 \iint_{D_1} |y-x^2|dxdy`，其中 $D_1: 0 \le x \le 1, 0 \le y \le 2$。
> >     $$ I = 2 \int_0^1 dx \int_0^2 |y-x^2|dy $$
> > 2.  **处理内层积分 (定积分技巧):** 内层积分 $\int_0^2 |y-x^2|dy$ 需要对 `y` 讨论。
> >     *   **常规处理 (分段)：** 以 $y=x^2$ 为界，积分为 $\int_0^{x^2} (x^2-y)dy + \int_{x^2}^2 (y-x^2)dy$。计算繁琐。
> >     *   **战术机动 (区间变换+几何)：** 令 $t=y-x^2$，则 $y=t+x^2$，$dy=dt$。当 $y=0, t=-x^2$；当 $y=2, t=2-x^2$。内层积分变为：
> >         $$ \int_{-x^2}^{2-x^2} |t|dt $$
> >     3.  **几何降维:** 该积分的几何意义是 `y=|t|` 的图像上，从 `t=-x²` 到 `t=2-x²` 所围成的面积。即两个等腰直角三角形的面积和。
> >         *   左侧三角形面积：$\frac{1}{2}(\text{底}) \times (\text{高}) = \frac{1}{2}(x^2)(x^2) = \frac{1}{2}x^4$。
> >         *   右侧三角形面积：$\frac{1}{2}(2-x^2)(2-x^2) = \frac{1}{2}(2-x^2)^2$。
> >     4.  **代回外层积分:**
> >         $$ I = 2 \int_0^1 [\frac{1}{2}x^4 + \frac{1}{2}(2-x^2)^2] dx = \int_0^1 [x^4 + (4-4x^2+x^4)] dx $$
> >         $$ I = \int_0^1 (2x^4 - 4x^2 + 4) dx = [ \frac{2}{5}x^5 - \frac{4}{3}x^3 + 4x ]_0^1 = \frac{2}{5} - \frac{4}{3} + 4 = \frac{46}{15} $$
