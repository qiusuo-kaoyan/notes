***
这道题主要考察了常见奇偶函数的判别、奇偶性的运算性质，以及利用奇偶性判断对称区间上的定积分为零的条件。

**题目：** 利用奇偶性判断对称区间定积分是否为零（奇偶函数、奇偶性运算）
![[181.png]]
**知识点类型：** 定积分、奇偶函数、奇偶性运算、对称区间

### 1. 核心知识点

*   **对称区间上的定积分：**
    *   若 $f(x)$ 为奇函数，则 $\int_{-a}^a f(x) dx = 0$。
    *   若 $f(x)$ 为偶函数，则 $\int_{-a}^a f(x) dx = 2\int_0^a f(x) dx$。
    *   本题问哪个积分不为零，即问哪个被积函数不是奇函数（或者是非奇非偶函数）。

*   **常见的奇函数：**
	* ![[181-奇1.png]]
	* ![[181-奇2.png]]
    1.  $y = x^k$ ($k$ 为奇数)，例如：$x, x^3, x^5, \dots$
    2.  $y = \sin x, y = \tan x, y = \arcsin x, y = \arctan x$
    3.  $y = \ln \frac{a+x}{a-x}, y = \ln \frac{x+a}{x-a}, y = \ln \frac{a-x}{a+x}, y = \ln \frac{x-a}{x+a}$ (其中 $a$ 为常数)
    4.  $f(x) - f(-x)$ 或 $f(-x) - f(x)$
    5.  $\frac{a^x - 1}{a^x + 1}, \frac{1 - a^x}{1 + a^x}, \frac{a^x + 1}{a^x - 1}, \frac{1 + a^x}{1 - a^x}$
    6.  $y = \ln(x \pm \sqrt{1+x^2})$ (注意：根号内必须是 $1+x^2$)

*   **常见的偶函数：**
    1.  $y = x^k$ ($k$ 为偶数)，例如：$x^2, x^4, \dots$
    2.  $y = \cos x$
    3.  $f(x) + f(-x)$
    4.  $f(|x|), f(x^2), f(x^{2k})$ (将自变量替换为偶函数形式)
    5.  $|f(x)|$ (若 $f(x)$ 具有奇偶性)

*   **奇偶性的运算：**
    *   **加减法：**
        *   奇 $\pm$ 奇 = 奇
        *   偶 $\pm$ 偶 = 偶
        *   奇 $\pm$ 偶 = 非奇非偶 (除非其中一个为零函数)
        *   **总结：** 同名加减，奇偶性不变；异名加减，非奇非偶。
    *   **乘除法：** (每次只能两个函数一起看)
        *   奇 $\times$ 奇 = 偶
        *   偶 $\times$ 偶 = 偶
        *   奇 $\times$ 偶 = 奇
        *   **总结：** 同名乘除为偶；异名乘除为奇。
    *   **==复合函数 $f(g(x))$：==**
	    * ![[181-内偶则偶内奇看外.png]]
        *   内偶则偶：若 $g(x)$ 为偶函数，则 $f(g(x))$ 为偶函数。
        *   内奇看外：若 $g(x)$ 为奇函数，则 $f(g(x))$ 的奇偶性与 $f(x)$ 相同。

*   **==判断函数是否为奇函数的技巧：==**
    1.  **看 $f(0)$：** 若奇函数在 $x=0$ 处有定义，则 $f(0) = 0$。如果 $f(0)$ 存在但不为 0，则该函数一定不是奇函数。
    2.  **看特殊值：**  例如，判断 $f(-1)$ 是否等于 $-f(1)$。若不等，则不是奇函数。

### 2. 解题步骤与分析

*   **选项 A：$\cos x \ln \frac{1-x}{1+x}$**
    *   $\cos x$：偶函数
    *   $\ln \frac{1-x}{1+x}$：奇函数
    *   偶 $\times$ 奇 = 奇函数。积分结果为 0。

*   **选项 B：$\frac{e^x - 1}{e^x + 1} \sin^2 x$**
    *   $\frac{e^x - 1}{e^x + 1}$：奇函数
    *   $\sin^2 x$：偶函数 ($\sin x$ 是奇函数，平方后是偶函数，或者看作 $\sin x \cdot \sin x$，奇 $\times$ 奇 = 偶)
    *   奇 $\times$ 偶 = 奇函数。积分结果为 0。

*   **选项 C：$\frac{\sqrt{1+x^2} - x}{\sqrt{1+x^2} + x}$**
    *   **方法一：判断 $f(0)$**
        *   $f(0) = \frac{\sqrt{1+0} - 0}{\sqrt{1+0} + 0} = \frac{1}{1} = 1 \ne 0$。所以不是奇函数。
    *   **方法二：判断 $f(-1)$ 与 $-f(1)$**
        *   $f(1) = \frac{\sqrt{2}-1}{\sqrt{2}+1}$
        *   $f(-1) = \frac{\sqrt{1+(-1)^2} - (-1)}{\sqrt{1+(-1)^2} + (-1)} = \frac{\sqrt{2}+1}{\sqrt{2}-1}$
        *   $f(-1) \ne -f(1)$ (实际上 $f(-1) = \frac{1}{f(1)}$)。所以不是奇函数。
    *   由于它不是奇函数，其在对称区间的积分不一定为零。

*   **选项 D：$x^2 \arctan x + \frac{x}{1+x^2}$**
    *   $x^2$：偶函数
    *   $\arctan x$：奇函数
    *   $x^2 \arctan x$：偶 $\times$ 奇 = 奇函数
    *   $x$：奇函数
    *   $1+x^2$：偶函数
    *   $\frac{x}{1+x^2}$：奇 $\div$ 偶 = 奇函数
    *   奇 + 奇 = 奇函数。积分结果为 0。

*   **结论：** 选项 C 的被积函数不是奇函数，其积分不为零。

### 3. 重点总结

*   熟练掌握常见奇偶函数的判别方法。
*   理解并能灵活运用奇偶性的运算法则（加减、乘除、复合）。
*   掌握判断函数是否为奇函数的两个实用技巧（$f(0)$ 和特殊值）。
*   明确对称区间上奇函数积分为零的性质。
*   在判断乘除法奇偶性时，务必注意“两个两个一起看”的规则。
