这道题以选择题的形式，深入考察了二阶常系数线性非齐次微分方程特解的求解，对比了[[深度解析：微分算子法求特解]]和待定系数法，并强调了题目条件的重要性。
#660重点题目 
**题目：** 二阶微分方程特解求解（微分算子法 vs 待定系数法）

**知识点类型：** 二阶常系数线性非齐次微分方程、特解、微分算子法、待定系数法、共轭复根

### 1. 核心知识点

*   **微分算子法（回顾）：**
    *   **型三：指数与函数乘积型 $f(x) = e^{kx}V(x)$**
        *   **法则：“指数外提，算子平移”**
            $$
            y^* = \frac{1}{F(D)} [e^{kx}V(x)] = e^{kx} \frac{1}{F(D+k)} V(x)
            $$
    *   **型二：三角函数型 $f(x) = \sin(ax)$ 或 $\cos(ax)$**
        *   **法则：“见 $D^2$ 就换 $-a^2$”**
    *   **特殊情况：分母为 0**
        *   **法则：“提 x 求导”**

*   **待定系数法（设特解形式）：**
    *   **原理：** 根据非齐次项 $f(x)$ 的形式，假设特解 $y^*$ 具有类似的形式，但带有待定系数。
    *   **右端项形式：** $f(x) = e^{\alpha x} [P_m(x)\cos(\beta x) + P_n(x)\sin(\beta x)]$
    *   **设特解 $y^*$ 的步骤：**
        1.  **基本形式：**
            $$
            y^* = e^{\alpha x} [Q_l(x)\cos(\beta x) + R_l(x)\sin(\beta x)]
            $$
            其中 $l = \max(m, n)$，$Q_l(x)$ 和 $R_l(x)$ 是待定的 $l$ 次多项式。
        2.  **乘以弹性系数 $x^k$：**
            *   写出特征方程，求解特征根 $\lambda$。
            *   判断 $\alpha + i\beta$ 是否为特征根。
                *   如果不是特征根，则 $k=0$（不乘 $x$）。
                *   如果是单特征根，则 $k=1$（乘以 $x$）。
                *   如果是重特征根，则 $k=2$（乘以 $x^2$）。
    *   **求待定系数：** 将设出的 $y^*$ 代入原微分方程，通过比较系数来确定待定系数。

### 2. 解题思路与步骤

**方法一：微分算子法（高效）**

1.  **写出特解的算子形式：**
    $$
    y^* = \frac{1}{D^2 - 2D + 5} [e^x \cos(2x)]
    $$

2.  **应用“指数外提，算子平移”法则（型三）：**
    *   $k=1$，将 $e^x$ 提出，算子中的 $D$ 替换为 $D+1$。
    $$
    y^* = e^x \frac{1}{(D+1)^2 - 2(D+1) + 5} \cos(2x)
    $$

3.  **化简分母的算子多项式：**
    $$
    (D+1)^2 - 2(D+1) + 5 = (D^2 + 2D + 1) - (2D + 2) + 5 = D^2 + 4
    $$

4.  **应用“见 $D^2$ 就换 $-a^2$”法则（型二）：**
    *   $a=2$，将 $D^2$ 替换为 $-2^2 = -4$。
    $$
    y^* = e^x \frac{1}{-4 + 4} \cos(2x)
    $$
    分母为 0。

5.  **应用“提 x 求导”法则：**
    *   对分母的算子多项式 $D^2+4$ 求导，得到 $2D$。
    $$
    y^* = e^x \cdot x \frac{1}{2D} \cos(2x)
    $$

6.  **计算积分：**
    $$
    y^* = \frac{x e^x}{2} \left( \frac{1}{D} \cos(2x) \right) = \frac{x e^x}{2} \int \cos(2x) dx
    $$
    $$
    = \frac{x e^x}{2} \left( \frac{1}{2}\sin(2x) \right) = \frac{1}{4} x e^x \sin(2x)
    $$

**方法二：待定系数法（传统）**

1.  **求解特征方程：**
    *   $\lambda^2 - 2\lambda + 5 = 0$
    *   $\lambda = \frac{2 \pm \sqrt{4 - 20}}{2} = 1 \pm 2i$

2.  **分析右端项形式：**
    *   $f(x) = e^x \cos(2x)$
    *   $\alpha = 1$, $\beta = 2$

3.  **判断弹性系数 $k$：**
    *   $\alpha + i\beta = 1 + 2i$，是特征方程的单根。
    *   所以需要乘以 $x$，即 $k=1$。

4.  **设特解 $y^*$：**
    *   $P_m(x) = 1$（0次多项式），$P_n(x) = 0$
    *   $l = \max(0, -1) = 0$
    *   $Q_l(x)$ 和 $R_l(x)$ 都是0次多项式，即常数 $A, B$。
    $$
    y^* = x^1 e^x (A\cos(2x) + B\sin(2x))
    $$

5.  **求待定系数 $A, B$：**
    *   将 $y^*$ 代入原方程，经过繁琐的求导和化简，比较系数可得：
        $$
        A = 0, \quad B = \frac{1}{4}
        $$

6.  **最终特解：**
    $$
    y^* = x e^x (0 \cdot \cos(2x) + \frac{1}{4}\sin(2x)) = \frac{1}{4} x e^x \sin(2x)
    $$

### 3. 题目选项分析

*   **题目条件：** $A, B$ 都是不等于零的常数。
*   **计算结果：** 最终的特解形式为 $y^* = C \cdot x e^x \sin(2x)$，其中 $C = \frac{1}{4}$。
    *   这个特解中，$\cos(2x)$ 项的系数为 0。
*   **选项 A：** $A e^x \cos(2x) + B e^x \sin(2x)$
    *   不含 $x$ 因子，形式错误。
*   **选项 B 和 C：** 形式错误。
*   **选项 D：** $A x e^x \sin(2x) + B x e^x \cos(2x)$
    *   这是唯一一个包含了正确因子 $x e^x \sin(2x)$ 的选项。
    *   题目说 $A, B$ 是非零常数，而我们的计算结果是 $A=0, B=1/4$。这里题目问的是**特解的形式**。我们的计算结果是**一个**特解，而D选项给出的是**一类**特解。
    *   我们的特解 $\frac{1}{4} x e^x \sin(2x)$ 属于D选项描述的形式（取 $A=1/4, B=0$），但题目要求 $A,B$ 非零，这似乎有矛盾。

### 4. 重点总结

*   **微分算子法 vs 待定系数法：**
    *   微分算子法在计算上通常更高效，特别是对于分母为 0 或形式复杂的右端项。
    *   待定系数法思路更传统，但计算量大，容易出错。
*   **共轭复根的处理：** 待定系数法中，当 $\alpha \pm i\beta$ 恰好是特征方程的共轭复根时，设特解时需要乘以 $x$。
*   **审题的重要性：** 题目中的附加条件（如本题的 $A, B$ 非零）可能会极大地影响最终的选择，需要仔细分析。
*   掌握两种方法，可以相互检验，提高解题的准确率。
