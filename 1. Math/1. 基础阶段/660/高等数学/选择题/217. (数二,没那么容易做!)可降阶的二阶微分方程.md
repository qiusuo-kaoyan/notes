*** 
**题目：** 可降阶的二阶微分方程求解（初值问题）
![[217-2.png]]
**知识点类型：** 二阶微分方程、可降阶微分方程、不显含 x、不显含 y、降阶法、链式法则、可分离变量

### 1. 核心知识点

*   **初值问题：** 求解满足特定初始条件的微分方程的特解的问题。
![[217-2-1.png]]
*   **可降阶的二阶微分方程的三种类型：**
    1.  **$y'' = f(x)$ 型：** 对 $x$ 积分两次即可。
    2.  **不显含 $y$ 型 ($y'' = f(x, y')$)：**
	   * ![[217-2-2.png]]
       *   **降阶方法：** 令 $p = y'$，则 $y'' = \frac{dp}{dx}$。
       *   **结果：** 得到一个关于 $p$ 和 $x$ 的一阶微分方程，解出 $p$ 后再积分求 $y$。
    1.  ==**不显含 $x$ 型 ($y'' = f(y, y')$)：**==
	   * ![[217-2-3.png]]
       *   **降阶方法：** 令 $p = y'$，则 $y'' = \frac{dp}{dx} = \frac{dp}{dy} \cdot \frac{dy}{dx} = p \frac{dp}{dy}$。
       *   **结果：** 得到一个关于 $p$ 和 $y$ 的一阶微分方程，解出 $p$ 后再积分求 $y$。

*   **链式求导法则的应用：** 在“不显含 $x$”的类型中，使用 $y'' = p \frac{dp}{dy}$ 是关键步骤，它将关于 $x$ 的导数转化为了关于 $y$ 的导数，从而消除了变量 $x$。

*   **可分离变量法：**  将所有包含 $x$ 的项和 $dx$ 移到等式一边，所有包含 $y$ 的项和 $dy$ 移到另一边，然后两边同时积分。

*   **积分常数的处理：**
    *   在不定积分中，如果其他项都是对数形式，可以将积分常数写成 $\ln C$ 的形式，以便于化简。
    *   在解二阶微分方程时，会出现两个任意常数，通常用 $C_1, C_2$ 表示。
    *   利用初始条件可以确定任意常数的值。

### 2. 解题步骤与分析（本题）

1.  **判断类型：**
    *   方程 $(1 + (y')^2) = 2yy''$ 中不显含 $x$，属于“不显含 $x$”的可降阶类型。

2.  **降阶：**
    *   令 $p = y'$，则 $y'' = p \frac{dp}{dy}$。
    *   代入原方程：$(1 + p^2) = 2y(p \frac{dp}{dy})$

3.  **求解关于 $p$ 和 $y$ 的一阶微分方程（可分离变量）：**
    *   分离变量：$\frac{1}{2y} dy = \frac{p}{1+p^2} dp$
    *   两边积分：$\int \frac{1}{2y} dy = \int \frac{p}{1+p^2} dp$
    *   $\frac{1}{2}\ln|y| = \frac{1}{2}\ln(1+p^2) + C_1$
    *   化简：$\ln|y| = \ln(1+p^2) + 2C_1 \implies \ln|y| = \ln(1+p^2) + \ln C_2$
    *   $y = C_2(1+p^2)$

4.  **利用初始条件确定 $C_2$：**
    *   初始条件：$y(1) = 1$，$y'(1) = -1$。
    *   当 $x=1$ 时，$y=1$，$p = y' = -1$。
    *   代入：$1 = C_2(1 + (-1)^2) \implies C_2 = \frac{1}{2}$
    *   得到：$y = \frac{1}{2}(1+p^2)$

5.  **求解关于 $y$ 和 $y'$ 的一阶微分方程：**
    *   $2y = 1 + p^2 \implies p^2 = 2y - 1 \implies p = \pm \sqrt{2y-1}$
    *   因为 $p = y'(1) = -1$ (负数)，所以取负号：$p = -\sqrt{2y-1}$
    *   $\frac{dy}{dx} = -\sqrt{2y-1}$

6.  **求解关于 $y$ 和 $x$ 的一阶微分方程（可分离变量）：**
    *   分离变量：$\frac{dy}{\sqrt{2y-1}} = -dx$
    *   两边积分：$\int (2y-1)^{-\frac{1}{2}} dy = \int -dx$
    *   $\frac{1}{2} \cdot 2(2y-1)^{\frac{1}{2}} = -x + C_3 \implies \sqrt{2y-1} = -x + C_3$

7.  **利用初始条件确定 $C_3$：**
    *   初始条件：$y(1) = 1$
    *   代入：$\sqrt{2(1)-1} = -1 + C_3 \implies C_3 = 2$

8.  **得到特解：**
    *   $\sqrt{2y-1} = 2-x$
    *   两边平方：$2y-1 = (2-x)^2 = 4 - 4x + x^2$
    *   $2y = x^2 - 4x + 5$
    *   $y = \frac{1}{2}(x^2 - 4x + 5)$

### 3. 重点总结

*   熟练掌握不显含 $x$ 和不显含 $y$ 的二阶微分方程的降阶方法。
*   理解链式求导法则在不显含 $x$ 类型中的关键作用。
*   能够准确判断和应用可分离变量法。
*   在求解过程中，要仔细处理积分常数，并利用初始条件将其确定。
*   注意细节，如开方时正负号的取舍，要根据初始条件来判断。
*   二阶微分方程需要两个初始条件才能确定特解。
