***
这道题主要讲解了欧拉方程的定义、标准解法以及其背后换元法的推导原理。

**题目：** 欧拉方程的求解（标准解法与原理推导）
![[222.png]]
**知识点类型：** 欧拉方程、换元法、算子、特征方程、常系数线性微分方程

### 1. 核心知识点

*   **欧拉方程的定义：**
    *   形如 $a_n x^n y^{(n)} + a_{n-1} x^{n-1} y^{(n-1)} + \dots + a_1 x y' + a_0 y = f(x)$ 的微分方程。
    *   **特征：** $y$ 的 $k$ 阶导数项 $y^{(k)}$ 前面乘以 $x$ 的 $k$ 次方项 $x^k$。

*   **欧拉方程的标准解法（三步走）：**

    1.  **换元：** 令 $x = e^t$，则 $t = \ln x$。
    2.  **替换：**
        *   $x y' \implies D y$
        *   $x^2 y'' \implies D(D-1)y$
        *   $x^3 y''' \implies D(D-1)(D-2)y$
        *   ...
        *   $x^k y^{(k)} \implies D(D-1)\dots(D-k+1)y$
        *   ==**注意：** $D$ 是对新变量 $t$ 的求导算子，即 $D = \frac{d}{dt}$。==
    3.  **解新方程并回代：**
        *   将原方程转化为一个关于 $y$ 和 $t$ 的**常系数线性微分方程**。
        *   解出 $y$ 关于 $t$ 的通解。
        *   将 $t = \ln x$ ==代回==，得到 $y$ 关于 $x$ 的通解。

*   **常系数线性微分方程的解法：**
    *   写出特征方程。
    *   解出特征根 $\lambda$。
    *   根据特征根的形式写出通解。

### 2. 解题步骤（本题）

1.  **判断类型：**
    *   方程 $x^2 y'' + 2xy' - 2y = 0$ 中，$y''$ 前面是 $x^2$，$y'$ 前面是 $x$，是欧拉方程。

2.  **换元与替换：**
    *   令 $x = e^t$，则 $t = \ln x$。
    *   $x^2 y'' \implies D(D-1)y = (D^2 - D)y$
    *   $x y' \implies D y$
    *   代入原方程：$(D^2 - D)y + 2Dy - 2y = 0$

3.  **解新方程：**
    *   整理：$(D^2 + D - 2)y = 0$
    *   特征方程：$\lambda^2 + \lambda - 2 = 0$
    *   解特征根：$(\lambda + 2)(\lambda - 1) = 0 \implies \lambda_1 = 1, \lambda_2 = -2$
    *   ==通解（关于 $t$）：==$y = C_1 e^{\lambda_1 t} + C_2 e^{\lambda_2 t} = C_1 e^t + C_2 e^{-2t}$

4.  **回代：**
    *   $y = C_1 e^{\ln x} + C_2 e^{-2 \ln x} = C_1 x + C_2 e^{\ln x^{-2}} = C_1 x + C_2 x^{-2}$
    *   最终通解：$y = C_1 x + \frac{C_2}{x^2}$

### ==3. 原理推导（换元+链式法则）==

*   **一阶导数：**
    $$
    y' = \frac{dy}{dx} = \frac{dy}{dt} \cdot \frac{dt}{dx} = \frac{dy}{dt} \cdot \frac{1}{x}
    $$
    所以，$x y' = \frac{dy}{dt} = Dy$

*   **二阶导数：**
    $$
    y'' = \frac{d}{dx}(y') = \frac{d}{dx}(\frac{1}{x}\frac{dy}{dt}) = -\frac{1}{x^2}\frac{dy}{dt} + \frac{1}{x}\frac{d}{dx}(\frac{dy}{dt})
    $$
    $$
    = -\frac{1}{x^2}\frac{dy}{dt} + \frac{1}{x} \left[\frac{d}{dt}(\frac{dy}{dt}) \cdot \frac{dt}{dx}\right] = -\frac{1}{x^2}\frac{dy}{dt} + \frac{1}{x} \frac{d^2y}{dt^2} \frac{1}{x} = \frac{1}{x^2}(\frac{d^2y}{dt^2} - \frac{dy}{dt})
    $$
    所以，$x^2 y'' = \frac{d^2y}{dt^2} - \frac{dy}{dt} = D^2y - Dy = D(D-1)y$

### 4. 重点总结

*   熟练掌握欧拉方程的定义和标准解法。
*   理解欧拉方程的核心思想：通过换元 $x = e^t$，将其转化为常系数线性微分方程。
*   记住 $x^k y^{(k)}$ 的替换规则，可以快速解题。
*   理解替换规则背后的链式求导法推导过程，有助于加深理解。
*   掌握常系数线性微分方程的解法，这是解欧拉方程的基础。
