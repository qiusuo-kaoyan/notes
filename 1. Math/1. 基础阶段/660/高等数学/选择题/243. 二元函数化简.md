***
这是一道综合性非常强的题目，全面考察了二元函数在原点的连续性、偏导数存在性、可微性以及偏导函数的连续性。
 
**题目：** 二元函数在原点的性质判断（可化简的分式函数）
![[243.png]]
**知识点类型：** 二元函数、连续性、偏导数、可微性、偏导数连续性、二重极限计算、化简、夹逼定理

### 1. 核心知识点与解题技巧

*   **二重极限计算中的化简：**
    *   在计算二重极限时，不要急于使用夹逼定理或取特殊路径，首先应观察函数是否可以**化简**。
    *   本题中，分子是平方差公式：$x^4 - y^4 = (x^2 - y^2)(x^2 + y^2)$。

*   **二元函数性质的层次关系：**
    *   **偏导数连续** $\implies$ **可微** $\implies$ **连续**
    *   **偏导数连续** $\implies$ **可微** $\implies$ **偏导数存在**
    *   **重要推论：** 如果能证明偏导数连续，那么可微性、连续性、偏导数存在性都自然成立。这是解决此类综合性选择题的**最快路径**。

*   **连续性判断：**
    *   **定义：** $\lim_{(x,y)\to(x_0,y_0)} f(x,y) = f(x_0,y_0)$。
    *   **方法：**
        1.  **化简法：**  先化简函数表达式，再求极限。
        2.  **夹逼定理：**  当无法直接求极限时，通过放缩构造夹逼条件。

*   **偏导数存在性判断：**
    *   **定义法：** 在分段点（如原点）求偏导，必须使用定义。
        $$
        f_x(0,0) = \lim_{\Delta x \to 0} \frac{f(\Delta x, 0) - f(0,0)}{\Delta x}
        $$

*   **可微性判断：**
    *   **定义法：**
        $$
        \lim_{(\Delta x, \Delta y)\to(0,0)} \frac{f(\Delta x, \Delta y) - f(0,0) - f_x(0,0)\Delta x - f_y(0,0)\Delta y}{\sqrt{(\Delta x)^2+(\Delta y)^2}} = 0
        $$

*   **偏导数连续性判断：**
    *   **步骤：**
        1.  求出偏导函数的表达式 $f_x(x,y)$ 和 $f_y(x,y)$。这通常需要分区域讨论：在分段点用定义，在其他点用求导公式。
        2.  求偏导函数在分段点的二重极限：$\lim_{(x,y)\to(x_0,y_0)} f_x(x,y)$。
        3.  将极限值与该点的偏导数值 $f_x(x_0,y_0)$ 进行比较。如果相等，则偏导数在该点连续。

### 2. 解题步骤与分析

**最优解法（化简法）：**

1.  **化简函数：**
    当 $(x,y) \ne (0,0)$ 时，
    $$
    f(x,y) = \frac{x^4-y^4}{x^2+y^2} = \frac{(x^2-y^2)(x^2+y^2)}{x^2+y^2} = x^2-y^2
    $$
    当 $(x,y) = (0,0)$ 时，$f(0,0)=0$。
    因为 $\lim_{(x,y)\to(0,0)} (x^2-y^2) = 0 = f(0,0)$，所以函数可以统一表示为：
    $$
    f(x,y) = x^2 - y^2
    $$

2.  **求偏导函数：**
    *   $f_x(x,y) = 2x$
    *   $f_y(x,y) = -2y$

3.  **判断偏导函数连续性：**
    *   $f_x(x,y) = 2x$ 和 $f_y(x,y) = -2y$ 都是初等函数，在整个定义域上连续。
    *   **结论：** 偏导数连续。

4.  **根据性质层次关系得出最终结论：**
    *   因为偏导数连续，所以函数可微，连续，且偏导数存在。
    *   **最终答案：** (D) 可微且 $f_x, f_y$ 连续。

**常规解法（按部就班）：**

1.  **连续性：**
    *   $\lim_{(x,y)\to(0,0)} \frac{x^4-y^4}{x^2+y^2} = \lim_{(x,y)\to(0,0)} (x^2-y^2) = 0 = f(0,0)$，**连续**。

2.  **偏导数存在性：**
    *   $f_x(0,0) = \lim_{\Delta x \to 0} \frac{f(\Delta x, 0) - f(0,0)}{\Delta x} = \lim_{\Delta x \to 0} \frac{(\Delta x)^2-0}{\Delta x} = 0$，**存在**。
    *   $f_y(0,0) = \lim_{\Delta y \to 0} \frac{f(0, \Delta y) - f(0,0)}{\Delta y} = \lim_{\Delta y \to 0} \frac{-(\Delta y)^2-0}{\Delta y} = 0$，**存在**。

3.  **可微性：**
    *   判断 $\lim_{(x,y)\to(0,0)} \frac{x^2-y^2 - 0 - 0\cdot x - 0\cdot y}{\sqrt{x^2+y^2}} = \lim_{(x,y)\to(0,0)} \frac{x^2-y^2}{\sqrt{x^2+y^2}}$ 是否为 0。
    *   使用极坐标 $x=r\cos\theta, y=r\sin\theta$：
        $$
        \lim_{r\to 0} \frac{r^2(\cos^2\theta - \sin^2\theta)}{r} = \lim_{r\to 0} r\cos(2\theta) = 0
        $$
    *   极限为0，故**可微**。

4.  **偏导函数连续性：**
    *   在 $(x,y)\ne(0,0)$ 处，$f_x(x,y) = \frac{\partial}{\partial x} (x^2-y^2) = 2x$。
    *   $\lim_{(x,y)\to(0,0)} f_x(x,y) = \lim_{(x,y)\to(0,0)} 2x = 0 = f_x(0,0)$，故 $f_x$ **连续**。
    *   同理可证 $f_y$ **连续**。

### 3. 重点总结

*   **化简优先**：在处理二元函数极限或性质判断时，应首先考虑是否可以化简函数表达式，这通常是最高效的方法。
*   **掌握性质层次**：熟记“偏导连续 $\implies$ 可微 $\implies$ 连续/偏导存在”的推导关系，可以快速解答选择题。
*   **方法多样性**：对于同一问题，应了解并掌握多种判断方法（化简、定义、夹逼、极坐标等），并根据题目特点选择最优解法。
*   本题的核心在于识别出函数可以化简，从而将一个复杂的分段函数问题转化为一个简单的初等函数问题。
