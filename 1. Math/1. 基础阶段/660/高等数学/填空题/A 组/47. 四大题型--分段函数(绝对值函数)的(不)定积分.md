这道题综合考察了分段函数、绝对值函数的不定积分和定积分，以及更复杂的带参数绝对值函数的定积分。视频中也回顾了分段函数不定积分和定积分的基本方法。

**题目：** 综合题（分段函数、绝对值函数的不定积分与定积分）

**知识点类型：** 不定积分、定积分、分段函数、绝对值函数、连续性、可导性、换元积分法、拆项法、带参数的定积分

### 1. 核心知识点与方法回顾

*   **分段函数不定积分：**

    1.  分段求不定积分，得到 $F_i(x) + C_i$。
    2.  利用连续性（分段点处左极限 = 右极限）确定常数关系。
    3.  统一积分常数。

*   **分段函数定积分：**  拆分积分区间，分别计算每一段的定积分。

*   **绝对值函数不定积分：**  本质上是分段函数不定积分，先去绝对值，再按照分段函数不定积分的步骤求解。

*   **绝对值函数定积分：**

    *   **方法一（通用）：**  令绝对值内的表达式等于 0，解出分界点，然后根据分界点拆分积分区间，去掉绝对值符号，再分别计算定积分。
    *   **方法二（特殊情况）：**  如果绝对值内的表达式在整个积分区间上的正负性是确定的，则可以直接去掉绝对值符号（正数不变，负数添负号）。

*   **常数的定积分：** $\int_a^b k dx = k(b-a)$，特别地，$\int_a^b 1 dx = b - a$。

*   **连续性与可导性：**  原函数（不定积分）一定连续，且在连续区间内可导，导数等于被积函数。

### 2. 例题分析

*   **例题 1（分段函数不定积分）：** $f(x) = \begin{cases} e^x, & x \ge 0 \\ \cos x, & x < 0 \end{cases}$

    1.  分段求不定积分：
        $$
        F(x) =
        \begin{cases}
        e^x + C_1, & x \ge 0 \\
        \sin x + C_2, & x < 0
        \end{cases}
        $$

    2.  利用连续性：
        *   $\lim_{x \to 0^-} F(x) = \sin 0 + C_2 = C_2$
        *   $\lim_{x \to 0^+} F(x) = e^0 + C_1 = 1 + C_1$
        *   $C_2 = 1 + C_1$

    3.  统一积分常数：
        *   令 $C_1 = C$，则 $C_2 = 1 + C$。
        $$
        F(x) =
        \begin{cases}
        e^x + C, & x \ge 0 \\
        \sin x + 1 + C, & x < 0
        \end{cases}
        $$

*   **例题 2（分段函数定积分）：** $\int_0^2 f(x) dx$，其中 $f(x) = \begin{cases} 1+x, & 0 \le x \le 1 \\ 3x^2, & 1 < x \le 2 \end{cases}$

    $$
    \int_0^2 f(x) dx = \int_0^1 (1+x) dx + \int_1^2 3x^2 dx = \left[x + \frac{1}{2}x^2\right]_0^1 + \left[x^3\right]_1^2 = \frac{3}{2} + (8-1) = \frac{17}{2}
    $$

*   **例题 3（绝对值函数不定积分）：** $f(x) = |x - |x+1||$

    1.  **去绝对值（从内到外）：**
        $$
        f(x) =
        \begin{cases}
        |x - (x+1)| = |-1| = 1, & x \ge -1 \\
        |x - (-x-1)| = |2x + 1|, & x < -1
        \end{cases}
        $$
        $$
         =
        \begin{cases}
              1, & x \ge -1 \\
             -(2x+1) & x < -1
        \end{cases}
        $$

    2.  分段求不定积分：
        $$
        F(x) =
        \begin{cases}
        x + C_1, & x \ge -1 \\
        -x^2 - x + C_2, & x < -1
        \end{cases}
        $$

    3.  利用连续性：
        *   $\lim_{x \to -1^-} F(x) = -(-1)^2 - (-1) + C_2 = C_2$
        *   $\lim_{x \to -1^+} F(x) = -1 + C_1$
        *   $C_2 = -1 + C_1$

    4.  统一积分常数：
        *   令 $C_2 = C$，则 $C_1 = C + 1$。
        $$
        F(x) =
        \begin{cases}
        x + 1 + C, & x \ge -1 \\
        -x^2 - x + C, & x < -1
        \end{cases}
        $$

*   **例题 4（绝对值函数定积分）：** $\int_0^2 |1-x| dx$

    1.  令 $|1-x| = 0$，解得 $x = 1$。
    2.  拆分积分区间：
        $$
        \int_0^2 |1-x| dx = \int_0^1 (1-x) dx + \int_1^2 (x-1) dx
        $$
    3.  计算定积分：
        $$
        \int_0^1 (1-x) dx + \int_1^2 (x-1) dx = \left[x - \frac{1}{2}x^2\right]_0^1 + \left[\frac{1}{2}x^2 - x\right]_1^2 = \frac{1}{2} + \frac{1}{2} = 1
        $$

*   **例题 5（带参数的绝对值函数定积分）：** $\int_0^1 |t^2 - x| dt$

    1.  **分情况讨论：**

        *   **$x \le 0$：**  $t^2 - x \ge 0$，直接去绝对值。
            $$
            \int_0^1 (t^2 - x) dt = \left[\frac{1}{3}t^3 - xt\right]_0^1 = \frac{1}{3} - x
            $$

        *   **$x \ge 1$：**  $t^2 - x \le 0$，去绝对值添负号。
            $$
            \int_0^1 (x - t^2) dt = \left[xt - \frac{1}{3}t^3\right]_0^1 = x - \frac{1}{3}
            $$

        *   **$0 < x < 1$：**  令 $t^2 - x = 0$，解得 $t = \sqrt{x}$（负值舍去）。
            $$
            \int_0^1 |t^2 - x| dt = \int_0^{\sqrt{x}} (x - t^2) dt + \int_{\sqrt{x}}^1 (t^2 - x) dt
            $$
            $$
            = \left[xt - \frac{1}{3}t^3\right]_0^{\sqrt{x}} + \left[\frac{1}{3}t^3 - xt\right]_{\sqrt{x}}^1 = (x^{3/2} - \frac{1}{3}x^{3/2}) + (\frac{1}{3} - x - (\frac{1}{3}x^{3/2} - x^{3/2}))
            $$
            $$
            = \frac{4}{3}x^{3/2} - x + \frac{1}{3}
            $$

    2.  **最终结果：**

        $$
        \int_0^1 |t^2 - x| dt =
        \begin{cases}
        \frac{1}{3} - x, & x \le 0 \\
        \frac{4}{3}x^{3/2} - x + \frac{1}{3}, & 0 < x < 1 \\
        x - \frac{1}{3}, & x \ge 1
        \end{cases}
        $$

### 3. 重点总结

*   分段函数不定积分和定积分的核心都是“分段处理”。
*   绝对值函数不定积分和定积分都可以转化为分段函数的问题。
*   求不定积分时，要利用连续性确定积分常数的关系，并统一成一个常数。
*   求定积分时，如果绝对值内的表达式在积分区间上的正负性不确定，要通过令其等于 0 来找到分界点，拆分积分区间。
*   对于带参数的定积分，可能需要根据参数的取值范围进行分类讨论。
*   熟练掌握基本积分公式和积分技巧（如凑微分、换元、拆项等）。
