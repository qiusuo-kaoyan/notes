这道题主要考察了分段函数不定积分的求法，强调了利用连续性确定积分常数的关系。

**题目：** 分段函数求不定积分（分段函数、不定积分、连续性）
![[46.png]]
**知识点类型：** 不定积分、分段函数、连续性、可导性、原函数

### 1. 核心知识点

*   **原函数与不定积分：**
    *   **原函数：**  如果 $F'(x) = f(x)$，则称 $F(x)$ 是 $f(x)$ 的一个原函数。
    *   **不定积分：**  $f(x)$ 的所有原函数称为 $f(x)$ 的不定积分，记作 $\int f(x) dx = F(x) + C$，其中 $C$ 为任意常数。

*   **分段函数求不定积分的步骤：**

    1.  **分段求不定积分：**  对每一段分别求不定积分，得到 $F_i(x) + C_i$。
    2.  **==利用连续性确定常数关系：==**
        *   ==因为原函数一定可导，可导一定连续，所以分段函数的原函数在分段点处一定连续。==
        *   利用分段点处的连续性（左极限 = 右极限 = 函数值），建立不同积分常数 $C_i$ 之间的关系。
    3.  **统一积分常数：**  将所有积分常数用同一个常数 $C$ 表示。

*   **连续性与可导性的关系：**  可导函数一定连续，但连续函数不一定可导。

*   **基本初等函数的原函数：**
    *   $\int \cos x dx = \sin x + C$
    *   $\int \sin x dx = -\cos x + C$
      *   $\int x^n dx = \frac{x^{n+1}}{n+1} + C$
    *   $\int \frac{1}{1+x^2} dx = \arctan x + C$

### 2. 解题步骤

1.  **分段求不定积分：**
    $$
    F(x) = \int f(x) dx =
    \begin{cases}
    \int (\sin x + 1) dx = -\cos x + x + C_1, & x > 0 \\
    \int \frac{1}{1+x^2} dx = \arctan x + C_2, & x \le 0
    \end{cases}
    $$

2.  **利用连续性确定常数关系：**
    *   因为 $F(x)$ 在 $x=0$ 处连续，所以 $\lim_{x \to 0^-} F(x) = \lim_{x \to 0^+} F(x) = F(0)$。
    *   $\lim_{x \to 0^-} F(x) = \lim_{x \to 0^-} (\arctan x + C_2) = C_2$
    *   $\lim_{x \to 0^+} F(x) = \lim_{x \to 0^+} (-\cos x + x + C_1) = C_1 - 1$
    *   $F(0) = \arctan 0 + C_2 = C_2$
    *   所以，$C_2 = C_1 - 1$

3.  **统一积分常数：**
    左极限:
    $$
    \lim_{x\to 0^-} F(x) = \lim_{x\to 0^-} (\arctan x + C_2) = C_2
    $$
    右极限:
     $$
    \lim_{x\to 0^+} F(x) = \lim_{x\to 0^+} (-\cos x + x + C_1) = -1 + C_1
    $$
    函数值:
    $$
    F(0) = \arctan 0 + C_2 = C_2
    $$
    由连续性: $C_2 = -1 + C_1$, 即 $C_2 + 1 = C_1$, 因此, 正确的答案是:

     $$
     F(x)  =
    \begin{cases}
    -\cos x + x + C, & x > 0 \\
    \arctan x + C - 1, & x \le 0
    \end{cases}
    $$

### 3. 重点总结

*   分段函数求不定积分的关键是分段求积分，然后利用连续性确定积分常数的关系。
*   ==原函数一定可导，可导一定连续==，这是确定积分常数关系的重要依据。
*   求一次不定积分，只能出现一个任意常数。
*   熟练掌握基本初等函数的原函数。
*   注意细节，避免计算错误。
