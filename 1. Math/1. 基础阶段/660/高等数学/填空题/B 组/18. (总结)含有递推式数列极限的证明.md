**题目：** 含递推式的数列极限存在性证明（单调有界准则、先斩后奏）
![[18.png]]
**知识点类型：** 数列极限、递推式、单调有界准则、先斩后奏、求导法、做差法、做比法、分离常数、数学归纳法、常见不等式

***

### 1. 核心知识点

*   **数列极限的两大主线：**
    1.  求==前 $n$ 项和==的数列极限。
    2.  含有==递推式==的数列极限存在性证明。

*   **含递推式的数列极限存在性证明的三种方法：**
    1.  **单调有界准则：** 证明数列单调且有界，则极限存在。
    2.  **先斩后奏：** 先假设极限存在并求出极限值，然后构造新数列（原数列减去极限值），证明新数列的极限为 0。
    3.  **构造新数列：** 适用于相邻三项递推关系，通过构造新数列将问题转化为更简单的形式。（本题未详细展开）
    4. **差分方程:** 更高阶的方法.(法 3 的高级版)

*   **单调有界准则：(默认先证单调)**

    *   **原理：** 单调有界数列必有极限。
        *   单调递增有上界的数列必有极限。
        *   单调递减有下界的数列必有极限。

    *   **证明单调性的方法：**
        1.  **==求导法：==**  如果递推式可以写成 $a_{n+1} = f(a_n)$ 的形式，则：
            *   $f'(x) > 0 \implies$ 数列 $\{a_n\}$ 单调。
                *   若 $a_1 < a_2$，则数列单调递增。
                *   若 $a_1 > a_2$，则数列单调递减。
            *   $f'(x) < 0 \implies$ 数列 $\{a_n\}$ 不单调。
            * ==**注意⚠️: 导数必须 > 0才能说明单调, < 0 不意味着单减!**==
        2.  **做差法：**  考察 $a_{n+1} - a_n$ 的符号。
            *   $a_{n+1} - a_n > 0 \implies$ 数列单调递增。
            *   $a_{n+1} - a_n < 0 \implies$ 数列单调递减。
            *    也可以考察$a_{n+1} - a_n$ 和 $a_n - a_{n-1}$是否同号.
        3.  **做比法：**  考察 $\frac{a_{n+1}}{a_n}$ 与 1 的大小关系（通常用于各项均为正数的情况）。
            *   $\frac{a_{n+1}}{a_n} > 1 \implies$ 数列单调递增。
            *   $\frac{a_{n+1}}{a_n} < 1 \implies$ 数列单调递减。
		![[18单调有界准则(单调性).png]]

    *   **证明有界性的方法：**
        1.  **分离常数：**  将递推式变形，分离出常数部分，观察剩余部分的变化趋势。(将分子凑成分母, 分离出来)
        2.  **数学归纳法：**  利用数学归纳法证明数列的每一项都小于或大于某个常数。(先把极限算出来,界就是极限)
        3.  **常见不等式：**  利用已知的不等式（如均值不等式、柯西不等式等）进行放缩。
        4.  **根据第一问的条件：**  如果题目有多个小问，第一问的结论或方法可能为证明有界性提供线索。

*   **先斩后奏：(单调有界准则难证的情况下考虑)**

    *   **步骤：**
        1.  **先斩（求极限）：**  假设数列极限存在，设为 $A$，对递推式两边取极限，解出 $A$ 的值（可能有多个，需要根据数列的性质进行取舍）。
        2.  **后奏（构造新数列并证明极限为 0. 注意绝对值, 类似于绝对收敛准则: 绝对收敛->极限收敛）：**
            *   构造新数列：$b_n = |a_n - A|$。
            *   证明 $\lim_{n \to \infty} b_n = 0$。通常利用**夹逼定理**，将 $|a_{n+1} - A|$ **放缩**为 $k|a_n - A|$ 的形式，其中 $0 < k < 1$。
            *   根据夹逼定理和递推关系，证明 $\lim_{n\to \infty} |a_n - A| = 0$, 从而得到$\lim_{n\to \infty}a_n = A$
    * ![[18 先斩后奏(单调有界不好证时考虑).png]]



### 2. 解题步骤与分析

**方法一：单调有界准则**

1.  **证明单调性：**

    *   **化简递推式(分离常数)：**
        $$
        x_n = \frac{1 + 2x_{n-1}}{1 + x_{n-1}} = \frac{2 + 2x_{n-1} - 1}{1+x_{n-1}} = 2 - \frac{1}{1 + x_{n-1}}
        $$

    *   **==求导法(为什么能求导,它的本质是什么?)：==**
        *   令 $f(x) = 2 - \frac{1}{1+x}$
        *   $f'(x) = \frac{1}{(1+x)^2} > 0$，所以数列单调。

    *   **判断单调性：**
        *   $x_0 = 0$
        *   $x_1 = \frac{1 + 2x_0}{1 + x_0} = 1 > x_0$
        *   所以数列单调递增。

2.  **证明有界性：**

    *   **分离常数：**
        $$
        x_n = 2 - \frac{1}{1 + x_{n-1}}
        $$
        因为 $x_{n-1} \ge 0$，所以 $1 + x_{n-1} \ge 1$，$\frac{1}{1 + x_{n-1}} \le 1$，$2 - \frac{1}{1 + x_{n-1}} \ge 1$
       又因为$x_n$单调递增, $x_0 = 0$, 所以显然$x_n \ge 0$
       从而得到$x_n = 2-\frac{1}{1+x_{n-1}} < 2$
    *   所以数列 $\{x_n\}$ 有上界 2。

3.  **求极限：**

    *   由单调有界准则，数列 $\{x_n\}$ 的极限存在，设为 $A$。
    *   对递推式 $x_n = \frac{1 + 2x_{n-1}}{1 + x_{n-1}}$ 两边取极限：
        $$
        A = \frac{1 + 2A}{1 + A}
        $$
    *   解方程：$A^2 + A = 1 + 2A \implies A^2 - A - 1 = 0$
    *   $A = \frac{1 \pm \sqrt{5}}{2}$，因为 $x_n \ge 0$，所以 $A = \frac{1 + \sqrt{5}}{2}$。

**==方法二：先斩后奏(一种反证的方法, 我很喜欢这种感觉!)==**

1.  **先斩（求极限）：**  同上，求出 $A = \frac{1 + \sqrt{5}}{2}$。

2.  **后奏（构造新数列并证明极限为 0, 关键是找到那个 0 到 1 之间的系数 k, 注意不等号方向!）：**

    *   构造新数列：$b_n = |x_n - A|$
    *   放缩(**注意: ==第二个等号不是猜的, 是直接带$A = 2 - \frac{1}{1+A}$ 得到的. 这里非常关键! <font color="#d83931">不带 A=... 的表达式绝对值内部非常难化简!</font>==**)：
        $$
        |x_{n+1} - A| = \left|2 - \frac{1}{1 + x_n} - A\right| = \left|2 - \frac{1}{1+x_n} - (2-\frac{1}{1+A})\right|
        $$
        $$
        = \left|\frac{1}{1+A} - \frac{1}{1+x_n}\right| = \left|\frac{1+x_n - (1+A)}{(1+A)(1+x_n)}\right| = \frac{|x_n - A|}{(1+A)(1+x_n)}
        $$
         因为$x_n \ge 0$, $A = \frac{1+\sqrt{5}}{2} > 0$, 所以$(1+A) > 1$, $(1+x_n) > 1$
        $$
         \frac{|x_n - A|}{(1+A)(1+x_n)} < \frac{1}{(1+A)}|x_n - A|
        $$
    *   递推：
      由于$A = \frac{1+\sqrt{5}}{2} > \frac{1}{2}$, 所以$1+A > \frac{3}{2}$, 所以$\frac{1}{1+A} < \frac{2}{3} < 1$
        $$
        |x_{n+1} - A| <  \frac{2}{3}|x_n - A| < (\frac{2}{3})^2 |x_{n-1} - A| < \dots < (\frac{2}{3})^n |x_1 - A|
        $$
        当 $n \to \infty$ 时，$(\frac{2}{3})^n |x_1 - A| \to 0$。

    *   由夹逼定理，$\lim_{n \to \infty} |x_n - A| = 0$，所以 $\lim_{n \to \infty} x_n = A = \frac{1 + \sqrt{5}}{2}$。

### 3. 重点总结

*   掌握含递推式的数列极限存在性证明的两种主要方法：单调有界准则和先斩后奏。
*   理解单调有界准则的原理和使用步骤，熟练掌握证明单调性和有界性的各种方法。
*   掌握先斩后奏的步骤，特别是构造新数列和利用夹逼定理证明新数列极限为 0 的技巧。
*   在处理递推关系时，经常需要对递推式进行变形（如本题中的分离常数），以便于分析和证明。
*   注意细节，如求出多个极限值时，要根据数列的性质进行取舍。
