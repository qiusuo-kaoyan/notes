**题目：** 较复杂的零比零型极限（三角函数、对数、等价无穷小、泰勒公式）
![[1. Math/1. 基础阶段/660/高等数学/Picture Set/11.png]]
**知识点类型：** 极限、三角函数、对数运算、等价无穷小代换、加减法等价无穷小代换的条件、泰勒展开、低阶吸收高阶

### 1. 核心知识点

*   **三角函数公式：** $\sec x = \frac{1}{\cos x}$

*   **对数运算性质：**
    *   $\ln a + \ln b = \ln(ab)$
    *   $\ln a - \ln b = \ln(\frac{a}{b})$

*   **特殊的等价无穷小：** 当 $框框 \to 1$ 时，$\ln(框框) \sim 框框 - 1$

*   **常用的等价无穷小 (当“框框” $\to 0$ 时)：**
    *   $\sin($“框框”$) \sim$ “框框”
    *   $1 - \cos(框框) \sim \frac{1}{2} (框框)^2$
    *    $\ln(1+框框)\sim 框框$

*   **加减法中等价无穷小代换的条件（重要）：**

    *   **不严谨但常用的说法：** 等价成最简形式后不抵消（不为 0）。
        * **最简形式:** 不能再被等价无穷小替换的形式.
    *   **严谨的定理：(加法是负, 减法是正)**
        *   若 $A \sim A'$，$B \sim B'$，则 $A + B \sim A' + B'$ 的条件是：$\lim \frac{A'}{B'}$ 存在且不等于 ==$-1$==。
        *   若 $A \sim A'$，$B \sim B'$，则 $A - B \sim A' - B'$ 的条件是：$\lim \frac{A'}{B'}$ 存在且不等于 ==$1$==。
    *   **例子:**
     如果$A \sim x$, $B\sim x+x^2$
     那么$A-B$不能等价替换为$x - (x+x^2)$, 因为$\lim_{x\to 0}\frac{x}{x+x^2} = 1$

*   **[[知识枢纽：泰勒公式与等价无穷小|泰勒展开]]：**
    *   $\ln(1 + x) = x - \frac{1}{2}x^2 + \frac{1}{3}x^3 - \dots$
    *   $\ln(1 + 框框) = 框框 - \frac{1}{2} (框框)^2 + \frac{1}{3} (框框)^3 - \dots$
    *   **精确到 n 阶：**  展开到 $x^n$ 项，且更高阶的项不会再出现 $x^n$。

*   **低阶吸收高阶：**  当 $x \to 0$ 时，多个幂函数相加减，可以等价于次数最低的那一项。
    *   例如：当 $x \to 0$ 时，$x^2 + x^4 \sim x^2$

### 2. 解题步骤

1.  **分母等价无穷小：**
    $$
    \lim_{x \to 0} \frac{\ln(1 + x + x^2) + \ln(1 - x + x^2)}{\sec x - \cos x} = \lim_{x \to 0} \frac{\ln(1 + x + x^2) + \ln(1 - x + x^2)}{\frac{1}{\cos x} - \cos x}
    $$
    $$
     = \lim_{x \to 0} \frac{\ln(1 + x + x^2) + \ln(1 - x + x^2)}{\frac{1-\cos ^2x}{\cos x}}
    $$
    因为$\cos x \to 1$, 所以
    $$
    = \lim_{x \to 0} \frac{\ln(1 + x + x^2) + \ln(1 - x + x^2)}{1-\cos ^2x} = \lim_{x\to 0}\frac{\ln(1 + x + x^2) + \ln(1 - x + x^2)}{x^2}
    $$

2.  **分子对数运算：**
    $$
    \ln(1 + x + x^2) + \ln(1 - x + x^2) = \ln[(1 + x + x^2)(1 - x + x^2)]
    $$
    $$
     = \ln[(1+x^2)^2 - x^2] = \ln(1 + 2x^2 + x^4 - x^2) = \ln(1 + x^2 + x^4)
    $$

3.  **分子等价无穷小：**
    $$
    \ln(1 + x^2 + x^4) \sim x^2 + x^4
    $$

4.  **低阶吸收高阶：**
    $$
    x^2 + x^4 \sim x^2
    $$

5.  **计算极限：**
    $$
    \lim_{x \to 0} \frac{x^2}{x^2} = 1
    $$

### 3. 易错点分析

*   **==加减法等价无穷小代换错误：==**  不能直接将 $\ln(1 + x + x^2)$ 等价为 $x + x^2$，将 $\ln(1 - x + x^2)$ 等价为 $-x + x^2$，因为它们做比的极限为 $-1$。

*   ==**泰勒展开不精确：**==  如果使用泰勒展开，只展开到 $x$ 项是不够的，因为更高阶的展开式中仍然可能包含 $x^2$ 项。必须展开到 $x^2$ 项，并确保更高阶的展开式中不再出现 $x^2$ 项，才算精确到二阶。

### 4. 重点总结

*   处理复杂的零比零型极限时，要综合运用各种技巧，包括三角函数公式、对数运算、等价无穷小代换、泰勒展开等。
*   深刻理解加减法中等价无穷小代换的条件，避免错误使用。
*   掌握泰勒展开的技巧，并注意展开的精度。
*   理解“低阶吸收高阶”的思想，简化幂函数的计算。
*   做题时要仔细分析，避免常见的错误。
