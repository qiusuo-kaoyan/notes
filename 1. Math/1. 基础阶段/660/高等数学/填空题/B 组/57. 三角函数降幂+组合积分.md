这道题主要考察了三角函数不定积分的计算，重点展示了两种方法：降幂法和组合积分法。同时也复习了三角函数的多个重要公式。

**题目：** 三角函数不定积分计算（降幂法、组合积分法）
![[57.png]]
**知识点类型：** 不定积分、三角函数、降幂公式、二倍角公式、组合积分

### 1. 核心知识点与公式

*   **降幂扩角公式：**
    *   $\cos^2 x = \frac{1 + \cos 2x}{2}$
    *   $\sin^2 x = \frac{1 - \cos 2x}{2}$
*   **二倍角公式：**
    *   $\sin 2x = 2 \sin x \cos x$
    *   $\cos 2x = \cos^2 x - \sin^2 x = 2\cos^2 x - 1 = 1 - 2\sin^2 x$

*   **平方和公式：** $\sin^2 x + \cos^2 x = 1$ （通常用于将 1 替换为 $\sin^2 x + \cos^2 x$）

*   **基本三角函数积分：**
    *   $\int \cos x dx = \sin x + C$
    *   $\int \sin x dx = -\cos x + C$
     *   $\int \cos kx dx = \frac{1}{k}\sin kx + C$
    *   $\int \sin kx dx = -\frac{1}{k}\cos kx + C$

*   **组合积分法：**  当两个积分 $\int f(x) dx$ 和 $\int g(x) dx$ 单独计算比较困难时，可以考虑计算 $\int (f(x) + g(x)) dx$ 和 $\int (f(x) - g(x)) dx$，然后通过加减运算得到原积分。

### 2. 解题方法与步骤

**方法一：降幂法**

*   **I1 = $\int \cos^4 x dx$**

    1.  降幂：
        $$
        \int \cos^4 x dx = \int (\cos^2 x)^2 dx = \int \left(\frac{1 + \cos 2x}{2}\right)^2 dx = \frac{1}{4} \int (1 + 2\cos 2x + \cos^2 2x) dx
        $$

    2.  再次降幂：
        $$
        \frac{1}{4} \int (1 + 2\cos 2x + \cos^2 2x) dx = \frac{1}{4} \int \left(1 + 2\cos 2x + \frac{1 + \cos 4x}{2}\right) dx
        $$

    3.  积分：
        $$
        \frac{1}{4} \int \left(\frac{3}{2} + 2\cos 2x + \frac{1}{2}\cos 4x\right) dx = \frac{1}{4} \left(\frac{3}{2}x + \sin 2x + \frac{1}{8}\sin 4x\right) + C = \frac{3}{8}x + \frac{1}{4}\sin 2x + \frac{1}{32}\sin 4x + C
        $$

*   **I2 = $\int \sin^4 x dx$**

    1.  降幂：
        $$
        \int \sin^4 x dx = \int (\sin^2 x)^2 dx = \int \left(\frac{1 - \cos 2x}{2}\right)^2 dx = \frac{1}{4} \int (1 - 2\cos 2x + \cos^2 2x) dx
        $$

    2.  再次降幂：
        $$
        \frac{1}{4} \int (1 - 2\cos 2x + \cos^2 2x) dx = \frac{1}{4} \int \left(1 - 2\cos 2x + \frac{1 + \cos 4x}{2}\right) dx
        $$

    3.  积分：
        $$
        \frac{1}{4} \int \left(\frac{3}{2} - 2\cos 2x + \frac{1}{2}\cos 4x\right) dx = \frac{1}{4} \left(\frac{3}{2}x - \sin 2x + \frac{1}{8}\sin 4x\right) + C = \frac{3}{8}x - \frac{1}{4}\sin 2x + \frac{1}{32}\sin 4x + C
        $$

**==方法二：组合积分法==**

1.  **计算 I1 - I2：**
    $$
    I1 - I2 = \int (\cos^4 x - \sin^4 x) dx = \int (\cos^2 x + \sin^2 x)(\cos^2 x - \sin^2 x) dx = \int \cos 2x dx = \frac{1}{2}\sin 2x + C_1
    $$

2.  **计算 I1 + I2：**
    $$
    I1 + I2 = \int (\cos^4 x + \sin^4 x) dx = \int [(\cos^2 x + \sin^2 x)^2 - 2\sin^2 x \cos^2 x] dx
    $$
    $$
    = \int \left(1 - \frac{1}{2}(2\sin x \cos x)^2\right) dx = \int \left(1 - \frac{1}{2}\sin^2 2x\right) dx = \int \left(1 - \frac{1}{2} \cdot \frac{1 - \cos 4x}{2}\right) dx
    $$
    $$
    = \int \left(\frac{3}{4} + \frac{1}{4}\cos 4x\right) dx = \frac{3}{4}x + \frac{1}{16}\sin 4x + C_2
    $$

3.  **求解 I1 和 I2：**
    *   $I1 = \frac{1}{2}[(I1 + I2) + (I1 - I2)] = \frac{3}{8}x + \frac{1}{32}\sin 4x + \frac{1}{4}\sin 2x + C$  (注意：这里的C是($C_1 + C_2$)/2)
    *   $I2 = \frac{1}{2}[(I1 + I2) - (I1 - I2)] = \frac{3}{8}x + \frac{1}{32}\sin 4x - \frac{1}{4}\sin 2x + C$ (注意: 这里的C是($C_2 - C_1$)/2)

### 3. 重点总结

*   对于三角函数高次幂的积分，降幂法是一种常用的方法。
*   熟练掌握降幂扩角公式和二倍角公式，以及它们之间的相互推导。
*   组合积分法是一种有效的技巧，可以简化某些积分的计算。
*   在计算不定积分时，最后一定要加上任意常数 $C$。
*   同一个题目中，不同的积分常数可以用不同的符号表示（如 $C_1$, $C_2$），以示区分。
*   $\sin^2 x + \cos^2 x = 1$ 的恒等式，通常用于将常数 1 替换成三角函数形式，以便于化简。
* 降幂法是更通用的方法.
