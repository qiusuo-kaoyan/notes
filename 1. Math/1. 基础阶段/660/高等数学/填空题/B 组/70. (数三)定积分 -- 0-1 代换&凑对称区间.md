这道题是一道定积分计算题，主要考察了[[深度解析：分部积分法]]（包括表格法）、换元法（凑对称区间、凑 0-1 区间）、以及一些化简技巧。视频中还提到了一个重要的结论：在凑微分后，如果被积函数只剩下 $\ln$ 或反三角函数，则应立即使用分部积分法。

**题目：** 定积分计算（分部积分、表格法、换元、凑对称区间、凑 0-1 区间）

**知识点类型：** 定积分、[[深度解析：分部积分法]]、表格法、换元积分法、凑对称区间、凑 0-1 区间、幂函数积分、三角函数

### 1. 核心知识点与技巧

*   **[[深度解析：分部积分法]]：** $\int u dv = uv - \int v du$
    *   **表格法：**  当被积函数是两个函数相乘，其中一个函数可以求导到 0，另一个函数容易积分时，可以使用表格法。
        *   表格的左列：对容易求导的函数反复求导，直到 0。
        *   表格的右列：对容易积分的函数反复积分。
        *   将左列和右列的对应项相乘，并交替添加正负号，最后将所有结果相加。

*   **换元积分法：**
    *   **凑对称区间：**  令 $x = t + \frac{a+b}{2}$，可以将积分区间 $[a, b]$ 转化为对称区间。
    *   **凑 0-1 区间：**  令 $x = a + t(b-a)$，可以将积分区间 $[a, b]$ 转化为 $[0, 1]$。

*   **幂函数积分：** $\int x^n dx = \frac{x^{n+1}}{n+1} + C$ ($n \ne -1$)

*   **重要结论：**  在凑微分后，如果被积函数只剩下 $\ln$ 或反三角函数，则应立即使用分部积分法。

* **化简的技巧**:
  * 加减常数, 凑出分母的形式.
* **常见的微分**:
 * $\frac{1}{x^2}dx = - d(\frac{1}{x})$
 * 更一般地, $\frac{1}{[f(x)]^2}d[f(x)] = - d(\frac{1}{f(x)})$

### 2. 解题方法与步骤

**方法一：分部积分表格法**

1.  **列表：**

    | 求导 (u)       | 积分 (dv)      | 符号 |
    | :------------- | :------------- | :--- |
    | $(x-a)^2$      | $(b-x)^3$      | +    |
    | $2(x-a)$       | $-\frac{1}{4}(b-x)^4$ | -    |
    | 2              | $\frac{1}{20}(b-x)^5$  | +    |
    | 0              | $-\frac{1}{120}(b-x)^6$ | -    |

2.  **计算：**
    $$
    \int_a^b (x-a)^2 (b-x)^3 dx = \left[ -\frac{1}{4}(x-a)^2(b-x)^4 - \frac{1}{10}(x-a)(b-x)^5 - \frac{1}{60}(b-x)^6 \right]_a^b
    $$
    $$
    = 0 - \left(0 + 0 - \frac{1}{60}(b-a)^6\right) = \frac{1}{60}(b-a)^6
    $$

**方法二：换元凑对称区间**

1.  **换元：** 令 $x = t + \frac{a+b}{2}$，则 $dx = dt$。
    *   $x = a \implies t = \frac{a-b}{2}$
    *   $x = b \implies t = \frac{b-a}{2}$
    *   $x - a = t + \frac{a+b}{2} - a = t + \frac{b-a}{2}$
    *   $b - x = b - t - \frac{a+b}{2} = \frac{b-a}{2} - t$

2.  **代入积分：**
    $$
    \int_a^b (x-a)^2 (b-x)^3 dx = \int_{\frac{a-b}{2}}^{\frac{b-a}{2}} \left(t + \frac{b-a}{2}\right)^2 \left(\frac{b-a}{2} - t\right)^3 dt
    $$

3.  **化简与计算：**  令 $\frac{b-a}{2} = A$，则
    $$
    \int_{-A}^A (t+A)^2 (A-t)^3 dt = \int_{-A}^A (t+A)^2 (A-t)^2(A-t) dt =  \int_{-A}^A (A^2 - t^2)^2 (A-t) dt
    $$
    $$
     = \int_{-A}^A A(A^2-t^2)^2 dt - \int_{-A}^A t(A^2 - t^2)^2 dt = A\int_{-A}^A (A^2 - t^2)^2 dt - 0
    $$
    $$
    = 2A\int_0^A (A^4 - 2A^2t^2 + t^4)dt = 2A[A^4t - \frac{2}{3}A^2t^3 + \frac{1}{5}t^5]_0^A
    $$
     $$
      = 2A(A^5 - \frac{2}{3}A^5 + \frac{1}{5}A^5) = 2A^6(1-\frac{2}{3} + \frac{1}{5}) = 2A^6 \cdot \frac{8}{15} = \frac{16}{15}A^6
     $$
     将$A = \frac{b-a}{2}$代入:
    $$
    \frac{16}{15} (\frac{b-a}{2})^6 = \frac{1}{60}(b-a)^6
    $$

**方法三：换元凑 0-1 区间**

1.  **换元：** 令 $x = a + t(b-a)$，则 $dx = (b-a) dt$。
    *   $x = a \implies t = 0$
    *   $x = b \implies t = 1$
    *   $x - a = t(b-a)$
    *   $b - x = b - a - t(b-a) = (1-t)(b-a)$

2.  **代入积分：**
    $$
    \int_a^b (x-a)^2 (b-x)^3 dx = \int_0^1 [t(b-a)]^2 [(1-t)(b-a)]^3 (b-a) dt = (b-a)^6 \int_0^1 t^2 (1-t)^3 dt
    $$

3.  **计算：**
    $$
    (b-a)^6 \int_0^1 t^2 (1-t)^3 dt = (b-a)^6 \int_0^1 t^2(1 - 3t + 3t^2 - t^3) dt
    $$
    $$
    = (b-a)^6 \int_0^1 (t^2 - 3t^3 + 3t^4 - t^5) dt = (b-a)^6 \left[\frac{1}{3}t^3 - \frac{3}{4}t^4 + \frac{3}{5}t^5 - \frac{1}{6}t^6\right]_0^1
    $$
    $$
     = (b-a)^6 (\frac{1}{3} - \frac{3}{4} + \frac{3}{5} - \frac{1}{6}) = \frac{1}{60}(b-a)^6
    $$

**方法四：分部积分**

1. **第一次分部积分:**
$$
 \int_a^b (x-a)^2(b-x)^3dx = \int_a^b (x-a)^2 d(-\frac{1}{4}(b-x)^4) = -\frac{1}{4}(x-a)^2(b-x)^4|_a^b + \frac{1}{4}\int_a^b (b-x)^4 d(x-a)^2
$$
$$
= 0 + \frac{1}{2}\int_a^b (x-a)(b-x)^4 dx
$$
2. **第二次分部积分:**
 $$
 \frac{1}{2}\int_a^b (x-a)(b-x)^4 dx =  \frac{1}{2}\int_a^b (x-a) d(-\frac{1}{5}(b-x)^5) = -\frac{1}{10}(x-a)(b-x)^5|_a^b + \frac{1}{10}\int_a^b (b-x)^5 d(x-a)
 $$
 $$
  = 0 - \frac{1}{10} \int_a^b (b-x)^5 d(b-x) = -\frac{1}{10} \cdot \frac{(b-x)^6}{-6}|_a^b = \frac{1}{60}(b-a)^6
 $$

### 3. 重点总结

*   对于本题，分部积分的表格法最为简便。
*   换元法（凑对称区间、凑 0-1 区间）也是有效的解题思路，但计算量较大。
*   直接分部积分也可以，但需要多次应用分部积分公式。
*   在计算过程中，要注意化简，并灵活运用代数技巧和函数性质。
*   掌握凑微分的常见形式和技巧。
*   熟练掌握有理函数拆分的方法。
*   对于 $d$ 后面加减常数不影响结果: $d(f(x) + C) = df(x)$
