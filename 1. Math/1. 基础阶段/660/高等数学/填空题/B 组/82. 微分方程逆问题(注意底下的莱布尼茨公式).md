***
如何根据已知的特解来反求二阶线性非齐次微分方程的通解甚至方程本身，这是微分方程解的结构理论的重要应用。

**题目：** 根据特解反求微分方程通解及方程本身（解的结构应用）
![[82.png]]
**知识点类型：** 二阶线性微分方程、齐次方程、非齐次方程、通解、特解、[[二阶线性微分方程解的结构|解的结构(忘了回去看)]]

### 1. 核心思路：通解是关键

> 无论是要求微分方程的通解，还是要求解微分方程本身，**核心都是先求出该非齐次方程的通解**。
>
> **非齐次方程通解的标准形式：**
> $$
> y(x) = \underbrace{C_1 y_1(x) + C_2 y_2(x)}_{\text{齐次方程的通解}} + \underbrace{y^*(x)}_{\text{非齐次方程的一个特解}}
> $$

### 2. 从通解到微分方程

*   **步骤：**
    1.  **确定特征根：** 齐次方程通解中的 $y_1(x)$ 和 $y_2(x)$ 的形式决定了特征方程的根 $\lambda_1, \lambda_2$。
        *   例如，如果齐次通解为 $C_1e^{2x} + C_2e^{-x}$，则特征根为 $\lambda_1 = 2, \lambda_2 = -1$。
    2.  **构造特征方程：** 根据特征根写出特征方程。
        *   $(\lambda - \lambda_1)(\lambda - \lambda_2) = 0$
        *   例如：$(\lambda - 2)(\lambda + 1) = \lambda^2 - \lambda - 2 = 0$
    3.  **还原齐次微分方程：** 根据特征方程写出对应的齐次微分方程。
        *   例如：$y'' - y' - 2y = 0$
    4.  **求右端项 $f(x)$：**
        *   将通解中的特解 $y^*(x)$ 代入非齐次方程 $y'' - y' - 2y = f(x)$ 的左侧。
        *   计算出的结果即为右端项 $f(x)$。

### 3. 如何从已知特解求通解

**方法一：观察法/猜测法（适用于形式简单、规律明显的特解）**

1.  **找特解 $y^*(x)$：**
    *   **原理：** 通解中的特解部分 $y^*(x)$ 是不含任意常数 $C_1, C_2$ 的，因此它会出现在所有给定的特解中。
    *   **操作：** 观察所有已知的特解，找出它们共同含有的、形式相同的部分，这部分就是 $y^*(x)$。
    *   **本题示例：**
        *   $y_1 = C_1e^{2x} + C_2e^{-x} + xe^x$
        *   $y_2 = C_1e^{2x} + C_2e^{-x} + xe^x$
        *   $y_3 = C_1e^{2x} + C_2e^{-x} + xe^x$
        *   共同部分是 $xe^x$，所以 $y^*(x) = xe^x$。

2.  **找齐次通解：**
    *   **原理：** 除去特解部分 $y^*(x)$ 后，剩下的部分就是齐次方程的解。
    *   **操作：** 观察剩下的部分，确定齐次方程的两个线性无关的特解形式，从而写出齐次通解。
    *   **本题示例：**
        *   除去 $xe^x$ 后，剩下的部分都具有 $e^{2x}$ 和 $e^{-x}$ 的形式。
        *   所以齐次方程的通解为 $C_1e^{2x} + C_2e^{-x}$。

3.  **组合成非齐次通解：**
    *   将齐次通解和非齐次特解相加。
    *   **本题示例：** $y(x) = C_1e^{2x} + C_2e^{-x} + xe^x$。

**方法二：利用解的结构定理（更严谨、普适性更强）**

1.  **构造齐次方程的解：**
    *   **原理：** 两个**非齐次**方程的解之**差**，是对应的**齐次**方程的解。
    *   **操作：** 将已知的非齐次特解两两相减，得到齐次方程的解。
    *   **本题示例：**
        *   $y_1 - y_2 = (2e^{2x} + xe^x) - (e^{2x} + e^{-x} + xe^x) = e^{2x} - e^{-x}$
        *   $y_1 - y_3 = (2e^{2x} + xe^x) - (e^{2x} - e^{-x} + xe^x) = e^{2x} + e^{-x}$
        *   这两个都是齐次方程的解。

2.  **构造齐次方程的通解：**
    *   **原理：** 齐次方程的解的任意线性组合仍是齐次方程的解。
    *   **操作：** 通过对上一步得到的齐次解进行线性组合，可以分离出两个线性无关的特解 $e^{2x}$ 和 $e^{-x}$。
        *   例如：$\frac{1}{2}[(e^{2x} - e^{-x}) + (e^{2x} + e^{-x})] = e^{2x}$
        *   $\frac{1}{2}[-(e^{2x} - e^{-x}) + (e^{2x} + e^{-x})] = e^{-x}$
    *   **结论：** 齐次方程的通解为 $C_1e^{2x} + C_2e^{-x}$。

3.  **构造非齐次方程的特解：**
    *   **原理：** 一个**非齐次**的解，加上（或减去）一个**齐次**的解，结果仍然是**非齐次**的解。
    *   **操作：** 用一个已知的非齐次特解（如 $y_2$），减去一个已知的齐次解（如 $e^{2x} - e^{-x}$），得到一个形式可能更简单的非齐次特解。
    *   **本题示例：**
        *   $y_2 - (e^{2x} - e^{-x}) = (e^{2x} + e^{-x} + xe^x) - (e^{2x} - e^{-x}) = 2e^{-x} + xe^x$（这是一个新的非齐次特解）
        *   $y_2 - e^{2x} = e^{-x} + xe^x$（这又是一个新的非齐次特解）
    *   **目标：** 找到一个形式最简单的非齐次特解，本题中通过观察法得到的 $xe^x$ 是最理想的。

4.  **组合成非齐次通解：**
    *   将齐次通解和找到的非齐次特解相加。
    *   $y(x) = C_1e^{2x} + C_2e^{-x} + xe^x$。

### 4. 重点总结

*   **核心是通解：** 解决这类问题的核心是利用已知特解求出非齐次方程的通解。
*   **两种方法：**
    *   **观察法** 简单直接，适用于规律明显的题目。
    *   **结构定理法** 更为严谨和普适，是解决复杂问题的基础。
*   **定理的灵活运用：** 要深刻理解并能灵活运用微分方程解的结构定理，尤其是：
    *   非齐次解之差是齐次解。
    *   非齐次解 $\pm$ 齐次解仍是非齐次解。
*   **还原微分方程：** 从通解的形式可以反推出特征方程，进而反推出微分方程的左端；将特解代入可以求出右端项。
*   **==莱布尼茨公式的应用：==**  在求 $y^*(x) = xe^x$ 的导数时，可以利用莱布尼茨公式快速求导。
   $$
   (uv)^{(n)} = \sum_{k=0}^n C_n^k u^{(n-k)}v^{(k)}
   $$
   在本题中, $y = xe^x$, $y^{(n)} = (n+x)e^x$
