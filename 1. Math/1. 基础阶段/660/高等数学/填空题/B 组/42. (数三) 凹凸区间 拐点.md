这道题考察了函数的单调区间、凹凸区间、极值、拐点的求法，以及一些细节问题，如定义域、极值点与拐点的表示、区间之间的连接符号等。

**题目：** 求函数的单调区间、凹凸区间、极值、拐点（定义域、单调性、凹凸性、极值、拐点）
![[42 (数三) 凹凸区间 拐点.png]]
**知识点类型：** 定义域、单调区间、凹凸区间、极值、拐点、导数、分段函数

### 1. 核心知识点

*   **定义域：**  函数自变量的取值范围。对于本题，$f(x) = x^{5/3} - x^{2/3}$ 的定义域为 $\mathbb{R}$。
*   **单调区间：**
    *   $f'(x) > 0$ 的区间为单调递增区间。
    *   $f'(x) < 0$ 的区间为单调递减区间。
*   **凹凸区间：**
    *   $f''(x) > 0$ 的区间为凹区间。
    *   $f''(x) < 0$ 的区间为凸区间。
*   **极值点：**  函数在某点处取得极大值或极小值，则该点为极值点（极值点是自变量的取值，==不是坐标==）。
*   **拐点：**  曲线凹凸性发生改变的点（拐点==是坐标==）。
    *   必要条件：$f''(x_0) = 0$ 或 $f''(x_0)$ 不存在。
    *   充分条件：$f''(x)$ 在 $x_0$ 两侧异号。

* **求导:**
  对于$f(x) = x^n$, $f'(x) = nx^{n-1}$

*   **分割点：**  用于划分单调区间和凹凸区间的点。
    *   对于单调区间：一阶导数的零点和不存在的点。
    *   对于凹凸区间：二阶导数的零点和不存在的点。

*   **==区间连接符号：==**
    *   定义域：可以用 "$\cup$" (并集) 连接。
    *   单调区间、凹凸区间：用 "," (逗号) 连接，不能用 "$\cup$"。

### 2. 解题步骤

1.  **求定义域：**  $f(x) = x^{5/3} - x^{2/3}$ 的定义域为 $\mathbb{R}$。

2.  **求一阶导数和二阶导数：**
    $$
    f(x) = x^{5/3} - x^{2/3}
    $$
    $$
    f'(x) = \frac{5}{3}x^{2/3} - \frac{2}{3}x^{-1/3}
    $$
    $$
    f''(x) = \frac{10}{9}x^{-1/3} + \frac{2}{9}x^{-4/3} = \frac{2}{9}x^{-4/3}(5x+1)
    $$

3.  **找分割点：**

    *   令 $f''(x) = 0$，解得 $x = -\frac{1}{5}$。
    *   $f''(x)$ 在 $x = 0$ 处不存在。
    *   所以分割点为 $x = -\frac{1}{5}$ 和 $x = 0$。

4.  **列表分析：**

    | $x$             | $(-\infty, -\frac{1}{5})$ | $-\frac{1}{5}$ | $(-\frac{1}{5}, 0)$ | $0$     | $(0, +\infty)$ |
    | :-------------- | :------------------------ | :------------- | :------------------ | :------ | :------------- |
    | $f''(x)$        | -                         | 0              | +                   | 不存在 | +             |
    | $f(x)$          | 凸                        | 拐点           | 凹                   |         | 凹             |
    | 单调性          |                           |                |                   |           |                 |

5.  **确定凹凸区间和拐点：**

    *   凸区间：$(-\infty, -\frac{1}{5})$
    *   凹区间：$(-\frac{1}{5}, 0)$ 和 $(0, +\infty)$
    *   拐点：$(-\frac{1}{5}, f(-\frac{1}{5}))$  (==注意：拐点是坐标，不是 $x$ 的值==)
      本题只要求拐点的横坐标, 所以拐点横坐标是$-\frac{1}{5}$

### 3. 重点总结与注意事项

*   **求单调区间、凹凸区间、极值、拐点的第一步是求定义域。**  定义域的错误会导致后续结果全部错误。
*   **极值点不是点，而是自变量的取值。**  例如，本题没有要求极值，如果有，应该写“$x = ...$ 是极大/小值点”。
*   **拐点是点，需要写成坐标形式。**  例如，本题的拐点应写为 $(-\frac{1}{5}, f(-\frac{1}{5}))$。
*   **单调区间和凹凸区间之间用逗号连接，不能用并集符号。**  并集符号只用于表示定义域。
*   **求导时要注意符号和指数的变化。**  视频中讲解时出现了多次符号和指数的错误，需要仔细核对。
*   **分割点包括导数为 0 的点和导数不存在的点。**  对于本题，$x=0$ 是一个容易被忽略的分割点。
*   **拐点判断不仅要看二阶导数是否为零，还要看二阶导数在零点两侧是否变号。** $x=0$虽然二阶导数不存在, 但是二阶导数在$x=0$两侧没有变号, 所以不是拐点.
