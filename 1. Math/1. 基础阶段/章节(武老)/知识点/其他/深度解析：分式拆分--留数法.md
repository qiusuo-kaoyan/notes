## 1. 分式拆分的目的与概念

### 1.1. 目的

将一个复杂的分式拆分成多个简单分式的和或差。

### 1.2. 通分与拆分

*   **通分：** 将多个分式合并成一个分式（分母相同）。
*   **拆分：** 通分的逆过程，将一个分式拆分成多个分式。

## 2. 留数法拆分步骤

### 2.1. 步骤一：分母因式分解

*   **目标：** 将分母在**实数范围**内分解成**不能再分解**的因式。
*   **不能再分解：** 分解成一次或二次质因式。
*   **质因式：** 不能再分解的因式。
    *   一次因式一定是质因式。
    *   二次因式中，$b^2 - 4ac < 0$ 的是质因式。
*   **例子：**
    *   $x^4 - 1 = (x^2 + 1)(x^2 - 1) = (x^2 + 1)(x+1)(x-1)$

### 2.2. 步骤二：根据分母因式形式逐个拆分

*   **原则：** 分子比分母低一次。
*   **拆分规则：**

    | 分母因式                  | 拆分形式                                  |
    | ------------------------- | ------------------------------------------ |
    | $ax + b$                  | $\frac{A}{ax+b}$                            |
    | $(ax + b)^2$             | $\frac{A_1}{ax+b} + \frac{A_2}{(ax+b)^2}$    |
    | $(ax + b)^3$             | $\frac{A_1}{ax+b} + \frac{A_2}{(ax+b)^2} + \frac{A_3}{(ax+b)^3}$ |
    | $ax^2 + bx + c$ (质因式) | $\frac{Bx + C}{ax^2 + bx + c}$               |
    | $(ax^2 + bx + c)^2$ (质因式) | $\frac{B_1x + C_1}{ax^2 + bx + c} + \frac{B_2x + C_2}{(ax^2 + bx + c)^2}$ |
* **注意：** 上述分母为$(ax+b)^n$和$(ax^2+bx+c)^n$的因式，虽然展开后是n次和2n次，但是我们仍然认为他们的次数是1次和2次，因为它们是由n个相同的一次因式和二次因式相乘得到的。

*   **例子：**
    $$
    \frac{x^4 + 1}{(x-1)^3(x^2+2x+2)^2(x+2)} = \frac{A_1}{x-1} + \frac{A_2}{(x-1)^2} + \frac{A_3}{(x-1)^3} + \frac{B_1x+C_1}{x^2+2x+2} + \frac{B_2x+C_2}{(x^2+2x+2)^2} + \frac{A_4}{x+2}
    $$

### 2.3. 步骤三：求待定系数

*   **三种方法：**
    1.  **待定系数法：** （不推荐，麻烦）通分后比较系数。
    2.  **留数法：** （核心方法）
        *   **普通留数法**
        *   **求导留数法**
    3.  **特殊值法：** （辅助方法）
        *   **代入 ∞：** 抓大头，看 $\frac{1}{x}$ 的系数。
        *   **代入 0：** 简化计算。
        *   **代入其他值：** (很少用到)

*   **强调：** 求待定系数的过程**不需要**写在答题卡上，直接写结果。

## 3. 留数法详解

### 3.1. 普通留数法

*   **适用情况：** 分母为一次因式，且没有重根。
*   **步骤：**
    1.  **令分母为 0：** 对于待定系数 $A_i$，令其对应的分母因式为 0，解出 $x = x_i$。
    2.  **盖住分母：** 在原分式中，盖住 $A_i$ 对应的分母因式。
    3.  **代入：** 将 $x = x_i$ 代入剩余部分，计算结果即为 $A_i$。

*   **例子：**
    $$
    \frac{x}{(x+1)(x-2)} = \frac{A_1}{x+1} + \frac{A_2}{x-2}
    $$
    *   求 $A_1$：令 $x+1=0$，解得 $x=-1$。盖住 $x+1$，代入 $x=-1$，得到 $A_1 = \frac{-1}{-1-2} = \frac{1}{3}$。
    *   求 $A_2$：令 $x-2=0$，解得 $x=2$。盖住 $x-2$，代入 $x=2$，得到 $A_2 = \frac{2}{2+1} = \frac{2}{3}$。

### 3.2. 求导留数法

*   **适用情况：** 分母为一次因式，但有重根。
*   **原因：** 普通留数法无法盖住所有重根，导致无法代入。
*   **步骤：**
    1.  **多盖：** 对于待定系数 $A_i$，如果其分母为 $(x-a)^k$，而 $A_i$ 对应的是 $(x-a)^m$（$m < k$），则可以多盖 $k-m$ 个 $(x-a)$。
    2.  **求导：** 将剩余部分求 $k-m$ 阶导。
    3.  **除阶乘：** 除以 $(k-m)!$。
    4.  **代入：** 将 $x=a$ 代入，计算结果即为 $A_i$。

*   **例子：**
    $$
    \frac{x}{(x-1)^2(x-2)} = \frac{A_1}{x-1} + \frac{A_2}{(x-1)^2} + \frac{A_3}{x-2}
    $$
    *   求$A_3$：普通留数法可求，$A_3=2$。
    *   求$A_2$：普通留数法可求，$A_2=1$。
    *   求 $A_1$：
        1.  多盖：分母为 $(x-1)^2$，可以多盖一个 $(x-1)$。
        2.  求导：剩余部分 $\frac{x}{x-2}$ 求一阶导，得到 $\frac{-2}{(x-2)^2}$。
        3.  除阶乘：除以 $1! = 1$。
        4.  代入：代入 $x=1$，得到 $A_1 = -2$。

*   **公式总结：**
    若分母中有因式 $(x-a)^k$, 待定系数为$A_i$所对应的分母是$(x-a)^m$, 且$m<k$, 则为了求出$A_i$：
    $$
    A_i = \frac{1}{(k-m)!}[\text{盖住}(x-a)^k\text{后剩余部分}]^{(k-m)}|_{x=a}
    $$

### 3.3. 特殊值法

*   **代入 ∞：**
    *   **适用情况：** 仅适用于分母为**一次**的情况。
    *   **方法：** 抓大头，比较等式两边 $\frac{1}{x}$ 的系数。
    *   **原理：** 当 $x \to \infty$ 时，只有分母为一次的项才会产生 $\frac{1}{x}$，其他项都是更高阶的无穷小。
*   **代入 0：**
    *   **适用情况：** 当代入 ∞ 后还剩下一个待定系数时使用。
    *   **方法：** 将 $x=0$ 代入等式两边，直接解出最后一个待定系数。
* **代入其他值：**
  *   **适用情况：** 当以上两种方法均无法解出所有系数时使用。
  *   **方法：** 代入其他能使计算简便的值。

*   **例子（续上例）：**

    在求出 $A_2 = 1$ 和 $A_3 = 2$ 后，求 $A_1$：

    1.  **代入 ∞：**
        *   左边：$\frac{x}{(x-1)^2(x-2)} \sim \frac{x}{x^3} = \frac{1}{x^2}$，$\frac{1}{x}$ 的系数为 0。
        *   右边：$\frac{A_1}{x-1} + \frac{1}{(x-1)^2} + \frac{2}{x-2} \sim \frac{A_1}{x} + \frac{2}{x} = \frac{A_1 + 2}{x}$。
        *   得到 $A_1 + 2 = 0$，即 $A_1 = -2$。

### 3.4. 例子

*   **例1：**
$$
\frac{1}{(x+1)(x-1)} = \frac{A_1}{x+1} + \frac{A_2}{x-1}
$$
通过留数法, 可以得到：
$$
A_1 = -\frac{1}{2}, A_2 = \frac{1}{2}
$$

*   **例2：**

$$
\frac{x}{(x+1)(x-2)} = \frac{A_1}{x+1} + \frac{A_2}{x-2}
$$

通过留数法, 可以得到：
$$
A_1 = \frac{1}{3}, A_2 = \frac{2}{3}
$$

*   **例3：**

$$
\frac{x-1}{x(2x+1)} = \frac{A_1}{x} + \frac{A_2}{2x+1}
$$

通过留数法, 可以得到：
$$
A_1 = -1, A_2 = 3
$$

*   **例4：**

$$
\frac{x}{(x-1)^2(x-2)} = \frac{A_1}{x-1} + \frac{A_2}{(x-1)^2} + \frac{A_3}{x-2}
$$
通过留数法+求导, 可以得到：
$$
A_1 = -2, A_2 = 1, A_3 = 2
$$

* **例5：**

$$
\frac{x}{(x-1)(x-2)^3} =  \frac{A_1}{x-1} + \frac{A_2}{x-2} + \frac{A_3}{(x-2)^2} + \frac{A_4}{(x-2)^3}
$$
通过留数法+求导+代入无穷, 可以得到：
$$
A_1 = 1, A_4 = 2, A_3 = 1, A_2 = -1
$$

*   **例6：**

$$
\frac{x^2 - x - 1}{(x+1)(x^2+3x+1)} = \frac{A_1}{x+1} + \frac{Bx+C}{x^2+3x+1}
$$

通过留数法+代入无穷+代入0, 可以得到：
$$
A_1 = -1, B = 2, C = 5
$$
注意: 这里的$x^2+3x+1$无法继续在实数范围内因式分解, 但是可以使用代入无穷和代入0的方法求解。

* **例7（2019年真题）：**

$$
\frac{3x+6}{(x-1)^2(x^2+x+1)} = \frac{A_1}{x-1} + \frac{A_2}{(x-1)^2} + \frac{Bx+C}{x^2+x+1}
$$
通过留数法+求导+代入无穷+代入0, 可以得到:
$$
A_2 = 3, A_1 = -2, B = 2, C = 1
$$

* **例8：**

$$
\frac{1}{(1+x)(1+x^2)} = \frac{A_1}{1+x} + \frac{Bx+C}{1+x^2}
$$
通过留数法+代入无穷+代入0, 可以得到:
$$
A_1 = \frac{1}{2}, B = -\frac{1}{2}, C = \frac{1}{2}
$$

* **例9：**

$$
\frac{x}{(x+1)^3} = \frac{A_1}{x+1} + \frac{A_2}{(x+1)^2} + \frac{A_3}{(x+1)^3}
$$
通过代入无穷+求导, 可以得到:
$$
A_1 = 0, A_3 = -1, A_2 = 1
$$

## 4. 总结

*   留数法是一种高效的分式拆分方法。
*   普通留数法适用于分母为一次因式且无重根的情况。
*   求导留数法适用于分母为一次因式且有重根的情况。
*   特殊值法（代入 ∞ 和 0）可以辅助求解。
*   代入 ∞ 时，只能比较 $\frac{1}{x}$ 的系数。
*   灵活运用各种方法，可以快速准确地完成分式拆分。
