**一句话总结: 精度问题**

### **一、基本原则：乘除可换，加减忌换**
等价无穷小替换的核心原则是**仅适用于乘除运算中的因子**，而加减运算一般不可直接替换。==这是因为加减运算可能导致高阶无穷小的抵消，进而改变极限结果。==例如：
- **错误案例**：$\lim_{x→0}\frac{\tan x - \sin x}{x^3}$  
  若直接替换$\tan x \sim x$和$\sin x \sim x$，则分子变为$x - x = 0$，得出错误极限$0$；而实际泰勒展开后分子为$\frac{1}{2}x^3 + o(x^3)$，正确极限为$\frac{1}{2}$。


### **二、加减替换的严格条件**
在**特定条件**下，加减运算中的等价替换可能成立，但需满足以下情况：

#### 1. **替换后整体不为零**
若替换后的表达式在加减运算中**不导致完全抵消**，即替换后的结果不为零，则可以替换。例如：
- $\lim_{x→0}\frac{x + \sin x}{x}$  
  替换$\sin x \sim x$后分子为$x + x = 2x$，结果为$2$。此处替换合法，因为替换后未完全抵消。

#### 2. **替换后的无穷小阶数相同且系数比非特定值**
设$\alpha \sim \alpha'$, $\beta \sim \beta'$，若$\lim_{x→a}\frac{\alpha'}{\beta'} \neq -1$（加法）或$\neq 1$（减法），则可替换：
- **加法条件**：$\lim_{x→a}\frac{\alpha'}{\beta'} \neq -1$  
  此时$\alpha + \beta \sim \alpha' + \beta'$。
- **减法条件**：$\lim_{x→a}\frac{\alpha'}{\beta'} \neq 1$  
  此时$\alpha - \beta \sim \alpha' - \beta'$。

#### 3. **高阶无穷小可忽略**
当加减运算中的高阶无穷小对极限无影响时，可进行替换。例如：
- $\lim_{x→0}\frac{x^2 + \sin x}{x}$  
  由于$x^2$为高阶无穷小，可忽略，直接替换$\sin x \sim x$，极限为$1$。

---

### **三、严格验证方法：[[知识枢纽：泰勒公式与等价无穷小|泰勒展开]]**
为确保替换的合法性，**泰勒展开**是最可靠的方法。通过展开到足够高阶，可明确高阶项的抵消情况：
- **正确操作**：$\tan x - \sin x = x + \frac{x^3}{3} - \left(x - \frac{x^3}{6}\right) + o(x^3) = \frac{x^3}{2} + o(x^3)$  
  此时分子为$\frac{1}{2}x^3$，可直接替换。

---

### **四、常见误区与注意事项**
1. **局部替换陷阱**  
   仅替换加减中的部分项（如$\sin x - x \sim x - x = 0$）会导致错误，必须整体分析。
2. **因子与非因子的区别**  
   只有作为乘除的**因子**（如$\sin x \cdot f(x)$中的$\sin x$）可替换，而$\sin x + f(x)$中的$\sin x$不可单独替换。
3. **极限存在性验证**  
   替换前需确认原式极限存在，否则可能导致错误。

---

### **五、总结表格**

| **情况**               | **是否可替换** | **条件/示例**                                                                 |
|------------------------|----------------|-----------------------------------------------------------------------------|
| 乘除中的因子           | ✅              | $\lim_{x→0}\frac{x \cdot \sin x}{x^3} = \lim_{x→0}\frac{x^2}{x^3} = \infty$ |
| 加减后整体不为零       | ✅              | $\lim_{x→0}\frac{x + \sin x}{x} = 2$                                        |
| 加减后高阶项抵消       | ❌              | $\tan x - \sin x$需展开到$x^3$项                                            |
| 替换后的系数比非-1或1 | ✅              | $\lim_{x→0}\frac{\alpha + \beta}{\gamma}$中$\alpha/\beta \neq -1$           |


---

通过上述分析可见，加减运算中的等价替换需严格验证条件或借助泰勒展开，以避免因高阶无穷小抵消导致的错误结果。
