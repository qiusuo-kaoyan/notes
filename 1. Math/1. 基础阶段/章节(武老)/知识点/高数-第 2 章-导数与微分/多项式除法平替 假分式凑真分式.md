#高等数学 
***

## 1. [问题背景](https://www.bilibili.com/video/BV1S14y1x7gC/?spm_id_from=333.1387.search.video_card.click&vd_source=18d354513d26eae91016959175f5e7f8)

*   **多项式除法：** 传统方法较复杂，易出错（如缺项需补零）。
*   **目标：** 将假分式（分子次数 ≥ 分母次数）化为真分式（分子次数 < 分母次数）。
*   **平替方法：** 通过“凑”的方式，避免传统的多项式除法。

## 2. 核心思想：凑

**重复操作：**

1.  **分子最高次 ÷ 分母最高次：** 得到一个商式。
2.  **商式 × 分母：** 构造一个与分子最高次相同的项。
3.  **补齐：** 在分子中减去/加上一些项，使分子与原分子相等。
4.  **拆分：** 将分式拆分为两部分：一个多项式 + 一个新的分式。
5.  **重复：** 对新的分式重复上述步骤，直到变为真分式。

**公式表示：**

将分子写成：

(分子的最高次 / 分母的最高次) * 分母 + 补齐项

## 3. 例题讲解

### 3.1. 例 1

化简：$\frac{x^3 - 3x^2 + 2}{x - 1}$

1.  **分子最高次 ÷ 分母最高次：** $x^3 / x = x^2$
2.  **$x^2$ × 分母：** $x^2(x - 1) = x^3 - x^2$
3.  **补齐：** 原分子 - $(x^3 - x^2) = -2x^2 + 2$
4.  **拆分：**
    $$
    \frac{x^3 - 3x^2 + 2}{x - 1} = \frac{x^2(x-1) - 2x^2 + 2}{x-1} = x^2 + \frac{-2x^2 + 2}{x-1}
    $$
5.  **重复：**
    *   $-2x^2 / x = -2x$
    *   $-2x(x-1) = -2x^2 + 2x$
    *   $(-2x^2 + 2) - (-2x^2 + 2x) = -2x + 2$
    $$
    x^2 + \frac{-2x^2 + 2}{x-1} = x^2 + \frac{-2x(x-1) -2x + 2}{x-1} = x^2 - 2x + \frac{-2x+2}{x-1}
    $$
6.  **重复：**
        * $-2x / x = -2$
        * $-2(x-1) = -2x + 2$
        * $(-2x+2) - (-2x+2) = 0$
     $$
     x^2 -2x + \frac{-2x+2}{x-1} = x^2 -2x + \frac{-2(x-1)}{x-1} = x^2 - 2x - 2
     $$

**最终结果：** $x^2 - 2x - 2$

### 3.2. 例 2

化简：$\frac{x^3}{x^2 - 2x - 3}$

1.  **$x^3 / x^2 = x$**
    $$
    \frac{x^3}{x^2 - 2x - 3} = \frac{x(x^2 - 2x - 3) + 2x^2 + 3x}{x^2 - 2x - 3} = x + \frac{2x^2 + 3x}{x^2 - 2x - 3}
    $$
2.  **$2x^2 / x^2 = 2$**
    $$
    x + \frac{2x^2 + 3x}{x^2 - 2x - 3} = x + \frac{2(x^2 - 2x - 3) + 7x + 6}{x^2 - 2x - 3} = x + 2 + \frac{7x+6}{x^2 - 2x - 3}
    $$

**最终结果：** $x + 2 + \frac{7x+6}{x^2 - 2x - 3}$ (已为真分式)

**注：**  $\frac{7x+6}{x^2 - 2x - 3}$ 可以进一步拆分为更简单的真分式之和（另有视频讲解）。

### 3.3 例3
化简:
$$\frac{\lambda^3 - 8\lambda + 8}{\lambda - 2}$$
1. $\lambda^3 / \lambda = \lambda^2$
$$
\frac{\lambda^3 - 8\lambda + 8}{\lambda - 2} = \frac{\lambda^2(\lambda - 2) + 2\lambda^2 - 8\lambda + 8}{\lambda - 2} = \lambda^2 + \frac{2\lambda^2 - 8\lambda + 8}{\lambda - 2}
$$
2. $2\lambda^2 / \lambda = 2\lambda$
$$
\lambda^2 + \frac{2\lambda^2 - 8\lambda + 8}{\lambda - 2} = \lambda^2 + \frac{2\lambda(\lambda - 2) -4\lambda + 8}{\lambda - 2} = \lambda^2 + 2\lambda + \frac{-4\lambda + 8}{\lambda - 2}
$$
3. $-4\lambda / \lambda = -4$
$$
\lambda^2 + 2\lambda + \frac{-4\lambda + 8}{\lambda - 2} = \lambda^2 + 2\lambda + \frac{-4(\lambda - 2)}{\lambda - 2} = \lambda^2 + 2\lambda - 4
$$
最终结果: $\lambda^2 + 2\lambda - 4$

## 4. 应用举例（解一元三次方程）

假设要解方程：$x^3 - 3x^2 + 2 = 0$

1.  **猜根：** 尝试 $x = 0, 1, -1$ 等，发现 $x = 1$ 是一个根。
2.  **因式分解：** $(x - 1)(...) = x^3 - 3x^2 + 2$
3.  **求另一因式：**
    $$
    (...) = \frac{x^3 - 3x^2 + 2}{x - 1}
    $$
    使用“凑”的方法，得到 $(...) = x^2 - 2x - 2$。
4.  **解方程：**
    $$
    (x - 1)(x^2 - 2x - 2) = 0
    $$
    分别解 $x - 1 = 0$ 和 $x^2 - 2x - 2 = 0$ 即可。

## 5. 总结

*   “凑”的方法是一种多项式除法的平替，避免了传统除法的繁琐。
*   核心思想是重复“分子最高次 ÷ 分母最高次 × 分母 + 补齐”的操作。
*   该方法可用于化简假分式、解方程等。
*   可以进一步将复杂的真分式, 拆分成多个简单的真分式之和.
