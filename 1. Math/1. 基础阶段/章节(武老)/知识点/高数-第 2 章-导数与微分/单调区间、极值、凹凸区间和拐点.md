***
**一句话: 拐点是点, 极值点是 $x$ 值**
### 1. **单调区间**
单调区间是指函数在某个区间内单调递增或单调递减。判断单调性的方法是通过一阶导数 $f'(x)$ 的符号：
- 如果 $f'(x) > 0$，则函数在该区间上单调递增；
- 如果 $f'(x) < 0$，则函数在该区间上单调递减。

**求解步骤**：
1. 求函数的导数 $f'(x)$；
2. 解方程 $f'(x) = 0$，找到临界点；
3. 将临界点与不可导点划定义域，得到若干个区间；
4. 在每个区间内判断 $f'(x)$ 的符号，从而确定单调区间。

**示例**：
对于函数 $y = x^2 \ln x$，其导数为 $y' = 2x \ln x + x$。令 $y' = 0$，解得 $x = 1$。通过分析 $y'$ 的符号，可以确定单调区间为：
- 单调递增区间：$(0, 1)$
- 单调递减区间：$(1, +\infty)$

### 2. **极值**
极值是函数在某一点附近取得的最大值或最小值。极值点通常出现在临界点（即 $f'(x) = 0$ 或 $f'(x)$ 不存在）处。

**求解步骤**：
1. 找到所有临界点；
2. 通过二阶导数 $f''(x)$ 判断极值点的性质：
   - 如果 $f''(x) < 0$，则该点为极大值点；
   - 如果 $f''(x) > 0$，则该点为极小值点；
   - 如果 $f''(x) = 0$，则需要进一步分析。

**示例**：
对于函数 $y = x^3 - 3x$，其导数为 $y' = 3x^2 - 3$，解得临界点为 $x = 1$ 和 $x = -1$。通过计算二阶导数 $y'' = 6x$，可以判断：
- $x = -1$ 为极大值点，极值为 $f(-1) = 4$
- $x = 1$ 为极小值点，极值为 $f(1) = -2$

### 3. **凹凸区间**
凹凸区间是指函数在某个区间内是凹函数还是凸函数。判断凹凸性的方法是通过二阶导数 $f''(x)$ 的符号：
- 如果 $f''(x) > 0$，则函数在该区间上是凹函数；
- 如果 $f''(x) < 0$，则函数在该区间上是凸函数。

**求解步骤**：
1. 求函数的二阶导数 $f''(x)$；
2. 解方程 $f''(x) = 0$，找到拐点；
3. 将拐点划定义域，得到若干个区间；
4. 在每个区间内判断 $f''(x)$ 的符号，从而确定凹凸区间。

**示例**：
对于函数 $y = x^3 - 3x$，其二阶导数为 $y'' = 6x$。令 $y'' = 0$，解得 $x = 0$。通过分析 $y''$ 的符号，可以确定：
- 凹区间：$(0, +\infty)$
- 凸区间：$(-\infty, 0)$
- 拐点：$(0, 0)$

### 4. **拐点**
拐点是函数凹凸性发生变化的点，通常出现在二阶导数 $f''(x)$ 为0或不存在的点处。

**求解步骤**：
1. 找到所有可能的拐点（即 $f''(x) = 0$ 或 $f''(x)$ 不存在）；
2. 检查拐点两侧的二阶导数符号是否发生变化；
3. 如果符号发生变化，则该点为拐点。

**示例**：
对于函数 $y = x^3 - 3x$，其拐点为 $(0, 0)$

### 总结
- **单调区间**：通过一阶导数的符号判断；
- **极值**：通过临界点和二阶导数判断；
- **凹凸区间**：通过二阶导数的符号判断；
- **拐点**：通过二阶导数为0或不存在的点判断。

这些概念在考研数学中非常重要，通常需要结合导数的分析来求解。通过系统地分析导数的符号和变化，可以准确判断函数的单调性、极值、凹凸性和拐点。
