这道题是一道经典的二阶常系数线性非齐次微分方程的初值问题，重点展示了如何用微分算子法求解形如 $xe^{kx}$ 的特解，这是微分算子法中综合性最强、计算最复杂的类型之一。

**题目：** 微分算子法求解 $e^{kx}P_n(x)$ 型特解（初值问题）
![[例 2.png]]
**知识点类型：** 二阶常系数线性非齐次微分方程、初值问题、齐次通解、非齐次特解、微分算子法、待定系数法、几何级数

### 1. 核心知识点回顾

*   **通解结构：** 通解 = 齐次通解 + 非齐次特解 ($Y = Y_c + y^*$)

*   **齐次通解求法：**
    *   特征方程：$\lambda^2 + 2\lambda + 1 = 0 \implies (\lambda+1)^2 = 0$
    *   特征根：$\lambda_1 = \lambda_2 = -1$ (二重实根)
    *   齐次通解：$Y_c = (C_1 + C_2 x)e^{-x}$

*   **微分算子法：**  $y^* = \frac{1}{f(D)}F(x)$

*   **类型三：$F(x) = e^{kx}V(x)$ (混合型)**
    *   **指数外提, 算子平移：** 将 $e^{kx}$ 移到算子左侧，代价是将算子中的 $D$ 替换为 $(D+k)$。
        $$
        y^* = \frac{1}{f(D)}e^{kx}V(x) = e^{kx}\frac{1}{f(D+k)}V(x)
        $$

*   **类型四：$F(x) = P_n(x)$ (多项式型)**
    *   **泰勒展开法：**
        1.  将算子 $\frac{1}{f(D)}$ 凑成几何级数求和公式的形式 $\frac{1}{1-q}$ 或 $\frac{1}{1+q}$。
        2.  将算子展开成 $D$ 的多项式。
        3.  将展开后的算子作用于多项式 $P_n(x)$，展开到 $D^n$ 即可。

### 2. 解题步骤（本题特解求解）

1.  **写出算子形式：**
    $$
    y^* = \frac{1}{D^2 + 2D + 1} (xe^x) = \frac{1}{(D+1)^2} (xe^x)
    $$

2.  **指数外提, 算子平移：** (这里 $k=1$, $V(x)=x$)
    $$
    y^* = e^x \frac{1}{((D+1)+1)^2} x = e^x \frac{1}{(D+2)^2} x
    $$

3.  **处理多项式部分（泰勒展开法）：**
    *   **凑几何级数形式：**
        $$
        \frac{1}{(D+2)^2} = \frac{1}{4(1 + \frac{D}{2})^2} = \frac{1}{4}(1 + \frac{D}{2})^{-2}
        $$
        可以使用 $(1+q)^{-2} = 1 - 2q + 3q^2 - \dots$ 的展开，或者更通用的方法：
        $$
        \frac{1}{(D+2)^2} = \frac{1}{4(1+\frac{D}{2})^2} = \frac{1}{4} \frac{1}{(1-(-\frac{D}{2}))^2}
        $$
       凑成$\frac{1}{1-q}$的形式:
         $$
         \frac{1}{D+2} = \frac{1}{2(1+\frac{D}{2})} = \frac{1}{2}\frac{1}{1-(-\frac{D}{2})} = \frac{1}{2}(1 - \frac{D}{2} + \frac{D^2}{4} - \dots)
         $$
         所以
         $$
         \frac{1}{(D+2)^2} = [\frac{1}{2}(1-\frac{D}{2}+\dots)]^2 = \frac{1}{4}(1-D+\dots)
         $$

    *   **展开算子：**
        因为 $V(x)=x$ 是一次多项式，所以算子展开到 $D$ 即可。
        $$
         \frac{1}{4}(1-D+\dots)
        $$

    *   **作用于多项式：**
        $$
        \frac{1}{4}(1-D)x = \frac{1}{4}(x - D(x)) = \frac{1}{4}(x-1)
        $$

4.  **组合得到特解：**
    $$
    y^* = e^x \cdot \frac{1}{4}(x-1) = \frac{1}{4}(x-1)e^x
    $$

5.  **确定通解并求解初值问题：**
    *   通解：$y = Y_c + y^* = (C_1 + C_2 x)e^{-x} + \frac{1}{4}(x-1)e^x$
    *   利用初始条件 $y(0)=0, y'(0)=0$ 求出 $C_1, C_2$。
    *   $y(0) = C_1 - \frac{1}{4} = 0 \implies C_1 = \frac{1}{4}$
    *   $y' = C_2 e^{-x} - (C_1 + C_2 x)e^{-x} + \frac{1}{4}e^x + \frac{1}{4}(x-1)e^x$
    *   $y'(0) = C_2 - C_1 + \frac{1}{4} - \frac{1}{4} = C_2 - C_1 = 0 \implies C_2 = C_1 = \frac{1}{4}$
    *   **最终特解：** $y = (\frac{1}{4} + \frac{1}{4}x)e^{-x} + \frac{1}{4}(x-1)e^x$

### 3. 重点总结

*   熟练掌握微分算子法处理四种基本类型的方法，特别是混合型和多项式型。
*   对于混合型 $e^{kx}V(x)$，关键是先使用指数函数移出法则，再处理剩余的 $V(x)$
*   对于多项式型，关键是将算子凑成几何级数求和的形式并进行展开。
*   在展开算子时，要注意展开的精度，对于 $n$ 次多项式，展开到 $D^n$ 即可。
*   微分算子法虽然步骤固定，但每一步的计算都需要细心，特别是符号和系数。
*   这是一个展示微分算子法威力的典型题目，相比待定系数法，计算量大大减少。
