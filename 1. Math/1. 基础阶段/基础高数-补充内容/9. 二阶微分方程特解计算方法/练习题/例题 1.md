这道题是一道综合题，将变限积分求导与二阶常系数线性非齐次微分方程的求解结合在一起，并展示了如何用微分算子法快速确定特解。

**题目：** 变限积分与二阶常系数线性非齐次微分方程综合（微分方程、微分算子）
![[例1.png]]
**知识点类型：** 变限积分求导、二阶常系数线性非齐次微分方程、特征方程、微分算子法、初值问题

### 1. 核心知识点

*   **变限积分求导：**
    *   **基本公式：** $\frac{d}{dx} \int_a^x f(t) dt = f(x)$
    *   **带参数处理：** 如果被积函数中含有 $x$，需要先将其提到积分号外或通过变量代换处理。
    *   **本题处理方法：**
        1.  先展开：$\int_0^x (x-t)f(t) dt = \int_0^x xf(t) dt - \int_0^x tf(t) dt$
        2.  提常数：$= x\int_0^x f(t) dt - \int_0^x tf(t) dt$
        3.  再求导：使用乘法求导法则对第一项求导。

*   **二阶常系数线性非齐次微分方程：**  $y'' + Ay' + By = F(x)$
    *   **通解结构：** 通解 = 齐次方程通解 + 非齐次方程特解 ($Y = Y_c + y^*$)
    *   **求齐次通解 $Y_c$：**
        1.  写出特征方程：$\lambda^2 + A\lambda + B = 0$
        2.  解出特征根 $\lambda$。
        3.  根据特征根的形式（实根、复根、重根）写出齐次通解。
    *   **求非齐次特解 $y^*$：**
        1.  **待定系数法：** 根据 $F(x)$ 的形式，假设特解的形式==（可能需要乘以弹性因子 $x^k$, 别设错了）==，然后代入原方程求解待定系数。
        2.  **微分算子法：**

*   **微分算子法（本题应用）：**
    *   **类型：** $F(x) = -\cos x$ (三角函数型)
    *   **算子形式：** $y^* = \frac{1}{D^2+1}(-\cos x) = - \frac{1}{D^2+1}\cos x$
    *   **计算：**
        *   将 $a=1$ 代入 $D^2$，得到 $D^2 = -a^2 = -1$。
        *   分母变为 $-1 + 1 = 0$。
        *   **分母为 0 的处理：** 分子乘以 $x$，分母对 $D$ 求导。
            $$
            y^* = -x \frac{1}{2D}\cos x = -\frac{x}{2} D^{-1}(\cos x) = -\frac{x}{2} \sin x
            $$

*   **初始条件的获取：**
    *   题目没有直接给出初始条件，但可以从原积分方程中推导出来。
    *   将 $x=0$ 代入原方程，可以求得 $f(0)$。
    *   将原方程求一次导，得到含有 $f'(x)$ 的方程，再代入 $x=0$ 求得 $f'(0)$

### 2. 解题步骤

1.  **化简并求导，得到微分方程：**
    *   原方程：$f(x) = \cos x - \int_0^x (x-t)f(t) dt = \cos x - x\int_0^x f(t)dt + \int_0^x tf(t)dt$
    *   第一次求导：$f'(x) = -\sin x - \int_0^x f(t)dt - xf(x) + xf(x) = -\sin x - \int_0^x f(t)dt$
    *   第二次求导：$f''(x) = -\cos x - f(x)$
    *   整理得：$f''(x) + f(x) = -\cos x$

2.  **求解微分方程：**

    *   **齐次通解：**
        *   $f''(x) + f(x) = 0$
        *   特征方程：$\lambda^2 + 1 = 0 \implies \lambda = \pm i$ (纯虚根, $\alpha=0, \beta=1$)
        *   $Y_c = e^{0x}(C_1 \cos(1x) + C_2 \sin(1x)) = C_1 \cos x + C_2 \sin x$

    *   **非齐次特解（微分算子法）：**
        *   $y^* = \frac{1}{D^2+1}(-\cos x) = -x \frac{1}{2D}\cos x = -\frac{x}{2} \sin x$

    *   **通解：** $f(x) = C_1 \cos x + C_2 \sin x - \frac{x}{2} \sin x$

3.  **确定初始条件：**
    *   $x=0$ 代入原方程：$f(0) = \cos 0 - 0 = 1$
    *   $x=0$ 代入第一次求导后的方程：$f'(0) = -\sin 0 - 0 = 0$

4.  **利用初始条件求特解：**
    *   $f(0) = C_1 \cos 0 + C_2 \sin 0 - 0 = C_1 = 1$
    *   $f'(x) = -C_1 \sin x + C_2 \cos x - \frac{1}{2}\sin x - \frac{x}{2}\cos x$
    *   $f'(0) = -0 + C_2 - 0 - 0 = C_2 = 0$
    *   得到 $C_1 = 1, C_2 = 0$

5.  **最终结果：** $f(x) = \cos x - \frac{x}{2} \sin x$

### 3. 重点总结

*   掌握将变限积分方程转化为微分方程的方法：通过一次或多次求导。
*   熟练运用二阶常系数线性非齐次微分方程的求解步骤。
*   微分算子法是求解特解的强大工具，特别是对于三角函数型，且分母为 0 的情况。
*   能够从积分方程中提取出求解特解所需的初始条件。
*   本题是一个很好的综合题，连接了变限积分、微分方程和初值问题，考察了综合分析和解决问题的能力。
