这道题是一道综合了微分方程求解和极限性质的题目，重点在于使用微分算子法求解特解，并通过极限条件来确定通解中的任意常数。

**题目：** 利用极限条件求解微分方程特解（微分算子法、极限性质）
![[例3.png]]
**知识点类型：** 二阶常系数线性非齐次微分方程、齐次通解、非齐次特解、微分算子法、极限性质

### 1. 核心知识点

*   **二阶常系数线性非齐次微分方程求解步骤：**
    1.  求齐次方程通解 $Y_c$。
    2.  求非齐次方程特解 $y^*$。
    3.  通解 $Y = Y_c + y^*$。
    4.  利用初始条件或极限条件确定任意常数。

*   **齐次通解求法：**
    *   特征方程：$\lambda^2 - 3\lambda + 2 = 0 \implies (\lambda-1)(\lambda-2) = 0$
    *   特征根：$\lambda_1 = 1, \lambda_2 = 2$ (两个不等的实根)
    *   齐次通解：$Y_c = C_1 e^x + C_2 e^{2x}$

*   **微分算子法（本题应用）：**
    *   **类型：** $F(x) = 10e^{-x}\sin x$ (混合型)
    *   **算子形式：** $y^* = \frac{1}{D^2 - 3D + 2} (10e^{-x}\sin x)$
    *   **计算步骤：**
        1.  **指数外提, 算子平移：** 将 $e^{-x}$ 移出，代价是将 $D$ 替换为 $(D-1)$。
            $$
            y^* = 10e^{-x} \frac{1}{(D-1)^2 - 3(D-1) + 2} \sin x = 10e^{-x} \frac{1}{D^2 - 5D + 6} \sin x
            $$
        2.  **处理三角函数(看见平方就换$-a^2$)：** 将 $D^2$ 替换为 $-a^2 = -1^2 = -1$
            $$
            y^* = 10e^{-x} \frac{1}{-1 - 5D + 6} \sin x = 10e^{-x} \frac{1}{5 - 5D} \sin x = 2e^{-x} \frac{1}{1-D} \sin x
            $$
        3.  **凑平方差：** 分子分母同乘 $(1+D)$。
            $$
            y^* = 2e^{-x} \frac{1+D}{(1-D)(1+D)} \sin x = 2e^{-x} \frac{1+D}{1-D^2} \sin x
            $$
        4.  **再次代入 $D^2 = -1$：**
            $$
            y^* = 2e^{-x} \frac{1+D}{1-(-1)} \sin x = e^{-x}(1+D)\sin x
            $$
        5.  **展开并计算：**
            $$
            y^* = e^{-x}(\sin x + D(\sin x)) = e^{-x}(\sin x + \cos x)
            $$

*   **极限性质：**
    *   当 $x \to +\infty$ 时，$\lim e^{kx} = +\infty$ (如果 $k>0$)。
    *   当 $x \to +\infty$ 时，$\lim e^{kx} = 0$ (如果 $k<0$)。
    *   如果 $\lim (A+B) = 0$，且 $A$ 和 $B$ 都是趋于 $+\infty$ 的项，则要使和为 0，必须使 $A$ 和 $B$ 的系数都为 0。

### 2. 解题步骤

1.  **求齐次通解：**
    *   $Y_c = C_1 e^x + C_2 e^{2x}$

2.  **求非齐次特解（微分算子法）：**
    *   $y^* = e^{-x}(\sin x + \cos x)$

3.  **写出通解：**
    *   $y(x) = Y_c + y^* = C_1 e^x + C_2 e^{2x} + e^{-x}(\sin x + \cos x)$

4.  **利用极限条件确定常数：**
    *   已知 $\lim_{x \to +\infty} y(x) = 0$。
    *   $\lim_{x \to +\infty} y(x) = \lim_{x \to +\infty} (C_1 e^x + C_2 e^{2x} + e^{-x}(\sin x + \cos x))$
    *   其中，$\lim_{x \to +\infty} e^{-x}(\sin x + \cos x) = 0$ (因为 $e^{-x} \to 0$，而三角函数有界)。
    *   要使整个极限为 0，必须有 $\lim_{x \to +\infty} (C_1 e^x + C_2 e^{2x}) = 0$。
    *   因为 $e^x \to +\infty$ 和 $e^{2x} \to +\infty$，所以唯一的可能是 $C_1 = 0$ 且 $C_2 = 0$。

5.  **最终结果（特解）：**
    *   $y(x) = e^{-x}(\sin x + \cos x)$

### 3. 重点总结

*   熟练掌握微分算子法求解混合型特解的步骤：先移出指数函数，再处理剩余部分。
*   理解利用极限条件来确定微分方程通解中任意常数的方法。
*   掌握指数函数在 $x \to +\infty$ 时的极限性质，这是确定常数的关键。
*   本题是一个很好的综合题，将微分方程求解与极限理论结合，考察了综合分析和解决问题的能力。
*   对于形如$A+B=0$的极限, 如果$A$和$B$都是趋于无穷的项, 则$A$和$B$的系数必须都为0, 这是解决本题的关键.
