好的，我们来提炼这道题的核心知识点。这道题是一道非常经典的综合题，它巧妙地将变限积分、微分方程、牛顿-莱布尼茨公式和微分算子法结合在了一起。

**题目：** 变限积分与二阶常系数线性非齐次微分方程的综合
![[例4.png]]
**知识点类型：** 变限积分求导、牛顿-莱布尼茨公式、二阶常系数线性非齐次微分方程、微分算子法、初值问题

### 1. 核心知识点

*   **变限积分求导：**
    *   **基本公式：** $\frac{d}{dx} \int_a^x f(t) dt = f(x)$
    *   **核心策略：** 看到题目中含有变限积分的等式，尤其是要求函数表达式时，一个重要的思路就是通过**求导**，将积分方程转化为**微分方程**。

*   **牛顿-莱布尼茨公式的应用（凑微分）：**
    *   **核心观察：** 识别被积函数是否是某个函数求导后的结果。
    *   **重要形式：** $\frac{d}{dx}[e^{ax}f(x)] = e^{ax}[f'(x) + af(x)]$
    *   **本题应用：** 被积函数为 $e^{-x}(f'(x)-f(x))$，可以观察到
        $$
        \frac{d}{dx}[e^{-x}f(x)] = (e^{-x})'f(x) + e^{-x}f'(x) = -e^{-x}f(x) + e^{-x}f'(x) = e^{-x}(f'(x)-f(x))
        $$
        因此，可以直接对 $e^{-x}f(x)$ 使用牛顿-莱布尼茨公式。

*   **二阶常系数线性非齐次微分方程：** $y'' + Ay' + By = F(x)$
    *   **通解结构：** 通解 = 齐次方程通解 + 非齐次方程特解 ($Y = Y_c + y^*$)
    *   **微分算子法：** 用于快速求解特解 $y^*$。

*   **从积分方程中寻找初始条件：**
    *   当需要确定通解中的任意常数时，可以从原积分方程中寻找“隐藏”的初始条件。
    *   例如，将积分下限的值（如 $x=0$）代入原方程和求导后的方程，可以得到 $f(0)$ 和 $f'(0)$ 的值。

### 2. 解题步骤

1.  **分析并简化目标定积分：**
    *   目标：$\int_0^{\pi/2} e^{-x}(g(x)-f(x))dx$
    *   已知 $g(x) = f'(x)$，代入得：$\int_0^{\pi/2} e^{-x}(f'(x)-f(x))dx$
    *   根据凑微分的观察，被积函数是 $e^{-x}f(x)$ 的导数。
    *   应用牛顿-莱布尼茨公式：
        $$
        \int_0^{\pi/2} e^{-x}(f'(x)-f(x))dx = [e^{-x}f(x)]_0^{\pi/2} = e^{-\pi/2}f(\pi/2) - e^0f(0)
        $$
    *   问题转化为求解函数 $f(x)$ 的表达式，并计算 $f(\pi/2)$ 和 $f(0)$。

2.  **建立微分方程：**
    *   已知 $g(x) = 1 + \int_0^x (1-f(t))dt$ 和 $f'(x) = g(x)$。
    *   对第一个式子两边求导：$g'(x) = 1 - f(x)$。
    *   对第二个式子两边求导：$f''(x) = g'(x)$。
    *   联立得到：$f''(x) = 1 - f(x)$，即 $f''(x) + f(x) = 1$。

3.  **求解微分方程：**
    *   **齐次通解：** $f''(x) + f(x) = 0$ 的特征方程为 $\lambda^2+1=0$，解得 $\lambda=\pm i$。
        $Y_c = C_1\cos x + C_2\sin x$
    *   **非齐次特解（微分算子法）：** $y^* = \frac{1}{D^2+1}(1) = \frac{1}{D^2+1}(e^{0x})$
        代入 $D=0$，$y^* = \frac{1}{0^2+1}(1) = 1$。
    *   **通解：** $f(x) = C_1\cos x + C_2\sin x + 1$。

4.  **确定初始条件：**
    *   从 $f'(x) = g(x) = 1 + \int_0^x (1-f(t))dt$
    *   令 $x=0$：$f'(0) = g(0) = 1 + 0 = 1$。
    *   题目中隐含条件 $f(0)=0$ (从视频上下文推断或作为标准初值问题的一部分)。
    *   **初始条件为：$f(0)=0, f'(0)=1$。**

5.  **求解特解：**
    *   $f(0) = C_1\cos 0 + C_2\sin 0 + 1 = C_1+1 = 0 \implies C_1 = -1$
    *   $f'(x) = -C_1\sin x + C_2\cos x$
    *   $f'(0) = -C_1(0) + C_2(1) = C_2 = 1$
    *   **最终特解为：** $f(x) = -\cos x + \sin x + 1$

6.  **计算最终结果：**
    *   $f(0) = 0$
    *   $f(\pi/2) = -\cos(\pi/2) + \sin(\pi/2) + 1 = 0 + 1 + 1 = 2$
    *   代入第一步的结果：$e^{-\pi/2}f(\pi/2) - f(0) = e^{-\pi/2}(2) - 0 = 2e^{-\pi/2}$

### 3. 重点总结

*   解决此类综合题的关键是**识别问题类型并进行转化**。本题的核心是将一个看似复杂的定积分问题，通过求导和凑微分，转化为一个标准的微分方程初值问题。
*   **“看到变限积分方程，想到求导”** 是一个非常重要的解题思路。
*   **微分算子法**是求解常系数线性非齐次微分方程特解的利器，应熟练掌握。
*   要善于从题目给出的积分方程中**挖掘隐藏的初始条件**。
*   **凑微分**的思想在积分计算中至关重要，能极大简化计算。
