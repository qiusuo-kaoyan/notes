# [[深度解析：微分算子法求特解]]

> [!abstract] 核心思想 (一句话总结)
> 将微分($\frac{d}{dx}$)和积分($\int$)运算符号化为代数算子 D，通过一系列代数运算法则（如平移、求导、泰勒展开），将求解特解的复杂过程转化为标准化的、高效的代数计算。

---

## 📝 方法详解

### 1. 引入微分算子 (D)

*   **定义：**
    *   **微分算子 $D$：** 代表求导运算，即 $D \equiv \frac{d}{dx}$。
        *   $Dy = y'$
        *   $D^2y = y''$
    *   **积分算子 $\frac{1}{D}$：** 代表积分运算，即 $\frac{1}{D} \equiv \int (\cdot) dx$。
        *   $\frac{1}{D}f(x) = \int f(x) dx$
*   **运算性质：**
    *   求导算子 $D$ 的多项式可以直接使用乘法分配律。
    *   积分算子 $\frac{1}{D}$ 的多项式**不能**直接拆开，需要特殊方法处理。

### 2. 求解特解的基本框架

对于二阶常系数线性非齐次方程 $y'' + ay' + by = f(x)$：

1.  **构造算子多项式 $F(D)$：**
    *   根据特征方程 $\lambda^2 + a\lambda + b = 0$，得到 $F(D) = D^2 + aD + b$。
2.  **写出特解的算子形式：**
    *   特解 $y^* = \frac{1}{F(D)} f(x)$。
3.  **计算并消去算子 $D$：**
    *   核心任务，根据 $f(x)$ 的类型使用相应规则计算。

### 3. 四大核心类型求解

#### 型一：指数函数型 $f(x) = e^{kx}$
*   **基本法则：“见 D 就换 k”**
    $$ y^* = \frac{1}{F(D)} e^{kx} = \frac{1}{F(k)} e^{kx} \quad (\text{前提: } F(k) \neq 0) $$
*   **特殊情况 ($F(k) = 0$)：“提 x 求导”**
    $$ y^* = x \frac{1}{F'(D)} e^{kx} $$

#### 型二：三角函数型 $f(x) = \sin(ax)$ 或 $\cos(ax)$
*   **基本法则：“见 $D^2$ 就换 $-a^2$”**
*   **处理一次项 D：“剩下的 D 用平方差”**，构造出新的 $D^2$。
*   **特殊情况 (分母为 0)：** 同理，“提 x 求导”。

#### 型三：指数与函数乘积型 $f(x) = e^{kx}V(x)$
*   **基本法则：“指数外提，算子平移” (Shift Theorem)**
    $$ y^* = \frac{1}{F(D)} [e^{kx}V(x)] = e^{kx} \frac{1}{F(D+k)} V(x) $$
    *   此方法是“降维打击”，将型三问题转化为求解 $V(x)$ 的问题（通常是型一、二、四）。

#### 型四：多项式函数型 $f(x) = P_n(x)$
*   **基本法则：“泰勒展开算子”**
    *   利用 $\frac{1}{1-q} = 1 + q + q^2 + \dots$ 将算子展开成 $D$ 的幂级数，展开到 $D^n$ 项即可。

### 4. 综合应用与考场技巧
*   **可加性：** 右端项为加和时，可拆开分别求特解再相加。
*   **考场技巧：** 对于简单问题，仍可用待定系数法心算。微分算子法作为“重武器”，用于处理复杂情况或验算，计算过程可写在草稿纸上。
