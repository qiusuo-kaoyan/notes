**660对应题目: [[18. (总结)含有递推式数列极限的证明]]**

## 1. 题型与思路

### 1.1. 两种题型

1.  **含有递推式的数列极限：** 给出 $a_{n+1}$ 与 $a_n$ 的关系式（递推式），证明极限存在并求极限。
2.  **n 项和的数列极限：** 证明 $\lim_{n \to \infty} \sum_{i=1}^n a_i$ 存在并求极限。(不讨论)

**重点：** 数列极限证明题通常作为压轴题，需要一套完整的逻辑体系，而不仅仅是掌握方法。

### 1.2. 核心思路：逻辑

做数列极限证明题，关键在于建立一套完整的逻辑，按步骤分析：

1.  **有无递推式：**
    *   无递推式：首要任务是**寻找递推式**。
    *   有递推式：进入下一步。
2.  **选择方法：** 按以下顺序考虑三种方法。

## 2. 方法


### 2.1. 总体思路

1.  **有无递推式：**
    *   **无递推式：** 先找递推式。
    *   **有递推式：** 按以下顺序考虑三种方法。

2.  **三种方法（按考虑顺序）：**
    1.  **单调有界准则：** 真题中最常考，题型最多变。
        *   先考虑单调性，再考虑有界性。
    2.  **先斩后奏（夹逼定理）：**
        *   先在草稿纸上求出极限值，再构造式子证明极限存在且等于该值。
    3.  **构造新数列：** 适用于递推式涉及三项 ($a_{n+2}$, $a_{n+1}$, $a_n$) 的情况。
        *   数三可用差分方程（更简单）。

### 2.1. 单调有界准则

*   **最常用，也最难。**
*   **准则内容：** 单调有界数列必有极限。
    *   单调递增数列有上界，则极限存在。
    *   单调递减数列有下界，则极限存在。
*   **思考顺序：**
    1.  **先证单调性：**
        *   **求导法（首选）：**
            *   将递推式 $a_{n+1} = f(a_n)$ 看作 $y = f(x)$。
            *   求导 $f'(x)$：
                *   $f'(x) > 0$：数列单调（具体单增还是单减看前两项）。
                *   $f'(x) < 0$：数列**不单调**（直接放弃单调有界准则）。
                *   $f'(x)$ 正负不确定：尝试做差或做比。
        *   **做差或做比：**
            *   **做差：** $a_{n+1} - a_n$
                *   与 0 比较：
                    *   $> 0$：单调递增
                    *   $< 0$：单调递减
                *   与 $a_n - a_{n-1}$ 比较：
                    *   同号：单调
                    *   异号：不单调（放弃单调有界准则）
            *   **做比：** $\frac{a_{n+1}}{a_n}$ 与 1 比较
                *   $> 1$：单调递增
                *   $< 1$：单调递减
            *   **选择：** 根据递推式，选择做差或做比中更简便的方法。
        * **单调性无法证明**: 如果求导法和做差/做比都无法确定单调性, 不要立即放弃单调有界, 应该先尝试证明有界性, 再利用有界性回头证明单调性.
    2.  **再证有界性：**
        *   **分离常数法（首选）：**
            *   将递推式 $a_{n+1} = f(a_n)$ 变形，分离出常数，判断有界性。
            *   例如：$a_{n+1} = \frac{a_n}{a_n + 1} = 1 - \frac{1}{a_n + 1}$
        *   **数学归纳法：**
            *   需要先猜出极限值（作为界）。
            *   根据首项与极限值的大小关系，判断是证明上界还是下界。
        *   **常见不等式：** 利用已知不等式进行放缩。
        *   **根据第一问条件：** 利用题目中第一问的结论（如最值、极值）证明有界性。

### 2.2. 先斩后奏（极限的定义）

*   **思路：**
    1.  **先斩：** 在草稿纸上，假设极限存在，设为 A，利用递推式两边取极限，解出 A。
    2.  **后奏：** 在答题纸上，构造 $|a_n - A|$，利用递推式和 A 的性质，逐步放缩，证明 $|a_n - A| \to 0$ (利用夹逼定理)。
*   **关键步骤：**
    1.  草稿纸上：$a_{n+1} = f(a_n)$ 两边取极限，得到 $A = f(A)$，解出 A（常数）。
    2.  答题纸上：
        *   构造 $|a_{n+1} - A|$。
        *   将 $A$ 替换为 $f(A)$ (因为 $A = f(A)$)。
        *   利用递推式和 $f(A)$，将 $|a_{n+1} - A|$ 放缩为 $k|a_n - A|$ 的形式，其中 $0 < k < 1$。
        *   逐步递推：$|a_{n+1} - A| \leq k|a_n - A| \leq k^2|a_{n-1} - A| \leq ... \leq k^n|a_1 - A|$
        *   夹逼定理：由于 $0 < k < 1$，$\lim_{n \to \infty} k^n = 0$，因此 $\lim_{n \to \infty} |a_n - A| = 0$，即 $\lim_{n \to \infty} a_n = A$。
* **注意：**
    *   A 是常数，不是设的极限, 不能用"假设极限存在且为A"作为前提条件, 这是循环论证.
    *   利用 $A = f(A)$ 将常数 A 转化为与递推式相关的形式，便于放缩。

### 2.3. 构造新数列

*   **适用情况：** 递推式涉及三项：$a_{n+2}$, $a_{n+1}$, $a_n$。
*   **思路：**
    1.  **构造关系式：** 通过递推式，找到 $a_{n+2} - a_{n+1}$ 与 $a_{n+1} - a_n$ 之间的关系。
    2.  **构造等比数列：** 设法将关系式变形为 $\frac{a_{n+2} - a_{n+1}}{a_{n+1} - a_n} = k$ (常数) 的形式。
    3.  **令 $c_n = a_{n+1} - a_n$：** 则 $\{c_n\}$ 是以 $k$ 为公比的等比数列。
    4.  **求出 $\{c_n\}$ 的通项：** $c_n = c_1 \cdot k^{n-1}$，其中 $c_1 = a_2 - a_1$。
    5.  **累加法：**
        *   $c_1 = a_2 - a_1$
        *   $c_2 = a_3 - a_2$
        *   ...
        *   $c_n = a_{n+1} - a_n$
        *   将以上各式相加，得到 $c_1 + c_2 + ... + c_n = a_{n+1} - a_1$
    6.  **求极限：**
        *   左边是等比数列求和，利用等比数列求和公式，并取极限（注意公比 $k$ 的绝对值小于 1）。
        *   右边取极限：$\lim_{n \to \infty} (a_{n+1} - a_1) = \lim_{n \to \infty} a_{n+1} - a_1 = \lim_{n \to \infty} a_n - a_1$
        *   从而求出 $\lim_{n \to \infty} a_n$。
* **数三的平替方法**: 差分方程. (数一数二不用)

## 3. 总结

1.  **拿到递推式，首选单调有界准则。**
2.  **单调有界准则：先单调，后有界。**
3.  **单调性：求导法优先，不行再做差/做比。**
4.  **有界性：分离常数优先，不行再数学归纳法、常见不等式、第一问结论。**
5.  **单调性无法证明时, 不要放弃单调有界, 先尝试证明有界性.**
6.  **单调性证明失败（如导数小于 0，或做差/做比异号），放弃单调有界，考虑先斩后奏。**
7.  **三项递推式，考虑构造新数列（或差分方程）。**
