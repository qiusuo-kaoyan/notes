# 考研数学基础阶段：习题册做不动怎么办？

核心围绕数学基础薄弱的同学，在考研数学基础复习阶段，如何选择合适的习题册以及应该采用何种学习方法。

## 1. 核心问题与困境

*   **问题描述：** 数学基础相对薄弱的同学，在基础阶段听课后，做配套习题时感到非常吃力，进度缓慢，甚至“做不动”。
*   **心理影响：**
    *   进度慢导致心理压力大。
    *   产生焦虑情绪。
    *   影响基础阶段的整体备考状态和信心。
*   **常见习题册的痛点分析 (针对基础薄弱同学)：**
    *   **《1000题》 (基础部分)：**
        *   常与张宇老师《基础30讲》配套。
        *   **痛点：** 对基础薄弱者难度偏高，做起来比较吃力。
    *   **《1800题》：**
         *   **痛点：** 题量过大，完成压力大，耗时太长，容易拖慢进度。
    *   **《660题》：**
        *   **定位问题：** 通常规划在强化阶段（如9月份二轮强化）使用，不完全适合纯基础阶段扫盲和巩固。

* **需求：** 需要一本难度和题量都**比较适中**，适合基础阶段打基础、培养信心的习题册。

## 2. 针对基础薄弱同学的习题册建议

*   **推荐：** 《基础700题》 (张宇、李良老师编写)
*   **核心定位：** 专门适合**基础薄弱**的同学在**基础阶段**使用的习题册。
*   **特点与优势：**
    *   **结构组成：**
        *   包含 高等数学、线性代数、概率论 三册。
    *   **题型全面：**
        *    包含选择题、填空题、解答题。
        *    **重点：** 证明题很少，减少基础阶段的刷题障碍和畏难情绪。基础阶段重点在于理解概念和基本运算，而非复杂证明。
    *   **难度适中：**
        *   难度与《1800题》基础篇的常规题目相当。
        *   没有过难的题目，便于建立信心和巩固基础概念。
    *   **题量适中：**
         *   题量相对较少，能够在合理时间内完成听课和做题的循环，保证进度。
    *   **普适性好：**
        *    无论跟随哪位老师的课程，都可以使用这本习题册进行练习。
    *   **目的：** 平衡难度和题量，避免因“做不动”或“题太多”导致的焦虑和进度拖延，保护备考状态。

## 3. 基础阶段数学学习方法详解 (核心步骤)

**三步走**: 强调==按步骤==、==留痕迹==、==重错题==。

### 3.1. 学习流程六步骤

1.  **听课 (Listen & Note)：**
    *   **要求：** 必须动笔，边听边记。
    *   **操作：**
        *   在书上及时 `圈画` 老师讲课的重点。
        *   书上没有但老师补充的内容，要及时在书上 `总结` 和 `补充`。
    *   **目的：** 集中注意力，初步标记核心知识点，形成自己的笔记体系。

2.  **回顾内容 (Review)：**
    *   **时机：** 一章的课听完之后，**开始做习题之前**。
    *   **操作：** 快速回顾一遍本章的概念、定理、公式及自己的笔记。
    *   **目的：** 唤醒记忆，整体把握本章知识结构，为做题做准备。

3.   **重做例题 (Redo Examples)：**
    *   **时机：** 回顾内容之后，做习题之前。
    *   **操作：** `独立地` 、不看答案地，重新做一遍老师上课圈画的重点例题。
    *    **目的：**
        *    检验自己是否真正听懂了课程内容和方法 (`检查自己到底听懂了没有`)。
        *    例题是概念和方法的直接应用，是习题的桥梁。

4.  **做章节习题 (Practice)：**
    *   **时机：** 完成上述三个步骤之后，再开始做对应章节的习题。
    *   **要求：** 必须**==“做题留痕”==**。
    *   **操作：**
        *   使用专门的笔记本。
        *   或者使用A4纸/草稿纸书写，并妥善收纳整理。
    *   **“做题留痕”的重要性：**
        *   `规范做题步骤`：有利于养成解答题的良好书写习惯。
        *   `方便后续复习`：特别是复习错题时，能清晰看到当初的错误思路和计算过程。

5.  **批改与标注 (Mark)：**
    *   **操作：** 做完题并批改订正后，在习题册的题号上进行清晰标注。
    *   **标注内容：**
        *    哪些题是完全不会做的。
        *    哪些题是做错的（计算错误/思路错误）。
    *   **目的：** 精确定位薄弱点，为后续复盘提供明确目标。

6.  **复盘与整理错题 (Analyze Mistakes)：**
    *   **核心：** `静下心来`，花时间去复盘、整理、`吃透` 错题。
    *   **重要性：** 演讲者认为是数学考取高分的 `关键`。
    *   **(演讲者提示：具体方法参考其之前的详细视频)**

### 3.2. 培养关键能力
数学考试不仅考察基础知识，更重要的是考察：
* **==计算能力==**: 准确、快速计算。
* **思维能力**: 逻辑推理、分析问题、选择方法。
这些能力都需要在平时做题过程中有意识地练习和培养。

## 4. 核心学习原则

*   ### **原则一：做题在于精，不在于多**
    *   **适用性：** 无论基础好坏，都应遵循此原则。
    *   **要求：**
        *   选定一本习题册，就 `踏踏实实` 地把它做完、做透。
        *   盲目追求数量、刷多本习题册但每本都浅尝辄止，效果不佳。
    *    **核心：** 把一本习题册彻底 `吃透` 远比做多本重要。
*   ### **原则二：高度重视错题的复盘与整理**
     *    **核心：** 提高数学成绩的关键在于从错误中学习，解决知识盲点和思维误区。错题是最宝贵的学习资源。花时间研究透彻错题，比盲目做新题更有效率。


## 5. 总结
针对基础薄弱同学做题难的问题：
1. 选择**难度和题量适中**的习题册（如《基础700题》），避免焦虑，保证进度。
2. 严格遵循 `听课 -> 回顾 -> 做例题 -> 做习题 -> 标注 -> 复盘` 的学习流程。
3. 坚持**做题留痕**，规范步骤，方便复习。
4. 秉持**“在精不在多”**和**“重视错题”**的核心原则，将选定的习题册彻底吃透。
