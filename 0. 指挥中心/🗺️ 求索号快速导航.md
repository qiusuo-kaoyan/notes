# 🗺️ 求索号快速导航

> **欢迎来到「求索号」考研知识母舰！**
>
> 这里不是一本传统的纯手写笔记，而是一套 **「人机协同考研」的方法论示范项目**。第一代舰长 Leo 通过自研的 **PZ 系列 Obsidian 模板** 与 AI 协同工作，将错题、思路、课程要点结构化输出，再经人工审校后汇入母舰。
>
> 如果你是第一次来到这艘船，**请从这里开始读**。

---

## 🚀 零基础上船（3 分钟）

如果你还没有 Fork 母舰并在本地打开，请先看这里：

👉 [**新舰长上船指南**](https://github.com/qiusuo-kaoyan/onboarding)

简要步骤：
1. **Fork** [qiusuo-kaoyan/notes](https://github.com/qiusuo-kaoyan/notes) 到你自己的 GitHub 账号
2. **Clone** 到本地：`git clone https://github.com/你的用户名/notes.git`
3. **用 Obsidian 打开** `notes` 文件夹
4. **安装必装插件**：Dataview、Templater、Kanban（详见上船指南）

### 1. 🧠 数学 (`1. Math/`)
高数、线代、概率论的完整知识体系与错题方法论。

| 子目录 | 内容 | 推荐阅读 |
|--------|------|----------|
| `1. Math/2. 强化阶段/` | 高等数学、线性代数、概率论强化笔记 | 按章节顺序阅读 |
| `1. Math/3. 真题阶段/` | 历年真题总结、错题分析报告、典型例题汇总 | 从「真题分析报告」开始 |

- `0. 指挥中心/复习导航仪/数学任务看板.md` — Kanban 式数学复习进度追踪

### 5. 🛠️ 模板与脚本 (`Templates/` & `Scripts/`)
这是求索号的核心工具链，体现「人机协同」的方法论价值。

| 文件 | 用途 |
|------|------|
| `Templates/Templater/PZ5-...舰长航行日志.md` | 每日日记模板 |
| `Templates/Templater/PZ6-...周度星航指挥室.md` | 周度复盘模板 |
| `Templates/Templater/PZ9-...AI首席战略参谋.md` | AI 周度战略分析模板 |
| `Templates/Templater/PZ10-...真题试炼场.md` | 真题总结模板 |
| `Scripts/kaoyanReview.js` | 数学错题复习看板脚本 |

> **P = Prompt，Z = Zenith**。PZ 系列模板 =「用提示工程攀上考研之巅」的笔记系统。
