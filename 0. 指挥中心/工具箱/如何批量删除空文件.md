在当前文件夹内打开 PowerShell 后，可以使用以下命令来清除空文件：

1. 先列出所有空文件，检查筛选是否正确：
   ```powershell
   Get-ChildItem -File | Where-Object { $_.Length -eq 0 }
   ```

2. 若筛选结果正确，再执行删除操作：
   ```powershell
   Get-ChildItem -File | Where-Object { $_.Length -eq 0 } | Remove-Item -Force
   ```

如果你担心误删，可以先用以下命令模拟删除过程：
```powershell
Get-ChildItem -File | Where-Object { $_.Length -eq 0 } | Remove-Item -Force -WhatIf
```

这样可以安全地清除当前文件夹内的空文件。
