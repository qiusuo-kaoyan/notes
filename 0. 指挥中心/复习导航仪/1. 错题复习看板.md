### 🎯 错题本 (按知识点聚合分析)
> [!TIP] 这是聚合视图，用于发现哪个知识点下的错题最多，适合战略性复习。

```dataview
TABLE WITHOUT ID
    rows.file.link AS "题目",
    length(rows) AS "错题数"
FROM "1. Math"
WHERE contains(type, "题目") AND contains(status, "错题")
GROUP BY choice(
    knowledge_points,
    knowledge_points, 
    join(knowledge_points, " & ")
)
SORT length(rows) DESC
```


## 🎯 错题本 (按薄弱知识点聚合)
> [!TIP] 这里会自动汇总所有标记为“错题”的题目，并按关联的知识点进行分组。优先攻击题目数量最多的知识点！

```dataview
TABLE WITHOUT ID
    rows.file.link AS "题目",
    rows.source AS "来源",
    rows.done_date AS "做错日期",
    rows.difficulty AS "难度 ⭐",
    rows.importance AS "重要性 🔥"
FROM "1. Math"
WHERE contains(type, "题目") AND contains(status, "错题")
GROUP BY join(knowledge_points, " & ")
SORT length(rows) DESC
```
